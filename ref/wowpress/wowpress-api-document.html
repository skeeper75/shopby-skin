Introduction

  • 와우프레스 오픈 API는 와우프레스 홈페이지의 기능을 외부 개발자가 쉽게 이용할 수 있도록 공개한 기술입니다.

  • 오픈 API를 활용하여 제품목록, 가격조회, 주문, 주문배송조회를 할 수 있습니다.

1. 개발문서

1.1. 제품상세 SPEC

  • 제품 상세정보는 제품의 주문페이지 구성을 위한 기본정보입니다.

  • 제품 주문에 필요한 파라미터를 생성하기 위한 필수 정보를 포함하고 있습니다
    다운로드

1.2. 제품가격 & 주문 SPEC

  • 주문상세정보를 바탕으로 규격,도수,재질에 따라 가격을 조회할 수 있습니다.

  • 주문에 필요한 파라미터 정보가 포함되어 있습니다.
    다운로드(2022.12.01)

1.3. Q & A 게시판 안내

2. 사전준비 사항

  • 회원가입 없이도 테스트 계정을 이용하여 개발 및 테스트 가능합니다.

  • 실제 API를 운영하고 사용하기 위해서는 회원가입을 하셔야 합니다.
    홈페이지에서 회원 가입 후 진행해 주세요.

2.1. 운영/개발 서버

  • 테스트 환경이 존재하지 않습니다

  • 실제 운영환경에서 모든 작업이 진행되므로 API사용에 주의하시고, API사용으로 인한 문제 발생 시, 사용자의 귀책 사유가 될 수 있습니다.

3. Authentication

3.1. 인증방식

  • 와우프레스 오픈 API는 JWT(Json Web Token)을 이용한 서명된 토큰 인증 방식을 사용합니다.

  • refresh Token 방식을 채용하지 않기 때문에 Token 유출에 주의 하세요.

  • 홈페이지 계정정보로 발급받은 토큰은 모든 API 요청에 사용됩니다.

  • JWT에 대한 내용은 https://jwt.io에서 확인가능합니다.

3.3. 토큰 유효기한(주의사항) 

한번 생성한 token의 사용유효기한은 1년입니다.
최소 1년에 1회 이상 재발급하여 사용해야 합니다.
운영서버 기동 시 토큰 발급 후, 전역변수로 사용하는 방식을 권장합니다.
한번 발급받은 토큰은 홈페이지 계정 비밀번호를 변경하더라도  유효기간내에는 유효합니다.
그렇기 때문에 타인에게 노출되지 않도록 주의 해야합니다.

6.3. 제품상세

  • 와우프레스 제품의 상세 내역을 조회할 수 있습니다.

  • 주문에 필요한 옵션정보 및 제약/제한 조건이 포함되어 있습니다.

  • 자세한 제약/제한 조건은 제품 상세정보 및 주문옵션 제약 조건을 참고해 주세요.

6.4. 제품가격조회

  • 제품 상세 및 제품옵션 정보로 부터 선택된 값으로 제품의 가격을 조회할 수 있습니다.

  • 제품의 가격이 수량에 정확히 비례하지 않습니다.

  • 개별 제품(또는 부자재)에 대한 제품 가격 조회 기능만 제공됩니다.

  • 제품의 가격이 조회 되지 않으면 주문이 불가능한 옵션이라고 생각하면 됩니다.

1) API 사용자가 와우프레스에서 제공되지 않은 옵션 선택
2) API 사용자가 와우프레스에서 갱신한 데이터(옵션 및 제품가격) 미반영
위 1) 과 2)의 사유로 발생한 모든 문제에 대해서는 API 사용자의 책임으로 귀속됩니다.

6.5. 주문하기

  • 제품가격이 조회된 제품의 주문을 할 수 있습니다.

  • 제품가격이 조회 되지 않는 요청은 주문 불가합니다.

  • 여러 주문을 동시에 주문할 수 있고 묶음가능한 제품은 자동 묶음배송됩니다.

  • 결제 또는 정산되지 않은 주문은 접수 및 생산이 진행되지 않습니다.

  • 배송방법(dlvymcd)만 선택가능하며, 택배사/배송호차는 수취인 주소에 따라 자동 설정됩니다.(* sd, sgg, umd 필수)

6.10. 주문취소

  • 정산되지 않은 주문(입금대기), 접수대기(접수전), 파일에러 상태의 주문만 취소가 가능합니다.

7.7. 선불택배불가제품

  • 택배비 산정에 어려움이 있어 선불택배 불가 제품이 있습니다.

  • 배쑝와쑝, 대리배송, 착불택배(롯데택배,경동택배)는 이용가능합니다.

  • 대리배송은 배쑝와쑝 가능지역만 선택가능합니다.

  • 대상은 현수막, 배너 제품 등

7.8. 묶음배송불가제품

  • 같은 묶음배송 그룹(명함,스티커)제품 주문 시 자동 묶음됩니다.

  • 책자, 봉투, 자석제품, 현수막 및 부자재 등의 묶음 불가 제품이 있습니다.

7.9. 롯데택배불가제품

  • 선불택배(롯데택배)선택이 불가하며, 배쑝와쑝(월배송), 대리배송, 착불택배(경동택배)만 가능한 제품

  • 제품번호 철제입간판, LED라이트배너, 플라잉배너, 스프링배너, 물통스프링배너, 롱스텐드게시대, 철제입간판, 스텐입간판, 아크릴입간판, 나무입간판 등

10. 결제안내

  • 오픈 API를 이용하여 주문을 하게 되면 주문은 입금대기 상태가 됩니다.

  • 선입금이 있는 경우, 주문 후에 정산이 자동으로 이뤄 집니다.

  • 정산이 100% 되지 않은 주문은 접수 및 생산이 진행이 되지 않습니다.

  • 미정산 금액을 결제(카드, 전용계좌 이체)하면 자동으로 정산이 이뤄지고 접수 및 생산이 진행됩니다.

  • API카드결제(빌링키결제)기능이 추가되었습니다. (2024.05.14)

10.1. 포인트사용

  • 포인트 사용은 주문시 적용합니다. ex) dcpointreq: 1000 주문하기참고

  • 회원정보조회를 통해서 포인트 잔액 조회 가능

  • 보유 포인트 범위내에서 사용해 주세요.

  • 여러주문에 포인트 사용 시 자동 적용 됨.

10.2. 결제방법

  • 홈페이지 및 모바일 웹/앱에서

  • NicePay 카드결제, 간편결제(NicePay, 카카오, 네이버), 계좌이체(전용계좌)로 결제 가능합니다.

10.3. 전용계좌

  • 전용계좌는 홈페이지에서 신청할 수 있습니다.

  • 개별 회원 전용으로 사용할 수 있는 계좌번호를 제공합니다.

  • 전용계좌에 입금하면 5 ~ 30분 이내에 입금확인 가능합니다.

  • 입금이 확인되면 입금대기 주문이 자동정산되어 접수 및 생산 진행됩니다.

10.4. API카드결제(빌링키결제)

  • 홈페이지에서 카드등록을 해야 사용가능합니다.
    카드등록

  • 빌링키 결제는 비밀번호 없이 결제가 가능하기 때문에 외부노출에 주의하세요.

  • 선입금 또는 미수금 결제 형태로 입금처리가 됩니다.

  • 정산되지 않은 주문이 있으면 결제후에 자동 정산이 이뤄지고 생산이 진행됩니다.
