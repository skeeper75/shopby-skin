## 마케팅 개인정보 수집/이용 동의 여부 조회

### 응답 값
- `isAgreed = true`: 동의
- `isAgreed = false`: 미동의
- `isAgreed = null`: 동의/미동의 설정 전 <-- 마케팅 목적의 개인정보 수집/이용 동의 약관 알림 노출 필요

### 동의 여부 판단 기준
SA 서비스 관리 > 약관/개인정보처리방침 관리 > 개인정보 수집/동의 항목(탭) > **마케팅 목적의 개인정보 수집/이용 동의** 설정에 따라 결정됩니다.

#### 조건별 반환 값
| 사용 여부 | 안내 노출 여부 | 반환 값 |
|:--------:|:------------:|:------:|
| 사용함    | 사용함        | `true(동의)` / `false(비동의)` / `null(동의/비동의 설정 전)` (회원 동의 데이터 기준) |
| 그 외     | -            | `false` |
