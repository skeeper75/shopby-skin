:::

## 서버 정보

| 환경 | Base URL |
|------|----------|
| Production | `https://shop-api.e-ncp.com` |

## 리소스 목록

| 리소스 | 설명 |
|--------|------|
| **Image** | 이미지 |
| **File** | 파일 |

## API 엔드포인트

## File — 파일

### 🔵 `POST` `/storage/temporary-files`

**파일 업로드**

> operationId: `upload-file`

**Parameters**

| 이름 | 위치 | 필수 | 타입 | 설명 |
|------|------|:----:|------|------|
| `Version` | header | ✅ | `string` | API 버전 |
| `clientId` | header | ✅ | `string` | 쇼핑몰 클라이언트 아이디 |
