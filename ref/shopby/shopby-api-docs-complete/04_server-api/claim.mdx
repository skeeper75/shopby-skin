## 부가설명 및 특이사항
관리자 지정 환불 클레임 신청시, 각 환불수단 별 최대로 입력 가능한 환불금액을\
        \ 반환합니다. <br/>
각 환불수단 별 금액 합계는 환불 금액 (refundAmt) 과 동일해야 합니다.<br/>
<br/>
\
        refundAmt 가 양수인 경우, 고객에게 환불되는 금액을 의미합니다.<br/>
refundAmt 가 음수인 경우, 고객이 추가로\
        \ 결제해야 하는 금액을 의미합니다. 이 경우, 모든 항목이 0으로 응답됩니다.<br/>
<br/>
반품 수거지 및 교환 배송지는\
        \ 최초 주문을 기준으로 합니다.<br/>
반품 및 교환 시 수거지 또는 배송지가 변경되면, 환불 금액이 계산된 값과 상이할 수 있\
        습니다.<br/>
<br/>
claimType : CANCEL, RETURN, EXCHANGE 중 하나입니다.
## ClaimType\
        \ 에 따른 파라미터
### 공통
claim, orderNo, responsibleObjectType, sellerPaysClaimedDelivery,\
        \ claimedOptions 파라미터는 필수로 입력해야 합니다.
### CANCEL
공통 파라미터 이외에 추가 파라미터는 없습니\
        다.
### RETURN
returnWayType 파라미터가 필수로 입력되어야 합니다.
### EXCHANGE
exchangeOption\
        \ 파라미터가 필수로 입력되어야 합니다.<br/>
반품교환 (출고 후 교환) 인 경우, returnWayType 은 필수로 입력되어\
        야 합니다.<br/>
claimedOptions 는 1개만 입력되어야 합니다.
