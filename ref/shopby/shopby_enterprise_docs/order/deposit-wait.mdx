# 입금대기 주문

<Callout type="info">
**주문자가 '무통장입금' 또는 '가상계좌'로 결제수단을 선택하고 상품을 주문한 다음, 결제금액을 입금하기 전 상태인 입금대기 상태의 주문 건을 조회합니다.**

**무통장 입금대기 주문 건은 운영자가 결제금액 입금을 확인하고, ‘입금확인 처리’를 수동으로 진행해 결제완료 상태로 변경해야 합니다.**

**입금대기 상태의 주문 건부터 상품의 재고가 차감됩니다.**
</Callout>

## 입금대기 주문리스트

**① 입금확인 처리**\
결제금액의 입금이 확인된 주문 건을 선택하고 \[입금확인 처리] 버튼 또는 \[입금확인]을 클릭하면 결제완료 상태로 변경(이동)됩니다.

* 결제완료 상태로 변경(이동)된 해당 주문 건은 결제완료 주문리스트에서 조회가 가능합니다.

**② 취소 처리**\
주문자가 입금 전 주문취소 요청을 하거나 입금이 미확인 된 경우, 해당 주문 건을 '취소 처리' 할 수 있습니다.

**③ '**&#xAC00;상계좌, 에스크로' 결제수단의 입금대기 주문 건은 PG 시스템에서 입금이 확인되면 자동으로 입금확인 처리되어 결제완료 상태로 변경(이동)됩니다.&#x20;

* &#x20;**'**&#xAC00;상계좌, 에스크로' 결제수단의 입금대기 주문 건은 수동으로 ‘입금확인 처리’ 가 불가합니다.
