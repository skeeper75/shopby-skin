# 결제실패 주문

<Callout type="info">
**주문자가 결제를 시도하던 중 결제 오류로 실패했거나, 스스로 포기 및 중단한 경우 등과 같이  주문접수(결제)가 완료되지 않은 주문 건을 조회합니다.**

**결제실패 주문 건은 삭제 처리 가능합니다.**
</Callout>

## 결제실패 주문리스트

**① 결제실패**\
결제가 실패되어 주문이 완료되지 않은 주문 건으로 시스템에서 확인된 실패사유가 표기됩니다.&#x20;

**② 결제포기**\
주문을 진행하다 포기한 주문 건으로 사유는 표기되지 않습니다.

**③ 삭제처리**\
\[삭제처리] 버튼을 클릭해 결제실패 주문 건을 삭제할 수 있습니다. \
삭제처리한 결제실패 주문 건은 복구되지 않는 점을 유의해 주세요.<br>
