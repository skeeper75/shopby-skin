# 구매확정 주문

<Callout type="info">
**주문자에게 상품이 배송완료된 구매확정 상태의 주문 건을 조회합니다.**

**구매확정 상태의 주문 건은 결제완료 상태부터 배송완료 상태까지 모든 단계의 변경(이동)이 가능합니다.**\ <mark style="color:red;">단, 에스크로 정책에 의해, 구매확정 상태의 에스크로 결제 주문 건은 모든 상태로 수동 변경(이동)이 불가한 점을 참고해 주세요.</mark>
</Callout>

## 구매확정 주문리스트

**① 주문상태 선택** \
주문상태 업데이트(변경)를 위해 주문상태 선택에서 변경을 희망하는 주문상태를 선택합니다.

**② 일괄처리** \
복수 주문 건을 선택한 주문상태로 일괄변경 처리 할 수 있습니다.

**③** 구매확정 상태 주문 건은 주문자가 직접 교환 및 반품 신청을 할 수 없습니다. \
운영자가 클레임 처리를 진행해야 하는 점을 참고해 주세요.
