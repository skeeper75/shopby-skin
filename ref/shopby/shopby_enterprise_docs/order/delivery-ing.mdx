# 배송중 주문

<Callout type="info">
**운영자가 상품을 출고처리하여 주문자에게 상품이 배송중인 상태의 주문 건을 조회합니다.**

**배송중 상태의 주문 건은 결제완료 상태부터 구매확정 상태까지 모든 단계의 변경(이동)이 가능합니다.**\ <mark style="color:red;">단, 에스크로 정책에 의해, 배송중 상태의 에스크로 결제 주문 건은 모든 상태로 수동 변경(이동)이 불가한 점을 참고해 주세요.</mark>
</Callout>

## 배송중 주문리스트

<figure><img src="/images/order/delivery-ing/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FPasC2JIerfXTTShlRcdg%2F%EB%B0%B0%EC%86%A1%EC%A4%91%EC%A3%BC%EB%AC%B8_%EC%A3%BC%EB%AC%B8%EB%A6%AC%EC%8A%A4%ED%8A%B8.png" alt=""/><figcaption><p>&#x3C;배송중 주문리스트 ></p></figcaption></figure>

**① 주문상태 선택** \
주문상태 업데이트(변경)를 위해 주문상태 선택에서 변경을 희망하는 주문상태를 선택합니다.

**② 일괄처리** \
복수 주문 건을 선택한 주문상태로 일괄변경 처리 할 수 있습니다.

**③ 배송완료**\
\[배송완료]를 클릭하면 배송중 다음 단계인 배송완료 상태로 변경(이동)되며, 배송완료 주문리스트에서 해당 주문 건을 조회합니다.

**④** 배송시작 이후 자동 배송완료 처리되는 기간을 지정하여 배송완료 상태 변경(이동)으로 자동 처리가 가능합니다.&#x20;

* 자동배송완료(국내배송) 기간 지정 : [서비스관리 > 쇼핑몰 관리 > 주문처리기간 설정](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/mall#undefined) 메뉴\
  📍 영업일 기준 / 1일 \~ 30일 중 선택
