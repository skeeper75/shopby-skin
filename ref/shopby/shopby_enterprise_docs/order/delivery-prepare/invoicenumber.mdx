# 송장번호 일괄 업로드

<Callout type="info">
**엑셀파일에 송장번호 정보를 입력하여 일괄 업로드로 적용할수 있습니다.**

**배송중 처리를 선택한 경우 엑셀파일 업로드와 동시에 배송중 상태로 변경(이동)되므로 배송중 주문리스트에서 해당 주문 건을 조회합니다.**
</Callout>

ⓐ \[송장업로드용 엑셀 다운로드] 버튼을 클릭하면 배송준비중 상태의 주문 건에 송장번호를 입력할 수 있는 엑셀파일을 생성할 수 있습니다.&#x20;

ⓑ 생성한 송장업로드용 엑셀 다운로드 파일의 \[다운로드]를 클릭하고, 사유확인과 암호화를 진행한 다음 엑셀파일 다운로드와 압축풀기를 합니다.&#x20;

ⓒ 엑셀파일 내 송장번호 항목에 정보를 입력하고 엑셀파일을 저장합니다.&#x20;

<figure><img src="/images/order/delivery-prepare/invoicenumber/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FnhVPBuP6FoLwOGc5j0R1%2F%EB%B0%B0%EC%86%A1%EC%A4%80%EB%B9%84%EC%A4%91%EC%A3%BC%EB%AC%B8_%EC%86%A1%EC%9E%A5%EB%B2%88%ED%98%B8%EC%9D%BC%EA%B4%84%EC%97%85%EB%A1%9C%EB%93%9C2%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;송장번호 일괄 업로드></p></figcaption></figure>

ⓓ 배송업체 지정에서 택배사를 선택합니다.

ⓔ 주문 건을 배송중 상태로 변경(이동)하려면 배송중 처리를 선택합니다.&#x20;

ⓕ \[파일찾기] 버튼으로 저장한 엑셀파일을 선택하고, \[업로드] 버튼을 클릭합니다.&#x20;

ⓖ 엑셀파일 업로드와 동시에 선택한 택배사가 송장번호 정보와 함께 주문 건에 적용됩니다.&#x20;
