# 결제완료 주문

<Callout type="info">
**주문자가 상품을 주문하고, 주문금액을 모두 결제한 결제완료 상태의 주문 건을 조회합니다.**

**결제완료 상태의 주문 건은 상품준비중 상태부터 구매확정 상태까지 변경(이동)이 가능합니다.**\ <mark style="color:red;">단, 에스크로 정책에 의해, 결제완료 상태의 에스크로 결제 주문 건은 '상품준비중\~배송중' 상태로 수동 변경(이동)은 가능하고, '배송완료'와 '구매확정' 상태로 변경이 불가한 점을 참고해 주세요.</mark>
</Callout>

## 결제완료 주문리스트

<figure><img src="/images/order/pay-done/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F4dImrR53zRAb2cfOrwxY%2F%EA%B2%B0%EC%A0%9C%EC%99%84%EB%A3%8C%EC%A3%BC%EB%AC%B8%EB%A6%AC%EC%8A%A4%ED%8A%B8%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;결제완료 주문리스트></p></figcaption></figure>

**① 주문상태 선택**\
주문상태 업데이트(변경)를 위해 주문상태 선택에서 변경을 희망하는 주문상태를 선택합니다.

**② 일괄처리**\
복수 주문 건을 선택한 주문상태로 일괄변경 처리 할 수 있습니다.&#x20;

**③ 상품준비중** \
\[상품준비중]을 클릭하면 결제완료 다음 단계인 상품준비중 상태로 변경(이동)되며, 상품준비중 주문리스트에서 해당 주문 건을 조회합니다.&#x20;

* 한 번에 여러 상품을 주문 한 경우 상품별 개별 상태 변경 처리도 가능합니다.&#x20;

**④ 품절취소처리**\
재고부족 상품(또는 상품의 옵션)을 결제한 경우 등 운영자가 해당 주문 건을 수동으로 품절취소처리하여 주문취소 상태로 변경할 수 있습니다.&#x20;

* 품절취소처리한 상품 및 상품의 옵션 상태는 '품절'로 변경되며, 재고수량도 '0'으로 처리됩니다.&#x20;
* 품절취소처리 이후 주문 건의 결제수단에 따라 주문상태(환불프로세스 포함)가 다르게 변경(이동)됩니다.

<Callout type="warning">
자동환불이 불가한 무통장입금 주문 건 + 품절취소처리\
결제완료 상태 → 취소처리\_환불보류 상태 (환불계좌 등 주문자 정보 확인 필요)&#x20;

자동환불이 가능한 PG 결제 주문 건 + 품절취소처\
결제완료 상태 → 취소처리\_환불완료 상태
</Callout>

**⑤** 세트상품은 '전체 품절취소처리'와 '부분 품절취소처리 + 부분 취소처리' 중에서 선택할 수 있습니다.
