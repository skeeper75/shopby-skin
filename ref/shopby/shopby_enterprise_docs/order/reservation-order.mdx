# 예약주문 관리

<Callout type="info">
**예약 판매 기간과 예약 배송시작일을 설정하여 등록한 예약판매 상품의 예약주문 건을 조회하고 관리합니다.**

**입금대기와 결제완료 상태의 예약주문 건이 예약주문 주문리스트에서 관리되며, '입금확인 처리' 할 수 있습니다.**

**예약 주문 건은 예약 배송시작일로 설정한 당일 새벽 1시에 주문관리가 가능하도록 원주문리스트 (입금대기/결제완료)로 자동 이동처리 됩니다.**

**자동 이동 처리된 예약 주문 건은 다른 주문상태로 변경(이동)이 가능하여 주문관리를 진행할 수 있습니다.**
</Callout>

## 예약주문 주문리스트

**① 입금확인 처리**\
결제금액의 입금이 확인된 예약 주문 건을 선택하고, \[입금확인 처리] 버튼을 클릭하면 결제완료 상태로 변경됩니다.

**② 주문메뉴로 이동**\
운영자가 예약 주문 건을 예약주문 주문리스트에서 원주문리스트(입금대기/결제완료)로 수동 이동처리 할 수 있습니다.

* 예약 배송시작일 이전 상품 출고가 가능한 예약주문 건의 주문관리가 가능하도록 주문메뉴로 이동합니다.
* '입금대기'와 '결제완료' 상태의 예약 주문 건을 선택하고 \[주문메뉴로 이동] 을 클릭하면, 각각의 원주문리스트(입금대기/결제완료)로 이동되어 확인 가능합니다.
* 수동 이동 처리된 예약 주문 건은 다른 주문상태로 변경(이동)이 가능하여 주문관리를 진행할 수 있습니다.

<Callout type="warning">
전체/입금대기/결제완료 주문리스트에서도 상세검색으로 예약 주문 건을 조회할 수 있고, 예약 주문 건 항목에 \[Y]로 별도 체크되어 노출됩니다.

단, 수동 이동 처리되지 않은 예약주문 건과 배송시작일이 도래하지 않은 예약 주문 건은 다른 주문상태로 변경(이동)이 불가능 한 점을 참고해 주세요.
</Callout>

#### **✅** <mark style="color:green;background-color:yellow;">**예약 재고**</mark>

* 예약재고는 예약판매기간 동안 사전 주문을 받기 위해 설정한 재고수량입니다.
* 예약재고는 상품의 재고(상품 실 재고, 판매가능재고)와 무관합니다.
* 예약판매 기간 동안의 예약판매 중 상품은 예약 재고를 기준으로 판매됩니다.
* 예약판매 기간 중 예약 재고수량이 ‘0’개가 되면 상품은 품절 처리됩니다.
* 판매 가능 재고수량은 예약판매에 영향을 주지 않습니다.&#x20;
* 판매 가능 재고수량이 '0'개인 경우,  예약 재고수량이 있다면 상품의 예약판매가 가능합니다.
