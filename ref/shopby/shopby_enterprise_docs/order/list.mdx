# 전체주문 조회

<Callout type="info">
**회원 및 비회원 고객이 주문한 상품 및 서비스의 전체 주문 건을 조회하고 주문상태 업데이트 및 거래명세서 출력 등의 주문관리 업무가 가능합니다.**

**'입금대기'부터 '구매확정'상태까지 원하는 주문상태로 수동 변경(이동)할 수 있습니다.**\ <mark style="color:red;">**단, 에스크로 정책에 의해, 에스크로 결제 주문 건은 '배송완료'와 '구매확정' 상태로 수동 변경(이동)이 불가한 점을 참고해 주세요.**</mark>

**검색분류, 주문상태, 정상/클레임주문, 결제수단 등 다양한 조건으로 주문 건을 검색하고, 배송번호와 주문번호 기준으로 주문리스트를 확인할 수 있습니다.**
</Callout>

## 전체주문 검색조건

<figure><img src="/images/order/list/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FuYuHtQPO6xd4RahV99vB%2F%EC%A0%84%EC%B2%B4%EC%A3%BC%EB%AC%B8%EC%A1%B0%ED%9A%8C_%EA%B2%80%EC%83%89%EC%A1%B0%EA%B1%B4%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;전체주문 조회></p></figcaption></figure>

**① 쇼핑몰** \
주문 건을 조회할 쇼핑몰을 선택합니다.

**② 파트너사** \
쇼핑몰 자체상품과 파트너사 공급상품의 주문 건을 각각 조회하고 관리할 수 있습니다.

**③ 검색어** \
주문번호, 주문자명, 회원번호, 상품번호, 주문자 연락처, 배송번호, 수령자명 등 다양한 검색어로 주문 건을조회합니다.

**④ 기간검색** \
주문일자로 검색하는 주문기간 검색은 최대 3개월까지 가능합니다.

**⑤ 주문/클레임 구분** \
주문이 접수된 이후 입금대기부터 구매확정 상태에 해당하는 정상주문과 취소/교환/반품/환불처리 주문 건에 해당하는 클레임 주문으로 구분하여 검색합니다.

* 정상주문과 클레임주문을 선택하면 각각의 주문에 해당하는 여러 상태 값이 노출되어 개별 검색이 가능합니다.

**⑥ 결제수단** \
쇼핑몰에 연동되어 있는 결제수단 중, 주문 건을 결제한 결제수단으로 검색할 수 있습니다.

**⑦ 상세검색** \
교환신청 접수여부, 주문접수 유형(일반/수기/외부연계 등), 회원등급, 회원그룹, 배송방법, 배송업체, 송장번호 등록여부, 예약주문, 배송보류주문 등 여러 조건으로 상세한 주문 건을 검색합니다.

<details>

<summary>🤚 <strong>주문접수 유형</strong></summary>

* 일반주문 : 주문자가 직접 쇼핑몰에서 상품을 주문한 주문 건
* 수기주문 : 운영자가 주문자의 회원정보를 이용하여 상품을 주문한 주문 건
* 외부연계주문 : 외부플랫폼 및 외부연동으로 상품이 주문 된 주문 건 \
  (예 : 네이버페이 주문형 주문 등)
* 예약주문 : 주문자가 예약판매기간의 상품을 주문한 주문 건
* 정기주문 : 정기결제(배송) 상품을 주문한 주문 건
* 배송보류주문 : 상품준비중/배송준비중 상태에서 출고준비/출고처리가 바로 불가할 경우 \
  운영자가 배송을 잠시 보류 처리한 주문 건

</details>

**⑧ 검색/초기화** \
모든 조건을 선택하고 \[조회] 버튼을 클릭하면 하단 조회 결과에 주문 건이 노출됩니다. \
\[초기화] 버튼을 클릭하면 입력된 검색어 및 검색조건이 기본값으로 초기화됩니다.

## 배송번호 기준 주문리스트

<figure><img src="/images/order/list/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FK9kXgOAdcplYRiDhK57m%2F%EC%A0%84%EC%B2%B4%EC%A3%BC%EB%AC%B8%EC%A1%B0%ED%9A%8C_%EB%B0%B0%EC%86%A1%EB%B2%88%ED%98%B8%EA%B8%B0%EC%A4%80%EB%A6%AC%EC%8A%A4%ED%8A%B8%20%EB%B3%B5%EC%82%AC4.png" alt=""/><figcaption><p>&#x3C;배송번호 기준 주문리스트></p></figcaption></figure>

**① 배송번호** \
주문한 상품에 등록되어 있는 배송비 템플릿에 따라 부여되는 번호로 주문한 상품을 함께 배송할 수 있는 \
단위 입니다.

**② 주문상태 선택** \
주문상태 선택에서 변경을 희망하는 주문상태를 선택하고, 상태 변경을 진행합니다.

**③ 일괄처리** \
복수 주문 건을 선택한 주문상태로 일괄변경 할 수 있습니다. \
이 때 특정 시점에 관계없이 특정 상태의 앞 단계로 복귀가 가능하고, 특정 단계를 건너뛸 수도 있습니다.

* 입금대기 상태 주문 건의 결제완료 상태 변경(입금확인 처리)은 입금대기 주문리스트에서 결제완료 \
  상태로만 변경 가능합니다.
* 에스크로 주문 건(가상계좌/실시간계좌이체)은 배송완료 및 구매확정 상태로 변경이 불가능합니다.

**④ 송장번호 저장** \
주문 건에 택배사를 선택하고 송장번호를 입력한 다음 \[송장번호 저장] 버튼을 클릭하면 해당 주문 \
건에 정보가 바로 적용됩니다.

* 택배사와 송장번호 정보는 결제완료 상태 주문 건부터 입력 및 적용이 가능합니다.

**⑤ SMS 발송** \
선택 또는 검색된 주문 건의 주문자에게 SMS를 발송할 수 있습니다.\
⚠️ <mark style="color:red;">광고성 내용을 발송하지 않도록 유의해 주세요.</mark>

* 사전에 반드시 SMS 발신번호등록과 SMS 포인트 충전을 완료해야 합니다.
* SMS 발송은 단문 SMS와 장문 SMS, 즉시 발송과 예약 발송이 가능합니다.
* 주문자 대상 SMS는 수신번호의 회원 여부 및 수신 동의 여부와 무관하게 발송 처리됩니다.

**⑥ 거래명세서 출력** \
해당 주문 건의 거래명세를 출력할 수 있는 창이 노출됩니다. \
거래명세서는 배송번호 기준으로만 출력됩니다.

**⑦ 보기** \
검색한 주문 건을 30, 50, 100, 200개 단위로 볼 수 있습니다.

**⑧** [**생성** ](#undefined-4)\
검색한 주문리스트가 엑셀파일로 생성되며, 압축된 엑셀파일을 다운로드 받을 수 있습니다.

**⑨ 톱니바퀴 아이콘** \
톱니바퀴 아이콘을 클릭하면 주문리스트 항목(그리드 컬럼)을 원하는 항목과 순서로 설정할 수 있고, \
주문상세창 노출 방법을 설정할 수 있습니다.

* 해당 설정은 로그인한 관리자 계정 별로 관리 됩니다.
* 그리드 컬럼 설정 변경 시 저장 버튼 클릭 시에만 설정 내용이 반영됩니다.

😊 하단 스크롤을 오른쪽으로 이동하면 더 많은 주문 정보 항목을 확인할 수 있습니다. \
&#x20;     주문상태 별 최종 변경 처리일시를 확인할 수 있고, 특정 상태 건너뛰기 시 최종 변경 상태 처리일시와 \
&#x20;     동일한 일자로 표기 됩니다.

## 주문번호 기준 주문리스트

<figure><img src="/images/order/list/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FIIXINLWN4uoQmrAUnIjv%2F%EC%A0%84%EC%B2%B4%EC%A3%BC%EB%AC%B8%EC%A1%B0%ED%9A%8C_%EC%A3%BC%EB%AC%B8%EB%B2%88%ED%98%B8%20%EA%B8%B0%EC%A4%80%EB%A6%AC%EC%8A%A4%ED%8A%B8%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;주문번호 기준 주문리스트></p></figcaption></figure>

**① 주문번호** \
쇼핑몰 주문 순서에 따라 순차적으로 부여되는 주문 건별 번호입니다. \[주문번호]를 클릭하면 해당 주문 \
건의 [주문상세 정보](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/order/list/detail) 페이지로 이동됩니다.

**② 주문자명** \
\[주문자명]을 클릭하면 해당 주문자의 회원정보와 주문 및 혜택내역 등을 확인하고 관리하는 고객관리시스\
템 CRM 창이 노출됩니다.

***

#### <mark style="color:blue;">■</mark> <mark style="color:blue;"></mark><mark style="color:blue;">**생성 : 엑셀 다운로드**</mark>

* 엑셀 다운로드 창은 어드민 메뉴에서 생성한 엑셀파일 상태 확인과 다운로드 기능을 제공합니다.
* 생성한 엑셀파일의 \[다운로드]를 클릭합니다. 엑셀 다운로드 창에서 다운로드 사유와 파일 압축풀기에 사용되는 비밀번호를 입력하고 \[요청] 버튼을 클릭합니다.

⚠️ 개인정보 안전성 확보조치 기준에 의거하여, 개인정보 다운로드 시 사유확인과 암호화가 필요합니다. \
&#x20;     압축 프로그램은 AES-256 알고리즘을 지원하는 프로그램을 사용하는 것을 권장합니다. \
&#x20;     (예: 반디집 등)
