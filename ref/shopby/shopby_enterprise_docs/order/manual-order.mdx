# 수기주문 관리

<Callout type="info">
**회원 및 비회원 고객이 쇼핑몰 이외에 외부채널을 통해 주문을 한 경우, 또는 여러 이슈로 주문이어려운 경우 운영자가 직접 주문을 등록할 수 있습니다.**

**수기주문 건의 결제는 쇼핑몰에 연동된 PG 서비사의 가상계좌 결제수단만 가능합니다.**

**등록된 수기주문 건은 회원 마이페이지 주문내역에서도 확인 가능합니다. 비회원은 주문번호로 조회해 주세요.**
</Callout>

## 수기주문 관리 (수기주문 등록)

### 회원구분 / 상품조회

<figure><img src="/images/order/manual-order/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FMwJ3HtZZDWAG9ZwJgxif%2F%EC%88%98%EA%B8%B0%EC%A3%BC%EB%AC%B8%EA%B4%80%EB%A6%AC%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;수기주문  회원구분 / 상품조회></p></figcaption></figure>

**① 쇼핑몰**\
수기주문을 등록할 쇼핑몰을 선택합니다.

**② 회원구분**

* 회원 : 회원을 선택하고 쇼핑몰 회원 아이디를 입력한 다음 \[조회] 버튼을 클릭하면, 주문대상 상품 조회가 가능하도록 \[상품 조회] 버튼이 활성화 됩니다.
* 비회원 : 비회원을 선택하면 주문대상 상품 조회가 가능하도록 \[상품 조회] 버튼이 활성화 됩니다.

**③ 상품조회**\
수기주문을 진행할 상품을 조회하고 등록합니다.

* 상품 조회 등록 : \[상품 조회] 버튼을 클릭하면 쇼핑몰에 등록한 전체 상품이 조회되는 상품조회 창이 노출됩니다. 수기주문을 진행할 상품을 선택하고 \[선택상품 추가] 버튼을 클릭합니다.
* 상품번호 직접 등록 : 상품번호를 입력하여 수기주문을 진행할 상품을 추가합니다. 상품번호를 입력하고 \[추가] 버튼을 클릭합니다.

### 주문(상품) 리스트

<figure><img src="/images/order/manual-order/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FKpNApk7CTmneqK1KCVSV%2F%EC%88%98%EA%B8%B0%EC%A3%BC%EB%AC%B8%EB%A6%AC%EC%8A%A4%ED%8A%B8%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;수기주문 리스트></p></figcaption></figure>

**④** 상품 조회에서 추가한 상품의 주문 리스트가 생성됩니다.&#x20;

**⑤** 상품을 확인하고 옵션과 수량을 선택한 다음, 판매가와 상품합계 금액을 확인합니다.&#x20;

**⑥** 주문 리스트에서 상품정보를 확인하고 \[주문확정] 버튼을 클릭하면, 배송비 항목이 노출됩니다.

<Callout type="danger">
품절 및 판매중지 상태 등의 상품은 주문 리스트에는 추가되지만 주문 확정 처리 되지 않는 점을 유의해 주세요.&#x20;
</Callout>

**⑦** \[주문수정] 버튼을 클릭하면 상품의 옵션과 수량 등을 수정할 수 있고,  \[선택 상품 삭제] 버튼으로 상품을삭제할 수 있습니다.

### 할인정보 / 주문자 정보

<figure><img src="/images/order/manual-order/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F2YwVDkaiLhu6aIDD09xT%2F%EC%88%98%EA%B8%B0%EC%A3%BC%EB%AC%B8%ED%95%A0%EC%9D%B8%2C%EC%A3%BC%EB%AC%B8%EC%9E%90%EC%A0%95%EB%B3%B4_%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;수기주문 할인정보 / 주문자 정보></p></figcaption></figure>

**⑧ 할인정보**\
회원이 보유한 쿠폰과 적립금을 조회하여 할인 적용할 수 있습니다. (비회원 할인 적용 불가)

* 쿠폰 할인금액 : \[쿠폰적용] 버튼을 클릭한 다음 쿠폰적용 창에서 회원이 보유한 상품 쿠폰 또는 장바구니 쿠폰(주문 쿠폰)을 확인하고 적용합니다.
* 적립금 사용 : 보유 적립금과 사용 가능 적립금을 확인하고, 사용할 적립금을 입력합니다. \[전액사용] 버튼을 클릭하면 사용 가능 적립금 전액이 자동 입력됩니다.

**⑨ 주문자 정보 (모든 항목 필수 입력값)**

* 회원 : 회원 정보가 자동 노출됩니다. 회원 정보에 없는 항목은 공란으로 노출되므로 모두 입력합니다.
* 비회원 : 공란으로 노출되어 있는모든 주문자 정보 항목을 입력해 주세요.&#x20;

### 배송지 정보 / 최종 결제금액

<figure><img src="/images/order/manual-order/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FVlCVnlUvl0c1XJrtINHJ%2F%EC%88%98%EA%B8%B0%EC%A3%BC%EB%AC%B8%ED%95%A0%EC%9D%B8%EB%B0%B0%EC%86%A1%2C%EC%B5%9C%EC%A2%85%EA%B2%B0%EC%A0%9C%EC%A0%95%EB%B3%B4_%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;수기주문 배송지 정보 / 최종 결제금액></p></figcaption></figure>

**⑩ 배송지 정보**

* 회원 : 주문내역이 있는 경우 회원의 최근 배송지 정보가 자동 노출됩니다. 최근 배송지가 없거나 새로운\
  배송지로 배송이 필요한 경우 신규 배송지를 선택하고  배송지 정보를 입력합니다.
* 비회원 : 공란으로 노출되어 있는 모든 배송지 정보를 입력합니다.

**⑪ 최종 결제금액**

* 총 결제금액을 확인하고 결제할 가상계좌 결제수단의 PG타입을 선택한 다음, \[결제하기] 버튼을 클릭합니다.
* 결제를 진행하고 나면, [수기주문 정상접수](#undefined-6) 팝업 창에서 가상계좌 상세정보 등을 포함한 수기 주문 접수 내역을 확인할 수 있습니다.

<Callout type="warning">
총 결제금액 = 총 상품합계 금액 + 총 배송비 – (총 할인적용가 = 쿠폰 할인금액 + 적립금 사용금액)
</Callout>

### 개인정보 수집⋅이용 동의

<figure><img src="/images/order/manual-order/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FQQetbAHgaRFQf6QJiG6u%2F%EC%88%98%EA%B8%B0%EC%A3%BC%EB%AC%B8_%EA%B0%9C%EC%9D%B8%EC%A0%95%EB%B3%B4%EC%88%98%EC%A7%91%EC%9D%B4%EC%9A%A9%EB%91%89%EC%9D%98_%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;수기주문 개인정보 수집⋅이용 동의></p></figcaption></figure>

**⑫ 개인정보 수집⋅이용 동의**\
수기 주문 건은 주문 필수 동의항목에 대해 고객이 동의한 것으로 간주 됩니다. 고객에게필수 동의항목을 고지하고 동의를 받는 절차를 별도로 진행하는 것을 권장드립니다.

***

#### <mark style="color:blue;">■</mark>  <mark style="color:blue;"></mark><mark style="color:blue;">**수기주문 정상접수**</mark>

* \[주문 상세로 가기] 버튼을 클릭하면 해당 주문의 주문  상세페이지로 이동합니다.
* \[수기주문 메뉴로 이동] 버튼을 클릭하면 새로운 수기 주문을 등록할 수 있는 수기주문 관리 화면으로 이동합니다.
