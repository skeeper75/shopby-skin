# 환불 관리

<Callout type="info">
**환불 요청된 주문 건을 확인할 수 있습니다.**

**주문 환불 처리 및 클레임 관련 주문상세정보를 확인할 수 있습니다.**
</Callout>

**😄&#x20;**<mark style="background-color:red;">**샵바이 도움말에서 '**</mark>[<mark style="background-color:red;">**클레임/정산 정책**</mark>](https://shopby-help.nhn-commerce.com/support/order/claim/claim-policy)<mark style="background-color:red;">**' 가이드를 자세하게 확인할 수 있어요.**</mark>

## 환불 관리 검색조건

<figure><img src="/images/claim-order/refund/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FSeKUdwlHKIoQmORz0C4O%2F%ED%99%98%EB%B6%88%EA%B4%80%EB%A6%AC_%EA%B2%80%EC%83%89%EC%A1%B0%EA%B1%B4%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;환불관리 검색조건></p></figcaption></figure>

**① 쇼핑몰** \
환불 주문 건을 조회할 쇼핑몰을 선택합니다.

**② 검색어** \
클레임번호, 주문번호, 회원번호, 상품번호 등 다양한 검색어로 교환 주문 건을조회합니다. 일부 검색어는 지원하지 않으며 정확한 검색어를 입력해 주세요.

<details>

<summary>🤚 <strong>검색어 선택</strong></summary>

* 클레임 번호 : 고객이 환불 신청할 때 생성된 클레임 번호입니다.
* 주문번호 : 고객이 주문을 완료하면 생성되는 주문번호입니다. 전체 주문 조회 메뉴에서 확인할 수 있습니다.
* 주문자명 : 주문한 고객의 이름입니다.
* 회원번호 : 회원가입 시 생성되는 고유한 번호입니다. 회원 조회 메뉴에서 확인할 수 있습니다.
* 주문자 연락처 : 주문자의 연락처입니다. 숫자만 입력해도 검색이 가능합니다.
* 수령자명 : 상품을 수령할 수령자의 이름입니다. 주문자와 다른 이름이 기재될 수 있습니다.
* 수령자 연락처 : 상품을 수령할 수령자의 연락처입니다. 주문자와 다른 연락처가 기재될 수 있습니다.
* 네이버페이 주문번호 : 네이버페이로 결제한 주문 건의 경우 주문번호에 네이버페이 주문번호가 추가로 생성됩니다. 샵바이 어드민 또는 네이버페이 센터에서 확인이 가능합니다.

&#x20;     📍 참고로 네이버페이 주문번호는 '네이버페이 주문형'에만 생성됩니다.\
&#x20;         '네이버페이 결제형'은 샵바이 주문번호 하나로 관리되므로, 주문번호로 검색해주세요.

</details>

**③ 기간 검색**\
기간을 설정하여 검색하는 기간 검색은 최대 3개월까지 가능합니다. \
환불 신청 일자와 환불 완료 일자 기준으로 검색할 수 있습니다.

**④** [**클레임 상태 (환불 관리)**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/claim-order/refund/condition)\
환불 주문 건의 현재 클레임 상태를 기준으로 검색할 수 있습니다.

**⑤ 플랫폼 구분**\
주문이 접수된 플랫폼 기준으로 검색할 수 있습니다.

**⑥ 결제수단**\
쇼핑몰에 연동되어 있는 결제수단 기준으로 검색할 수 있습니다.

<Callout type="warning">
샵바이에서 사용 가능한 모든 결제수단이 노출되므로, 쇼핑몰에 연동되어있지 않은 결제수단도포함되어 있는 점을 참고해 주세요.
</Callout>

**⑦ 상세검색**\
주문접수 유형(일반/수기/외부연계 등), 회원등급, 회원그룹, 배송방법, 배송업체, 송장번호 등록여부, 예약주문, 배송보류주문 등 여러 조건으로 상세한 주문 건을 검색합니다.

<details>

<summary>🤚 <strong>주문 구분</strong></summary>

* 일반 주문 : 쇼핑몰에서 접수된 일반 주문입니다.
* 수기 주문 : 별도의 채널(전화  등)로 접수하여 서비스 어드민 내 ‘수기 주문 관리’ 에서 등록한 주문입니다.
* 외부 연계 주문 : 타 플랫폼(쿠팡, 스마트스토어 등) 및 쇼핑몰에서의 주문이 샵바이에 연동되어 들어온 주문입니다.
* 정기 주문 : 정기결제 상품에 대한 주문입니다.

</details>

## 환불 관리 검색결과(리스트)

<div data-full-width="false"><figure><img src="/images/claim-order/refund/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FCUrkbm76VONqq4TuTXb8%2F%ED%81%B4%EB%A0%88%EC%9E%84%EA%B4%80%EB%A6%AC_%ED%99%98%EB%B6%88%EA%B2%B0%EA%B3%BC%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;환불관리 검색결과></p></figcaption></figure></div>

**① SMS발송**\
주문자에게 SMS를 발송할 수 있습니다.

<details>

<summary>🤚 <strong>SMS 발송 유형</strong></summary>

* 선택 주문 발송 : 선택한 주문 건의 주문자에게만 SMS를 발송합니다.
* 검색 주문 발송 : 검색된 주문 건 전체의 주문자에게 SMS를 발송합니다.

</details>

**② 클레임번호**\
클레임 주문건에 대한 고유한 번호입니다. \[클레임번호] 클릭 시 환불 처리 내역을 확인할 수 있습니다.

**③ 주문번호**\
클레임 주문건의 원주문번호입니다. \[주문번호] 클릭 시 주문 상세 정보를 확인할 수 있습니다.

**④ 주문자명**\
주문자의 이름입니다.  \[주문자명] 클릭 시 회원 상세 정보를 확인할 수 있습니다.

**⑤** [**그리드 개인설정 (톱니바퀴 아이콘)**](#id)\
검색결과 오른쪽 상단 톱니바퀴 아이콘 클릭하면 '그리드 개인설정' 창이 노출됩니다.

***

#### <mark style="color:blue;">**■ 그리드 개인설정**</mark>

* 환불 관리 검색결과 노출항목(그리드 컬럼)과 순서, 주문상세창 노출 방법을 설정할 수 있습니다.
* 주문상세창 노출 설정 : 주문번호 클릭 시 노출되는 주문상세창의 노출 방법을 설정할 수 있습니다.
* 그리드 컬럼 설정 변경 시 \[저장] 버튼을 반드시 클릭해야 설정 내용이 반영됩니다.

<details>

<summary>🤚 <strong>주문상세창 노출 설정</strong></summary>

* 탭으로 띄우기 : 주문번호 클릭 시 새로운 탭으로 주문 내역이 노출됩니다. 클릭 당 하나의 탭으로 주문번호가 띄워집니다.
* 하나의 탭으로 띄우기 : 주문번호 클릭 시 새로운 탭으로 주문 내역이 노출됩니다. 기존에 클릭했던 주문번호가 있을 경우 해당 탭에서 정보가 변경됩니다.
* 새 창으로 띄우기 : 주문번호 클릭 시 새 창으로 주문 내역이 노출됩니다. 클릭 당 하나의 창으로 주문번호가 띄워집니다.
* 하나의 새 창으로 띄우기 : 주문번호 클릭 시 새 창으로 주문 내역이 노출됩니다. 기존에 클릭했던 주문번호가 있을 경우 해당 창에서 정보가 변경됩니다.

</details>
