# 취소처리내역

<Callout type="info">
**클레임처리 \[상세보기] 또는 클레임번호를 클릭하면 노출되는 '취소처리내역' 화면에서,** \
**해당 클레임 건의 취소상품 관련 정보를 확인하고 클레임을 처리할 수 있습니다.**

**환불수단에 따라 운영자가 환불금액을 설정할 수 있는 관리자지정 환불방법을 이용할 수 있어요.**\
**단, 클레임 상태 및 취소상품 조건에 따라 관리자지정 환불방법 사용이 불가한 점을 참고해 주세요.** &#x20;

**😄보다 자세한 내용은** [**샵바이 > 관리자지정환불 가이드**](https://shopby-help.nhn-commerce.com/support/order/claim/refundmethod)**에서 확인 가능합니다.**
</Callout>

✅ <mark style="background-color:yellow;">관리자지정 환불방법은 입금 전 취소신청, 결제 후 취소신청, 취소신청 상태에서 이용할 수 있어요.</mark>

**① 취소상품**

* 상품번호를 클릭하면 해당 상품의 상품 수정페이지로 이동해요.
* \[변경] 버튼을 클릭하면 '취소구분/사유' 와 '귀책대상'을 변경할 수 있도록 입력란이 활성화돼요. \
  변경하고 반드시 \[완료] 버튼을 클릭해 주세요.

**② 취소상품 금액상세**

* 취소상품 합계금액과 취소상품에 적용된 추가할인, 상품쿠폰 및 사은품 정보가 확인됩니다.
* 상품 합계금액에서 추가할인 및 상품쿠폰재계산 금액이 차감된 취소금액합계가 산정되어 표시돼요.

**③ 할인/배송비재계산**

* 주문쿠폰이 취소상품에 적용된 할인금액과 상품취소 발생 배송비 차액이 확인됩니다.
* 주문쿠폰재계산 금액에서 배송비 차액을 더한 할인/배송비 재계산 금액이 산정되어 표시돼요.

**④ 환불예상금액**&#x20;

* 취소상품 금액합계와 할인/배송비 재계산 합계금액을 합산한 환불금액이 확인됩니다.
* \[변경] 버튼을 클릭하면 환불방법, 환불(반환)금액을 설정할 수 있도록 해당 입력란이 활성화 돼요.\
  환불금액을 설정하고 반드시 \[완료] 버튼을 클릭해 주세요.
* 환불방법을 해당 클레임 주문 건의 **최초 결제수단, 현금환불, 관리자지정환불** 중에서 선택할 수 있어요.\
  📍<mark style="color:orange;">관리자지정환불은 환불방법 셀렉트박스를 선택하면 노출돼요.</mark><br>
* 관리자지정환불 선택 시, 적립금반환 / PG환불액 / 현금환불액을 환불(반환)가능 잔여금액 만큼 각각 \
  설정할 수 있어요.
  * 환불(반환)가능한 최대 금액이 개별 표시됩니다.
  * PG환불액과 현금환불액은 함께 설정할 수 없어요.
* 환불계좌번호
  * 현금환불액을 설정한 경우 고객의 환불계좌정보를 필수로 입력해야 해요.

**⑤ 지급(예정)적립금재계산**

* 취소 상품을 제외하고, 해당 주문 건에 다시 지급될 적립금 재계산 금액이 산정되어 표시돼요.
