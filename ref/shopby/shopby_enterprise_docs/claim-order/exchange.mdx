# 교환 관리

<Callout type="info">
**교환 요청된 주문 건을 확인할 수 있습니다.**

**주문 교환 처리 및 클레임 관련 주문상세정보를 확인할 수 있습니다.**
</Callout>

**😄&#x20;**<mark style="background-color:red;">**샵바이 도움말에서 '**</mark>[<mark style="background-color:red;">**교환 프로세스**</mark>](https://shopby-help.nhn-commerce.com/support/order/claim#undefined-1)<mark style="background-color:red;">**' 와 '**</mark>[<mark style="background-color:red;">**클레임/정산 정책**</mark>](https://shopby-help.nhn-commerce.com/support/order/claim/claim-policy)<mark style="background-color:red;">**' 가이드를 자세하게 확인할 수 있어요.**</mark>

## 교환 관리 검색조건

<figure><img src="/images/claim-order/exchange/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FlgABvOw7p1QL1DS92zf7%2F%EA%B5%90%ED%99%98%EA%B4%80%EB%A6%AC_%EA%B2%80%EC%83%89%EC%A1%B0%EA%B1%B4%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;교환 관리 검색조건></p></figcaption></figure>

<Callout type="warning">
<mark style="background-color:green;">**교환 신청일로부터  10영업일 이상 경과**</mark>된 경우, <mark style="background-color:green;">**교환 수거 완료일로부터 1영업일 이상 경과**</mark>된 경우 '교환 지연' 건으로 분류됩니다.\
교환 처리가 지연된 건수가 표시되며, 숫자를 클릭하면 교환지연 클레임건만 결과로 확인할 수 있습니다.
</Callout>

**① 쇼핑몰** \
교환 주문 건을 조회할 쇼핑몰을 선택합니다.

**② 파트너사** \
쇼핑몰 자체상품과 파트너사 공급상품의 취소 주문 건을 각각 조회하고 관리할 수 있습니다.

**③ 검색어**\
클레임번호, 주문번호, 회원번호, 상품번호 등 다양한 검색어로 교환 주문 건을조회합니다. 일부 검색어는 지원하지 않으며 정확한 검색어를 입력해 주세요.

<details>

<summary>🤚 <strong>검색어 선택</strong></summary>

* 클레임번호 : 고객이 주문 건을 교환 신청할 때 생성된  클레임 번호입니다.
* 주문번호 : 고객이 주문을 완료하면 생성되는 주문번호입니다. 전체주문 조회 메뉴에서 확인할 수 있습니다.
* 주문자명 : 주문한 고객의 이름입니다.
* 주문자 연락처 : 주문자의 연락처입니다. 숫자만 입력해도 검색이 가능합니다.
* 회원번호 : 회원가입 시 생성되는 고유한 번호입니다. 회원 조회 메뉴에서 확인할 수 있습니다.
* 상품번호 : 상품 등록 시 생성되는 고유한번호입니다. 상품정보 조회/수정 메뉴에서 확인할 수 있습니다.
* 상품명 : 상품의 이름입니다. 상품정보 조회/수정 메뉴에서 확인할 수 있습니다. 상품명의 일부 단어만으로도 검색이 가능합니다.
* 상품관리코드 : 상품 등록 시 입력한 상품관리코드입니다. 상품 등록 시 입력하지 않았다면 검색이 불가합니다.
* 옵션값 : 상품에 대한 옵션값입니다. 옵션 항목 내 옵션값으로 검색할 수 있습니다.\
  예) 상품 → 신발 / 옵션명 → 사이즈 / 옵션값 → 230, 240
* 옵션관리코드 : 옵션 등록 시 입력한옵션관리코드입니다. 옵션 등록 시 입력하지 않았다면 검색이 불가합니다.
* 송장번호 : 해당 주문 건에 할당한 배송장번호입니다.
* 수령자명 : 상품을 수령할 수령자의 이름입니다.  주문/결제 시 주문자와 다른 이름을 수령자명으로 기재할 수 있으므로, 검색에 참고해 주세요.
* 수령자 연락처 : 상품을 수령할 수령자의 연락처입니다. 주문/결제 시 주문자와 다른 연락처를 수령자 연락처로기재할 수 있으므로, 검색에 참고해 주세요.
* 네이버페이 주문번호 : 네이버페이로 결제한 주문 건의 경우 주문번호에 네이버페이 주문번호가 추가로 생성됩니다. 샵바이 어드민 또는 네이버페이 센터에서 확인이 가능합니다.

&#x20;     📍 참고로 네이버페이 주문번호는 '네이버페이 주문형'에만 생성됩니다.\
&#x20;         '네이버페이 결제형'은 샵바이 주문번호 하나로 관리되므로, 주문번호로 검색해주세요.

</details>

**④ 기간검색** \
기간을 설정하여 검색하는 기간 검색은 최대 3개월까지 가능합니다. \
교환 신청 일자와 교 완료 일자 기준으로 검색할 수 있습니다.

**⑤** [**클레임상태 (교환 관리)**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/claim-order/exchange/condition)\
교환 주문 건의 현재 클레임 상태를 기준으로 검색할 수 있습니다.

**⑥ 플랫폼 구분** \
주문이 접수된 플랫폼 기준으로 검색할 수 있습니다.

**⑦ 결제수단** \
쇼핑몰에 연동되어 있는 결제수단 기준으로 검색할 수 있습니다.

<Callout type="warning">
샵바이에서 사용 가능한 모든 결제수단이 노출되므로, 쇼핑몰에 연동되어있지 않은 결제수단도 포함되어 있는 점을 참고해 주세요.
</Callout>

**⑧ 상세검색**\
주문접수 유형(주문구분 : 일반/수기/외부연계 등), 회원등급, 회원그룹, 배송방법, 배송업체, 송장번호 등록여부, 예약주문, 배송보류주문 등 여러 조건으로 상세한 주문 건을 검색합니다.

<details>

<summary>🤚 <strong>주문 구분</strong></summary>

* 일반 주문 : 쇼핑몰에 접수된 일반 주문입니다.&#x20;
* 수기 주문 : 별도의 채널(유선 및 주문신청서식등)로 주문을 접수하여 어드민 '수기 주문 관리' 에 수기로 등록한 주문입니다.&#x20;
* 외부 연계 주문 : 외부 플랫폼(쿠팡, 스마트스토어 등) 및 타 쇼핑몰의 주문이 샵바이에 연동되어 등록된 주문입니다.&#x20;
* 정기 주문 : 정기결제(배송) 상품의 주문입니다.

</details>

## 교환 관리 검색결과(리스트)

<figure><img src="/images/claim-order/exchange/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FCShMioZBskF73t1d1VUh%2F%ED%81%B4%EB%A0%88%EC%9E%84%EA%B4%80%EB%A6%AC_%EA%B5%90%ED%99%98%EA%B4%80%EB%A6%AC%EA%B2%80%EC%83%89%EA%B2%B0%EA%B3%BC%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;교환관리 검색결과(리스트)></p></figcaption></figure>

**① SMS 발송** \
주문자에게 SMS를 발송할 수 있습니다.

<details>

<summary>🤚 <strong>SMS 발송 유형</strong></summary>

* 선택 주문 발송 : 선택한 주문 건의 주문자에게만 SMS를 발송합니다.
* 검색 주문 발송 : 검색된 주문 건의 전체주문자에게 SMS를 발송합니다.

</details>

**② 클레임처리(교환처리내역)**\
\[상세 보기] 클릭 시 '교환처리내역' 화면이 노출되며, 교환상품 관련 정보 확인과 클레임을 처리할 수 있어요.

<Callout type="warning">
교환 신청\[승인대기] 상태의 클레임 건은 교한승인을 바로 처리할 수 있는 \[교환승인]이 노출돼요.
</Callout>

**③ 클레임상태**\
해당 교환 주문 건의 현재 클레임 관리 상태가 확인됩니다.

**④ 클레임번호(교환처리내역)**\
클레임 주문건에 대한 고유한 번호입니다. \
\[클레임번호] 클릭 시  '교환처리내역' 화면이 노출되며, 교환상품 관련 정보 확인과 클레임을 처리할 수 있어요.

**⑤ 주문번호**\
클레임 주문건의 원주문번호입니다. \[주문번호] 클릭 시 주문 상세 정보를 확인할 수 있습니다.\
\
**⑥ 주문자명**\
주문자의 이름입니다.  \[주문자명] 클릭 시 회원 상세 정보를 확인할 수 있습니다.

**⑦** [**그리드 개인설정(톱니바퀴 아이콘)**](#undefined-4)\
검색결과 오른쪽 상단 톱니바퀴 아이콘 클릭하면 '그리드 개인설정' 창이 노출됩니다.

<Callout type="danger">
**환불수단에 따라 운영자가 환불금액을 설정할 수 있는 관리자지정 환불방법을 이용할 수 있어요.**\
**단, 클레임 상태 및 교환상품 조건에 따라 관리자지정 환불방법 사용이 불가한 점을 참고해 주세요.**\ <mark style="background-color:yellow;">관리자지정 환불방법은 운영자가 교환신청을 진행한 클레임 건의 출고 전 교환신청, 출고 후 교환신청 상태에서 이용할 수 있어요.</mark>

**😄보다 자세한 내용은** [**샵바이 > 관리자지정환불 가이드**](https://shopby-help.nhn-commerce.com/support/order/claim/refundmethod)**에서 확인 가능합니다.**
</Callout>

***

#### <mark style="color:blue;">**■ 그리드 개인설정**</mark>

* 교환 관리 검색결과 노출항목(그리드 컬럼)과 순서, 주문상세창 노출 방법을 설정할 수 있습니다.
* 주문상세창 노출 설정 : 주문번호 클릭 시 노출되는 주문상세창의 노출 방법을 설정할 수 있습니다.
* 그리드 컬럼 설정 변경 시 \[저장] 버튼을 반드시 클릭해야 설정 내용이 반영됩니다.

<details>

<summary>🤚 <strong>주문상세창 노출 설정</strong></summary>

* 탭으로 띄우기 : 주문번호 클릭 시 새로운 탭으로 주문 내역이 노출됩니다. 클릭 당 하나의 탭으로 주문번호가 띄워집니다.
* 하나의 탭으로 띄우기 : 주문번호 클릭 시 새로운 탭으로 주문 내역이 노출됩니다. 기존에 클릭했던 주문번호가 있을 경우 해당 탭에서 정보가 변경됩니다.
* 새 창으로 띄우기 : 주문번호 클릭 시 새 창으로 주문 내역이 노출됩니다. 클릭 당 하나의 창으로 주문번호가 띄워집니다.
* 하나의 새 창으로 띄우기 : 주문번호 클릭 시 새 창으로 주문 내역이 노출됩니다. 기존에 클릭했던 주문번호가 있을 경우 해당 창에서 정보가 변경됩니다.

</details>
