# 회원통계

<Callout type="info">
**쇼핑몰 회원의 기간별 회원가입 현황을 통계 데이터로 확인할 수 있습니다.**
</Callout>

## 기간별 회원현황 검색조건

<figure><img src="/images/statistic/member/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F2EQgtW3rgO1Rsqs6ATKY%2F%ED%9A%8C%EC%9B%90%ED%86%B5%EA%B3%84%20%EA%B2%80%EC%83%89%EC%A1%B0%EA%B1%B4.jpg" alt=""/><figcaption><p>&#x3C;기간별 회원현황 검색조건></p></figcaption></figure>

**① 쇼핑몰** \
회원을 조회할 쇼핑몰을 선택합니다.

**② 회원유형**\
회원 가입 유형으로 검색이 가능합니다.\
쇼핑몰 회원, 연동아이디회원, 오픈아이디 회원(페이코, 네이버, 카카오, 페이스북, 라인) 중 선택하여 조회합니다.

* 쇼핑몰회원 : 쇼핑몰 회원가입을 통해 가입한 회원
* 연동아이디회원 : 외부 회원 연동을 이용하는 업체 대상 API로 회원가입과 로그인이 가능한 회원
* 오픈아이디회원 : SNS 간편로그인을 통해 가입한 회원 (네이버, 카카오, 페이코, 페이스북, 라인)

**③ 회원성별**\
회원의 성별을 선택하여 조회합니다.

**④ 플랫폼 구분**\
회원가입을 진행한 플랫폼 기준으로 검색할 수 있습니다.

**⑤** **조회기간**\
기간 별, 월 별로 지정하여 검색하는 조회기간 검색은 최대 3개월까지 가능합니다.<br>
