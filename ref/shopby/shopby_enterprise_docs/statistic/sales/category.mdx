# 카테고리 판매현황

<Callout type="info">
**쇼핑몰의 표준카테고리별 판매현황을 통계 데이터로 확인할 수 있습니다.**
</Callout>

## 카테고리 판매현황 검색조건

<figure><img src="/images/statistic/sales/category/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F59vbiYzFPihmB3jlvUxQ%2F%ED%8C%90%EB%A7%A4%ED%86%B5%EA%B3%84_%EC%B9%B4%ED%85%8C%EA%B3%A0%EB%A6%AC%20%ED%8C%90%EB%A7%A4%ED%98%84%ED%99%A9%20%EA%B2%80%EC%83%89%EC%A1%B0%EA%B1%B4.jpg" alt=""/><figcaption><p>&#x3C;카테고리 판매현황 검색조건></p></figcaption></figure>

**① 쇼핑몰** \
카테고리 판매현황을 조회할 쇼핑몰을 선택합니다.

**② 파트너**\
파트너사를 검색하여, 파트너사 별 카테고리 판매현황을 조회합니다.

**③ 브랜드**\
브랜드를 검색하여, 브랜드별 카테고리 판매현황을 조회합니다.

**④ 회원유형** \
회원가입 유형 별 카테고리 판매현황을 조회합니다.

**⑤ 회원성별** \
회원의 성별을 선택하여 카테고리 판매현황을 조회합니다.

**⑥ 회원등급** \
회원등급 별 카테고리 판매현황을 조회합니다.

**⑦ 회원그룹** \
회원그룹 별 카테고리 판매현황을 조회합니다.

**⑧ 플랫폼구분**\
판매가 진행된 플랫폼 기준으로 카테고리 판매현황을 조회합니다.

**⑨ 클레임구분** \
취소/교환/반품/환불 등의 클레임 건을 제외 또는 포함하여 카테고리 판매현황을 조회할 수 있습니다.

**⑩ 표준카테고리**\
표준카테고리를 선택하여 해당 카테고리 판매현황을 조회합니다.

**⑪ 기간** \
기간 별, 월 별로 지정하여 검색하는 조회기간 검색은 최대 3개월까지 가능합니다.
