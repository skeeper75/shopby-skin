# 파트너 검색/등록

<Callout type="info">
**shop by에 등록된 파트너사 정보를 검색하여 쇼핑몰과 거래(계약) 등록을 진행하는 파트너 검색/등록 메뉴입니다.**
</Callout>

## 사업자 정보 검색

<figure><img src="/images/partner/register/contract/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FCXaHPjRQWwUDNZMlWy0b%2F%ED%8C%8C%ED%8A%B8%EB%84%88%EC%82%AC%20%EA%B2%80%EC%83%89%20%EB%93%B1%EB%A1%9D%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;파트너 검색/등록></p></figcaption></figure>

**① 회사명**\
검색을 진행할 파트너사의 회사명을 입력합니다.

**② 사업자등록번호**\
검색을 진행할 파트너사의 사업자등록번호를을 입력합니다.

**③** \[조회] 버튼을 클릭해 검색한 파트너사 정보를 확인해 주세요.

**④** 쇼핑몰 거래(계약) 등록을 진행할 파트너사를 선택하고 \[다음] 버튼을 클릭합니다.

## 세부사항 확인/입력

<figure><img src="/images/partner/register/contract/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FEwuMYn6QiFUAAphqrMlR%2F%ED%8C%8C%ED%8A%B8%EB%84%88%EC%82%AC%20%EA%B2%80%EC%83%89%20%EB%93%B1%EB%A1%9D%20%EB%B3%B5%EC%82%AC2.jpg" alt=""/><figcaption><p>&#x3C;파트너 검색/등록></p></figcaption></figure>

**① 기본정보**\
검색한  파트너사의 국가와 주소를 확인합니다.

**② 사업자 기본정보** \
파트너사 신규 등록과정에서 파트너사가 직접 입력한 사업자 세부 정보를 확인합니다.\
📍파트너번호는 파트너사 등록 시 자동으로 부여되는 고유번호입니다.

**③ 정산**\
파트너사 신규 등록과정에서 파트너사가 직접 입력한 정산 담당자 정보와 정산 계좌정보 등의 정산관련 정보를 확인합니다.

**④ 쇼핑몰 계약 담당자**\
파트너사 신규 등록과정에서 설정한 마스터 계정 ID와 쇼핑몰 거래(계약) 담당자 관련 정보를 확인합니다.

**⑤ 파트너 등록 세부사항**\
검색한 파트너사와의 거래(계약) 등록에 필요한 파트너사 등록 세부 정보를 입력합니다.\
📍[파트너사 신규/등록 > 파트너  등록 세부사항](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/partner/register/new) 메뉴와 동일한 과정으로 내용을 참고해 주세요. &#x20;

* 거래 쇼핑몰 : 해당 파트너사와 거래(계약)를 진행할 쇼핑몰을 선택합니다.
* 입점 계약서 사용 : 파트너사 동의가 필요한 입점 계약 내용을 등록할 수 있는입점 계약서 사용 여부를 설정합니다.
* 담담 MD명 : 해당  파트너사를 담당하는 쇼핑몰 담당 MD 운영자를 선택합니다.
* 수수료율 : 파트너사 판매 정산에 반영되는 수수료율을 입력합니다.
* 전달 메모 : 파트너어드민 마스터 운영자 계정 등록 메일에 함께 안내되는 전달 사항을 입력합니다.

**⑥** \[파트너 등록] 버튼을 클릭해 파트너 검색/등록을 완료해 주세요.<br>
