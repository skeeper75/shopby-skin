# 파트너 복사등록

<Callout type="info">
**shop by에 등록된 파트너사 정보를 활용하여 파트너사 추가 계정 정보를 등록하고, 쇼핑몰과 거래(계약) 등록을 진행하는 파트너 복사등록 메뉴입니다.**
</Callout>

## 사업자 정보 검색

<figure><img src="/images/partner/register/copy/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FKFhHoOGWevNFsnDbIpBy%2F%ED%8C%8C%ED%8A%B8%EB%84%88%EC%82%AC%20%EB%B3%B5%EC%82%AC%20%EB%93%B1%EB%A1%9D.jpg" alt=""/><figcaption><p>&#x3C;파트너 복사등록></p></figcaption></figure>

**① 회사명**\
검색을 진행할 파트너사의 회사명을 입력합니다.

**② 사업자등록번호**\
검색을 진행할 파트너사의 사업자등록번호를을 입력합니다.

**③** \[조회] 버튼을 클릭해 검색한 파트너사 정보를 확인해 주세요.

**④** 쇼핑몰 거래(계약) 등록을 진행할 파트너사를 선택하고 \[다음] 버튼을 클릭합니다.

## 세부사항 확인/입력

<figure><img src="/images/partner/register/copy/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FhqeEuELBawFYuDcap7bD%2F%ED%8C%8C%ED%8A%B8%EB%84%88%EC%82%AC%20%EB%93%B1%EB%A1%9D%20%EB%B3%B5%EC%82%AC%EB%93%B1%EB%A1%9D%20%EC%84%B8%EB%B6%80%EC%82%AC%ED%95%AD%20%ED%99%95%EC%9D%B8%20%EA%B4%80%EB%A6%AC.jpg" alt=""/><figcaption><p>&#x3C;파트너 복사등록></p></figcaption></figure>

**① 기본정보**\
파트너사 신규 등록과정에서 파트너사가 직접 입력한 사업자 세부정보를 확인합니다.\
📍파트너번호는 파트너사 등록 시 자동으로 부여되는 고유번호입니다.

**② 파트너 등록 세부사항**\
검색한 파트너사와의 거래(계약) 등록에 필요한 파트너사 등록 세부 정보를 입력합니다.\
📍[파트너사 신규/등록 > 파트너  등록 세부사항](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/partner/register/new) 메뉴와 동일한 과정으로 내용을 참고해 주세요. &#x20;

* 신규 파트너명 : 새롭게 지정할 파트너사 이름을 입력하고, \[중복확인] 버튼으로 사용 여부를 확인합니다.
* 마스터 운영자명 : 파트너사 세부 정보 및 계정 등록을 진행할 파트너사의 마스터 운영자명을 입력합니다.
* e메일 주소 : 입력한 e-메일 주소로 '파트너어드민 마스터 운영자 계정 등록' 메일이 발송됩니다.
* 거래 쇼핑몰 : 해당 파트너사와 거래(계약)를 진행할 쇼핑몰을 선택합니다.
* 입점 계약서 사용 : 입점 계약서 사용 여부를 설정합니다.
* 담담 MD명 : 해당 파트너사를 담당하는 쇼핑몰 담당 MD 운영자를 선택합니다.
* 수수료율 : 파트너사 판매 정산에 반영되는 수수료율을 입력합니다.
* 전달 메모 : 파트너어드민 마스터 운영자 계정 등록 메일에 함께 안내되는 전달 사항을 입력합니다.

**③** \[파트너 등록] 버튼을 클릭해 파트너 복사등록을 완료해 주세요.
