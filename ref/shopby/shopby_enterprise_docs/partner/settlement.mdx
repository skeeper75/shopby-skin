# 매출 내역 조회

<Callout type="info">
**검색조건에 설정한 기간별 파트너사 매출 내역을 조회할 수 있습니다.**&#x20;

**매출 내역은 조회일자를 기준으로, 해당 조회일의 전일자 매출 데이터까지만 집계됩니다.**&#x20;

**설정한 정산기준기점을 바탕으로 매출 데이터가 집계되며, 정산기준시점에 도래하지 않은 판매 건은 매출 내역에 집계되지 않아요.**

**정산기준시점 설정은  파트너사 정산 관련 설정은** [**서비스관리 > 쇼핑몰 수정 >  정산 설정**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/mall/modification#undefined-14) **메뉴에서 가능해요.**
</Callout>

## 매출 내역 조회 검색조건

<figure><img src="/images/partner/settlement/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F5Fd1Y9nvpNUcbOmsAB97%2F%EB%A7%A4%EC%B6%9C%EB%82%B4%EC%97%AD%EC%A1%B0%ED%9A%8C%20%EA%B2%80%EC%83%89%EC%A1%B0%EA%B1%B4%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;매출 내역 조회 검색조건></p></figcaption></figure>

**① 쇼핑몰** \
파트너사 매출 내역을 조회할 쇼핑몰을 선택합니다.

**② 파트너구분** \
매출 내역을 조회할 파트너사 유형을 '국내파트너'와 '해외 파트너' 중에서 선택해 주세요.

**③ 기간** \
매출 내역 조회 기간을 '월별' 조회와 '일별' 조회 중에서 선택합니다.\
📍'일별' 기간 조회를 선택하면, 조회 대상 시작일과 종료일을 설정할 수 있어요.

## 매출 합계(매출 내역 조회 리스트)

<figure><img src="/images/partner/settlement/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2Fh8LEscvQauiuGzLn1Hkv%2F%EB%A7%A4%EC%B6%9C%EB%82%B4%EC%97%AD%EC%A1%B0%ED%9A%8C%20%EB%A6%AC%EC%8A%A4%ED%8A%B8%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;매출 합계(매출 내역 조회 리스트)></p></figcaption></figure>

<Callout type="warning">
**매출합계**\
설정한 조회기간의 파트너사 매출 합계 상세 현황이 확인됩니다.\
\
**일별 매출 상세**\
설정한 조회기간에 해당하는 파트너사 일별 매출 상세 현황이 확인됩니다.
</Callout>

**① 조회일자**\
설정한 매출 내역 조회기간이 확인됩니다.\
'월별' 기간 조회로 선택하면, 해당 월의 시작일과 종료일이 표기되어어요.

**② 판매수량**\
조회일자에 해당하는 총 판매수량이 확인됩니다.

**③ 환불수량**\
조회일자에 해당하는 총 환불수량이 확인됩니다.

**④ 판매액(A)**\
조회일자에 해당하는 판매 합계금액이 확인됩니다.\
📍판매금액 = 상품 판매가 - 즉시할인 금액 + 옵션가

**⑤ 판매수수료(B)**\
조회일자에 해당하는 판매수수료 합계금액이 확인됩니다.

* 판매수수료 → 주문시점에 상품에 적용된 상품수수료 기준으로 산출
* 판매수수료 계산 방식\
  공급가 직접 입력 방식 → 판매액 - 공급가\
  상품수수료율 방식  → 판매액 X 수수료율\
  사입상품 → 판매액 - 매입가

**⑥ 배송비(C)**\
조회일자에 해당하는 배송비 합계금액이 확인됩니다.\
📍배송비 = 주문 건에 발생된 배송비 총 금액

**⑦ 추가/상품/사은품할인파트너부담(D)**\
조회일자에 해당하는 추가/상품/사은품 등의 프로모션 할인금액 중, 파트너사 부담금이 확인됩니다.

**⑧ 공제금액(E)**\
조회일자에 해당하는 파트너사 공제금액 합계가 확인됩니다.\
📍공제금액 = 상품금액조정 파트너부담 + 배송비조정금액 파트너부담 + 쇼핑몰배송비 - 환불보류 금액조정 \
📍공제금액 계산식 항목은 월별 및 일별 매출 현황 > 상세보기 다운로드 > 엑셀파일 생성 > 엑셀파일에서\
&#x20;    자세히 확인할 수 있어요.<br>

**⑨ 정산예상금액 (F=A=B+C-D-E)**\
조회일자에 해당하는 정산예상 합계금액이 확인됩니다. \
📍정산예상금액 = 판매액 - 판매수수료 + 배송비 - 추가/상품/사은품할인파트너부담 - 공제금액

**⑩ 상세보기 (다운로드)**\
조회일자에 해당하는 파트너사 매출 내역 상세 현황 데이터를 엑셀파일로 다운로드 받을 수 있습니다.
