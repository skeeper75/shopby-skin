# 정산 현황 조회

<Callout type="info">
**설정한 월별 정산예정일 기준으로, 전월 발생한 매출에 따른 정산예상금액을 조회할 수 있습니다.**

**정산기준시점에 도래하지 않은 판매 건은 정산예상금액에 집계되지 않아요.**\
**(예 : 정산기준시점 '구매확정'으로 설정 → 구매확정 전 상태 판매 건은 정산예상금액에서 집계 제외)**

**파트너사 정산 관련 설정은** [**서비스관리 > 쇼핑몰 수정 >  정산 설정**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/mall/modification#undefined-14) **메뉴에서 가능하며, 설정한 정산기준기점 / 정산예정일 / 지급비율을 바탕으로 정산 데이터가 집계됩니다.**
</Callout>

## 정산 현황 조회 검색조건

**① 쇼핑몰**\
파트너사 정산 현황을 조회할 쇼핑몰을 선택합니다.&#x20;

**② 파트너구분**\
정산 현황을 조회할 파트너사 유형을 '국내파트너'와 '해외 파트너' 중에서 선택해 주세요.

**③ 기간**\
정산 현황 조회 대상 '년도'와 '월'을 선택합니다.

## 정산 현황 조회 내역(정산 캘린더)

<figure><img src="/images/partner/settlement-list/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FO57t8ZCtHk9w56t9MDtt%2F%EC%A0%95%EC%82%B0%ED%98%84%ED%99%A9%EC%A1%B0%ED%9A%8C%20%EB%82%B4%EC%97%AD%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;정산 현황 조회 내역(캘린더)></p></figcaption></figure>

**①** 조회한 월 별 정산 현황이 캘린더로 확인됩니다.

**②** 월 별 정산 캘린더에서 <mark style="color:blue;">**확정내역은 파란색**</mark> / <mark style="color:orange;">**미확정내역은 주황색**</mark>으로 표시됩니다.

**③** 정산내역을 클릭하면, 정산예상금액 상세 현황을 리스트로 확인할 수 있습니다.<br>

## 정산예상금액 상세(리스트)

<figure><img src="/images/partner/settlement-list/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FbnXcZlkog0exxhVHjODc%2F%ED%8C%8C%ED%8A%B8%EB%84%88%EC%82%AC%20%EC%A0%95%EC%82%B0%ED%98%84%ED%99%A9%20%EC%83%81%EC%84%B8%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;정산예상금액 상세></p></figcaption></figure>

**① 정산예정일**\
파트너사 정산예정일이 확인됩니다.

**② 정산유형**\
파트너사 정산 주기는 월 정산 고정방식입니다.

**③ 확정여부**\
설정한 정산예정일이 경과되면 '미확정'에서 '확정'으로 확정여부가 자동 표기됩니다.\
📍정산예정일은 정산 데이터에 영향을 주지 않으며, 정산 캘린더 표기 및 확정여부에 활용되는 점을 참고해 주세요.

**④ 집계기간**\
정산예상금액 산출을 위한 매출 데이터 집계 기간이 확인됩니다.

**⑤ 판매액(A)**\
파트너사의 전월 판매 합계금액이 확인됩니다.\
📍판매금액 = 상품 판매가 - 즉시할인 금액 + 옵션가

**⑥ 판매수수료(B)**\
파트너사의 전월 판매수수료 합계금액이 확인됩니다.\
📍판매수수료 → 주문시점에 상품에 적용된 상품수수료 기준으로 산출

**⑦ 배송비(C)**\
파트너사의 전월 배송비 합계금액이 확인됩니다.\
📍배송비 = 판매 건에 발생된 배송비 총 금액

**⑧ 추가/상품/사은품할인파트너부담(D)**\
파트너사의 전월 판매 건에 적용된 추가/상품/사은품 등의 프로모션 할인금액 중, 파트너사 부담금이 확인됩니다.

**⑨ 공제금액(E)**\
파트너사의 전월 공제금액 합계가 확인됩니다.\
📍공제금액 = 상품금액조정 파트너부담 + 배송비조정금액 파트너부담 + 쇼핑몰배송비 - 환불보류 금액조정 \
📍공제금액 계산식 항목은 파트너별 다운로드 > 엑셀파일 생성 > 엑셀파일에서 자세히 확인할 수 있어요.

**⑩ 정산예상금액(F)=(A-B+C-D-E)**\
파트너사의 정산예상 합계금액이 확인됩니다.\
📍정산예상금액 =  판매액 - 판매수수료 + 배송비 - 추가/상품/사은품할인파트너부담 - 공제금액

**⑪ 은행명**\
파트너사가 등록한 정산 정보 항목의 계좌 은행명이 확인됩니다.

**⑫ 계좌번호**\
파트너사가 등록한 정산 정보 항목의 계좌번호가 확인됩니다.

**⑬ 예금주**\
파트너사가 등록한 정산 정보 항목의 계좌 예금주가 확인됩니다.

**⑭ 파트너별 다운로드**\
파트너사별 월 정산 현황 세부 데이터를 엑셀파일로 다운로드받을 수 있습니다.&#x20;

**⑮ 옵션별 다운로드**\
상품옵션별 월 정산 현황 세부 데이터를 엑셀파일로 다운로드받을 수 있습니다.
