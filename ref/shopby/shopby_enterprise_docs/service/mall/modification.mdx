# 쇼핑몰 수정

## 쇼핑몰 기본정보 <a href="#undefined-1" id="undefined-1"></a>

<figure><img src="/images/service/mall/modification/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F8yyBroLk77gZ4dlSzHQm%2F%EC%87%BC%ED%95%91%EB%AA%B0%EA%B4%80%EB%A6%AC%20%EC%87%BC%ED%95%91%EB%AA%B0%EA%B8%B0%EB%B3%B8%EC%A0%95%EB%B3%B4%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;쇼핑몰 기본정보></p></figcaption></figure>

**① 쇼핑몰번호**\
쇼핑몰 고유 번호가 확인됩니다.

**② 쇼핑몰명**\
쇼핑몰 이름을 입력합니다. \
📍쇼핑몰 이름은 웹 브라우저 상단 탭에 노출됩니다.

**③ 언어**\
쇼핑몰에서 사용할 언어를 선택할 수 있습니다. (언어팩 미제공)

**④ 시간대 선택**\
쇼핑몰 운영 시간대를 설정할 수 있습니다. \
국내몰을 운영하는 경우 'UTC+9 대한민국' 시간대를 선택해 주세요.

**⑤ 사용 화폐/환율**\
쇼핑몰에서 사용할 화폐단위와 환율을 설정할 수 있습니다.&#x20;

**⑥ 국가별 최대 주문금액**\
국가별 주문 가능한 최대 주문금액을 설정할 수 있습니다. \
설정한 국가를 배송지로 선택해 주문하는 경우 주문금액이 제한되며, 원화 입력 시 사용 화폐/환율에 설정한 화폐의 환율로 자동 계산되어 표기됩니다.\
📍\[+] 버튼을 클릭하면 국가별 최대 주문금액 입력란이 추가되고, \[-] 버튼을 클릭해 입력란을 삭제합니다.

**⑦ 사용 여부**\
해당 쇼핑몰의 사용 여부를 설정할 수 있습니다.\
📍'사용 안 함' 으로 설정하면 쇼핑몰 접속이 불가한 점을 유의해 주세요.

## 쇼핑몰 도메인 정보 <a href="#undefined-2" id="undefined-2"></a>

**① PC 웹 도메인**\
PC 플랫폼에서 쇼핑몰 접속 시 사용할 쇼핑몰 URL입니다.\
&#x20;\[바로가기] 버튼을 클릭하면 입력한 PC 쇼핑몰 주소로 이동합니다.\
📍대표 도메인을 신청 및 연결한 경우, 대표 도메인 주소를 입력해 주세요.

**② 모바일 웹 도메인**\

\[바로가기] 버튼을 클릭하면 입력한한 모바일 쇼핑몰 주소로 이동합니다.

AOS 앱스토어에서 앱을 확인할 수 있는 URL입니다.

**④ iOS 모바일 앱 정보**\
iOS 앱스토어에서 앱을 확인할 수 있는 URL입니다.

## 국내 사업자 계약 서류 관리 <a href="#undefined-3" id="undefined-3"></a>

<figure><img src="/images/service/mall/modification/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FABh27sTJ9BmHvpaCdd1I%2F%EC%87%BC%ED%95%91%EB%AA%B0%EA%B4%80%EB%A6%AC%20%EA%B5%AD%EB%82%B4%EC%82%AC%EC%97%85%EC%9E%90%EC%84%9C%EB%A5%98%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;국내 사업자 계약 서류 관리></p></figcaption></figure>

**① 계약 서류**\
국내 파트너사 계약에서 사용할 서류를 등록할 수 있습니다. \
등록한 서류는 파트너사 등록 과정에서 파트너사에게 확인됩니다.\
사업자등록증, 통장사본, 법인인감증명서는 기본 서류로 설정되어 있으며, 이외에 추가로 필요한 서류를 \[파일찾기] 버튼을 클릭해 등록해 주세요.\
추가 등록한 서류는 파트너사가 다운로드 받아 확인하고, 제출할 수 있는 기능이 제공됩니다.<br>

* \[+] 버튼을 클릭해 계약 서류 등록란을 추가할 수 있고, \[-] 버튼을 클릭해 삭제할 수 있습니다.
* 등록한 계약서류를 \[다운로드] 버튼을 클릭해 다운로드 받을 수 있습니다.

**② 안내 사항**\
국내 파트너사 계약 시 파트너사에게 안내가 필요한 내용을 작성할 수 있습니다. \
파트너사 등록 과정에서 파트너사에게 해당 내용이 노출됩니다.

## 입점 표준 계약서 <a href="#undefined-5" id="undefined-5"></a>

<figure><img src="/images/service/mall/modification/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FIhUBCGDEcABrfWL8UU04%2F%EC%87%BC%ED%95%91%EB%AA%B0%20%EA%B4%80%EB%A6%AC%20%EC%9E%85%EC%A0%90%ED%91%9C%EC%A4%80%EA%B3%84%EC%95%BD%EC%84%9C%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;입점 표준 계약서></p></figcaption></figure>

**①** 파트너사 입점 계약 시, 파트너사의 동의가 필요한 계약 내용 등을 등록할 수 있습니다.

<Callout type="warning">
해당 내용은 파트너사에게 입점계약서 \[내용보기] 항목으로 안내되고, \[동의] 버튼이 노출됩니다.\
입점 계약 동의는 '파트너사 입점 계약 승인'으로 활용할 수 있습니다.\
\
파트너사 등록(신규/검색/복사) 시, 등록한 입점계약서 사용 여부를 설정할 수 있습니다.\
'사용'으로 설정해야 파트너사가 입점계약서를 확인하고 '동의' 처리할 수 있는 점을 참고해 주세요.
</Callout>

**②** \[등록] 버튼을 클릭하면 '[입점 표준 계약서](#undefined)' 창이 노출되며, 내용 입력란에 계약 내용을 작성해 주세요.\

**③** 등록 이후 \[변경] 및 \[삭제]가 가능합니다.

## 회원 인증 설정 <a href="#undefined-9" id="undefined-9"></a>

✅ **쇼핑몰 (일반)회원과 오픈아이디(간편로그인) 회원이 본인임을 인증하는 본인 인증 기능을 설정합니다.**

<figure><img src="/images/service/mall/modification/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F6l60wx8mf4UZ4lBvtmNn%2F%EC%87%BC%ED%95%91%EB%AA%B0%20%EA%B4%80%EB%A6%AC%20%ED%9A%8C%EC%9B%90%20%EC%9D%B8%EC%A6%9D%20%EC%84%A4%EC%A0%95%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;회원 인증 설정></p></figcaption></figure>

**① 쇼핑몰회원**\
쇼핑몰 회원의 본인 인증 사용 여부를 설정합니다.\
📍<mark style="color:red;">쇼핑몰회원 > 사용 안 함 : 쇼핑몰에서 더 이상 회원가입 진행이 되지 않도록 설정합니다.</mark>

* 인증 사용여부 : 쇼핑몰 회원의  본인확인인증 사용여부를 설정합니다.
* 인증종류 설정 : 본인확인 인증을 진행하는 방식을 설정합니다.

<Callout type="warning">
SMS인증 : 회원에게 문자로 일회성 인증번호를 발송하여 인증하는 방식으로, SMS 사용 설정과  SMS 포인트를 충전해야 합니다.<br>

휴대폰인증 : 통신사 휴대폰인증 방식으로 쇼핑몰에 휴대폰 본인확인 부가서비스를 연동해야 사용할 수 있습니다. \
**📍'**[**회원 관리 > 회원가입 항목 관리**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/member/collection-setting)**' 메뉴의&#x20;**<mark style="color:blue;">**회원가입 항목 설정을 '사용함'으로 설정한 경우,**</mark>\
&#x20;     <mark style="color:blue;background-color:orange;">**이름/휴대폰번호/생년월일/성별 항목을 모두 '사용'으로 설정해야 휴대폰인증이 가능합니다.**</mark>

이메일인증 : 회원의 이메일주소로 일회성 인증번호를 발송하여 인증하는 방식입니다.\

**① 회원가입 승인 사용설정**\
'승인 후 가입'으로설정하면, 운영자가 회원가입 승인 처리를 수동으로진행해야 회원가입이 완료됩니다. \

📍회원가입일 기준 60일 이내에 회원가입 승인 처리를 진행하지 않으면, 회원가입 정보가 자동 삭제 처리되므로 유의해 주세요.
