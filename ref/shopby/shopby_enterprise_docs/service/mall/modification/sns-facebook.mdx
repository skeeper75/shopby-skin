# 페이스북 간편가입

<Callout type="info">
**페이스북 간편 회원가입을 설정하여 쇼핑몰에서 회원 가입을 진행할 수 있어요.** \
**등록(저장)한 간편 회원가입 설정 값 수정은 서비스 관리 > 외부 서비스 설정 > 간편로그인 연동 항목에서 가능해요.**\
\
**2023년 2월 1일부터 페이스북 간편가입 서비스는 인증이 완료된, 비즈니스 포트폴리오(계정)와 연결해야 사용이 가능합니다.**\
**보다 자세한 내용은** [**페이스북 비즈니스 지원 센터 비즈니스 인증 방법**](https://www.facebook.com/business/help/2058515294227817?id=180505742745347) **참고해 주시기 바랍니다.**
</Callout>

**① 개발자센터**\
[페이스북 개발자센터 - Facebook](https://developers.facebook.com/?locale=ko_KR)에 접속해 주세요.

**② Meta for Developers 개발자(등록) 계정 생성**\
개발자센터 페이지에서 로그인을 해 주세요. \
상단 '시작하기' 메뉴를 클릭하고, 개발자(등록) 계정을 생성합니다.\
📍페이스북 간편가입 연동 설정은 페이스북 회원만 가능합니다. 페이스북 로그인 정보가 없다면 먼저 회원\
&#x20;   가입을 진행하고 로그인 해 주세요.

* 약관 및 정책 동의 : 내용을 확인하고 \[다음] 버튼을 클릭해 '동의' 처리합니다.
* 계정 인증 : 입력한 휴대폰 번호 또는 e-mail 주소로 인증 코드가 발송됩니다. 전송받은 코드를 입력해 인증을 완료합니다.
* 직업 선택 : 계정 생성 담당자의 역할(직업)을 선택하고, \[등록 완료] 버튼을 클릭해 개발자 등록을 완료해주세요.

**③ 앱 만들기**\
\[앱 만들기] 버튼을 클릭해 주세요.

* 앱에 필요한 기능 선택 : 'Facebook 로그인을 통한 인증 및 사용자의 데이터 요청'을 선택하고, \[다음] 버튼을 클릭합니다.
* 앱 이름 추가 : 앱 관리에서 사용되는 앱 이름을 작성해 주세요.&#x20;
* 앱 연락처 이메일 : 앱 관련 안내사항을 받을 수 있는 이메일 주소를 작성해 주세요. 고객에게는 노출되지 않습니다.
* 앱 게시 및 데이터 액세스를 유지하기 위한 요구사항 항목을 확인하고, \[앱 만들기] 버튼을 클릭해 주세요.&#x20;
* 보안을 위한 페이스북 비밀번호를 재 입력하고, \[제출] 버튼을 클릭해 앱 만들기를 완료해 주세요.

**④ 앱 정보 설정**\
앱 설정 > 기본 설정 메뉴로 이동합니다.

* 표시 이름 : 앱 이름이 확인됩니다. 페이스북 간편 회원가입 및 로그인 등에서 고객에게 노출되는 정보로 수정할 수 있어요.
* 앱 도메인 : 페이스북 간편가입 연동을 사용할 쇼핑몰 도메인을 입력해 주세요.\
  예) <https://쇼핑몰도메인> ⇀ <https://abcde.com\\>
  ⚠️ <mark style="color:orange;">페이스북 간편가입은 보안서버가 적용된 쇼핑몰 도메인에 연동할 수 있어요.</mark><br>
* 개인정보처리방침 URL : 쇼핑몰 개인정보처리방침 페이지 URL을 입력해 주세요.
* 서비스 약관 URL : 쇼핑몰 서비스 이용약관 페이지 URL을 입력해 주세요.
* 앱 아이콘 : 고객이 페이스북 간편 회원가입 단계에서 확인하는 쇼핑몰 로고를 등록합니다.\
  📍512 X 512 \~ 1024 X 1024 사이즈의 JPG, GIF, PNG 이미지 파일을 업로드 해주세요.
* 카테고리 : 쇼핑몰 운영 카테고리를 선택합니다.
* 사용자 데이터 삭제 : 고객에게 데이터 삭제 요청 방법을 제공하는 용도로, 쇼핑몰 도메인을 입력합니다.
* \[변경 내용 저장] 버튼을 클릭해 기본 설정을 완료합니다.

**⑤ 이용 사례 맞춤 설정**\
이용 사례 > 인증 및 계정 만들기 항목에서 \[맞춤 설정] 버튼을 클릭하고, 맞춤 설정 > 설정 화면으로 이동합니다.

* 유효한 OAuth 리디렉션 URI : 리디렉션 URI는 페이스 로그인 이후 고객 인증 결과를 전달받기 위한 주소입니다. 쇼핑몰 도메인을 포함하여 아래와 같이 작성해 주세요.

<Callout type="warning">
사용하는 스킨에 따라 리디렉션 URI가 상이하므로, 반드시 스킨 종류를 확인해 주세요. \
기본 스킨 외 별도의 스킨을 구매하여 사용하는 경우, 스킨 제작사에게 리디렉션 URI를 문의해 주세요. \
헤드리스 운영방식은 자체 사용중인 리디렉션 URI를 입력합니다.\ <br>

<mark style="color:blue;background-color:yellow;">오로라, 어나더 : 개별형 스킨은 PC와 모바일 각각 총 2개의 Callback URL을 입력해 주세요.</mark>

* PC : <https://쇼핑몰도메인/callback/auth-callback.html>
* 모바일 \
  ⓐ 도메인 m. : <https://m.쇼핑몰도메인/callback/auth-callback.html> \
  ⓑ 도메인 m- : <https://m-쇼핑몰도메인/callback/auth-callback.html> \
  ⓒ 도메인 /m : <https://쇼핑몰도메인/m/callback/auth-callback.html><br>

<mark style="color:blue;background-color:yellow;">오로라(리액트) : 통합형 스킨은 PC와 모바일 모두 동일합니다.</mark>

* PC/모바일 : <https://쇼핑몰도메인/callback/auth-callback>
</Callout>

* JavaScript SDK로 로그인 : \[아니오] 버튼을 클릭해, JavaScript SDK로 로그인을 \[예] 로 활성화 합니다.

* JavaScript SDK에 허용된 도메인 : 쇼핑몰 도메인을 입력합니다.

* \[변경 내용 저장] 버튼을 클릭해 맞춤 설정을 완료해 주세요.

**⑥ 앱 인증**\
검수 > 인증 > 비즈니스 인증 화면에서 \[인증 시작] 버튼을 클릭해 비즈니스 인증을 진행합니다.

* 비즈니스 상세 정보를 입력하고, 증빙 서류를 제출한 다음 인증 검토를 요청합니다.
* 비즈니스 인증은 요청 이후,  최소 10분 \~ 최대 14일(영업일 기준) 이내에 처리되는 점을 참고해 주세요.

**⑦ 앱 게시**\
비즈니스 인증이 완료되면, 게시 메뉴로 이동해 모든 설정 사항을 검토 및 확인합니다.\
\[데이터 처리 질문으로 이동]을 클릭해 데이터 처리에 관한 질문에 답변을 완료하고, 게시 화면 하단 \[게시] 버튼을 클릭해 주세요.

**⑧ 앱 ID / 앱 시크릿 코드 복사**\
앱 설정 > 기본 설정 메뉴로 이동하면,  '앱 ID' 와 '앱 시크릿 코드'가 확인됩니다.\
앱 ID / 앱 시크릿 코드를 복사해 주세요.\
📍앱 시크릿 코드의 '보기'를 클릭하 전체를 확인할 수 있어요.

**⑨ 앱 ID / 앱 시크릿 코드 입력**\
shop by 어드민 > 서비스 관리 > 쇼핑몰 수정 > SNS 간편 회원가입 설정 항목으로 이동합니다.\
페이스북 간편가입 '선택' 체크박스를 선택하면 입력란이 활성화 되고,\
복사한 <mark style="background-color:orange;">앱 ID는 Client ID</mark> 입력란에, <mark style="background-color:green;">앱 시크릿 코드는 Secret Key</mark> 입력란에 붙여넣기 해 주세요.

**⑩** 쇼핑몰 수정 화면, 하단 \[변경사항 저장] 버튼을 클릭해 설정을 완료합니다.&#x20;

**⑪** 쇼핑몰에서 적용된 페이스북 회원가입 및 로그인을 확인해 주세요.\
📍쇼핑몰 회원가입 페이지 : 간편 회원가입 > \[페이스북 아이디로 회원가입]\
&#x20;    쇼핑몰 로그인 페이지 : 간편로그인 > \[페이스북 아이디로 로그인]
