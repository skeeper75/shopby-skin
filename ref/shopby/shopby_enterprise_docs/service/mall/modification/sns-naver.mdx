# 네이버 간편가입

<Callout type="info">
**네이버 간편 회원가입을 설정하여 쇼핑몰에서 회원 가입을 진행할 수 있어요.** \
**등록(저장)한 Client ID / (Client)Secret Key 값의 수정은 서비스 관리 > 외부 서비스 설정 > 간편로그인 연동 항목에서 가능해요.**

**모든 네이버 아이디 고객의 간편 회원가입 및 로그인의 정상적인 이용을 위해 '**[**네이버 로그인 검수**](#undefined)**'가 필요합니다.**\
**보다 자세한 내용은** [**네이버 로그인 사전 검수 가이드**](https://developers.naver.com/docs/login/verify/verify.md)**를 참고해 주세요.**
</Callout>

**① 개발자센터** \
[네이버 개발자 센터 - NAVER Developers](https://developers.naver.com/main/)에 접속해 주세요.

**② 애플리케이션(Application) 로그인**\
개발자센터 페이지 > 상단 'Application' 메뉴를 클릭하고 로그인 해 주세요.\
📍네이버 간편가입 연동 설정은 네이버 회원만 가능합니다. 네이버 로그인 정보가 없다면 먼저 회원가입을\
&#x20;    진행하고 로그인 해 주세요.

**③ 애플리케이션 등록**\
애플리케이션 화면에서 \[애플리케이션 등록]을 클릭해 주세요.

* 애플리케이션 이름 : 네이버 간편 회원가입 및 로그인 등에서 고객에게 노출되는 정보입니다.
* 사용 API : '네이버 로그인'을 선택해 주세요. \
  '제공 정보 선택' 의 필수/추가 개인정보 항목은 쇼핑몰 정책을 반영해 설정해 주세요.

  **📍**제공 정보를 선택하지 않으면 shop by 어드민에서 고객 정보를 확인할 수 없습니다.
* 로그인 오픈 API 서비스 환경 : 네이버 간편가입 연동을 사용할 쇼핑몰 플랫폼을 선택합니다. \
  📍Android 및 iOS에서 설치하는 별도 네이티브앱 없이, 샵바이 쇼핑몰에서 네이버 간편가입을\
  &#x20;   사용하면  'PC 웹' 항목만 선택해 주세요.
* 서비스 URL : 네이버 간편가입 연동을 사용할 쇼핑몰 도메인을 입력해 주세요.\
  예) <https://쇼핑몰도메인> ⇀ <https://abcde.com>
* 네이버 로그인 Callback URL : Callback URL은 네이버 로그인 이후 고객 인증 결과를 전달받기 위한 주소입니다. 쇼핑몰 도메인을 포함하여 아래와 같이 작성해 주세요.

<Callout type="warning">
사용하는 스킨에 따라 Callback URL이 상이하므로, 반드시 스킨 종류를 확인해 주세요. \
기본 스킨 외 별도의 스킨을 구매하여 사용하는 경우, 스킨 제작사에게 Callback URL을 문의해 주세요. \
헤드리스 운영방식은 자체 사용중인 Callback URL을 입력합니다.\ <br>

<mark style="color:blue;background-color:yellow;">오로라, 어나더 : 개별형 스킨은 PC와 모바일 각각 총 2개의 Callback URL을 입력해 주세요.</mark>

* PC : <https://쇼핑몰도메인/callback/auth-callback.html>
* 모바일 \
  ⓐ 도메인 m. : <https://m.쇼핑몰도메인/callback/auth-callback.html> \
  ⓑ 도메인 m- : <https://m-쇼핑몰도메인/callback/auth-callback.html> \
  ⓒ 도메인 /m : <https://쇼핑몰도메인/m/callback/auth-callback.html><br>

<mark style="color:blue;background-color:yellow;">오로라(리액트) : 통합형 스킨은 PC와 모바일 모두 동일합니다.</mark>

* PC/모바일 : <https://쇼핑몰도메인/callback/auth-callback>
</Callout>

* 서비스 이용 동의 : 관련 내용을 모두 확인하고 '동의' 체크박스를 선택합니다.

* 애플리케이션 등록 설정 사항을 모두 확인하고, \[등록하기] 버튼을 클릭해 완료해 주세요.

**④ Client ID / Client Secret 복사** \
내 애플리케이션 > 개요 > 애플리케이션 정보에서서 'Client ID' 와 'Client Secret' 값이 확인됩니다. \
Client ID / Client Secret 값을 복사해 주세요. \
📍Client Secret 값의 '보기'를 클릭하면 전체를 확인할 수 있어요.

**⑤ Client ID / Client Secret 입력**\
shop by 어드민 > 서비스 관리 > 쇼핑몰 수정 > SNS 간편 회원가입 설정 항목으로 이동합니다.\
네이버 간편가입 '선택' 체크박스를 선택하면 입력란이 활성화 되고,\
복사한 <mark style="background-color:orange;">Client ID는 Client ID</mark> 입력란에, <mark style="background-color:green;">Client Secret 값은 Secret Key</mark> 입력란에 붙여넣기 해 주세요.

**⑥** 쇼핑몰 수정 화면, 하단 \[변경사항 저장] 버튼을 클릭해 설정을 완료합니다.&#x20;

**⑦** 쇼핑몰에서 적용된 네이버 회원가입 및 로그인을 확인해 주세요.\
📍쇼핑몰 회원가입 페이지 : 간편 회원가입 > \[네이버 아이디로 회원가입]\
&#x20;    쇼핑몰 로그인 페이지 : 간편로그인 > \[네이버 아이디로 로그인]

<Callout type="warning">
네이버 개발자센터 > 내 애플리케이션 > 개요 화면에서 네이버 로그인 개발상태가 '개발 중'으로 확인됩니다.\
'개발 중' 상태의 애플리케이션은 멤버관리 메뉴에 등록한 관리자 및 테스터 아이디만 간편 회원가입 및 로그인이 가능해요.\
모든 네이버 아이디 고객의 간편 회원가입 및 로그인의 정상적인 이용을 위해 '[네이버 로그인 검수](#undefined)' 요청을 반드시 진행해 주시기 바랍니다.\
\
\
**네이버 로그인 검수 준비 사항**\
쇼핑몰에서 \[네이버 아이디로 회원가입] 버튼을 클릭해, 회원가입 및 로그인 완료까지의 모든 단계별 화면을 캡처해 주세요.\
플리케이션 등록 시 '제공 정보 선택'에서 설정한 개인정보 항목이 노출되는 쇼핑몰의 모든 화면을 캡처해 주세요.
</Callout>

### 네이버 로그인 검수 <a href="#undefined" id="undefined"></a>

**① 로고 이미지 등록**\
내 애플리케이션 > API 설정 > 로그인 오픈 API 서비스 환경 > 로고 이미지 항목으로 이동합니다. \
고객이 네이버 간편 회원가입 단계에서 확인하는 쇼핑몰 로고를 등록하고, 하단 \[수정] 버튼을 클릭합니다.\
📍140 X 140 사이즈의 JPG, GIF, PNG 이미지 파일을 업로드 해주세요.

**② 네이버 로그인 검수 상태**\
애플리케이션 개발 상태가 '개발 중'으로 확인되고, \[검수요청 하러 가기] 버튼을 클릭해 네이버 로그인 검수상태 화면으로 이동해 주세요.

**③ 검수 신청 전 가이드 확인**\
[**검수 요청 가이드**](https://developers.naver.com/docs/login/verify/verify.md) 페이지로 이동해 안내 내용을 확인하고, '확인' 체크박스를 선택합니다.

**④ 제공 정보 활용처 확인**\
'제공 정보 선택'에서 설정한 개인정보 항목이 확인됩니다.\
해당 캡처 이미지 파일을 업로드 하고, 첨부 여부 체크박스를 선택해 업로드 완료를 확인해 주세요.\
📍'제공 정보 선택'에서 개인정보 항목을 설정하지 않으면, 해당 영역이 노출되지 않아요.

**⑤ 서비스 적용 형태 확인**\
**'**&#xB124;이버 로그인을 통한 신규 회원 가입에 적용'을 선택해 주세요.

**⑥ 로그인 이용 절차 캡처파일 업로드**\
해당 캡처 이미지 파일을 업로드 합니다.

**⑦ 추가 제출 서류 확인**\
쇼핑몰에 해당 사항이 없다면 '없음'을 선택해 주세요.\
📍해당 사항이 있는 경우 선택하고, 추가 제출 서류를 업로드 해주세요.

**⑧ 검수 요청**\
\[검수요청] 버튼을 클릭해 검수요청을 완료하면, 검수 상태가 '검수요청' 중으로 확인됩니다.

**⑨ 검수 결과 확인**\
애플리케이션(Application) > 계정 설정 메뉴에 설정한 사용자 이메일로 검수 결과가 발송됩니다.\
📍검수 결과는 1일 \~ 1주일 정도 소요될 수 있어요.
