# 카카오 싱크

<Callout type="info">
**카카오 싱크는 카카오 로그인 절차를 간소화 하는 부가서비스로, 빠른 회원가입과 동시에 카카오톡 채널 친구 추가가 가능하여, 쇼핑몰 마케팅 용도로 활용할 수 있어요.**

**카카오 싱크는 사업자 등록증을 보유한 사업자 계정만 가입할 수 있는 비즈니스 서비스로, 심사가 완료된 카카오 비즈니스 채널 연결이 필요합니다. (카카오 비즈니스 채널은 별도 신청(가입) 및 심사 필요)**&#x20;

**​보다 자세한 내용은** [**카카오 싱크 >  도입하기 > 카카오싱크 도입절차**](https://kakaobusiness.gitbook.io/main/tool/kakaosync/phase-in#id-1) **가이드와** [**\[프리미엄\] 카카오 신청 가이드**](https://workspace-help.nhn-commerce.com/recommendedcontents/contents/prm_kakao-sync)**를 참고해 주세요.**
</Callout>

<Callout type="warning">
**카카오 싱크 연동 전, 필요사항**<br>

[서비스 관리 > 기초정보 관리  > 서비스 기본정보](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/undefined#undefined) 항목과 카카오 싱크 연동 계정 정보를 동일하게 등록해 주세요.\
[서비스 관리 > 약관/개인정보처리방침](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/terms) 메뉴에 카카오 싱크와 관련된 내용을 기입해 주세요.\
[서비스 관리 > 쇼핑몰 수정 > SNS 간편 회원가입 로그인 URL 설정](https://nhn-commerce.gitbook.io/shopby_premium_manual/service/mall/modification#sns-url) 항목에 쇼핑몰에 맞는 Callback URL을 정확히 입력해 주세요.
</Callout>

<mark style="color:orange;">⚠️ 이미 카카오 간편가입(로그인) 기능을 사용 중인 쇼핑몰은 카카오 싱크 연동 완료 시, 카카오 간편가입</mark>\
&#x20;     <mark style="color:orange;">기능은 자동으로 비활성화 처리되며</mark> <mark style="color:orange;background-color:green;">쇼핑몰 회원은 카카오 싱크로만 가입 및 로그인이 가능</mark><mark style="color:orange;">합니다.</mark>

**① 카카오 싱크 앱 설치**\
NHN커머스 > 앱스토어에서 '[카카오 싱크 - shop by](https://apps.nhn-commerce.com/apps/255)' 앱을 먼저 설치해 주세요.\
📍설치한 앱은 shop by 어드민 > 앱 서비스 > 앱 리스트에서 확인할 수 있어요.

**② 카카오 싱크 설정**\
shop by 어드민 > 서비스 관리 > 쇼핑몰 수정 > SNS 간편 회원가입 설정 항목으로 이동합니다.\
\[카카오 싱크 설정] 버튼을 클릭해 주세요.

**③ 카카오 싱크 연동**\
카카오 싱크 설정 화면에서 \[카카오 싱크 연동] 버튼을 클릭해 주세요.

**④ 카카오 계정 로그인**\
카카오 싱크를 사용할 '카카오 비즈니스 계정'으로 로그인 해주세요.\
📍카카오 싱크 연동 설정은 카카오 회원만 가능합니다. 카카오 로그인 정보가 없다면 먼저 카카오 비즈니스         \
&#x20;    계정 가입을 진행하고 로그인 해 주세&#xC694;**.**

**⑤ 카카오 싱크 간편 설정**\
카카오 싱크 간편 설정 화면에서 안내 내용을 확인하고, \[동의하고 계속] 버튼을 클릭해 주세요.

<Callout type="warning">
카카오 개발자센터에서 앱을 등록하여 카카오 간편가입(로그인) 기능을 사용 중인 쇼핑몰은, \ <mark style="background-color:blue;">해당 앱에 카카오 싱크를 적용해야 기존 카카오 간편가입 회원의 쇼핑몰 이용이 가능</mark>한 점을 참고해 주세요.\ <mark style="background-color:blue;">다른 앱에 카카오 싱크를 적용하면</mark> 기존 카카오 간편가입 회원은 로그인이 불가하고, <mark style="background-color:blue;">신규 회원으로 가입 처리되요.</mark>
</Callout>

**⑥ 앱 설정 (기존 앱 적용 / 신규 앱 생성)**

* 권한이 있는 앱 목록에 등록한 앱 정보가 확인됩니다.\
  등록한 앱 중에서 카카오 싱크를 적용할 앱을 선택하고 \[다음] 버튼을 클릭해 주세요.
* 등록한 앱이 없다면, \[+ 신규 앱 생성하기]를 클릭합니다.\
  신규 앱의 쇼핑몰 정보(쇼핑몰 로고, 쇼핑몰 이름, 회사 이름)을 입력해 주세요.\
  📍 보다 자세한 내용은 [카카오 싱크 > 도입하기 > 2-2. 간편 설정 팝업](https://kakaobusiness.gitbook.io/main/tool/kakaosync/phase-in#id-2-2) 가이드의 1) 간편가입 설정 내용을 \
  &#x20;     참고해 주세요.

**⑦ 카카오 채널 선택 (기존 채널 적용 / 신규 채널 생성)**

* 권한이 있는 채널 목록에 보유한 카카오 채널 정보가 확인됩니다.\
  보유한 채널 중에서 카카오 싱크를 적용할 채널을 선택하고 \[다음] 버튼을 클릭해 주세요.
* 보유한 채널이 없다면, \[+ 신규 채널 생성하기]를 클릭합니다.\
  간편 가입 동의 화면에 노출되는 카카오톡 채널 정보(쇼핑몰 프로필 사진, 채널 이름, 검색용 아이디, 운영 카테고리)를 설정하고, 개인정보제공 항목 간편 검수를 진행하여 채널 생성을 완료합니다.\
  📍 보다 자세한 내용은 [카카오 싱크 > 도입하기 > 2-2. 간편 설정 팝업](https://kakaobusiness.gitbook.io/main/tool/kakaosync/phase-in#id-2-2) 가이드의 2) 카카오톡 채널 설정,\
  &#x20;     3\) 카카오싱크 적용 > 개인정보제공 항목 간편 검수 내용을 참고해 주세요.

**⑧ 카카오 싱크 자동 설정 확인 / 약관 동의 및 완료**\
카카오 싱크를 적용할 앱과 채널 정보를 확인하고, 자동 설정 관련 및 약관 동의 내용도 확인해 주세요.\
확인 '동의' 체크박스를 선택하고, \[완료] 버튼을 클릭해 카카오 싱크 설정을 완료합니다.

**⑨ 카카오 싱크 연동 정보 확인**\
카카오 싱크 설정 화면에서 자동 입력된, 연동 정보 - Rest API Key 값이 확인됩니다.\
📍Rest API Key 값이 확인되지 않으면, 카카오 싱크 설정 화면을 새로고침(F5) 해 주세요.

**⑩** 쇼핑몰에서 적용된 카카오 싱크를 확인해 주세요. \
📍쇼핑몰 회원가입 및 로그인 페이지 - \[카카오로 시작하기]  &#x20;
