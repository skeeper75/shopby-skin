# 페이코 간편가입

<Callout type="info">
**페이코 간편 회원가입을 설정하여 쇼핑몰에서 회원 가입을 진행할 수 있어요.**\
**등록(저장)한 Client ID / (Client)Secret Key 값의 수정은 서비스 관리 > 외부 서비스 설정 > 간편로그인 연동 항목에서 가능해요.**
</Callout>

**① 개발자센터**\
[**PAYCO** 로그인 개발자센터](https://developers.payco.com/)에 접속해 주세요.

**② 애플리케이션 관리 로그인**\
개발자센터 페이지 > 오른쪽 상단 '애플리케이션 관리' 메뉴를 클릭하고 로그인 해 주세요.\
📍페이코 간편가입 연동 설정은 페이코 회원만 가능합니다. 페이코 로그인 정보가 없다면 먼저 회원가입을 \
&#x20;    진행하고 로그인 해 주세요.

**③ 애플리케이션 등록**\
애플리케이션 관리 화면에서 내 애플리케이션 메뉴의 \[+ 애플리케이션 등록] 버튼을 클릭해 주세요.

* 앱 아이콘 : 고객이 페이코 간편 회원가입 단계에서 확인하는 쇼핑몰 로고를 등록합니다. \
  **📍**140 X 140 사이즈의 JPG, GIF, PNG 이미지 파일을 업로드 해주세요.
* 애플리케이션 이름 : 페이코 애플리케이션 관리에서 사용되며, 고객에게 미노출됩니다.&#x20;
* 회사명 : 페이코 간편 회원가입 및 로그인, 개인정보 제공 동의 내용 등에서 고객에게 노출되는 정보입니다.
* 플랫폼 : 페이코 간편가입 연동을 사용할 쇼핑몰 플랫폼을 선택합니다.\
  📍Android 및 iOS에서 설치하는 별도 네이티브앱 없이, 샵바이 쇼핑몰에서 페이코 간편가입을\
  &#x20;   사용하면  '웹사이트' 항목만 선택해 주세요.
* 웹 서비스 URL : 페이코 간편가입 연동을 사용할 쇼핑몰 도메인을 입력해 주세요. \
  예) <https://쇼핑몰도메인> ⇀ <https://abcde.com>
* Callback URL : Callback URL은 페이코 로그인 이후 고객 인증 결과를 전달받기 위한 주소입니다.\
  쇼핑몰 도메인을 포함하여 아래와 같이 작성해 주세요.\
  📍[서비스 관리 > 쇼핑몰 수정 > SNS 간편 회원가입 로그인 URL 설정](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/mall/modification/..#sns-url) 정보와 동일

<Callout type="warning">
사용하는 스킨에 따라 Callback URL이 상이하므로, 반드시 스킨 종류를 확인해 주세요. \
기본 스킨 외 별도의 스킨을 구매하여 사용하는 경우, 스킨 제작사에게 Callback URL을 문의해 주세요. \
헤드리스 운영방식은 자체 사용중인 Callback URL을 입력합니다.\ <br>

<mark style="color:blue;background-color:yellow;">오로라, 어나더 : 개별형 스킨은 PC와 모바일 각각 총 2개의 Callback URL을 입력해 주세요.</mark>

* PC : <https://쇼핑몰도메인/callback/auth-callback.html>
* 모바일 \
  ⓐ 도메인 m. : <https://m.쇼핑몰도메인/callback/auth-callback.html> \
  ⓑ 도메인 m- : <https://m-쇼핑몰도메인/callback/auth-callback.html> \
  ⓒ 도메인 /m : <https://쇼핑몰도메인/m/callback/auth-callback.html><br>

<mark style="color:blue;background-color:yellow;">오로라(리액트) : 통합형 스킨은 PC와 모바일 모두 동일합니다.</mark>

* PC/모바일 : <https://쇼핑몰도메인/callback/auth-callback>
</Callout>

* 개인정보 이용 항목 : 쇼핑몰 회원가입 단계에서 고객에게 수집하는 개인정보 항목을 선택합니다.\
  '선택'으로 설정한 항목은 고객이 해당 개인정보 제공 여부를 선택하도록 설정됩니다.

* PAYCO 서비스 연동 : 'PAYCO 로그인만 연동'을 선택해 주세요.

* 애플리케이션 등록 설정 사항을 모두 확인하고, \[등록] 버튼을 클릭해 완료합니다.

**④ Client ID / Client Secret 복사**\
등록한 애플리케이션 설정 화면에서 애플리케이션 관리자의 'Client ID' 와 'Client Secret' 값이 확인됩니다. Client ID / Client Secret 값을 복사해 주세요.\
📍Client Secret 값의 '보기'를 클릭하면 전체를 확인할 수 있어요.

**⑤ Client ID / Client Secret 입력**\
shop by 어드민 > 서비스 관리 > 쇼핑몰 수정 > SNS 간편 회원가입 설정 항목으로 이동합니다.\
페이코 간편가입 '선택' 체크박스를 선택하면 입력란이 활성화 되고,\
복사한 <mark style="background-color:orange;">Client ID는 Client ID</mark> 입력란에, <mark style="background-color:green;">Client Secret 값은 Secret Key</mark> 입력란에 붙여넣기 해 주세요.

**⑥** 쇼핑몰 수정 화면, 하단 \[변경사항 저장] 버튼을 클릭해 설정을 완료합니다.&#x20;

**⑦** 쇼핑몰에서 적용된 페이코 회원가입 및 로그인을 확인해 주세요.\
📍쇼핑몰 회원가입 페이지 : 간편 회원가입 > \[페이코 아이디로 회원가입]\
&#x20;    쇼핑몰 로그인 페이지 : 간편로그인 > \[페이코 아이디로 로그인]
