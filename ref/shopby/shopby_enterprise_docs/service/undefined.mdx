# 기초정보 관리

<Callout type="info">
**사업자등록증 및 통신판매신고증 등 쇼핑몰 운영서류의 행정정보와 접속 IP, 개인정보관리 책임자 정보를 설정할 수 있습니다.**

**서비스 기본정보와 개인정보 관리 책임자 정보에 입력한 내용은 쇼핑몰 하단 영역에 노출됩니다.**
</Callout>

## 서비스 기본정보

<figure><img src="/images/service/undefined/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FofyKOFQ3w1e6ms6C0LLl%2F%EA%B8%B0%EC%B4%88%EC%A0%95%EB%B3%B4%EA%B4%80%EB%A6%AC-%EC%84%9C%EB%B9%84%EC%8A%A4%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;서비스 기본정보></p></figcaption></figure>

**① 회사명**\
사업자등록증 상의 상호 또는 법인(단체 )명을 입력합니다.

**② 사업자등록번호**\
사업자등록증 상의 10자리 등록번호를 입력합니다.

**③ 대표자명**\
사업자등록증 상의 대표자 성명을 입력합니다.

**④ 업종 / 업태**\
사업자등록증 상의 사업의 종류에 기재된 업종과 업태를 입력합니다.\
📍다수의 업종과 업태를 등록한 경우 쇼핑몰 운영에 적합한 항목으로 입력해 주세요.

**⑤ 대표 이메일**\
대표 이메일을 입력합니다.

**⑦ 통신판매업 신고번호**\
통신판매업신고증 상의 신고번호를 입력합니다.

**⑧ 주소**\
사업자등록증 상의 사업장 소재지를 입력합니다.

## 접속 IP주소 등록

<figure><img src="/images/service/undefined/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F1iMwQ03JNWnecPOpY7O1%2F%EA%B8%B0%EC%B4%88%EC%A0%95%EB%B3%B4%EA%B4%80%EB%A6%AC-%EC%A0%91%EC%86%8Dip%EC%84%A4%EC%A0%95%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;접속 IP 설정></p></figcaption></figure>

**①** 쇼핑몰 어드민 접속 IP 주소를 등록하여 관리할 수 있습니다.

**②** 어드민 접속 시 IP 주소를 통해 회사 내부와 외부를 구분하여 어드민 접속을 관리할 수 있습니다.\
📍등록하지 않은 IP 주소는 외부 접속으로 처리되는 점을 참고해 주세요.

**③** 외부에서 접속하는 운영자 계정은 SMS 추가 인증 절차를 진행합니다.\
📍운영자계정 정보에 등록된 휴대폰번호로 SMS 추가 인증이 진행 돼요.
