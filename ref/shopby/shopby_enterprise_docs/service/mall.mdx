# 쇼핑몰 관리

<Callout type="info">
**쇼핑몰 정보를 수정 및 관리하고, 신규 쇼핑몰 추가 등록이 가능한 쇼핑몰 관리 메뉴입니다.**

**쇼핑몰 기본정보, 도메인정보, 계약서류 관리, 운영정보 설정 등 쇼핑몰 정책사항을 설정할 수 있습니다.**
</Callout>

## 쇼핑몰 관리 <a href="#undefined" id="undefined"></a>

<figure><img src="/images/service/mall/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FYaelxuRyHL9oaXfFgFVy%2F%EC%87%BC%ED%95%91%EB%AA%B0%20%EA%B4%80%EB%A6%AC%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;쇼핑몰 관리 리스트></p></figcaption></figure>

**① 쇼핑몰 번호**\
쇼핑몰 번호는 샵바이 솔루션 신청 시 부여되는 고유한 번호입니다.\
쇼핑몰 번호를 클릭하면 쇼핑몰 수정 페이지로 이동합니다.

**② PC 웹**\
&#x20;\[바로가기] 버튼을 클릭하면 PC 쇼핑몰 메인 페이지로 이동됩니다.\
📍PC 쇼핑몰 메인 페이지로 이동하려면, '[쇼핑몰 도메인 정보](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/modification#undefined-2)' 항목에 PC 웹 도메인 주소가 입력되어야 합니다.

**③ 모바일 웹**\
&#x20;\[바로가기] 버튼을 클릭하면 모바일 쇼핑몰 메인 페이지로 이동됩니다.\
📍모바일 쇼핑몰 메인 페이지로 이동하려면, '[쇼핑몰 도메인 정보](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/modification#undefined-2)' 항목에 모바일 웹 도메인 주소가 입력되어야 합니다.

**④** [**신규 쇼핑몰 등록**](#undefined-1)\
\[신규 쇼핑몰 등록] 버튼을 클릭하면 새로운 쇼핑몰을 추가할 수 있는 '쇼핑몰 등록' 창이 노출됩니다.

### 쇼핑몰 등록 <a href="#undefined-1" id="undefined-1"></a>

<div align="left"><figure><img src="/images/service/mall/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F5roFERlz7wuZQUmrDJUI%2F%EC%8B%A0%EA%B7%9C%EC%87%BC%ED%95%91%EB%AA%B0%20%EB%93%B1%EB%A1%9D%20%EB%B3%B5%EC%82%AC.jpg" alt="" width="563"/><figcaption></figcaption></figure></div>

**① 쇼핑몰 주소**\
추가 등록하는 신규 쇼핑몰에서 사용할 쇼핑몰 기본 도메인을 입력하고, \[중복확인] 버튼으로 쇼핑몰 주소 사용여부를 확인합니다.

**② 정기결제 주기**\
1, 3, 6, 12개월의 정기결제 주기 중에서 선택할 수 있습니다.\
📍 정기결제 주기에 따라 이용금액의 할인율이 상이하므로 참고해 주세요.

**③ 정기결제 예정일**\
선택한 정기결제 주기에 따라 정기결제 예정일이 안내됩니다.

**④ 서비스 정책**\
쇼핑몰 사용기간 만료 및 정기결제 관련 정책사항, 취소/환불규정 등을 모두 확인해 주세요.

**⑤ 결제 금액**\
선택한 정기결제 주기에 따른 결제 금액이 확인됩니다.

**⑥** 확인 및 동의항목 체크박스를 선택하고 \[정기결제 신청] 버튼을 클릭해 쇼핑몰 주문/결제를 진행합니다.

<Callout type="warning">
**정기결제**

사전에 등록한 카드로 원하는 서비스를 정기결제 주기에 맞추어 자동으로 결제 처리하는 서비스입니다.\
추가쇼핑몰 이이용 금액은 정기결제 방식만 가능하며, 정기결제 서비스는 원클릭 결제수단이 등록되어 있어야 사용 가능합니다.\
\ <br>

**원클릭 결제**

결제 카드를 미리 등록하고, 결제 비밀번호만 입력해서 사용하는 새로운 결제수단입니다.\
NHN커머스 > 마이페이지 > 회원/결제관리 / 원클릭 결제 관리 메뉴에서 자동결제 카드와 6자리의 결제 비밀번호를 등록해 이용합니다.
</Callout>
