# 쇼핑몰/회사소개

<Callout type="info">
**쇼핑몰에 쇼핑몰 소개 또는 회사 소개 콘텐츠를 노출할 수 있습니다.**

**'사용함'으로 설정하면, 회사소개 메뉴가 쇼핑몰 하단 영역에 노출되고 '사용 안함'으로 설정하면 미노출 처리됩니다. (기본스킨 기준)**
</Callout>

## 쇼핑몰/회사소개

**①** '사용함'을 선택하면, 쇼핑몰/회사소개 내용을 등록할 수 있는 에디터 창이 나타납니다.

**②** 에디터 기능을 활용하여 쇼핑몰 및 회사 소개 콘텐츠를 등록합니다.

**③** 내용 입력란에 직접 작성할 수 있고, 이미지로 제작하여 업로드 할 수 있습니다.

**④** \[변경사항 저장] 버튼을 클릭 해 쇼핑몰/회사소개 등록을 완료합니다.\
📍변경사항 수정 및 저장 이후에는, 이전 내용으로 복원이 불가한 점을 참고해 주세요.
