# 전자금융거래이용약관

<Callout type="info">
**쇼핑몰 전자금융 거래에 관한 기본적인 사항을 이용약관으로 설정하고, 고객에게 안내할 수 있습니다.**

**샵바이 프리미엄에서는공정거래위원회에서 제공하는 전자금융거래 표준약관을 기본내용으로 제공하고 있습니다.** \
**😊전자금융거래 이용약관은 쇼핑몰 운영 및 정책에 따라 수정하여 활용해 주세요.**
</Callout>

에디터 창 내용 입력란에 쇼핑몰 전자금융거래 이용약관을 작성해 주세요.

**②** **시행일** \
변경 및 추가한 이용약관이 적용되는 시행 일자는 캘린더로 설정합니다.

**③** [**변경이력**<br>](#undefined-1)\[변경이력] 버튼을 클릭하면, '전자금융거래이용약관 변경이력' 창이 노출됩니다.

**④ 기본제공내용 불러오기** \
\[기본제공내용 불러오기 ] 버튼을 클릭하면, 기본 제공되는 표준약관이 약관 내용에 반영됩니다.

**⑤** \[저장] 버튼을 클릭해 전자금융거래이용약관 설정을 완료해 주세요.&#x20;
