# 정기결제(배송)이용약관

<Callout type="info">
**정기결제(배송) 기능을 사용하는 쇼핑몰은, 정기결제(배송) 이용약관과 자동 승인 이용약관을 설정하고, 고객에게 안내합니다.**

**신용카드 등 자동 승인 결제 기능을 사용하는 쇼핑몰은, 자동 승인 이용약관을 설정해 주세요.**
</Callout>

## 정기결제(배송) 이용약관

**① 정기결제(배송) / 자동승인 이용약관**\
에디터 창 내용 입력란에 정기결제(배송) / 자동승인 이용약관을 각각 작성해 주세요.

**②** **시행일**\
변경 및 추가한 이용약관이 적용되는 시행 일자는 캘린더로 설정합니다.&#x20;

**③** \[저장] 버튼을 클릭해 이용약관 설정을 완료해 주세요.
