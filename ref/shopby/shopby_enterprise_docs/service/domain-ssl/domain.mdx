# 도메인 연결 리스트

<Callout type="info">
**도메인은 인터넷 상에서의 '쇼핑몰 주소' 로,**\
**shop by 신청 시 '쇼핑몰아이디.shopby.co.kr' 형태의 기본 도메인이 자동으로 생성되요.**

**기본 도메인으로 쇼핑몰 운영이 가능하지만, 쇼핑몰 및 브랜드의 특징을 잘 나타내는 대표 도메인을 등록하면 고객이 쉽게 기억하고 편리하게 쇼핑몰을 방문할 수 있어요.**

**shop by 어드민 페이지 내에서 도메인을 구매(연결)하거나, 타사에서 구매한 도메인을 쇼핑몰에 연결할 수 있습니다.**

**도메인 연결 시 'm.연결도메인' 형태로 모바일 도메인이 자동 생성되고, 별도 관리가 필요하지 않습니다.**
</Callout>

## 도메인 구매

✅ <mark style="background-color:yellow;">\[신규 도메인 구매] 버튼을 클릭하면 도메인을 구매할 수 있는 '도메인 구매' 창이 노출됩니다.</mark>

**① 쇼핑몰 선택**\
신규 도메인을 구매하여 연결할 쇼핑몰을 선택합니다.

**② 도메인 검색 및 선택** \
한글, 영문자, 숫자, 하이픈(-) 등을 조합하여 희망하는는 도메인을 입력하고, \[검색] 버튼을 클릭해 주세요.

<Callout type="warning">
**도메인 검색 시,&#x20;**<mark style="background-color:blue;">**최상위 도메인 (co.kr/com/net 등)**</mark>**을 제외하고 입력해 주세요.**

한글 도메인은 브라우저 환경에 따라 연결이 다소 불안정할 수 있고, e-메일 사용 및 특정 외부서비스에서 이용이 불가할 수 있어 영문 도메인 사용을 권장합니다.
</Callout>

**③ 검색결과 (추천순)**\
다양한 도메인이 검색결과 리스트에 추천순으로 안내됩니다.

* 등록여부가 '등록가능'으로 되어있는 도메인을 구매할 수 있습니다. 도메인을 확인하고 선택해 주세요. 📍'등록불가' 도메인은 다른 소유자가 이미 사용중이거나 보유중인 도메인입니다.
* \[더보기] 버튼을 클릭하면 더 많은 도메인 검색결과가 확인됩니다.

**④ 등록할 도메인 및 기간/가격 선택**\
구매할 도메인을 최종 확인하고 소유기간을 선택하면, 결제금액이 안내됩니다.

* 도메인 소유 기간은 1/2/3/4/5년 중에서 선택할 수 있고, 소유 기간에 따라 가격이 산정되어 노출됩니다.
* 도메인 결제금액은 vat 포함입니다.

**⑤ 도메인 소유자 정보**\
도메인을 소유하는 소유자 정보를 입력해 주세요.\
소유자 정보는 등록 후 변경이 불가하며, 다른 소유자에게 노출됩니다.

<details>

<summary> <strong>🤚 도메인 소유자 정보 항목</strong></summary>

* **소유자 명 (한글) (영문)**\
  소유자가 개인이면 이름을, 사업자인 경우 회사명을 입력해 주세요.\
  소유자명은 도메인 소유권 정보이므로, 신분증에 기재된 정보로 작성해야 합니다.
* **우편번호 / 주소(한글) (영문)**\
  \[검색] 버튼을 클릭하면 나타나는 '주소찾기' 창에서 지번 또는 도로명을 입력하여 주소를 \
  검색합니다. (나머지 주소도 입력해 주세요.)
* **이메일 주소**\
  도메인 구매 관련 안내 메일을 수신받을 이메일 주소를 입력합니다.
* **휴대폰 번호**\
  휴대폰 번호를 입력합니다.
* **전화번호**\
  입력 선택 사항으로, 추가 전화번호 정보 등이 필요한 경우 활용할 수 있어요.

</details>

⑥ **약관 동의**\
개인정보 수집 및 이용안내 / 처리 및 위⋅수탁 등에 관한 내용을 확인하고 '동의' 체크박스를 클릭합니다.

**⑦** \[다음] 버튼을 클릭하면, 주문/결제 화면으로 이동합니다.

<div align="left"><figure><img src="/images/service/domain-ssl/domain/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FuYRAVCQ1bYJTRyDELrIL%2F%EB%8F%84%EB%A9%94%EC%9D%B8%EC%A3%BC%EB%AC%B8%EA%B2%B0%EC%A0%9C%20%EB%B3%B5%EC%82%AC.jpg" alt="" width="332"/><figcaption><p>&#x3C;도메인 주문/결제></p></figcaption></figure></div>

**⑧ 주문/결제**\
주문/결제 화면에서 주문 상품과 총 결제금액 등 주문/결제 정보를 확인해 주세요.

* 결제를 진행할 카드(등록 결제 수단)를 선택하고, 결제 '동의' 체크박스를 클릭합니다.
* \[결제하기] 버튼을 클릭해 결제를 완료하면, 구매한 도메인이 쇼핑몰에 대표 도메인으로 연결됩니다.

<Callout type="warning">
도메인 주문/결제는 통합회원만 가능합니다. \
결제 수단에 등록한 카드가 없다면 \[카드 등록하기] 버튼을 클릭해 카드 정보를 먼저 등록하고, 결제해 주세요.
</Callout>

## 도메인 연결 리스트

<figure><img src="/images/service/domain-ssl/domain/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FCUkuf8mec3g4nqG5hRU0%2F%EB%8F%84%EB%A9%94%EC%9D%B8%EC%97%B0%EA%B2%B0%EB%A6%AC%EC%8A%A4%ED%8A%B8%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;도메인 연결 리스트></p></figcaption></figure>

**① 도메인** \
쇼핑몰에 연결된 대표 도메인과 기본 도메인, 추가 도메인 등이 확인됩니다.

<details>

<summary><strong>🤚 도메인 종류</strong></summary>

* **\[대표] 도메인**\
  쇼핑몰을 대표하는 도메인입니다.\
  shop by 최초 신청 시 기본 도메인이 대표 도메인으로 설정되며, 이후 신규 도메인을 구매하면 해당 도메인이 대표 도메인으로 자동 연결됩니다. \
  📍대표 도메인은 변경이 가능해요.
* **\[기본] 도메인**\
  shop by 최초 신청 시 '쇼핑몰아이디.shopby.co.kr' 형태로 생성되고, 추가 쇼핑몰 등록 시 \
  운영자가 입력한 쇼핑몰 주소로 생성됩니다.\
  📍기본 도메인은 보안서버가 설치되어 있으므로 별도 설치가 필요하지 않아요.
* **추가 도메인**\
  쇼핑몰에 연결한 도메인 중 대표 및 기본 도메인을 제외한 그 밖의 도메인을 의미해요.\ <mark style="color:blue;">고객이 추가 도메인으로 쇼핑몰에 접속하면, 대표 도메인으로 리다이렉트 처리되어</mark>\ <mark style="color:blue;">쇼핑몰에 자동 접속됩니다.</mark>

</details>

**② 연결상태** \
도메인 연결상태를 '연결완료', '연결 중', '기간만료', 연결해지' 로 구분합니다.\
📍기본 도메인은 연결상태가 표시되지 않아요.

<details>

<summary> <strong>🤚 도메인 연결상태</strong></summary>

* **연결완료**\
  쇼핑몰과 도메인이 정상 연결되어 도메인 사용이 가능한 상태입니다.
* **연결 중**\
  도메인 연결 신청을 진행하여 쇼핑몰에 연결 대기 중인 상태로, 연결완료까지 최대 1분 정도\
  소요됩니다.&#x20;
* **기간만료**\
  도메인 소유 기간에 따라 사용기간이 만료된 상태입니다.\
  📍만료일이 경과한 도메인이 연결된 쇼핑몰은 정상 접속이 불가합니다.\
  📍도메인에 따라 기간만료 이후 일정기간 도메인 연장 유예기간이 제공됩니다.\
  📍연장 유예기간을 초과하면, 해당 도메인의 연장 및 연결이 불가한 점을 유의해 주세요.
* **연결해지**\
  도메인을 운영자가 수동으로 연결해지하여 쇼핑몰과 도메인이 연결되지 않은 상태, \
  또는 도메인 사용기간 만료로 자동 연결해지된 상태입니다.\
  📍운영자가 수동으로 연결해지 하였지만, 사용기간이 유효한 도메인은 다시 연결하여 \
  &#x20;    사용할 수 있어요.\
  📍사용기간이 만료된 도메인은 쇼핑몰과 연결이 불가해요.

</details>

**③ 만료일** \
도메인의 사용기간 만료일을 확인할 수 있습니다.

<Callout type="danger">
<mark style="background-color:blue;">**shop by 어드민 페이지 내에서 구매하고 연결한 도메인만**</mark>**&#x20;만료일이 확인됩니다.**\
**타사에서 구매하여 연결한 도메인은 만료일이 노출되지 않아 확인 및 만료 관련 안내가 불가합니다.**\
**→ 만료일 : ( - ) 표시**
</Callout>

**④** [**관리 (도메인 상세)**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/domain-ssl/undefined#undefined)\
\[관리] 버튼을 클릭하면 해당 도메인의 상세 정보 및 구매내역을 확인하고 관리할 수 있습니다.

**⑤** [**대표 도메인 변경**](#undefined-5)\
\[대표 도메인 변경] 버튼을 클릭하면, 쇼핑몰 대표 도메인을 변경할 수 있습니다.

**⑥** [**도메인 연결 신청**](#undefined-6)\
\[도메인 연결 신청] 버튼을 클릭하면, 쇼핑몰에 도메인을 연결할 수 있습니다.\
📍shop by 어드민 페이지 내에서 구매한 도메인 이외에, 다른 도메인을 쇼핑몰에 연결하는 기능으로 \
&#x20;   도메인 연결은 최대 5개까지 가능한 점을 참고해 주세요.

<br>

#### <mark style="color:blue;">**■ 대표 도메인 변경**</mark>

* \[대표 도메인 변경] 버튼을 클릭하면 '대표 도메인 변경' 창이 노출됩니다.
* 쇼핑몰에 연결된 도메인 중에서 대표 도메인으로 변경할 도메인을 선택하고, \[확인] 버튼을 클릭해 변경을 완료합니다.&#x20;

#### <mark style="color:blue;">**■ 도메인 연결 신청**</mark>

* \[도메인 연결 신청] 버튼을 클릭하면 '도메인 연결 신청' 창이 노출됩니다.
* 도메인을 연결할 쇼핑몰을 선택하고 연결한 도메인을 입력한 다음, \[연결신청] 버튼을 클릭해 연결신청을 완료합니다. \
  📍<mark style="color:red;background-color:green;">www (2차도메인) 등을 제외</mark><mark style="color:red;">한 '도메인 주소(예:abc.com)'만 입력해야 정상으로 연결됩니다.</mark>\
  📍shop by 어드민 페이지 내에서 구매한 도메인은 자동 연결되므로, 별도 연결 신청을 하지 않아도 되요.

<Callout type="danger">
<mark style="background-color:blue;">**타사에서 구매한 도메인 연결 (타사 네임서버 업체 도메인)**</mark>

**해당 도메인이 shop by 웹서버 IP로 연결되도록, 네임서버 업체에서 DNS 레코드 설정을 반드시**\
**먼저 진행하고 도메인 연결 신청을 해 주세요.**

* **레코드 : A**\
  **호스트 : www**\
  **IP 주소 : 125.6.38.49**
* **레코드 : A**\
  **호스트 : @**\
  **IP 주소 : 125.6.38.49**
* **레코드 : A**\
  **호스트 : m**\
  **IP 주소 : 125.6.38.49**
* **DNS 레코드는 'A'레코드 타입으로 'www\`, '@\`, 'm' 호스트를 모두 추가해 주세요.**
</Callout>
