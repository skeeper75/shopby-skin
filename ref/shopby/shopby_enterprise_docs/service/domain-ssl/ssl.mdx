# 보안서버 설치 리스트

<Callout type="info">
**보안서버는 웹 브라우저(쇼핑몰 사용자)와 웹 서버(쇼핑몰 서버) 사이의 통신 데이터를 암호화하여, 해킹 발생에도 개인정보 유출을 방지할 수 있는 보안 장치에요.**  \
😊보안서버 미설치 http\:// ⇀ 보안서버 <mark style="color:orange;">설치</mark> http<mark style="color:orange;">s</mark>://

**shop by 기본 도메인은 보안서버가 적용되어 있어 별도 설치가 필요하지 않지만,**\
**구매한 도메인을 쇼핑몰에 연결하여 사용하면, 보안서버 설치가 반드시 필요해요.**

**쇼핑몰에 연결된 도메인 별 보안서버 설치 정보를 확인하고, 미설치 상태 시 신청(구매)과 설치를 진행해 주세요.**\
**보안서버는 PC 도메인과 모바일 도메인 각각 개별 설치해야 합니다.**

**타사 보안서버는 shop by 어드민 페이지 내에서 설치가 불가하므로, NHN커머스 1:1 문의로 수동 설치를 요청해 주세요.**
</Callout>

⚠️  온라인 쇼핑몰 및 개인정보 취급 사이트는 보안서버 설치 적용이 법적으로 의무화 되어 있습니다. \
&#x20;      ⇀ 개인정보보호법 제29조(안전조치의무) 및 동법 시행령 제30조(개인정보의 안전성 확보 조치)

## 보안서버 신청

**✅** <mark style="background-color:yellow;">미설치 상태 도메인의 \[보안서버를 설치해주세요] 버튼을 클릭하면 보안서버를 신청(구매)할 수 있는</mark>\
&#x20;     <mark style="background-color:yellow;">'보안서버 신청' 창이 노출됩니다. 또는 \[관리] 버튼을 클릭해 '보안서버 상세' 화면에서 \[보안서버 신청]</mark>\
&#x20;     <mark style="background-color:yellow;">버튼을 클릭해 주세요.</mark>

**① 도메인 선택** \
보안서버를 신청(구매)하여 설치할 쇼핑몰을 확인합니다.

**② 설치할 보안서버 선택** \
설치할 보안서버 상품 종류를 선택하고, 결제금액을 확인합니다.

<Callout type="warning">
**Sectigo 베이직 DV 싱글 : 1개 도메인 설치** \
**Sectigo 베이직 DV 와일드카드 : 서브도메인 무제한 설치**

**보안서버 설치 사용기간은 1년 단위만 가능한 점을 참고해 주세요.**\
**보안서버 결제금액은 vat 포함입니다.**
</Callout>

**③ 보안서버 소유자 정보** \
보안서버를 소유하는 소유자 정보를 입력해 주세요.&#x20;

<details>

<summary> <strong>🤚 보안서버 소유자 정보 항목</strong></summary>

* **회사명 (한글) (영문)** \
  보안서버를 구매하여 소유하는 회사명을 작성합니다.\
  소유자가 개인이면 소유자명과 동일하게 작성해 주세요.
* **소유자 명 (한글) (영문)**\
  소유자명은 보안서버 소유권 정보이므로, 신분증에 기재된 정보로 작성해야 합니다.
* **우편번호 / 주소(한글) (영문)**\
  \[검색] 버튼을 클릭하면 나타나는 '주소찾기' 창에서 지번 또는 도로명을 입력하여 주소를 \
  검색합니다. (나머지 주소도 입력해 주세요.)
* **이메일 주소**\
  보안서버 구매 관련 안내 메일을 수신받을 이메일 주소를 입력합니다.
* **휴대폰 번호**\
  휴대폰 번호를 입력합니다.
* **전화번호**\
  입력 선택 사항으로, 추가 전화번호 정보 등이 필요한 경우 활용할 수 있어요.

</details>

**④ 약관 동의** \
보안서버 신청 관련 안내사항 / 개인정보 수집 및 이용안내 / 처리 및 위⋅수탁 등에 관한 내용을 확인하고,\
'동의' 체크박스를 클릭합니다.

**⑤** \[다음] 버튼을 클릭하면, 주문/결제 화면으로 이동합니다.

<div align="left"><figure><img src="/images/service/domain-ssl/ssl/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2Fp45qy2dvxD5yTVo1MwCw%2F%EB%B3%B4%EC%95%88%EC%84%9C%EB%B2%84%20%EC%A3%BC%EB%AC%B8%20%EA%B2%B0%EC%A0%9C%20%EB%B3%B5%EC%82%AC.jpg" alt="" width="332"/><figcaption><p>&#x3C;보안서버 주문/결제></p></figcaption></figure></div>

**⑥ 주문/결제** \
주문/결제 화면에서 주문 상품과 총 결제금액 등 주문/결제 정보를 확인해 주세요.

* 결제를 진행할 카드(등록 결제 수단)를 선택하고, 결제 '동의' 체크박스를 클릭합니다.
* \[결제하기] 버튼을 클릭해 결제를 완료하면, 보안서버 설치(구매)가 완료됩니다.

<Callout type="warning">
보안서버 주문/결제는 통합회원만 가능합니다. \
결제 수단에 등록한 카드가 없다면 \[카드 등록하기] 버튼을 클릭해 카드 정보를 먼저 등록하고, 결제해 주세요.
</Callout>

<Callout type="danger">
<mark style="background-color:blue;">**타사에서 구매하여 연결한 도메인에 신청(구매)한 보안서버를 설치**</mark>**하려면 다음과 같이 진행해 주세요.**<br>

**ⓐ NHN커머스 1:1 문의로 구매한 보안서버의 'CNAME' 요청 문의글을 남겨주세요.**\
**ⓑ 답변을 통해 'CNAME'을 확인하고, 도메인 구매 업체(기관)에 'CNAME' 을 등록해 주세요.**\
**ⓒ NHN커머스 1:1 문의로 'CNAME 등록완료 여부' 를 문의해 정상 등록을 최종 확인합니다.**
</Callout>

## 도메인 별 보안서버 설치 리스트

<figure><img src="/images/service/domain-ssl/ssl/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FhzQKnlWJFjyXiyXkT5hC%2F%EB%8F%84%EB%A9%94%EC%9D%B8%EB%B3%84%20%EB%B3%B4%EC%95%88%EC%84%9C%EB%B2%84%20%EC%84%A4%EC%B9%98%EB%A6%AC%EC%8A%A4%ED%8A%B8%20%EB%B3%B5%EC%82%AC2%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;도메인 별 보안서버 설치 리스트></p></figcaption></figure>

**① 도메인** \
쇼핑몰에 연결된 대표 도메인과 기본 도메인, 추가 도메인 등이 확인됩니다.

**② 상품명**\
해당 도메인에 설치한 보안서버 상품명을 확인할 수 있습니다.

**③ 설치상태**\
보안서버 설치상태를 '미설치', '설치중', '설치완료', '기간만료' 로 구분합니다.

<details>

<summary><strong>🤚 보안서버 설치상태</strong></summary>

* **미설치**\
  보안서버의 신청(구매) 및 설치가 되지 않은 상태입니다.
* **설치중**\
  보안서버 신청(구매)을 완료하여, 발급 및 설치가 진행 중인 상태입니다.
* **설치완료**\
  보안서버가 정상적으로 설치되어 사용이 가능한 상태입니다.
* **기간만료**\
  보안서버 사용기간이 만료된 상태입니다. \
  📍만료일이 경과한 보안서버는 연장이 불가하며, 보안서버 신청(구매) 및 설치를 다시 해야해요.

</details>

**④ 만료일**\
보안서버 만료일을 확인할 수 있습니다.

<Callout type="danger">
<mark style="background-color:blue;">**shop by 어드민 페이지 내에서 신청(구매) 및 설치한 보안서버만**</mark>**&#x20;만료일이 확인됩니다.**\
**타사 보안서버는 만료일이 노출되지 않는 점을 참고해 주세요. → 만료일 : ( - ) 표시**
</Callout>

**⑤** [**관리 (보안서버 상세)**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/domain-ssl/ssl/undefined)\
\[관리] 버튼을 클릭하면 해당 보안서버의 상세 정보 및 구매내역을 확인하고 관리할 수 있습니다.
