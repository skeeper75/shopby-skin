# 도메인 상세

<Callout type="info">
**쇼핑몰에 연결한 도메인의 상세 정보 및 구매내역 등을 확인하고, 연장 및 연결해지 등 도메인을 관리합니다.**

**도메인 연결 시 자동 생성된 모바일 도메인(m.연결도메인)은 서브도메인으로 표시되며, 별도 관리가 필요하지 않습니다.**
</Callout>

**① 쇼핑몰 정보**\
해당 도메인이 연결된 쇼핑몰명과 쇼핑몰 고유번호가 확인됩니다.

**② 도메인 관리**

* 구분 : '구매'한 도메인과 '연결'한 도메인으로 구분되어 관리합니다.

<details>

<summary><strong>🤚 도메인 구분</strong></summary>

* 구매 : shop by 어드민 페이지 내에서 구매하여 자동 연결된 도메인
* 연결 : 타사 구매 도메인 등 도메인 연결 신청으로 연결된 도메인

</details>

* 도메인 : 해당 도메인의 종류(대표/기본 등) 와 도메인 주소를 확인할 수 있습니다.
* 서비스 상태 : '서비스 중' 또는 '사용기간 만료'로 해당 도메인의 서비스 상태가 확인됩니다.
* 연결상태 : 해당 도메인의 연결상태에 따라 '연결완료', '기간만료', '연결해지'로 표시됩니다.
* 만료일 : 해당 도메인의 사용기간 만료일이 확인됩니다.
* 연결신청일 : 쇼핑몰에 해당 도메인 연결을 신청한 일자가 확인됩니다.
* [도메인 연장](#undefined-2) : \[연장] 버튼을 클릭해 해당 도메인의 사용기간을 연장합니다.\
  📍<mark style="color:red;">도메인 연장은 만료일이 도래하기 전, 언제든지 가능합니다.</mark>
* [도메인 연결 해지](#undefined-3) : \[연결해지] 버튼을 클릭해 해당 도메인과 쇼핑몰 연결을 수동으로 해지합니다.\
  📍쇼핑몰과 해당 도메인의 연결을 해지하는 기능으로, 도메인이 삭제되는 것은 아니에요.\
  📍만료일이 경과하지 않은 도메인 또는 다른 쇼핑몰에서 사용하고 있지 않은 도메인은 재연결이 가능합니다.&#x20;

<Callout type="danger">
<mark style="background-color:blue;">**shop by 어드민 페이지 내에서 구매하고 연결한 도메인만**</mark>**&#x20;만료일 확인과 연장 처리가 가능해요.**

**타사에서 구매하여 연결한 도메인은 만료일이 노출되지 않아 확인 및 만료 관련 안내가 불가합니다.**\
**→ 만료일 : ( - ) 표시**\
**타사에서 구매한 도메인의 만료일 확인 및 연장 처리 등은 해당 도메인 구매 업체에서 반드시 확인하고 진행해 주세요.**
</Callout>

**③ 서브도메인 관리**\
도메인 연결 시 자동 생성된 모바일 도메인(m.연결도메인)으로, 별도 관리가 필요하지 않습니&#xB2E4;**.**

**④** [**구매내역**](#undefined-4)\
shop by 어드민 페이지 내에서 구매한 도메인의 구매내역 정보를 \[상세보기] 버튼을 클릭해, 보다 자세하게확인할 수 있습니다.\
📍NHN커머스 > 마에페이지 >  회원결제관리 > 결제 관리 > 결제내역관리 메뉴에서도 확인할 수 있어요.\
📍타사에서 구매한 도메인은 '구매내역' 항목이 미노출됩니다.

<br>

***

#### <mark style="color:blue;">**■ 도메인 연장**</mark>&#x20;

* \[도메인 연장] 버튼을 클릭하면. 도메인 사용기간 연장 구매가 가능한 '도메인 연장' 창이 노출됩니다.
* 연장할 도메인을 최종 확인하고 소유기간을 선택하면, 결제금액이 안내됩니다.
* 도메인 소유자 정보인 소유자명과 이메일 주소를 확인합니다.
* 개인정보 수집 및 이용안내 / 처리 및 위⋅수탁 등에 관한 내용을 확인하고 '동의' 체크박스를 클릭한 다음, \[다음] 버튼을 클릭해 주세요.
* 주문/결제 화면에서 결제를 진행하여 도메인 연장을 완료합니다.

#### <mark style="color:blue;">**■ 도메인 연결 해지**</mark>

* \[연결해지] 버튼을 클릭하면 쇼핑몰과 도메인 연결을 수동 해지 처리할 수 있는 '도메인 연결해지' 창이 노출됩니다.
* 연결을 해지할 도메인과 하단 주의사항을 반드시 확인해 주세요.
* '주의사항 확인 및 동의' 체크박스를 클릭한 다음, \[연결 신청 해지] 버튼을 클릭하면 연결해지 처리됩니다.\
  📍대표 도메인은 연결 해지가 불가한 점을 참고해 주세요.

#### <mark style="color:blue;">**■ 구매내역 상세**</mark>

* 구매내역의 \[상세보기] 버튼을 클릭하면 구매한 도메인의 소유 기간, 결제 정보, 소유자 정보 등\
  상세 정보를 확인할 수 있는 '구매내역 상세' 창이 노출됩니다.
* 결제 정보에서 '신규'와 '연장'으로 도메인 구매 현황을 확인할 수 있습니다.
