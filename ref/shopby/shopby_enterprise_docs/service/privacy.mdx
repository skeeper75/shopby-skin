# 개인정보접속기록 조회

<Callout type="info">
**개인정보의 안정성 확보에 필요한 조치 관련 사항에 따라,  회원정보 접속기록과 운영자정보 접속기록을 확인하고 관리합니다.**
</Callout>

## 개인정보접속기록 조회 검색조건

<figure><img src="/images/service/privacy/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2Fl5iS4SKegZfHXhYUwuD0%2F%EA%B0%9C%EC%9D%B8%EC%A0%95%EB%B3%B4%EC%A0%91%EC%86%8D%EA%B8%B0%EB%A1%9D%20%EC%A1%B0%ED%9A%8C%20%EA%B2%80%EC%83%89%EC%A1%B0%EA%B1%B4.jpg" alt=""/><figcaption><p>&#x3C;개인정보접속기록 조회 검색조건></p></figcaption></figure>

**① 쇼핑몰**\
개인정보접속기록 조회 대상 쇼핑몰을 선택합니다.

* 전체 쇼핑몰 대상 처리 내역 : 특정 쇼핑몰을 선택하지 않고 (쇼핑몰 구분없이) 정보를 조회한 기록
* 쇼핑몰명 : 해당 쇼핑몰을 선택하여 정보를 조회한 기록

<div align="left"><figure><img src="/images/service/privacy/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FsDUy3AyNazpIFmGa80x6%2F%EA%B0%9C%EC%9D%B8%EC%A0%95%EB%B3%B4%EC%A0%91%EC%86%8D%EA%B8%B0%EB%A1%9D%EC%A1%B0%ED%9A%8C_%EC%87%BC%ED%95%91%EB%AA%B0%EC%84%A0%ED%83%9D_%ED%99%94%EB%A9%B4%EC%98%88%EC%8B%9C.jpg" alt=""/><figcaption><p>&#x3C;쇼핑몰 선택 화면 예시></p></figcaption></figure></div>

**② 구분** \
회원정보 접속기록과 운영자정보 접속 기록을 선택해 해당 기록을 조회합니다.

* 회원정보 : 쇼핑몰 회원정보 관련 접속기록을 확인하고 관리합니다. (회원관리정보, 주문/배송 등)
* 운영자정보 : 쇼핑몰 운영자에 대한 개인정보 접속기록을 확인하고 관리합니다.

<Callout type="danger">
쇼핑몰이 1개인 경우, 어드민 메뉴에서 쇼핑몰을 선택하지 않고 정보를 조회하였더라도\
\&#xNAN;**'전체 쇼핑몰 대상 처리 내역'**&#xC5D0;서는 **회원정보 접속기록 조회가 불가**하고,\
\&#xNAN;**'쇼핑몰명'**&#xC744; 선택해야 **조회가 가능**한 점을 참고해 주세요.
</Callout>

**③ 운영자 ID**\
운영자  ID를 입력해 접속기록 조회가 가능합니다.

**④ 기간**\
기간을 지정하여 검색하는 조회기간 검색은 최대 6개월까지 가능합니다.

## 회원정보  접속기록 검색결과

<figure><img src="/images/service/privacy/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2Fu22GacbukVGjiLDGpkWf%2F%ED%9A%8C%EC%9B%90%EC%A0%95%EB%B3%B4%20%EC%A0%91%EC%86%8D%EA%B8%B0%EB%A1%9D%20%EA%B2%80%EC%83%89%EA%B2%B0%EA%B3%BC.jpg" alt=""/><figcaption><p>&#x3C;회원정보  접속기록 검색결과></p></figcaption></figure>

**① 수행업무**\
접속 URL에서 수행한 업무가 확인됩니다.\
📍수행업무 항목 : 조회, 등록, 수정, 삭제, 엑셀 생성, 엑셀 업로드, 엑셀 다운로드

**②** [**수행업무 상세**](#undefined-3)\
\[상세] 버튼을 클릭하면 해당 접속기록 및 업무 수행 상세정보를 확인할 수 있습니다.

## 운영자정보 접속기록 검색결과

<figure><img src="/images/service/privacy/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2Fa49MRluRzt6IYZq0zpub%2F%EC%9A%B4%EC%98%81%EC%9E%90%EC%A0%95%EB%B3%B4%20%EC%A0%91%EC%86%8D%EA%B8%B0%EB%A1%9D%20%EA%B2%80%EC%83%89%EA%B2%B0%EA%B3%BC.jpg" alt=""/><figcaption><p>&#x3C;운영자정보 접속기록 검색결과></p></figcaption></figure>

**① 수행업무**\
접속 URL에서 수행한 업무가 확인됩니다.\
📍수행업무 항목 : 조회, 등록, 수정, 삭제, 차단, 로그인, 로그인(실패), 로그아웃, 인증, 인증(실패)

**②** [**수행업무 상세**](#undefined-4)\
\[상세] 버튼을 클릭하면 해당 접속기록 및 업무 수행 상세정보를 확인할 수 있습니다.
