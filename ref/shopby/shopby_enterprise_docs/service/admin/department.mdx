# 부서/직책관리

<Callout type="info">
**쇼핑몰 운영자 계정 정보에 활용되는 부서와 직책을 추가하고 관리합니다.**
</Callout>

## 부서관리

<figure><img src="/images/service/admin/department/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2Fx5q3Bce1j7s2OEjQl3ue%2F%EB%B6%80%EC%84%9C%EA%B4%80%EB%A6%AC%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;부서관리></p></figcaption></figure>

**① 부서추가** \
새로운 부서추가를 위해 \[+부서 추가] 버튼을 클릭합니다. \
부서명 리스트에 \[새 부서명을 입력하세요]로 부서명 입력란이 추가되고, 상세설정 영역에서 부서명을 입력해주세요.

**② 삭제** \
삭제할 부서명을 선택하고 \[삭제] 버튼을 클릭합니다.

**③ 상/하 이동 아이콘 (부서명 순서변경)**

* 순서 변경이 필요한 부서명을 선택합니다.
* ∧(상) 이동 아이콘을 클릭하면 현재 위치보다 한 단계 위, ∨(하) 이동 아이콘은 한 단계 아래로 해당 부서명이 이동됩니다.

<br>

## 직책관리

**① 직책추가** \
새로운 직추가를 위해 \[+직책 추가] 버튼을 클릭합니다. \
직책명 리스트에 \[새 직책명을 입력하세요]로 직책명 입력란이 추가되고, 상세설정 영역에서 직책명을 입력해주세요.

**② 삭제** \
삭제할 직책명을 선택하고 \[삭제] 버튼을 클릭합니다.

**③ 상/하 이동 아이콘 (직책명 순서변경)**

* 순서 변경이 필요한 직책명을 선택합니다.
* ∧(상) 이동 아이콘을 클릭하면 현재 위치보다 한 단계 위, ∨(하) 이동 아이콘은 한 단계 아래로 해당 직책명이 이동됩니다.
