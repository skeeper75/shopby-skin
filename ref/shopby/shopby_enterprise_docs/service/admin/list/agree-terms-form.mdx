# 운영자 계정 등록 안내 (e-메일)

ⓐ '서비스 어드민 운영자 계정 등록' 안내 e-메일을 확인해 주세요. 메일 본문 내용의 \[운영자 계정 등록] 버튼을 클릭하면 운영자 계정등록 화면으로 이동합니다.

<div align="left"><figure><img src="/images/service/admin/list/agree-terms-form/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F4fYjoCKmyMmAu70iCLJ3%2F%EC%9A%B4%EC%98%81%EC%9E%90%20%EA%B3%84%EC%A0%95%20%EC%9D%B4%EB%A9%94%EC%9D%BC%20%EB%B3%B5%EC%82%AC%202.png" alt=""/><figcaption><p>&#x3C;e-메일 수신 확인></p></figcaption></figure></div>

ⓑ 운영자 계정등록 화면에서 '개인정보 수집 및 이용' 약관 내용을 확인합니다. 하단 영역 약관동의 체크박스를 선택하고 \[다음] 버튼을 클릭해 주세요.

<figure><img src="/images/service/admin/list/agree-terms-form/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FYOkEIJJd7AZcUQknQZkG%2F%EC%9A%B4%EC%98%81%EC%9E%90%20%EA%B3%84%EC%A0%95%EB%93%B1%EB%A1%9D1%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;운영자 계정등록 약관동의></p></figcaption></figure>

ⓒ  운영자 정보 입력과 휴대폰번호 인증을 진행하고, \[다음] 버튼을 클릭합니다.

* 쇼핑몰 어드민 로그인에 사용할 아이디를 입력하고, \[중복확인] 버튼으로 사용 여부를 확인합니다. \
  아이디는 영문 소문자, 숫자를 조합하여 5\~20자리로 설정할 수 있습니다. (특수문자 입력 불가)\
  📍아이디 첫번째 문자는 영문으로 시작해 주세요.
* 쇼핑몰 어드민 로그인에 사용할 비밀번호를 입력하고 비밀번호 확인을 진행합니다.\
  비밀번호는 영문, 숫자, 특수문자를 조합하여 8\~20자리로 설정이 가능합니다.\
  📍특수문자는  !@#$%^&+=-\_.( )만 사용이 가능해요.&#x20;
* 운영자명은 신규 운영자 계정 등록 과정에서 입력한 이름이 연동되어 노출됩니다.
* 직급과 소속부서/직책을 설정해 주세요.
* 전화번호와 휴대폰번호를 입력합니다.\
  📍등록한 휴대폰번호는 어드민 외부 접속 시 2차 인증 번호로 활용 돼요.
* \[인증번호발송] 버튼을 클릭하면 휴대폰으로 인증코드가 발송됩니다.
* 인증코드를 입력하고 \[확인] 버튼으로 본인 인증을 완료합니다.

<figure><img src="/images/service/admin/list/agree-terms-form/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FY1sgWthZzx2xMnHKGE4k%2F%EC%9A%B4%EC%98%81%EC%9E%90%20%EA%B3%84%EC%A0%95%EB%93%B1%EB%A1%9D2%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;운영자 계정등록 운영자 정보 입력과 인증></p></figcaption></figure>

ⓓ  '운영자 계정등록 완료' 안내 메시지와 설정한 운영자 정보가 확인됩니다. \[로그인] 버튼을 클릭해 샵바이 어드민 로그인 화면으로 이동해 주세요.

<figure><img src="/images/service/admin/list/agree-terms-form/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FglscAGcPzvNbzgbr6ePY%2F%EC%9A%B4%EC%98%81%EC%9E%90%20%EA%B3%84%EC%A0%95%EB%93%B1%EB%A1%9D3%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption></figcaption></figure>

ⓔ 샵바이 어드민 로그인 화면에서 설정한 아이디와 비밀번호를 입력하고 \[로그인] 합니다.

<Callout type="warning">
추가 등록한 운영자는 '일반회원'으로 로그인을 진행해 주세요.
</Callout>
