# 운영자 계정관리

<Callout type="info">
**신규 운영자 계정을 등록하고 관리할 수 있습니다.**

**샵바이 솔루션을 신청한 대표 운영자의 계정 정보가 확인됩니다.**
</Callout>

## 운영자 계정관리

<figure><img src="/images/service/admin/list/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FXnI7u8fxi9Vq2jQZaoIh%2F%EC%9A%B4%EC%98%81%EC%9E%90%20%EA%B3%84%EC%A0%95%EA%B4%80%EB%A6%AC%20%EB%B3%B5%EC%82%AC%202.jpg" alt=""/><figcaption><p>&#x3C;운영자 계정관리></p></figcaption></figure>

**① 검색어**\
운영자명, 아이디, 소속부서 등 다양한 검색어로 운영자를 조회합니다. 일부 검색어는 지원하지 않으며 정확한 검색어를 입력해 주세요.&#x20;

**② 운영자 삭제** \
퇴사 및 탈퇴 등의 사유로 사용하지 않는 운영자 계정을 삭제할 수 있습니다.&#x20;

**③** [**신규 운영자 계정 등록**](#undefined-2)\
새로운 운영자 계정을 등록합니다.

**④ 운영자 계정 수정**\
조회결과(리스트)에서 \[아이디]를 클릭하면, 운영자 계정 정보를 수정할 수 있는 수정 페이지로 이동됩니다.

**⑤ 상태**\
운영자 별 계정 등록 상태가 확인됩니다.

* 등록대기 : 신규 운영자 계정 등록 이후 헤당 운영자가 아이디/비밀번호 설정 등 추가 설정을 진행하지 않아 계정 등록이 완료되지 않은 상태
* 등록완료 : 해당 운영자가 직접 아이디/비밀번호 설정 등으로 운영자 계정 등록을 완료한 상태

##

### 신규 운영자 계정 등록

<figure><img src="/images/service/admin/list/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FTfmYIXFyq3gQL2CeMT3o%2F%EC%9A%B4%EC%98%81%20%EA%B3%84%EC%A0%95%20%EB%93%B1%EB%A1%9D%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;운영자 계정 등록></p></figcaption></figure>

**① 계정유형**\
마스터 운영자와 일반 운영자 중에서 계정유형을 선택합니다.

* 마스터 : 모든 어드민 메뉴에 권한이 부여됩니다.
* 일반 : 권한그룹 등록을 필수로 설정 해야하고, 해당 권한그룹만 권한이 부여됩니다.\
  📍일반 유형을 선택하면 SA 권한설정 항목이 노출됩니다. 권한그룹을 설정해 주세요. \
  &#x20;     (마스터 유형 선택 시 미노출)

**② 운영자명**\
운영자 이름을 입력합니다. (한글, 영문 ,숫자입력 가능 / 특수문자 입력 불가)

**③ e-메일 주소**\
입력한 e-메일 주소로 운영자 계정 등록 방법이 안내되어 있는 '운영자 계정 등록' 안내 메일이 발송됩니다.\
해당 운영자가 직접 메일 본문 내용을 확인하고, 운영자 인증확인 및 아이디/비밀번호 설정 등으로 등록완료를 해야합니다.&#x20;

**④ 외부접속**\
서비스관리 > 기초정보 관리 > 접속 IP 설정 메뉴에 등록한 회사 내부 사용 IP가 아닌, 외부에서의 어드민 접속 사용여부를 설정할 수 있습니다.

* 가능 : 해당 운영자가 SMS 추가 인증을 한 다음 외부에서 어드민 접속이 가능합니다.&#x20;
* 불가 : 해당 운영자의 외부 접속이 불가합니다.

&#x20;**⑤ 계정 만료일**\
등록하는 운영자 계정의 만료일을 설정할 수 있습니다. 만료일이 경과되면 해당 운영자 계정으로 어드민 접속이 불가합니다.\
📍 운영자 계정 수정 페이지에서 계정 만료일을 수정할 수 있습니다.

**⑥** \[저장] 버튼을 클릭해 운영자 계정정보 등록을 완료합니다.
