# 정산 보정 관리 (enterprise)

<Callout type="warning">
**위탁 판매 상품의 주문데이터를 기반으로 적재된 정산데이터의 판매수수료율이나 공급가를 수정할 수 있습니다.**\
**위탁 판매 상품의 주문이 아닌 정산데이터는 변경할 수 없습니다.**\
**주문별 정산데이터의 판매수수료율 또는 공급가를 확인할 수 있는 쇼핑몰 관리자 경로입니다.**

\*쇼핑몰 정산데이터: [서비스 관리>쇼핑몰 관리](https://service.shopby.co.kr/mall/list)>\[정산데이터 생성]\
\*파트너 정산데이터: [파트너 관리>매출 내역 조회](https://service.shopby.co.kr/settlement/partner)>\[엑셀 다운로드], [파트너 관리>정산 현황 조회](https://service.shopby.co.kr/settlement/list)>\[엑셀 다운로드]
</Callout>

<Callout type="danger">
**샵바이에서 제공하는 정산데이터에는 쇼핑몰 정산데이터와 파트너 정산데이터가 있습니다.**\
**쇼핑몰 정산데이터는 특정 쇼핑몰에서 발생한 주문 기준의 정산데이터이며,** \
**파트너 정산데이터는 특정 파트너사 상품이 판매된 주문 기준의 정산데이터입니다.**\
**정산데이터는 주문이 발생한 시점의 주문데이터 기반으로 적재되며, 적재 시점은 정산데이터에 따라 상이합니다.**
</Callout>

## 정산 보정 관리

* 정산 보정 기능은 다음의 두 가지 경우에 유용하게 사용됩니다.
  * 1\. 특정 기간 내 발생한 주문의 판매수수료 보정이 필요해요
  * 2\. 특정 주문에 한하여 판매수수료 보정이 필요해요

\
**1. 특정 기간 내 발생한 주문의 판매수수료 보정이 필요해요**

* 파트너와 사전 협의를 통해 일정 기간에 발생한 주문에 대한 판매수수료를 평소보다 높이거나, 낮추게 되었다면 일정 기간에 판매된 주문데이터를 확인한 뒤 보정해 주시면 됩니다.

\
**2. 특정 주문에 한하여 판매수수료 보정이 필요해요**

* 상품에 판매수수료가 반영되지 않은 상태 또는 오기재된 상태에서 발생한 주문의 판매수수료 보정이 필요하다면 주문데이터를 확인한 뒤 보정해 주시면 됩니다.

\
정산 보정 관리 메뉴에서 제공하는 기능과 정산데이터를 보정하는 방법에 대해 안내해 드리겠습니다.\
정산 보정 관리 메뉴에서는 크게 1) 정산 보정 목록 조회 2) 정산 보정 3) 보정데이터 결과 확인 3가지 기능을 제공합니다.

</Callout>

보정목록이 저장되어 ② 누가, 언제, 어느 쇼핑몰의 데이터를 보정했는지 확인하실 수 있습니다.\
어느 주문에 포함된 데이터가 보정되었는지, 보정데이터 중 성공/실패된 데이터는 무엇인지 등은 요청 상세 화면에서 확인 가능합니다.

## 정산 보정

정산 보정은 정해진 엑셀 양식을 업로드하여 요청하는 방식만 지원하며, 엑셀 양식에 포함된 정보의 보정만 허용합니다.

<figure><img src="/images/service/settlement-adjustment/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FFUL2T2egTB6pDiPvjVGU%2Fimage.png" alt=""/><figcaption></figcaption></figure>

① 엑셀 양식은 정산 보정 화면에서 다운로드할 수 있습니다.\
정산 보정 시에는 정산데이터 뿐만 아니라 주문 및 통계데이터에 적재된 데이터도 함께 보정될 뿐만 아니라 수수료율 기준으로 계산되어 저장된 판매수수료 금액, 정산예정금액, 총 상품판매금액, 쇼핑몰 수익 등 재계산 되어야 하는 데이터도 자동으로 재계산 됩니다.

## 보정데이터 결과 확인

<figure><img src="/images/service/settlement-adjustment/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2Fcwjlh6iDzAeOFlaw2nWB%2Fimage.png" alt=""/><figcaption></figcaption></figure>

① 주문상품옵션번호 기준으로 적재된 모든 쇼핑몰 정산, 파트너 정산, 주문, 통계데이터가 모두 각각 보정됨으로써 데이터 보정 결과도 각각 확인할 수 있습니다.\
보정 요청된 주문이 아직 정산데이터나 통계데이터에 적재되지 않은 상태라면 주문데이터에 대한 보정 결과만 확인 가능하고, 보정 성공 이후 적재되는 정산데이터와 통계데이터에는 보정 내용이 반영됩니다.\
만약 주문데이터 보정이 실패되었다면 이후 동일한 주문상품옵션번호로 적재되는 모든 데이터에 보정 내용이 반영되지 않으니 유의해 주시기 바랍니다.

\
보정 실패한 데이터는 ② 실패 사유를 확인한 뒤 다시 시도해 주시기 바랍니다. \
동일한 주문상품옵션번호에 대해 보정 결과가 일부는 성공하고, 일부는 실패했다면 일시적인 오류일 확률이 높아 동일한 내용으로 한 번 더 보정해 주시기 바랍니다. \
③ 엑셀파일을 다운로드해 다시 보정 필요한 데이터만 남겨 손쉽게 보정할 수 있습니다.

데이터 보정이 실패되는 사유는 다음과 같습니다. 기능 사용에 참고해 주시기 바랍니다.&#x20;

| 실패 사유                              | 설명                                                                    |
| ---------------------------------- | --------------------------------------------------------------------- |
| 존재하지 않는 주문                         | 존재하지 않는 주문번호인 경우                                                      |
| 주문번호와 주문상품옵션번호 불일치                 | 주문번호와 주문상품옵션번호가 일치하지 않은 경우                                            |
| 변경 불가 상품(사입상품)                     | 주문상품의 판매방식이 사입 상품인 경우                                                 |
| 보정데이터 미입력                          | 주문번호와 주문상품옵션번호만 입력되고, 수수료율 또는 공급가는 입력하지 않은 경우                         |
| 보정데이터 입력 오류                        | 공급가와 수수료율을 모두 입력한 경우                                                  |
| 수수료율 변경 불가 상품(공급가 입력 상품)           | 주문상품의 판매수수료가 공급가인 경우                                                  |
| 공급가 변경 불가 상품(수수료율 입력 상품)           | 주문상품의 판매수수료가 수수료율인 경우                                                 |
| 수수료율 오기재(0\~99.9 입력 가능)            | 입력된 수수료율이 0\~99.9 범위를 벗어난 경우                                          |
| 공급가 오기재(공급가에 . 입력 불가)              | <p>공급가에 . 이 포함된 경우<br>\* 원화 또는 엔화로 운영되는 쇼핑몰만 해당</p>                   |
| 공급가 오기재(0\~1,000,000,000 미만 입력 가능) | <p>공급가를 1,000,000,000 이상의 숫자로 입력한 경우<br>\* 원화 또는 엔화로 운영되는 쇼핑몰만 해당</p> |
| 공급가 기재 범위 초과                       | <p>공급가를 10자리 이상 입력한 경우<br>\* 달러로 운영되는 쇼핑몰만 해당</p>                     |
| 장애 및 오류 발생                         | 통신 장애 등의 사유로 보정 실패한 경우                                                |
| 기타                                 | 위 설명에 해당하지 않는 사유로 실패한 경우                                              |
