# 상품이미지 대량등록

<Callout type="info">
**상품 이미지를 일괄 등록할 수 있습니다.**
</Callout>

## 상품이미지 업로드 방법/파일명설정

<figure><img src="/images/product/bulkimage/add/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FIlOKs7t56og7BHWGBLCQ%2Fimage.png" alt=""/><figcaption></figcaption></figure>

**① 상품이미지 업로드 방법**\
이미지 업로드 시 '파일명'과 '이미지 항목+파일명' 을 선택 후 업로드 가능합니다.\
\[?] 버튼 클릭 시 상품이미지 대량등록 업로드 방법에 대한 '[상품이미지 대량등록 업로드 방법 가이드](#undefined-1)' 확인이 가능합니다.

* 이미지 업로드 방법은 '파일명' 이 기본 값으로 설정됩니다.&#x20;
* 이미지 업로드 방법을 '이미지 항목+파일명' 으로 선택 시 '이미지 항목+파일명 업로드 요청 설정' 메뉴가 노출됩니다.&#x20;

② 파일명 설정\
이미지 업로드 방법의 규칙에 따라 '상품번호' 또는 '상품관리코드'를 선택 후 파일명을 설정합니다.\
파일명 설정 규칙 및 예시는 이미지 업로드 방식에 따라 메뉴 하단에 노출되거나 '[상품이미지 대량등록 업로드 방법 가이드](#undefined-1)' 통해 확인이 가능합니다.&#x20;

③ 이미지 등록 \
\[이미지등록] 버튼 클릭 시 '[이미지 등록](#undefined-2)'이 가능한 페이지가 팝업 형태로 노출됩니다.

① 파일추가 : \[파일추가] 버튼 클릭 후 파일 추가가 가능합니다. 상단 '업로드할 파일 또는 폴더를 여기에 끌어다 놓으세요' 문구 위 업로드를 희망하는 파일 및 폴더를 드롭다운 하는 방식으로도 업로드가 가능합니다.

② 파일삭제 : 업로드한 파일을 선택 후 \[파일삭제] 버튼 클릭 시 해당 파일이 삭제됩니다.

③ 업로드 : \[업로드] 버튼 클릭 시 업로드한 파일 및 폴더가 최종 업로드 됩니다.

④ 취소 : \[취소] 버튼 클릭 시 이미지 업로드 창이 종료 됩니다.
