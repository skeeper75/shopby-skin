# 입출고주소 관리

<Callout type="info">
**쇼핑몰 운영 방식 및 상품 판매 방식에 따라 상품이 출고되는 주소지와 반품 및 교환 상품을 접수받는 주소지가 상이할 수 있기 때문에,  여러 개의 입출고 주소지를 미리 등록해 놓고 상품에 따른 여러 종류의 배송비 템플릿에 적용할 수 있습니다.**
</Callout>

## 입출고주소 관리 리스트

<figure><img src="/images/product/delivery/warehouse/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FV2JGSGZFUltcnaymebnn%2F%EC%9E%85%EC%B6%9C%EA%B3%A0%EC%A3%BC%EC%86%8C%EA%B4%80%EB%A6%AC%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;입출고주소 관리 리스트></p></figcaption></figure>

**① 입출고 주소등록** \
\[입출고 주소등록] 버튼을 클릭하면 신규 주소지를 등록할수 있는 '신규주소 등록' 창이 노출됩니다.

**② 선택주소 삭제** \
주소지의 체크박스를 선택하면, 리스트 오른쪽 상단에 \[선택주소 삭제] 버튼이 노출됩니다. \
삭제가 필요한 주소지의 체크박스를 선택해 삭제를 진행해 주세요.

**③ 수정** \
\[주소명]을 클릭하면 해당 주소지 설정 항목을 수정할 수 있는 '주소정수' 창이 노출됩니다.

**④ 대표여부**\
입출고 주소지 기본값인 대표 주소지(대표 출고지, 대표 반품/교환지)가 확인됩니다.

### 입출고 주소 등록

<div align="left"><figure><img src="/images/product/delivery/warehouse/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FqyfuGm5KkwkCI1OVDMgM%2F%EC%9E%85%EC%B6%9C%EA%B3%A0%EC%A3%BC%EC%86%8C%EA%B4%80%EB%A6%AC%20%EC%8B%A0%EA%B7%9C%EC%A3%BC%EC%86%8C%EB%93%B1%EB%A1%9D%20%EB%B3%B5%EC%82%AC.jpg" alt="" width="563"/><figcaption><p>&#x3C;신규주소 등록></p></figcaption></figure></div>

**① 주소명**\
신규 주소지의 이름을 입력합니다.

**② 국가**\
신규 주소지의 국가를 선택합니다.

**③ 대체문구**\
대체문구 체크박스를 선택하면 하단에 대체문구 입력란이 노출됩니다.

<Callout type="warning">
주소 대신 안내되어야 하는 다른 문구 및 내용이 있다면 200자 이내의 대체문구를 입력하여 안내할 수 있습니다.
</Callout>

**④ 주소**\
\[우편번호 찾기] 버튼을 클릭하면 '주소찾기' 창이 노출됩니다.\
'주소찾기' 창에서 지번 또는 도로명 주소를 입력하고, \[조회] 버튼을 클릭합니다. \
조회결과 영역에서 도로명 또는 지번을 선택하고, 주소 항목에서 나머지 주소를 입력해 주세요.

**⑤ 대표여부**\
대표출고지와 대표 반품/교환지 체크박스를 선택하면, 해당 주소지는 입출고 주소지의 기본값으로 설정됩니다.

**⑥** \[저장] 버튼을 클릭 해 신규 주소지 등록을 완료합니다.
