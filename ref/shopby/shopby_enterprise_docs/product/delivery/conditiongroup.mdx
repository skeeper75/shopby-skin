# 배송비 템플릿관리

<Callout type="info">
**상품에 적용하는 배송정보를 규칙으로 만든 배송비 템플릿을 등록하고 관리하는 배송비 템플릿관리 메뉴입니다.**

**쇼핑몰 운영과 상품 판매 전, 배송비 템플릿을 미리 등록해 놓아야 상품 등록 과정에서 배송비 설정이 가능하고, 상품 구매 시 해당 배송비까지 포함되어 고객 결제금액에 바로 적용됩니다.**

**등록된 복수의 배송비 템플릿을 그룹으로 묶어 묶음배송으로 활용할 수 있습니다.**
</Callout>

## 배송비 템플릿 묶음그룹 리스트

<figure><img src="/images/product/delivery/conditiongroup/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FOOZ4lEkkD3rMu17IIDfE%2F%EB%B0%B0%EC%86%A1%EB%B9%84%ED%85%9C%ED%94%8C%EB%A6%BF%EA%B4%80%EB%A6%AC%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;배송비 템플릿 관리></p></figcaption></figure>

**①**  기본 (묶음)그룹에 지정된 배송비 템플릿 수가 확인되며, 기본 (묶음)그룹도 삭제와 순서 변경, 그룹명 수정 등 모든 상세 설정이 가능합니다.

**② 묶음그룹 추가**\
새로운 배송비 템플릿 묶음그룹의 등록을 위해 \[+ 묶음그룹 추가] 버튼을 클릭합니다. 배송비 템플릿 그룹 리스트 하단에 \[새 그룹] 으로 새로운 배송비 템플릿 묶음그룹이 추가됩니다.

<Callout type="warning">
**묶음그룹이란 묶음배송이 가능한 배송비 템플릿을 모은 그룹을 의미합니다.**

묶음배송은 동일한 묶음그룹 내의 배송비 템플릿 주문 건만 적용되며, 다른 묶음그룹의 배송비 템플릿 주문 건 간의 묶음배송은 불가합니다.
</Callout>

**③ 삭제** \
삭제할 배송비 템플릿 묶음그룹을 선택하고 \[삭제] 버튼을 클릭합니다.

**④ 상/하 이동 아이콘 (순서변경)** \
이동 아이콘으로 배송비 템플릿 묶음그룹 순서를 변경할 수 있습니다.

* 순서 변경이 필요한 배송비 템플릿 묶음그룹을 선택합니다.
* ∧(상) 이동 아이콘을 클릭하면 현재 위치보다 한 단계 위, ∨(하) 이동 아이콘은 한 단계 아래로 해당 배송비 템플릿 묶음그룹이 이동됩니다.

## 그룹 상세 설정

<figure><img src="/images/product/delivery/conditiongroup/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2Ft5Z87oeoOlA1atD0OitZ%2F%EB%B0%B0%EC%86%A1%EB%B9%84%ED%85%9C%ED%94%8C%EB%A6%BF%20%EA%B7%B8%EB%A3%B9%EC%83%81%EC%84%B8%EC%84%A4%EC%A0%95%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;배송비 묶음그룹 상세 설정></p></figcaption></figure>

**① 그룹명** \
해당 묶음그룹의 이름을 입력합니다.

**② 배송비 템플릿**&#x20;

* \[+ 배송비 템플릿 추가] 버튼을 클릭해 해당 묶음그룹에 등록할 배송비 템플릿을 추가합니다.
* \[[배송비 템플릿명](#undefined-9)]을 클릭하면 해당 배송비 템플릿을 수정할 수 있습니다.
* 적용상품의 \[[보기](#undefined-9)]를 클릭하면, 해당 배송비 템플릿이 적용된 상품 리스트가 확인됩니다.
* \[삭제]를 클릭해 해당 묶음그룹에서 배송비 템플릿을 삭제할 수 있습니다.&#x20;

**③ 묶음배송비 산출방식** \
묶음배송 적용 시, 선택한 묶음배송비 산출방식에 따라 배송비를 부과합니다.

* 최소부과 : 각 상품별로 배송비를 산출하여 그 중 가장 적은 배송비를 부과합니다.
* 최대부과 : 각 상품별로 배송비를 산출하여 그 중 가장 많은 배송비를 부과합니다.

**④ 결제방식** \
'선결제'와 '착불' 중, 해당 묶음그룹에 적용할 배송비 결제방식을 선택합니다.

**⑤ 지역별 추가 배송비** \
일부 국가 및 지역에 추가 배송비를 부과할 수 있습니다. \
📍추가 배송비를 설정한 국가 및 지역은 기본으로 설정한 배송비에추가 배송비가 더 적용됩니다.

* 국가 및 지역별 추가 배송비를 부과하지 않는 경우 '사용안함'을 선택합니다.
* 국가 및 지역별 추가 배송비를 부과하지 않는 경우 '사용함'을 선택하고, 추가 배송비를 부과할 국가 및 지역을 선택합니다.

<Callout type="warning">
추가 배송비를 부과할 국가 및 지역은 [배송비 관리 > 지역별 추가배송비 관리 메뉴](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/product/delivery/areafee)에서 먼저 등록하거나, \[새로등록] 버튼을 클릭해 해당 화면에서 추가해 주세요.
</Callout>

**⑥ 배송불가 국가 설정**\
특정 국가를 배송 제외 국가로 설정할 수 있습니다. \
배송불가 처리를 희망하는 국가의 체크박스를 선택해 주세요.\
📍복수 설정 또는 미설정도 가능합니다.

**⑦** \[변경사항 저장] 버튼을 클릭해 새로운 묶음그룹을 생성합니다.

### 배송비 템플릿 등록

<figure><img src="/images/product/delivery/conditiongroup/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FilcWJhAOp4zLpyuG3zx3%2F%EB%B0%B0%EC%86%A1%EB%B9%84%ED%85%9C%ED%94%8C%EB%A6%BF%EB%93%B1%EB%A1%9D.jpg" alt=""/><figcaption><p>&#x3C;배송비 템플릿  등록></p></figcaption></figure>

**① 배송비템플릿명**\
배송비 템플릿 이름을 입력합니다.

**② 배송방법**\
택배/등기/소포와 직접배송(화물배달) 중에서 배송이 진행되는 방법을 선택합니다.

* 택배/등기/소포를 선택한 다음 배송업체를 선택합니다.
* 택배사 리스트에서 이용중인 택배사를 선택해 주세요. \
  📍만약 이용중인 택배사가 리스트에 없는 경우, 리스트 최하단 ‘기타’를 선택해 주세요.

**③ 배송비유형**\
**무료 / 조건부 무료 / 유료(고정 배송비) / 수랑비례** 중에서 기본 배송비 계산 방식을 설정합니다.

* 무료 : 배송비가 0원으로 계산됩니다.
* 조건부 무료 : 주문 금액에 따라 입력한 배송비가 계산되는 방식으로, 기준 금액 미만이면 배송비가 계산되고 기준 금액 이상이면 배송비가 무료입니다.
  * 예) 50,000원 이상 배송비 무료 / 50,000원 미만 시 배송비 3,000원
* 유료(고정 배송비) : 주문 수량 및 결제금액에 상관없이 입력한 배송비가 무조건 계산됩니다.
* 수량비례 : 주문 상품의 품목별 수량에 따라 입력한 배송비가 계산되는 방식입니다.
  * 예) 2개당 배송비 3,000원\
    &#x20;     A상품 2개 + B상품-옵션a 1개 + B상품-옵션b 2개 : 구매수량 총 5개 → 배송비 9,000원

**④ 출고지**\
상품이 출고되는 주소를 선택합니다.

**⑤ 반품/교환지**\
상품이 반품 및 교환되는 경우 입고되는 주소를 선택합니다.\
📍출고지와 반품/교환지는 [배송비 관리 > 입출고주소 관리](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/product/delivery/warehouse) 메뉴에서 먼저 등록하거나, \[새로등록] 버튼을 클릭해 해당 화면에서 추가해 주세요.

**⑥ 반품/교환배송비**\
반품 및 교환 상품에 부과할 배송비를 편도 기준으로 입력합니다.\
📍 '0'원으로 설정하는 경우 반품 및 교환이 무료로 진행되는 점을 반드시 유의해 주세요.

**⑦ 기본배송비**\
기본배송비를 선택하면, 상품 등록 시 해당 배송비 템플릿이 기본 배송비로 설정됩니다.

⑧ \[확인] 버튼을 클릭해 배송비 템플릿을 등록합니다.
