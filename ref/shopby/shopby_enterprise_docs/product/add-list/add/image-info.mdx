# 이미지정보

<figure><img src="/images/product/add-list/add/image-info/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FfMyglk2k91DKI3XJaOuE%2F%EC%83%81%ED%92%88%EB%93%B1%EB%A1%9D%20%EC%9D%B4%EB%AF%B8%EC%A7%80%EC%A0%95%EB%B3%B41%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;이미지 정보></p></figcaption></figure>

**① 대표이미지**\
상품을 대표하는 대표이미지를 등록합니다.\
📍 대표이미지는 1개 등록이 가능합니다.<br>

**② 추가이미지**\
상품의 추가이미지는 최대 10개까지 등록이 가능합니다.\
\[이미지 추가] 버튼을 클릭하면 상품의 추가이미지를 등록할 수 있는 이미지 등록란이 노출됩니다.\
📍 ︿ (상) ﹀ (하) 이동 아이콘으로 추가이미지 노출 순서를 변경할 수 있어요.<br>

**③ 리스트 이미지**\
쇼핑몰 메인 페이지 및 전시카테고리 페이지 등과 같은 상품 리스팅 페이지에 노출되는 리스트 이미지는\
1개 등록이 가능합니다.&#x20;

<Callout type="warning">
**상품 이미지는 URL 입력 / 이미지 등록 / 동영상 등록 방식으로 이미지를 등록할 수 있습니다.**\
등록 가능한 이미지 확장자는 JPG, JPEG, GIF, PNG, BMP, WebP 이며 최대 10MB까지 등록 가능해요.

* URL 입력 : 외부 저장소에 있는 이미지를 상품 이미지로 등록할 수 있습니다. \
  \[URL 입력] 버튼을 활성화하여 이미지가 위치한 주소 경로(URL)를 입력합니다 .
* 이미지 등록 : \[파일찾기] 버튼을 클릭 해 상품 이미지를 첨부하여 등록할 수 있습니다.
* 동영상 URL <img src="/images/product/add-list/add/image-info/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FoTe3FZtVw7Vx0YwMW055%2Fapi2.png" alt="" data-size="line"/> : 상품 이미지를 동영상 컨텐츠로 등록할 수 있습니다. \
  \[URL 입력] 버튼을 활성화 하고, 동영상 URL의 체크박스를 선택한 다음 동영상이 위치한 주소 경로(URL)를 입력합니다.
</Callout>

**④ 상품 상세(상단)**\
쇼핑몰 상품 상세 페이지 상단 영역에 노출되는 컨텐츠를 에디터 기능으로 등록할 수 있습니다.\
📍쇼핑몰 관련 공통 공지 및 판매 상품 추가 안내 등으로 활용할 수 있어요.<br>

**⑤ 상품 상세**\
쇼핑몰 상품 상세 페이지 영역에 상품의 상세 정보를 에디터 기능으로 등록할 수 있습니다.

* 등록된 옵션 이미지 사용 : 상품 옵션 이미지를 상세 페이지에 노출하는 기능으로, 옵션별 대표 및 상세 이미지가 1개 이상 필수로 등록되어 있어야 설정이 가능합니다. \
  📍옵션별 대표 및 상세 이미지 등록은 \
  &#x20;    판매정보 > 옵션등록 > [옵션 이미지 일괄등록](https://nhn-commerce.gitbook.io/shopby_premium_manual/product/add-list/add/sale-info/option#undefined-8) 또는 [옵션별 이미지 등록](https://nhn-commerce.gitbook.io/shopby_premium_manual/product/add-list/add/sale-info/option#undefined-9)에서 가능해요.

**⑥ 상품 상세(하단)**\
쇼핑몰 상품 상세 페이지 하단  영역에 노출되는 컨텐츠를 에디터 기능으로 등록할 수 있습니다.\
📍쇼핑몰 관련 공통 공지 및 판매 상품 추가 안내 등으로 활용할 수 있어요.

**⑦ 배송안내 / AS안내 / 교환안내 / 환불안내**\
배송안내 / AS안내 / 교환안내 / 환불안내의 '사용함'을 선택하면, 배송/AS/교환/환불 관련 상품 이용 안내 사항을 작성할 수 있는 각각의 에디터 창 입력란이 노출됩니다.

* '기본 템플릿 사용'을 선택하면 샘플로 제공하는 기본 안내문구가 에디터 창에 노출되며,  \[수정] 버튼을 클릭해 '[안내 템플릿 수정'](#undefined) 창에서 등록하는 상품 및 쇼핑몰 운영사항을 반영한 내용으로 수정하여 사용합니다.

<Callout type="warning">
배송안내 / AS안내 / 교환안내 / 환불안내는 고객에게 반드시 안내되어야 하는 상품 이용 필수 안내사항으로 상품 상세 페이지에 반드시 노출되어야 합니다.
</Callout>
