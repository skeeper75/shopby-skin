# 옵션 등록

## 옵션입력

### 분리형 / 일치형

<figure><img src="/images/product/add-list/add/sale-info/option/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F3WhBTpqdw5CmOCnY2mJn%2F%EC%98%B5%EC%85%98%EB%93%B1%EB%A1%9D_%EC%98%B5%EC%85%98%EC%9E%85%EB%A0%A5_%EB%B6%84%EB%A6%AC%EC%9D%BC%EC%B2%B4.jpg" alt=""/><figcaption><p>&#x3C;옵션입력 - 분리형/일치형></p></figcaption></figure>

**①** [**다른상품 옵션 불러오기**](#undefined-7)\
옵션이 등록된 상품의 옵션을 불러와 적용하는 방식으로, 옵션 등록 상품이 있는 경우에 사용 가능합니다.

**② 조합형 엑셀양식 다운로드**\
\[조합형 엑셀양식 다운로드] 버튼을 클릭하면, 옵션정보 항목을 입력할 수 있는 엑셀 양식 파일이 다운로드 됩니다.

<Callout type="warning">
다운로드 받은 엑셀 양식 파일에서 옵션 정보 항목별 데이터의 상세 입력 방법과 예시가 확인됩니다.
</Callout>

**③ 엑셀 일괄등록** \
\[엑셀 일괄 등록] 버튼을 클릭 해 옵션 정보 항목을 입력한 엑셀 양식 파일을 업로드 합니다.

<Callout type="danger">
**엑셀 양식에 옵션 정보 항목을 입력하고, 데이터가 없는 '행'은 완전히 삭제하고 업로드를 진행해 주세요.** \
**빈 값 행이 많은 엑셀 파일을 등록하면, 행 개수만큼 옵션 정보로 인식하여 처리 소요 시간에 지장을 주고 오류가 발생할 수 있습니다.**

\
ⓐ 엑셀 파일 내 스크롤을 가장 아래로 이동합니다.\
ⓑ 데이터가 없는 행의 범위를 확인합니다.\
ⓒ 행을 모두 선택하고 삭제합니다. \
ⓓ 엑셀 파일을 저장하고 \[엑셀 일괄 등록]으로 업로드 합니다.
</Callout>

<div align="left"><figure><img src="/images/product/add-list/add/sale-info/option/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FFO4aDOEZUvi0rxCVjCFo%2F%EC%98%B5%EC%85%98%EC%9D%BC%EA%B4%84%EB%93%B1%EB%A1%9D%20%EC%97%85%EB%A1%9C%EB%93%9C%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;엑셀 일괄 등록 - 빈 값 행 삭제></p></figcaption></figure></div>

**④ 옵션 사용 설정**\
선택형 옵션과 텍스트 옵션의 사용 개수를 선택합니다. \
📍 다수의 선택형 옵션을 사용하면 쇼핑몰 화면에  옵션이 조합되어 노출됩니다.

* 선택형 옵션 : 운영자가 등록한 옵션을 주문자가 선택하는 방식으로, 최대 5개까지 선택 가능합니다.
* 텍스트 옵션 : 주문자가 옵션의 내용을 직접 입력하는 방식으로, 최대 5개까지 선택 가능합니다.

**⑤ 옵션 선택방식**

* 분리형 : 입력된 옵션을 각각 선택할 수 있도록 옵션의 개별 선택 박스가 생성됩니다.
* 일체형 : 모든 옵션이 조합된 하나의 옵션 선택 박스가 생성됩니다 .&#x20;

**⑥ 옵션명**\
옵션값을 대표하는 속성으로 옵션의 이름을 입력합니다.\
📍 예) 사이즈, 컬러, 용량 등<br>

**⑦  옵션값**\
옵션의 세부 정보로, 주문자가 선택할 수 있는 옵션의 종류를 입력합니다.\
📍 예)  230m, S, 화이트, 500ml 등<br>

**⑧ 옵션가**\
옵션의 종류를 선택했을 때 상품 판매가에 추가 및 차감되는 옵션의 가격을 입력합니다. \
추가 금액은 숫자만 입력하고, 차감 금액은 (–)를 숫자와 함께 입력해 주세요.

**⑨ 매입가**\
해당 옵션(상품)을 매입한 실제 금액을 입력하여 내부 관리용으로 활용합니다.

**⑩ 재고수량**\
옵션별 재고 수량을 각각 입력합니다.\
📍입력한 옵션별 재고 수량의 총 합계가 해당 상품의 전체 재고 수량으로 산정되요.

**⑪ 옵션관리코드 / 추가관리코드**\
내부 관리용으로 활용하는 옵션관리코드 및 추가관리코드를 등록할 수 있습니다. \
📍옵션관리코드와 추가관리코드는 [상품관리 > 상품재고관리 메뉴](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/stock-management#undefined-1)에서도 수정할 수 있어요.

**⑫ \[ + ]  /  \[ - ]**\
선택형 옵션 개수를 추가하거나, 한 개의 옵션에 여러 옵션값이 있는 경우 \[ + ] 버튼을 클릭하면 입력란이 추가되어 활용할 수 있습니다.  \[ - ] 버튼을 클릭하면 입력란이 삭제됩니다.

**⑬ 옵션 정보 적용**\
옵션 정보 항목을 입력 한 다음, \[옵션 정보적용] 버튼을 클릭해 주세요.

<Callout type="warning">
\[옵션 정보 적용] 버튼을 클릭하면, 입력한 옵션 정보들이 서로 조합되어 옵션 목록에 노출되고, 옵션을 최종 등록할 수 있습니다.
</Callout>

### 분리형(필수/선택)

**✅&#x20;**<mark style="color:green;background-color:yellow;">**필수옵션 : 상품 구매를 위해 반드시 1개 이상 선택해야 하는 옵션이에요.**</mark>\
\&#xNAN;**✅&#x20;**<mark style="color:green;background-color:yellow;">**선택옵션 : 고객의 필요에 의해 선택할 수 있는 옵션으로, 추가상품 판매 등으로 활용할 수 있어요.**</mark>

<figure><img src="/images/product/add-list/add/sale-info/option/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FwmpICORoOT3r4TXWrH8h%2F%EC%98%B5%EC%85%98%EB%93%B1%EB%A1%9D_%EC%98%B5%EC%85%98%EC%9E%85%EB%A0%A5_%EB%B6%84%EB%A6%AC_%ED%95%84%EC%88%98%EC%84%A0%ED%83%9D.jpg" alt=""/><figcaption><p>&#x3C;옵션입력 - 분리형(필수/선택)></p></figcaption></figure>

**① 옵션 사용 설정**\
분리형(필수/선택) 옵션방식의 선택형 옵션 사용 개수는 2개로 고정되어 있습니다. \
📍필수옵션 1개 / 선택옵션 1개

**② 옵션명**\
분리형(필수/선택) 옵션방식의 옵션명은 '필수옵션'과 '선택옵션'으로 기본 옵션명이 제공됩니다.\
📍 필수/선택 옵션명은 수정이 불가한 점을 참고해 주세요.

**③  옵션값**\
옵션의 세부 정보로, 주문자가 선택할 수 있는 옵션의 종류를 입력합니다.

**④ 옵션가**\
옵션의 종류를 선택했을 때 상품 판매가에 추가 및 차감되는 옵션의 가격을 입력합니다. \
추가 금액은 숫자만 입력하고, 차감 금액은 (–)를 숫자와 함께 입력해 주세요.

**⑤ 옵션 정보 적용**\
옵션 정보 항목을 입력 한 다음, \[옵션 정보적용] 버튼을 클릭해 주세요.

<Callout type="warning">
\[옵션 정보 적용] 버튼을 클릭하면, 입력한 필수옵션과 선택옵션 정보가 반영되어 옵션 목록에 노출되고, 옵션을 최종 등록할 수 있습니다.
</Callout>

## 옵션 목록

### 분리형 / 일치형

<figure><img src="/images/product/add-list/add/sale-info/option/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F2O77MJlruDuAgjTmJmwu%2F%EC%98%B5%EC%85%98%EB%93%B1%EB%A1%9D_%EC%98%B5%EC%85%98%EB%AA%A9%EB%A1%9D_%EB%B6%84%EB%A6%AC.%EC%9D%BC%EC%B9%98.jpg" alt=""/><figcaption><p>&#x3C;옵션목록 - 분리형/일치형></p></figcaption></figure>

**① 선택삭제**\
옵션 정보 항목을 선택해 삭제할 수 있습니다.

**② 선택일괄수정**\
특정 옵션을 선택하고 '옵션가'와 '재고수량'을 입력한 다음, \[선택일괄수정] 버튼을 클릭해 해당 항목을 일괄수정 처리할 수 있습니다.

**③ 품절상태**\
옵션 재고수량에 따른 옵션 (품절)상태 값입니다.

* 옵션 재고수량이 '0' 인 경우, 자동으로 '품절 '상태로 변경됩니다.&#x20;
* '품절' 상태로 수동 변경하면, 옵션 재고수량이 '0'으로 변경됩니다.
* '일시품절' 상태로 수동 변경하면, 옵션 재고수량은 유지되며 옵션 상태는 '품절' 로 변경됩니다.
* '일시품절' 상태에서 '정상' 상태로 수동 변경이 가능하고, 옵션 재고수량으로 판매가 가능합니다.

**④** [**이미지일괄수정 (옵션 이미지 일괄 등록)**](#undefined-8)\
\[이미지일괄수정] 버튼을 클릭해 모든 옵션 이미지를 일괄 등록 및 수정 할 수 있습니다.

**⑤** [**이미지 등록 / 변경**](#undefined-9)\
\[등록] 및 \[변경] 버튼을 클릭하면, '옵션 이미지 등록/수정' 창이 노출되며, 옵션 이미지를 등록(수정)할 수 있습니다.

**⑥ 사용여부**\
옵션 사용여부를 'Y' 와 'N' 으로 설정합니다.\
사용여부를  'N' 으로 설정하면, 쇼핑몰에서 옵션 선택 및 구매가 불가능합니다.

**⑦** \[엑셀다운로드] 버튼을 클릭하면, 옵션 목록의 옵션 정보를 엑셀 파일로 다운로드 받을 수 있습니다.

### 분리형(필수/선택)

<figure><img src="/images/product/add-list/add/sale-info/option/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F9acKJpE7NBf6cyqs5bqa%2F%EC%98%B5%EC%85%98%EB%93%B1%EB%A1%9D_%EC%98%B5%EC%85%98%EB%AA%A9%EB%A1%9D_%EB%B6%84%EB%A6%AC_%ED%95%84%EC%88%98%EC%84%A0%ED%83%9D.jpg" alt=""/><figcaption><p>&#x3C;옵션목록 - 분리형(필수/선택)></p></figcaption></figure>

**① 필수여부**\
필수옵션(빨간색)과 선택옵션(파란색)으로 설정된 옵션 필수여부가 확인됩니다.\
📍옵션 목록에서 필수옵션과 선택옵션의 설정 변경이 가능해요.

<Callout type="danger">
**분리형(필수/선택) 옵션**을 등록한 상품은 상품 복사 등에서 **재고수량 연동이 불가해요.**&#x20;

**세트 상품의 세트 옵션**을 구성하는 경우, **분리형(필수/선택) 옵션을 설정할 수 없어요.**

상품의 **최소구매수량과 최대구매수량**은 **필수옵션 구매수량을 기준**으로 적용돼요.\
선택옵션은 판매 가능 재고 범위 내에서 구매수량에 제한이 없어요.

상품의 **구매수량별 추가할인 혜택**은 **필수옵션 구매수량과 필수옵션 금액을 기준**으로 적용돼요.\
쿠폰 및 적립금 할인 혜택은 필수옵션과 선택옵션 모두 해당돼요.
</Callout>

## 텍스트 옵션

<figure><img src="/images/product/add-list/add/sale-info/option/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F1OjA2dJrANP5j8wKmZqd%2F%ED%85%8D%EC%8A%A4%ED%8A%B8%EC%98%B5%EC%85%98%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;텍스트옵션></p></figcaption></figure>

**① 옵션명**\
옵션값을 대표하는 옵션의 이름을 입력합니다. 최대 25자까지 입력이 가능해요.\
📍 예) 선물 메세지, 이름 입력등<br>

**② 사용여부**\
텍스트 옵션의 사용 여부를 'Y'와 'N'으로 설정합니다.&#x20;

**③ 필수여부**\
텍스트 옵션의 필수(입력) 여부를 '필수'와 '선택'으로 설정합니다. \
'필수'로 설정하면 주문자가 상품 구매 시, 텍스트 옵션을 필수로 입력해야 주문/결제 페이지로 이동할 수 있어요.\
📍 텍스트 옵션을 반드시 입력해야 하는 상품에 활용할 수 있어요.\
&#x20;   예) 네임 tag, 주차번호판 등<br>

**④ 매칭타입**

* 상품별 매칭 : 해당 상품의 모든 옵션에 텍스트 옵션을 동일하게 적용
* 옵션별 매칭 : 각각의 개별 옵션에 텍스트 옵션이 적용

**⑤ 삭제**\
\[삭제] 버튼을 클릭해 텍스트 옵션을 삭제할 수 있습니다.

* 일괄 등록(수정)  전, 이미지 파일명 예시와 등록 안내를 확인해 주세요.
  * 옵션 이미지 일괄 등록은 이미지 파일명을 옵션 번호와 일치하게 제작하고, \[파일찾기]를 클릭해 \
