# 판매정보

<figure><img src="/images/product/add-list/add/sale-info/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FLJoYe8Z2toFcb65EPN3w%2F%EC%83%81%ED%92%88%EA%B4%80%EB%A6%AC_%EC%83%81%ED%92%88%EB%93%B1%EB%A1%9D_%ED%8C%90%EB%A7%A4%EC%A0%95%EB%B3%B41.png" alt=""/><figcaption><p>&#x3C;판매정보></p></figcaption></figure>

**① 예약판매**\
설정한 예약판매 기간 동안에만 상품 주문을 예약을 받고, 설정한 예약재고 수량만 판매할 수 있는 예약판매 기능입니다. 등록하는 상품을 예약판매로 판매하려면 '설정함' 을 선택해 주세요.

* 예약판매 기간 : 예약판매 시작일과 종료일을 설정합니다.\
  📍 예약판매 시작일이 경과하면, 예약판매 설정 항목의 변경 및 예약판매 시작일 수정이 불가해요.
* 예약배송 시작일 : 예약판매 주문 건의 출고일자인 예약배송 시작일을 설정합니다.
* 예약재고 수량 : 예약판매 대상 상품의 재고 수량을 입력합니다. \
  📍 예약재고 수량은 상품 실 재고수량과 무관합니다. \
  **📍** 예약판매 기간 동안 재고가 모두 차감되어 예약재고 수량이 '0'으로 처리되면, 해당 상품은 '품절상태'로 변경됩니다.\
  📍 상품 실 재고수량이 '0' 으로 처리되어 있어도, 예약재고 수량이 존재하면 예약판매 진행이 가능해요.

<Callout type="warning">
예약판매에서 설정한 예약배송 시작일이 도래하면 예약판매  주문 건이 '주문관리 입금대기/결제완료 주문' 메뉴로 이동 됩니다.&#x20;
</Callout>

**② 판매기간**\
등록하는 상품의 판매기간을 설정합니다.

* 상시판매 : 상품의 판매기간에 관계없이 상품의 재고수량만큼 상품을 판매합니다.\
  📍 상시판매의 판매 시작일은 '상품등록일-00:00', 판매 종료일은 '2999-12-31 23:59'로 자동 설정돼요.
* 기간지정 판매 : 판매 시작일과 종료일을 설정하여 설정한 기간동안 상품의 재고수량만큼 상품을 판매합니다. <br>

**③ 판매수수료**\
판매방식이 '위탁'인 상품에 설정하는 항목으로, 파트너사와 사전에 협의된 수수료를 설정하여 정산 시 활용합니다.

* 상품 수수료 : 등록하는 상품에 개별 수수료를 적용할 수 있습니다. \
  0\~99% 사이 값으로 소수점 첫째자리까지 입력 가능합니다.&#x20;
* 카테고리 수수료 : 등록하는 상품에 설정한 표준카테고리 수수료가 적용됩니다. 표준카테고리 수수료는 [상품관리 > 표준카테고리 수수료관리 ](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/product/category-commissionrate)메뉴에서 설정할 수 있습니다.
* 파트너 수수료 : 파트너사에 설정된 수수료가 적용되며, 파트너사 수수료 설정은 [파트너관리 > 파트너 조회 > 파트너 수정](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/partner/list) 메뉴의 거래쇼핑몰 항목에서 가능합니다.&#x20;
* 공급가 입력 : 등록하는 상품의 판매수수료를 수수료율이 아닌 공급가로 입력해 적용할 수 있습니다.

<Callout type="warning">
'사입' 판매방식 상품은 판매수수료를 설정하지 않습니다.
</Callout>

**④ 판매가**\
할인이 적용되지 않은 상품의 판매가(판매금액)를 입력합니다.

* 가격대체문구<img src="/images/product/add-list/add/sale-info/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FfCtrqbCqqHtkUFZ6PsUe%2FAPI1.png" alt="" data-size="line"/>: 상품 판매상태가 '판매중지'인 경우 상품의 판매를 대신하는 안내 문구로, '사용'으로 선택하면 가격대체 문구 입력란이 노출됩니다.
* 단위별 가격 등록 : 등록하는 상품의 단위 정보 및 판매가의 설정이 필요한 경우, 단위별 가격 등록을 설정합니다.\
  📍 예) 100g 당 1,000원<br>

**⑤ 즉시할인**\
등록하는 상품의 판매가에 할인을 설정할 수 있으며, 할인이 적용되는 기간 설정도 가능합니다.

* 정액 (원) 할인 : 할인금액을 직접 입력하는 즉시할인 방식
* 정률 (%) 할인 : 할인률을 입력하는 즉시할인 방식\
  📍 정률 (%) 할인 설정 시, 원 단위는 절사되어 적용돼요.<br>

**⑥ 즉시할인가**\
판매가에서 즉시할인을 차감한 금액이 즉시할인가로 자동 산정되어 표시됩니다.\
📍 즉시할인가 : 즉시할인이 적용된 금액 (= 판매가 - 즉시할인금액)으로 상품이 판매돼요.<br>

**⑦ 공급가**\
판매방식이 '위탁'인 상품에서 설정한 판매수수료에 따라 공급가가 자동 산정되어 표시됩니다. &#x20;

<Callout type="warning">
판매방식이 '사입'인 상품은 해당 항목명이 '매입가'로 변경됩니다.\
매입가는실제 상품을 매입한 금액을 입력하여 내부 관리용으로 활용합니다.
</Callout>

**⑧ 개별 적립금 설정**\
등록하는 상품을 구매하면 지급되는 상품별 구매 적립금을 '%' 단위의 적립률로 설정할 수 있습니다. \
' 설정함'을 선택하고 적립률을 입력합니다.

<Callout type="danger">
상품 등록 시 상품 개별 적립금을 설정하지 않으면, 쇼핑몰에 설정된 상품 구매 - 기본 적립률이 적용되어 적립금이 지급됩니다. \
( [운영관리 > 적립금 설정 관리 > 적립금 지급 설정](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/management/accumulation-setting#undefined-1) > 구매 적립금 > 상품 구매 적립금 지급 > 기본 적립률)

상품 개별 적립률과 기본 적립률이 동시에 설정되어 있다면, 상품 개별 적립률이 적용되어 적립금이 지급되는 점을 유의해 주세요.&#x20;
</Callout>

**⑨ 적립금 사용**\
회원이 보유한 적립금을 등록하는 상품 결제에 사용하는 적립금 사용 여부를 설정할 수 있습니다.

<Callout type="danger">
적립금 사용이 가능한 상품과 불가한 상품을 동시 주문하는 경우, **불가한 상품의 금액을 제외한 만큼만** 적립금 사용이 가능한 점을 참고해 주세요.
</Callout>

**⑩ 프로모션**\
등록하는 상품에 할인 및 쿠폰, 사은품 등의 혜택을 적용하는 프로모션 설정입니다. \
'가능'을 선택하고 상품에 적용할 프로모션 종류를 선택합니다. \
📍 추가할인, 쿠폰, 사은품을 일괄 선택하면, 선택한 모든 프로모션의 적용이 가능합니다.

<Callout type="danger">
파트너사 계약 시 프로모션 동의여부를 '동의 안 함'으로 설정한 경우, 등록하는 상품의 프로모션 설정에 관계없이 해당 상품은 프로모션 대상에서 제외됩니다.&#x20;
</Callout>

**⑪ 재고수량 (판매 가능 재고)**\
등록하는 상품의 판매 가능한 실 재고수량을 입력합니다. \
옵션이 있는 상품 등록 시, 옵션 별 재고수량이 해당 재고수량과 자동 합산되어 표시됩니다.

**⑫ 재입고 알림 사용 설정**\
등록하는 상품이 품절된 경우, 해당 상품의 재입고 알림 사용 여부를 설정할 수 있습니다.

<Callout type="warning">
**재입고 알림 '사용함' 설정 상품이 재입고 되더라도, 알림이 자동 발송처리 되지 않습니다.**

[**상품관리 > 상품 재입고 알림 신청 관리**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/product/restock) **메뉴에서, 고객이 신청한 '재입고 알림 신청 상품'을 확인하고 별도 알림을 발송해야 합니다.**\
**쇼핑몰 화면에서 고객이 상품 재입고 알림 신청 기능을 사용할 수 있도록** <img src="/images/product/add-list/add/sale-info/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FoTe3FZtVw7Vx0YwMW055%2Fapi2.png" alt="" data-size="line"/>**구현이 필요합니다.**
</Callout>

**⑬ 최소구매수량**\
최소 구매 수량은 고객이 구매해야 하는 해당 상품의 최소 개수 입니다.\
최소 구매 수량은 2개 이상 부터 설정이 가능합니다.

**⑭ 최대구매수량**\
최대 구매 수량은 고객이 구매해야 하는 해당 상품의 최대 개수 입니다.\
회원 아이디 1인 / 1회 / 기간제한 총 3개의설정 조건중에서 선택합니다.\
📍 비회원은 1회 최대 구매 수량 제한 설정만 적용이 가능합니다.

**⑮ 옵션 사용 여부**\
등록하는 상품의 옵션이 있는 경우 '사용함'을 선택합니다.

**⑯** [**옵션 (옵션 등록)**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/product/add-list/add/sale-info/option)\
\[옵션 등록] 버튼을 클릭하면 옵션 등록 및 관리가 가능한 '옵션 등록' 창이 노출되고,\
옵션 등록을 완료하면 \[옵션 변경]과 \[옵션 초기화] 버튼이 노출됩니다.

* \[옵션 변경] 버튼을 클릭하면 등록한 옵션 정보의 수정이 가능한 '[옵션 수정](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/product/add-list/add/option#undefined-10)' 창이 노출됩니다.\
  📍옵션 변경 : 등록한 옵션 정보의 추가 / 삭제 / 변경 등 옵션 수정 진행
* \[옵션 초기화] 버튼을 클릭하면 등록한 옵션 정보가 모두 초기화 처리됩니다.

**⑰ 상품정보고시**\
등록하는 상품이 해당하는 상품군에 대한 필수 정보 항목을 입력하여, 고객에게 안내하는 상품정보고시 기능입니다. '사용함'을 선택하고 \[등록] 버튼을 클릭하면, '상품정보고시' 창이 노출됩니다.

<details>

<summary>🤚 <strong>상품정보고시</strong></summary>

전자상거래법에 따라 전자상거래를 통해 많이 거래되는 상품군을 약 30여개로 분류한 품목별 상품 정보를 고객에게 의무적으로 제공해야 하는 공정거래위원회가 공고한 법적 규정사항입니다.

</details>

**⑱ 인증정보**\
등록하는 상품이 안전관리대상 및 어린이제품 등과 같이 인증대상 상품인 경우 인증관련 정보를 입력하여 고객에게 안내하는 인증정보 기능입니다. '인증대상'을 선택하고 \[등록] 버튼을 클릭하면, '인증정보' 창이 노출됩니다.\
📍 인증대상 상품에 해당하는 표준카테고리와 매칭되어 있어야, 인증대상 \[등록] 버튼이 활성화되고, '인증    \
&#x20;   정보' 창이 노출됩니다.<br>

**⑲ 원산지**\
등록하는 상품의 원산지를 설정할 수 있습니다.\
📍 등록하는 상품에  복수 원산지가  존재하는 경우, '원산지 다른 상품 함께 등록' 을 체크해 주세요.

**⑳ 매입처 상품명**\
내부 관리용으로 사용되는 상품의 매입처 상품 이름을 입력합니다.

**㉑ 제조일자 / 유효일자**\
등록하는 상품의 제조일자와 유효일자를 설정할 수 있습니다.

**㉒ 부가세**\
등록하는 상품이 해당하는 부가세 종류를 선택합니다.

* 과세상품 : 부가세가 포함된 부가세 신고대상 상품
* 영세상품 : 매입세액 환급대상 상품
* 면세상품 : 부가세 신고대상에서 제외되는 상품<br>

**㉓ HS CODE**\
HS CODE는 국제무역 상품의 관세분류 및 수출입 통계 목적을 위해 품목에 숫자 코드를 부여하여 분류한 값으로, 세계관세기구(WCO)에서 관리하고 있습니다.

**㉔ 주문환불**\
등록하는 상품이 주문 이후,  취소/교환/반품 신청이 불가하도록 개별 설정할 수 있습니다.\
'환불불가'를 선택하면, 취소/교환/반품 클레임의 환불불가 개별 설정이 노출됩니다.

**㉕ 아이콘**\
상시 노출 또는 기간을 지정하여 아이콘을 노출할 수 있습니다.&#x20;

* 아이콘은 [상품관리 > 아이콘 관리](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/product/sticker) 메뉴에 등록된 아이콘만 설정이 가능합니다.&#x20;
* 상시 노출 아이콘과 기간 노출 아이콘을 모두 사용하는 경우, 상시 노출 아이콘 >>>>> 기간노출 아이콘 순으로 노출됩니다.

**㉖ 추가상품 전용**\
추가상품 전용으로만 사용할 추가상품 전용 상품을 설정할 수 있습니다.\
'사용함'을 선택하면, 추가상품과 연관된 아래 항목값이 자동으로 설정됩니다.

* 기본정보 > URL로만 접근 : 사용함
* 판매정보 > 추가상품 설정 : 사용 안 함\
  📍 위 상품은 반드시 본 상품과 함께 구매만 가능하며, 쇼핑몰에서 단독 검색 시 노출이 제한됩니다.

**㉗ 추가상품 설정**\
본 상품 구매 시, 함께 구매 가능한 추가상품을 설정할 수 있습니다.\
'설정함'을 선택하면, 선택한 추가상품이 본 상품의 상세 페이지에 같이 노출됩니다.

* 추가상품 표시명 : 쇼핑몰에 노출할 추가상품 선택 영역의 표시명을 입력할 수 있습니다.
* 추가상품 선택 : 기등록된 상품 중에서 추가상품으로 노출할 상품을 선택할 수 있습니다.
  * 상품 선택이 완료되더라도, 상품의 실제 노출여부에 따라 쇼핑몰에 노출되게 됩니다.\
    📍노출 가능한 상품이더라도, 본 상품과 같이 결제 가능한 수단이 존재하지 않는다면 미노출됩니다.
  * 선택한 상품의 상품의 진열순서를 조정할 수 있고, 최초에는 상품 등록일 오름차순으로 정렬됩니다.<br>
