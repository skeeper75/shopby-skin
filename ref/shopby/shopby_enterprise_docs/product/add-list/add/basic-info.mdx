# 기본정보

**① 쇼핑몰**\
상품을 등록할 쇼핑몰을 선택합니다.

**② 상품군**\
배송 진행에 따른 상품군을 선택합니다.

* 배송상품군 : 렌탈상품을 포함하여 배송이 진행되는 모든 상품군
* 서비스상품군 : 온라인 및 오프라인에서 사용 가능한 티켓 및 쿠폰 상품 등과 같은 배송 미진행 상품군<br>

**③ 상품유형**\
판매 유형에 따른 상품유형을 '일반상품'과 '렌탈상품' 중에서 선택합니다.\
✅ 렌탈상품 판매<img src="/images/product/add-list/add/basic-info/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FfCtrqbCqqHtkUFZ6PsUe%2FAPI1.png" alt="" data-size="line"/> : 고가의 상품이나 장기 분납서비스 등을 구독 결제 방식으로 판매하는 유형입니다. \
&#x20;     렌탈상품은 기본설정 항목이 일부 추가됩니다.

<details>

<summary>🤚 <strong>렌탈상품 판매유형</strong>  </summary>

* **렌탈상품 기본설정 추가 항목** \
  제조사명 : 15자 이내 한글, 영문 대/소문자, 숫자만 입력 가능\
  제품모델명 : 20자 이내 한글, 영문 대/소문자, 숫자만 입력 가능\
  제품일련번호 : 20자 이내 한글, 영문 대/소문자, 숫자만 입력 가능\
  📍 제조사명, 제품모델명, 제품일련번호는 렌탈 PG 서비스사 '이니렌탈' 에서 사용되는 \
  &#x20;     항목으로 쇼핑몰에 별도 노출되지 않습니다.\ <br>
* **렌탈상품 PG 서비스 신청**\
  렌탈상품을 판매하려면 렌탈결제 PG 서비스사와 별도의 렌탈결제 서비스 계약을 진행하고, 쇼핑몰에 별도 '**렌탈결제 Key값**' 등을 세팅해야 합니다.\
  📍 현재 정기결제가 가능한 PG 서비스는 '이니렌탈'입니다.\
  📍 이니렌탈 PG는 이니시스 PG 사용여부와 관계없이 사용 가능해요.\ <br>
* **렌탈상품 PG 서비스 계약 및 세팅 프로세스** \
  📍 PG사 렌탈결제 계약 진행 → \
  &#x20;     PG사로부터 전달받은 '**렌탈결제 Key값**' 을 NHN커머스 1:1 문의 통해 접수 → \
  &#x20;    '렌탈결제 Key값' 세팅 완료(영업일 기준 2\~3일 소요)

**④ 파트너사**

* 쇼핑몰 자체 상품 : 등록하는 상품의 (판매)주체가 쇼핑몰 자체인 경우 선택합니다.
* 파트너사 : 등록하는 상품의 (판매)주체가 파트너사인 경우 선택합니다. \
  📍 [파트너 관리 > 파트너 조회](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/partner/list) 메뉴에서 shop by 등록여부가 '등록완료' 상태인 파트너사가 조회되므로, \
  &#x20;   사전에 파트너사 계약으로 등록을완료해 주세요. <br>

**⑤ 판매방식**

* 사입 : 쇼핑몰 운영자가 상품을 직접 매입하거나 공급하여 판매하는 방식입니다.
* 위탁 : 쇼핑몰에 입점한 파트너사의 상품을 공급받아 위탁하여 판매하는 방식입니다. \
  위탁 판매방식 선택 시 수수료 쉐어 등을 위한 판매수수료 설정이 가능해요.<br>

**⑥ 담당자**\
등록하는 상품의 담당자명을 검색하여 입력합니다.\
📍 상품 담당자는 [설정 > 관리정책 운영자 관리](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/admin) 메뉴에 운영자로 등록되어 있어야 검색이 가능합니다.

**⑦ 표준카테고리**\
네이버쇼핑 EP 등 외부 연동 카테고리 정보로 활용되는 표준카테고리를 찾기 또는 입력(검색)으로 설정해 주세요.

<Callout type="warning">
표준카테고리는 쇼핑몰에 노출되지 않으며, 별도의 추가/수정/ 삭제 등이 불가합니다.\
[표준카테고리 수수료관리](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/product/category-commissionrate) 메뉴에서 표준카테고리 별 수수료를 설정할 수 있습니다.
</Callout>

**⑧ 전시카테고리**\
쇼핑몰 메인 페이지에 노출되는 카테고리인 전시카테고리를 찾기 또는 입력(검색)으로 설정해 주세요.\
📍 전시카테고리는 [ 전시관리 > 전시카테고리 관리 ](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/appearance/display-category)메뉴에 사전에 등록되어 있어야 등록하는 상품에 설정이 가능합니다.\
📍 한 개의 상품에 다수의 전시카테고리와 중복매칭도 가능합니다.<br>

**⑨ 플랫폼별 노출설정**\
등록하는 상품을 노출할 플랫폼을 설정할 수 있습니다. \
'설정'을 선택하고 플랫폼을 선택합니다.

**⑩ 회원등급 노출 설정**\
등록하는 상품을 노출할 회원등급을 별도 설정할 수 있습니다. \
전체 회원등급 및 특정 회원등급의 선택이 가능합니다.

**⑪ 회원그룹 노출 설정**\
등록하는 상품을 노출할 회원그룹을 별도 설정할 수 있습니다. \
전체 회원그룹 및 특정 회원그룹의 선택이 가능합니다.

**⑫ URL로만 접근**\
'사용함'으로 설정하면, 등록하는 상품의 전시 카테고리와 쇼핑몰 내 검색 결과에 노출되지 않으며, \
해당 상품 상세 페이지 URL 주소로만 상품의 노출(조회)가 가능합니다.

**⑬ 상품명**\
등록하는 상품의 이름을 입력합니&#xB2E4;**.**\
상품명은 최대 255자까지 입력이 가능하고, 입력 문자에 제한이 없습니다.\
📍 입력한 문자 중 일부는 PG사 정책에 따라 통신과정에서 제거될 수 있는 점을 유의해 주세요.\
📍 '영문상품명 등록' 체크박스를 선택하면 입력란이 노출돼요.

**⑭ 홍보문구**\
등록하는 상품의 추가정보 및 이벤트용도 등으로 활용하는 홍보문구는 '사용함'을 선택하면, 입력란이 노출됩니다.  노출기간 설정을 체크하면 홍보문구의 노출기간 설정도 가능해요.\
📍 홍보문구는 상세 페이지 및 카테고리 페이지 등에서 노출됩니다. (스킨 별 상이)

**⑮ 브랜드**\
등록하는 상품의 브랜드를 설정할 수 있습니다. 브랜드 찾기 또는 입력(검색)으로 설정해 주세요.

**⑯ 결제수단 설정**\
등록하는 상품에 사용 가능 및 불가능한 결제수단과 비회원 구매 불가를 개별 설정할 수 있습니다.

<Callout type="danger">
네이버페이 규칙에 의해 판매가 허락되지 않는 '취급제한 상품'과 현행 법령상 매매가 금지된 '취급불가 상품'은 네이버페이 결제수단으로 결제할 수 없습니다. \
쇼핑몰에 네이버페이 결제 수단이 연동되어 있는 경우, 해당 상품 등록 시 '네이버페이 결제 불가'를 반드시 설정해야 합니다.
</Callout>

<details>

<summary>🤚 <strong>네이버페이 결제수단에 따른 결제제한 조건</strong></summary>

* **네이버페이 주문형**\ <mark style="color:red;">**즉시할인가(판매가-즉시할인금액)+옵션가 = 금액**</mark>이 <mark style="color:red;">**즉시할인가에 -50%를 초과하는 상품**</mark>과\ <mark style="color:red;">**서비스 상품군**</mark>\
  예) 판매가 10,000원 / 즉시할인금액 2,000원 ⇀ 즉시할인가 8,000원\
  ⓐ 옵션가 -4,500원 \
  &#x20;    즉시할인가 8,000원 + 옵션가 -4,500원 = 3,500원\
  &#x20;    결제금액 3,500이 즉시할인가 8,000원의 -50%를 초과하므로 '<mark style="background-color:green;">결제불가능</mark>'\
  ⓑ 옵션가 -3,500원\
  &#x20;    즉시할인가 8,000원 + 옵션가 -3,500원 = 4,500원\
  &#x20;    결제금액 4,500이 즉시할인가 8,000원의 -50%를 초과하지 않으므로 '<mark style="background-color:orange;">결제가능</mark>'\ <br>
* **네이버페이 결제형**\ <mark style="color:red;">**총 결제금액 100원 미만**</mark>

</details>

**⑰ 성인인증**\
등록하는 상품이 성인인증 대상 상품인 경우 '사용함'을 설정합니다. \
'사용함' 설정 시 해당 상품은 성인인증이 완료된 회원에게만 구매가 가능하도록 상품 이미지가 노출됩니다.\
📍 쇼핑몰에 회원 본인인증 사용 설정 및  본인인증 부가서비스가 가입이 되어 있어야 합니다.\
&#x20;     (쇼핑몰 회원 인증 설정 : [서비스관리 >  쇼핑몰관리 > 쇼핑몰 수정](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/mall))\
📍 표준카테고리를 '성인 카테고리'로 선택하면, 성인인증 항목이 ‘사용함’ 으로 자동 설정됩니다.

**⑱ 장바구니**\
등록하는 상품의 장바구니 담기 기능을 제한할 수 있습니다.\
'불가능'으로 설정하면, 해당 상품은 장바구니 버튼(기능)이 미노출 됩니다.

**⑲ 검색어**\
쇼핑몰 검색창에서 상품 검색 키워드로 활용되는 상품 관련 검색어를 입력합니다.

**⑳ 단축 URL 사용여부**\
'사용함'으로 설정하면, 등록하는 상품의 상세 페이지 단축 URL이 생성되며, 단축 URL은 [상품관리  > 상품정보 조회/수정](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/product/list) 메뉴의 상품 리스트에서도 확인이 가능합니다 .\
📍 쇼핑몰 운영방식이 '헤드리스' 인 경우 단축 URL 사용이 불가합니다. (해당 설정 항목 '사용 안 함' 처리)

<Callout type="danger">
대표 도메인이 변경되어도, 단축 URL은 자동으로 변경되지 않는 점을 유의해주세요.
</Callout>

\
**㉑ 관련상품 설정**\
등록하는 상품과 관련된 상품을 해당 상품 상세페이지 상단 관련상품영역에 노출할 수 있습니다.

* 전시카테고리  : 해당 상품과 동일한 전시카테고리에 매칭된 상품을 관련상품으로 노출할 수 있으며, \
  자동진열(상품 최신등록순, 상품 판매량순, 상품후기 순 중에서 선택)로 진열순서 설정이 가능합니다.
* 개별상품 : \[상품조회] 버튼을 클릭 해 '상품조회' 창에서 관련상품을 개별 설정합니다. \
  자동진열(상품 최신등록순, 상품 판매량순, 상품후기 순 중에서 선택)과 수동진열 중에서 진열순서를 선택하여 설정합니다.
