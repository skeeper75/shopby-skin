# 표준카테고리수수료 관리

<Callout type="info">
<mark style="background-color:red;">**표준카테고리는 네이버쇼핑 등 외부 서비스에 상품 정보를 연동할 수 있도록 미리 분류(설정)해 놓은 카테고리입니다.**</mark>

**표준카테고리의 카테고리 번호 및 카테고리명은 추가/수정/삭제가 불가합니다.**

**표준카테고리에 수수료를 개별 설정하는 기능이 제공되며, 특정 카테고리는 성인카테고리로 설정이 되어 있습니다.**&#x20;
</Callout>

## 표준카테고리수수료 관리

**① 표준카테고리 목록**\
샵바이에서 제공하는 표준카테고리 전체가 목록으로 확인됩니다.

* 표준카테고리는 4depth로 구성되어  있습니다.
* 표준카테고리별 최하위 depth에서 개별 수수료 설정이 가능합니다.

**② 수수료**\
표준카테고리 수수료를 설정할 수 있습니다. \
설정 된 수수료는 상품 등록 시 판매수수료를 카테고리 수수료로 설정한 경우 반영됩니다.\
📍기본값은 0.0%로 설정되어 있습니다.

**③ 성인카테고리**\
성인인증 대상 상품에 해당하는 카데고리는 자동으로 '설정'이 체크되어 있습니다.\
📍예 :  식품 > 주류 > 전통주

**④** [**엑셀다운로드**](#undefined-1)\
\[엑셀다운로드] 버튼을 클릭하면,  표준카테고리 목록의 상세설정 항목이 엑셀파일로 다운로드 됩니다.

**⑤** \[변경사항 저장] 버튼을 클릭해 표준카테고리 수수료 상세 설정을 완료합니다.

<Callout type="danger">
**상품 등록 시 표준카테고리를 '음반'과 'DVD'로 선택하면, 상품 정보 설정 항목인 '제조일'이 '발매일'로 변경됩니다.**

**성인 카테고리로 설정된 상품은 비회원과 성인 미인증 회원에게는 결제가 제한됩니다.** \
**다만 상품 리스트와 상품 상세 페이지에서 해당 상품의 접근이 가능한 점을 참고해 주세요.**
</Callout>
