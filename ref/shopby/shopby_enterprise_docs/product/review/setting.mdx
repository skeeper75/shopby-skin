# 상품후기 설정

<Callout type="info">
**상품후기 게시판의 상세 정보를 관리하고, 상품후기 위젯 출력 및 안내문구 노출 등의 추가설정이 가능한 상품후기 설정 메뉴입니다.**
</Callout>

## 상품후기 게시판 상세설정

<figure><img src="/images/product/review/setting/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FK0wCjEMWI3BVgJYhSBNt%2F%EC%83%81%ED%92%88%ED%9B%84%EA%B8%B0%20%EC%84%A4%EC%A0%95_%EA%B2%8C%EC%8B%9C%ED%8C%90%EC%83%81%EC%84%B8%EC%84%A4%EC%A0%95%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;상품후기 게시판 상세설정></p></figcaption></figure>

**① 게시판명** \
상품후기 게시판 이름을 입력합니다. 게시판 이름은 쇼핑몰에 노출됩니다.

**② 게시판 설명** \
내부 관리 용도인 게시판 설명을 입력합니다.

**③ 첨부파일** \
상품후기 게시판의 첨부파일 사용여부를 설정해 주세요.

**④ 댓글 기능**\
상품후기 게시판의 댓글 작성 기능 사용여부를 설정해 주세요.

**⑤ 적립금 사용여부**\
[**운영관리 > 적립금 설정 관리**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/management/accumulation-setting) 메뉴에서 설정한 상품후기 적립금 지급 설정 사항이 확인됩니다.<br>

## 상품후기 게시판 추가설정

<figure><img src="/images/product/review/setting/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FBe6FHnEGqedy2NR0W8Pq%2F%EC%83%81%ED%92%88%ED%9B%84%EA%B8%B0%20%EC%84%A4%EC%A0%95_%EA%B2%8C%EC%8B%9C%ED%8C%90%EC%B6%94%EA%B0%80%EC%84%A4%EC%A0%95%20%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;상품후기 게시판 추가설정></p></figcaption></figure>

**① 페이지당 게시물 수**\
유형별 상품후기 게시판 페이지에 노출되는 상품후기 게시글 개수를 설정할 수 있습니다.\
📍 4개, 8개, 12개, 16개, 20개 중에서 페이지에 노출되는 상품후기 게시글 개수를 선택합니다.

<details>

<summary><strong>🤚유형별 상품후기 게시판</strong></summary>

* 전체 상품후기 : 쇼핑몰에 등록한 상품에 작성된 상품후기 전체가 노출됩니다.
* 포토 상품후기 : 이미지를 첨부하여 작성한 포토 상품후기만 노출됩니다.
* 상품기준 상품후기 : 상품 별 기준, 상품후기가 노출됩니다.

</details>

**② 위젯설정**

* 전체 상품후기, 포토 상품후기, 상품기준 상품후기를 위젯으로 생성하여 유형별 상품후기 게시판에 노출할 수 있습니다.
* 위젯 사용여부 설정과 위젯으로 출력할 상품후기 개수를 설정합니다.\
  📍 1개 \~  20개 중에서 위젯 출력 상품후기 개수를 선택합니다.
* \[미리보기] 버튼을 클릭하면 유형별 상품후기 게시판의 '위젯 미리보기' 창이 노출됩니다.

**③ 상품후기 혜택 안내 문구**\
상품후기 작성 영역에서 고객에게 '혜택안내'로 노출되는 상품후기 작성 혜택 사항을 입력합니다.

**④ 적립금 혜택 안내문구**\
고객이 상품후기 작성 시점에 작성한 상품후기 글자수가 적립금 지급이 가능한 글자 수 미만으로 입력된 경우 안내되는 문구를 입력합니다.

**⑤ 상품후기 작성 안내문구**\
상품후기 작성 영역에 기본으로 표시되는 안내 문구를 입력해 주세요.

<Callout type="warning">
상품후기 혜택, 적립금 혜택, 상품후기 작성 안내문구 모두 공백포함 최대 250자까지 작성이이 가능합니다.
</Callout>

**⑥ 포토 상품후기 디스플레이 유형 설정**\
이미지를 첨부한 포토 상품후기 노출의디스플레이 유형을 두 가지 타입 중에서 설정할 수 있습니다.

**⑦ 포토 상품후기 모아보기**&#x20;

* 작성된 상품후기 중에서 포토 상품후기만 모아보기를 할 수 있도록 설정할 수 있습니다.
* 포토 상품후기가 없는 경우 포토 상품후기 작성을 유도할 수 있는 안내 문구 등을 오른쪽 입력란에 작성하여 노출하세요.

**⑧ 후기 추천**\
다른 회원이 작성한 상품후기를 추천할 수 있는 기능으로 ‘사용함’ 을 설정하면, 상품후기 페이지 정렬 기준에 ‘추천순’이 추가됩니다.
