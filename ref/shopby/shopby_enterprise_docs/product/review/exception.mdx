# 상품후기 작성 예외상품 설정

<Callout type="info">
**특정 상품의 상품후기 작성을 제한하는 기능으로, 상품후기 작성을 원하지 않는 상품은 예외 상품으로 등록해야 합니다.**

**상품후기 작성 예외상품은 상품후기 작성이 불가하며, 만약 설정하기 전 작성된 상품후기가 있는 경우 상품후기 전시상태에 따라 쇼핑몰에 상품후기가 노출됩니다.**
</Callout>

## 상품후기 작성 예외상품 설정

**① 상품등록**\
상품후기 작성이 불가한 '상품후기 작성 예외상품'을 등록합니다.

* 상품조회 등록 :  \[상품조회] 버튼을 클릭해 나타나는 '상품조회' 창에서 상품을 조회하고 등록합니다.
* 상품번호 직접 등록 : 상품번호를  입력하고, \[상품추가] 버튼을 클릭해 예외상품을 등록합니다.
* 상품번호 일괄 등록 : \[엑셀양식다운로드] 버튼을 클릭하면 상품번호 일괄업로드 양식이 다운로드 됩니다. 해당 양식에 예외상품으로 등록할 상품번호를 입력하여 \[엑셀 일괄등록] 할 수 있습니다.

**② 선택상품 삭제**\
상품후기 예외상품으로 설정한 상품을 삭제하려면, \[선택상품 삭제] 버튼 및 \[삭제]를 클릭해 삭제합니다.

**③** \[저장] 버튼을 클릭해 상품평 작성 예외상품 설을 완료합니다.
