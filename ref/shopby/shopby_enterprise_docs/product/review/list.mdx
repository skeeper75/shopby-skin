# 상품후기

<Callout type="info">
**구매자가 등록한 상품평을 조회하고, 상품평에 댓글을 작성할 수 있는 상품후기 메뉴입니다.**\
**(단, 상품후기는 회원만 등록이 가능합니다.)**

**상품후기는 우수 상품후기와 일반 상품후기로 구분하여 관리가 가능하고, 우수 상품후기는 상품후기 리스트에서 베스트순 정렬 시 상위 노출됩니다.**

**구매내역이 없는 고객은 상품평 등록이 불가합니다.**
</Callout>

## 상품후기 검색조건

<figure><img src="/images/product/review/list/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FzloPUDnuqMR6aJNWE2RK%2F%EC%83%81%ED%92%88%ED%8F%89%EA%B4%80%EB%A6%AC%20%EA%B2%80%EC%83%89%EC%A1%B0%EA%B1%B4.jpg" alt=""/><figcaption><p>&#x3C;상품후기 검색조건></p></figcaption></figure>

**① 쇼핑몰**\
상품후기를 조회할 쇼핑몰을 선택합니다.&#x20;

**② 등록일**\
&#x20;상품후기를 등록한 일자를 기준으 상품후기를 검색합니다.

**③ 검색어**\
상품후기 작성자명, 상번호,  상품명, 상품후기번호, 파트너사 등의 검색어로 상품후기를 검색합니다.\
📍작성자명은 전체를 입력해야 검색이 가능합니다.

**④ 상세검색**\
상품후기가 쇼핑몰에 노출된 전시상태 기준, 상품후기 유형, 상품후기 구분, 평점 등의 다양한 검색값으로 상품후기를 검색합니다.

## 상품후기 검색결과 (상품후기 리스트)

<figure><img src="/images/product/review/list/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FI1iAAcDMSyaCBcXuurqR%2F%EC%83%81%ED%92%88%ED%8F%89%EA%B4%80%EB%A6%AC%EA%B2%80%EC%83%89%EA%B2%B0%EA%B3%BC%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;상품후기 검색결과 ></p></figcaption></figure>

**①** [**전시상태 변경**](#undefined-3)\
상품후기를 선택하고 \[전시상태 변경] 버튼을 클릭하면, 상품후기 노출여부를 변경(설정)할 수 있는 '전시상태 변경' 창이 노출됩니다.

<details>

<summary><strong>🤚전시상태</strong></summary>

* **전시 중**\
  회원이 작성한 상품후기가 쇼핑몰에 노출되어 다른 회원이 해당 상품후기를 확인할 수 있는 상태입니다.
* **블라인드**\
  전시 중인 상품후기를 운영자가 미노출 처리한 상태입니다.\
  📍블라인드 상태 상품후기는 쇼핑몰에서 미노출되며, 해당 상품후기를 작성한 회원의 \
  &#x20;    마이페이지 > 상품후기 리스트에서는 확인됩니다.
* **삭제**\
  회원이 상품후기를 직접 삭제 처리한 상태입니다.\
  📍운영자는 상품후기 삭제가 불가능 점을 참고해 주세요.

</details>

**②** [**상품후기 구분 변경**](#undefined-4)\
회원이 작성한 상품후기를 ‘일반 상품후기(기본값)’ 또는 ‘우수 상품후기’로 설정할 수 있습니다.

**③** [**상품후기번호**](#undefined-5)\
\[상품후기번호]를 클릭하면 해당 상품후기의 상세세정보가 확인되는 '상품후기 상세정보' 창이 노출됩니다.

**④ 유형**\
상품후기 작성 시 이미지 첨부 파일 여부에 따라 '포토'와 '일반'으로 구분됩니다.

**⑤ 평점**\
회원이 상품후기 작성 시 체크한 상품의 점수입니다.

**⑥ 댓글**\
해당 상품후기에 작성된 댓글 개수가 노출됩니다.

**⑦ 추천**\
해당 상품후기를 추천한 추천 건수가 노출됩니다.

**⑧ 신고**\
해당 상품후기를 신고한 신고 건수가 노출됩니다.

**⑨** [**변경이력**](#undefined-6)\
\[보기]를 클릭하면 상품후기 설정 항목을 변경한 이력정보가 확인됩니다.

***

#### <mark style="color:blue;">**■ 전시상태 변경**</mark>

* 쇼핑몰에 상품후기가 노출되는 '전시 중' 상태와, 미노출되는 '블라인드' 상태로 전시상태를 변경(설정) 할 수 있습니다.
* \[저장] 버튼을 클릭 해 전시상태 변경을 완료해 주세요.

#### <mark style="color:blue;">**■ 상품후기 상세정보**</mark>

* 상품후기번호, 상품후기작성자, 상품후기 등록일, 상품후기 유형과 평점 등 회원이 상품후기 작성 시 설정한 상품후기 상세정보를 확인할 수 있습니다.
* 회원이 작성한 상품후기의 자세한 내용과 포토후기 작성 시 첨부한 이미지도 확인됩니다.&#x20;
* 해당 상품후기의 상품후기 구분과 전시상태를 변경(설정)할 수 있습니다.
* 상품후기 댓글 입력란에 댓글을 작성하고, 댓글 항목의 \[저장] 버튼을 클릭해 댓글을 저장합니다.\
  📍운영자가 작성한 댓글 이외에 추가 댓글이 작성된 경우 총 댓글 개수가  표기됩니다.
* 하단 \[저장] 버튼을 클릭 해 상품후기 구분 및전시상태 변경(설정), 댓글 입력 등을 최종 완료해주세요.
