# 상품등록

<Callout type="info">
**상품등록 유형에 따라 설정 항목이 상이하므로, 상품등록 유형을 반드시 확인해 주세요.**
</Callout>

## 상품등록 유형

<figure><img src="/images/product/add-list/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FxlO5XtEeyUztOA6LDetD%2F1.%20%EC%83%81%ED%92%88%EB%93%B1%EB%A1%9D_%EC%9C%A0%ED%98%95.png" alt=""/><figcaption></figcaption></figure>

**①** [**일반 상품**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/product/add-list/add)\
쇼핑몰에서 판매할 상품을 개별 등록 할 수 있으며 상품 등록 시 필요한 정보 기재 후 등록 가능합니다.

**②** [**세트 상품**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/product/add-list/setadd)\
쇼핑몰에 등록된 상품의 옵션을 조합하여 세트 상품으로 등록할 수 있습니다.

**③** [**대량 상품 등록**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/product/add-list/bulkadd)\
엑셀 양식 파일을 통해 상품을 대량으로 일괄 등록할 수 있습니다.
