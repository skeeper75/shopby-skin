# 상품문의 설정

<Callout type="info">
**상품문의 게시판의 상세 정보를 관리하는 상품문의 설정 메뉴입니다.**
</Callout>

## 상품문의 설정

<figure><img src="/images/product/inquiry/setting/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FZwDrfxzlps3M8ZVfP4Lh%2F%EC%83%81%ED%92%88%EB%AC%B8%EC%9D%98%EC%84%A4%EC%A0%95%EB%B3%B5%EC%82%AC.jpg" alt=""/><figcaption><p>&#x3C;상품문의 설정></p></figcaption></figure>

**① 게시판명** \
상품문의 게시판 이름을 입력합니다. 게시판 이름은 쇼핑몰에 노출됩니다.

**② 게시판 설명** \
내부 관리 용도인 게시판 설명을 입력합니다.

**③ 첨부파일** \
상품문의 게시판의 첨부파일 사용여부를 설정해 주세요.

**④** \[저장] 버튼을 클릭해 상품문의 게시판 설정을 완료합니다.
