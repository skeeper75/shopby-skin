# 아이콘 관리

<Callout type="info">
**이미지 및 텍스트 형식의 아이콘을 등록하고 관리하는 아이콘 관리 메뉴입니다.**

**상품에 아이콘을 적용하려면 해당 메뉴에서 아이콘을 먼저 등록해 주세요.**
</Callout>

## 아이콘 관리

**①** [**아이콘 등록**](#undefined-1)\
\[아이콘 등록] 버튼을 클릭하면 텍스트 또는 이미지 형식의 아이콘 등록이 가능한 '아이콘 등록' 창이 노출됩니다.

**② 선택삭제**\
등록한 아이콘을 선택하고 \[선택삭제] 버튼을 클릭해 아이콘을 삭제할수 있습니다. \
📍\[삭제]를 클릭해도 아이콘 삭제가 가능합니다.

**③ 아이콘 진열(노출)순서 조정**\
📍해당 진열순서는 상품 등록 > 판매 정보 > 아이콘 항목에 노출되는 순서입니다.

* 상/하 이동 아이콘으로 등록한 아이콘의 노출순서를 조정할 수 있습니다.
* 아이콘을 선택하고 진열(노출) 위치 순번을 입력한 다음, \[이동] 버튼을 클릭해 아이콘의 노출순서를\
  지정할 수 있습니다.

<Callout type="warning">
︿ (상) : 현재 위치보다 한 단계 위로 이동\
﹀ (하) : 현재 위치보다 한 단계 아래로 이동\
︽ (최상) : 진열순서 최상단으로 이동\
︾ (최하) : 진열순서 최하단으로 이동
</Callout>

**④** [**수정**](#undefined-2)\
해당 아이콘을 수정할 수 있는 '아이콘 수정' 창이 노출됩니다.

**⑤ 사용여부**\
해당 아이콘의 사용여부를 설정할 수 있습니다.\
📍적용상품이 있는 아이콘의 사용여부를 '사용 안함'으로 변경하면, 상품에 아이콘이 노출되지 않아요.

**⑥** [**적용상품**](#undefined-3)\
적용상품 건수를 클릭하면, 해당 아이콘이 적용된 상품 정보를 일괄 확인할 수 있는 '적용상품 보기' 창이 노출됩니다.

**⑦** \[저장] 버튼을 클릭해 아이콘 등록 및 수정 등 아이콘 관리를 완료해 주세요.

***

#### <mark style="color:blue;">■ 아이콘 등록</mark>

* 사용여부 : 해당 아이콘 사용여부를 설정합니다.
* 이미지 : 아이콘 이미지를 제작하고,  \[파일찾기] 버튼을 클릭해 아이콘 이미지를 업로드 합니다.\
  📍아이콘 이미지 제작 시, 사용 가능한 파일형식 및 용량 등을 반드시 사전에 확인해 주세요.
* 텍스트 : 아이콘으로 노출할 텍스트를 **아이콘 설정 입력란**에 20자 이내로 입력합니다.
* \[등록] 버튼을 클릭해 아이콘 등록을 완료합니다.

***

#### <mark style="color:blue;">■ 아이콘 수정</mark>

* 아이콘 사용여부, 아이콘 형식, 아이콘명, 아이콘 설정 등 등록한 아이콘 정보를 수정할 수 있습니다.
* \[삭제] 버튼을 클릭하면 해당 아이콘이 삭제됩니다.
* \[취소] 버튼을 클릭하면 '아이콘 수정' 창이 닫힙니다.

#### <mark style="color:blue;">■ 적용상품 보기</mark>&#x20;

* 아이콘이 적용된 상품 리스트가 확인됩니다.
* \[수정] 을 클릭하면 해당 상품의 상품 수정페이지로 이동합니다.
* \[상품번호]를 클릭하면 해당 상품의 상품 상세페이지로 이동합니다.
* 아이콘 적용상품 검색 : 상품명/상품번호 등의 검색어, 브랜드, 상품 승인 및 전시상태 등 다양한 상세 검색 조건으로 아이콘이 적용된 상품을 해당 창에서 검색할 수 있습니다.
