# 기획전 관리

<Callout type="info">
**시즌 할인 혜택, 특별 행사 이벤트 상품을 별도 페이지로 제작하여 판매하는 기획전의 등록 및 관리가 가능한 기획전 관리 메뉴입니다.**
</Callout>

## 기획전 관리 검색조건

<figure><img src="/images/appearance/event/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FZtU19VQru8TXI6GMr63o%2F%EA%B8%B0%ED%9A%8D%EC%A0%84%EA%B4%80%EB%A6%AC%20%EA%B2%80%EC%83%89%EC%A1%B0%EA%B1%B4%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;기획전 관리 검색조건></p></figcaption></figure>

**① 쇼핑몰**\
등록한 기획전을 조회할 쇼핑몰을 선택합니다.

**② 검색어**\
기획전명, 기획전 번호, 기획전 ID, 담당자 등의 검색어로 등록한 기획전을 조회합니다.\
다중 검색과 일부 검색어는 지원하지 않습니다. 정확한 검색어를 입력해 주세요.

**③ 기간검색**\
등록일, 전시기간 기준으로 등록한 등록한 기획전을 조회합니다.

**④ 진행상태**\
등록한 기획전을 진행대기, 진행 중, 진행종료 등의 진행상태로 검색할 수 있습니다

* 진행대기 : 기획전시작일이 도래하지 않은 상태
* 진행 중 : 기획전 시작일이 도래하여 기획전이 진행 중인 상태
* 진행종료 : 기획전 종료일이 지나 기획전이 종료된 상태

**⑤ 기획전 유형**\
기획전 운영 형태인 기획전 유형으로 등록한 기획전을 조회합니다.

* 일반 : 쇼핑몰에서 운영되는 기획전
* 외부링크 : 외부에서 진행하는 기획전 URL을 쇼핑몰 기획전 페이지에 연동하여 운영

## 기획전 검색결과 (리스트)

<figure><img src="/images/appearance/event/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FwVsjN7wsjYzyecZKqLuJ%2F%EA%B8%B0%ED%9A%8D%EC%A0%84%EA%B4%80%EB%A6%AC%20%EA%B2%80%EC%83%89%EA%B2%B0%EA%B3%BC%20%EB%A6%AC%EC%8A%A4%ED%8A%B8%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;기획전 검색결과 (리스트)></p></figcaption></figure>

**①** [**기획전 등록**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/appearance/event/addform)\
기획전을 신규로 등록할 수 있습니다.

**② 수정 (기획전 수정)**\
등록된 기획전 수정이 가능한 기획전 수정페이지로 이동됩니다.

**③ 복사 (기획전 복사)**\
기획전 등록 시 설정한 기획전 정보를 복사하여 동일한 기획전을 (복사)등록할 수 있는 기획전 복사 페이지로 이동됩니다.

**④ 기획전 번호**\
기획전 등록 시 생성되는 고유한 번호입니다.

**⑤ 기획전 URL**\
해당 기획전 페이지의 상세 URL이 확인됩니다.

**⑥ 진행상태**\
기획전 시작일과 종료일에 따른 기획전 진행상태가 확인됩니다.

**⑦ 유형**\
등록된 기획전의 운영 형태가 확인됩니다.

**⑧ 적용대상 상품**\
&#x20;\[엑셀 다운로드] 버튼을 클릭하면 기획전에 적용한 상품 리스트를 엑셀 파일로 다운로드 받을 수 있습니다.
