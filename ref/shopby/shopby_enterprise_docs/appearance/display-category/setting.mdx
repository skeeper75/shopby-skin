# 전시카테고리 설정

<Callout type="info">
**쇼핑몰 메인 페이지 카테고리 영역 및 햄버거 메뉴 등에 상품과 매칭되어 노출되는 전시카테고리를 등록하고 관리할 수 있습니다.**

**전시카테고리는 상품 등록 설정 항목으로, 상품 등록 전에 전시카테고리를 생성해야 합니다.**

**전시카테고리는 개수에 제한없이 생성이 가능합니다.**
</Callout>

## 전시카테고리 설정

<figure><img src="/images/appearance/display-category/setting/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F0UKA4oiSx0vzITb45B4E%2F%EC%A0%84%EC%8B%9C%EC%B9%B4%ED%85%8C%EA%B3%A0%EB%A6%AC%20%EC%84%A4%EC%A0%95%EA%B4%80%EB%A6%AC%20%EB%B3%B5%EC%82%AC%202.png" alt=""/><figcaption><p>&#x3C;전시카테고리 설정></p></figcaption></figure>

**① 전시카테고리 리스트**\
전시카테고리 추가 및 삭제, 수정과 순서 변경 등 모든 상세 설정이 가능합니다.

**② 카테고리 추가**\
새로운 전시카테고리 추가를 위해 \[+카테고리 추가] 버튼을 클릭합니다.\
전시카테고리 리스트에  \[신규 카테고리]로 새로운 전시카테고리가 추가됩니다.

**③** [**하위 카테고리 추가**](#undefined-2)\
여러 개의 전시카테고리 중, 하위 카테고리로 사용할 전시카테고리를 클릭한 다음, 상위 전시카테고리명에드래그 합니다.

<Callout type="warning">
매칭된 상품이 없는 상위 카테고리에만 하위 카테고리 추가가 가능합니다. \
📍상위 카테고리에 매칭된 상품이 있는 경우 하위 카테고리 추가 불가

하위 카테고리는 5depth까지 추가할 수 있습니다.
</Callout>

**④ 삭제**\
삭제할 전시카테고리를 선택하고 \[삭제] 버튼을 클릭합니다.&#x20;

<Callout type="danger">
매칭된 상품이 없는 전시카테고리의 삭제만 가능합니다. &#x20;
</Callout>

**⑤  전시카테고리 순서변경**\
순서 변경이 필요한 전시카테고리를 선택하고, 이동하고 싶은 위치로 드래그하여 이동합니다.

<Callout type="warning">
**전시카테고리 리스트 노출순으로 쇼핑몰에 반영되는 점을 참고해 주세요.**
</Callout>

**⑥ 카테고리 번호**\
전시카테고리 저장 시 자동으로 부여되는 고유 번호입니다.

**⑦ 카테고리 URL**\
전시카테고리 저장 시 자동으로 생성되는 고유 URL입니다.

**⑧ 카테고리명**\
새로운 카테고리인 신규 카테고리에 이름을 입력하거나, 기존 전시카테고리명을 수정할 수 있습니다.\
📍카테고리명은 공백 포함 20자까지 입력 가능합니다.

**⑨ 전시여부**\
쇼핑몰 전시카테고리 영역에 해당 전시카테고리 노출을 설정합니다.

**⑩ 카테고리 아이콘**\
전시카테고리명 대신 아이콘 이미지를 등록하여 노출할 수 있습니다.\
📍아이콘 이미지는 512KB 미만의 이미지1개만 등록 이가능합니다.

**⑪ 상단 이미지** <img src="/images/appearance/display-category/setting/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FfCtrqbCqqHtkUFZ6PsUe%2FAPI1.png" alt="" data-size="line"/>\
전시카테고리 페이지 상단 영역에 이미지를 노출할 수 있습니다. \
'등록'을 선택하면 나타나는 에디터 기능을 활용하여 이미지를 등록할 수 있습니다.&#x20;

**⑫** \[변경사항 저장] 버튼을 클릭해 전시카테고리 설정을 완료합니다.

<Callout type="warning">
**전시카테고리 별 설정 사항은 반드시 각각의 \[저장] 버튼을 클릭해 반영해 주세요.**\
**\[저장] 버튼을 클릭하지 않고, 다른 전시카테고리로 이동하면 해당 설정 사항이 반영되지 않으므로 유의해 주시기 바랍니다.**
</Callout>

**⑬** [엑셀 일괄등록](#undefined-2)\
\[엑셀 일괄등록] 버튼을 클릭하면 나타나는 '전시카테고리 일괄등록' 창에서 전시카테고리를 일괄 등록(추가)할 수 있습니다.

**⑭** 엑셀 다운로드\
\[엑셀 다운로드] 버튼을 클릭하면 현재 설정되어 있는 전시카테고리 정보가 입력된 엑셀파일이 다운로드 됩니다.

***

#### <mark style="color:blue;">**■ 하위 카테고리 추가**</mark>

* 하위 카테고리로 사용할 전시카테고리를 선택하고, 상위 카테고리명에 드래그하면 해당 상위 카테고리의 하위 카테고리로 생성(추가)됩니다.
* 하위 카테고리를 상위 카테고리 위치로 드래그하여 이동하면, 하위 카테고리에서 상위 카테고리로 변경됩니다.

***

#### <mark style="color:blue;">**■ 전시카테고리 일괄등록**</mark>

* '전시카테고리 일괄등록' 창에서 \[일괄등록 샘플 다운로드] 버튼을 클릭해 엑셀양식 파일을 다운로드 받을 수 있습니다.
* 엑셀양식에 안내된 작성가이드를 확인하고, 전시카테고리 정보를 입력합니다.
* 전시카테고리 엑셀양식을 \[파일찾기] 버튼으로 업로드하고, \[저장] 버튼을 클릭해 전시카테고리 일괄등록을 완료해 주세요.
