# 브랜드 관리

<Callout type="info">
**상품과 매칭되어 노출되는 브랜드를 등록하고 관리할 수 있습니다.**

**브랜드는 상품 등록 설정 항목으로 상품 등록 전에, 해당 메뉴에서 브랜드를 생성하거나 상품 등록 과정에서 브랜드를 바로 입력할 수도 있습니다.**

**브랜드는 개수에 제한없이 생성이 가능합니다.**
</Callout>

## 브랜드 설정

<figure><img src="/images/appearance/undefined/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FNkBS2VIpoZU1tp0R564t%2F%EB%B8%8C%EB%9E%9C%EB%93%9C%EC%84%A4%EC%A0%95.jpg" alt=""/><figcaption><p>&#x3C;브랜드 설정></p></figcaption></figure>

**① 브랜드 리스트**\
등록한 브랜드가 확인되고, 브랜드 수정 및 하위 브랜드 추가 등의 상세 설정이 가능합니다.\
📍상품 등록 과정에서 입력한 브랜드는, 브랜드 리스트에 바로 추가되어 확인됩니다.\
📍브랜드명 기준 '가나다순' 으로 정렬됩니다.

**② 1차 브랜드 추가**\
새로운 브랜드 추가를 위해 \[+ 1차 브랜드추가] 버튼을 클릭합니다.\
브랜드 리스트에 \[신규 브랜드]로 새로운 브랜드가 추가됩니다.

**③ 하위 브랜드 추가**\
하위 브랜드를 추가(등록)할 1차 브랜드를 클릭하고, \[+] 버튼을 클릭해 하위 브랜드를 추가합니다.\
하위 브랜드가 추가된 1차 브랜드는 \[∧] 버튼과 \[∨] 버튼으로, 하위 브랜드 열기와 닫기가 가능합니다.

<Callout type="warning">
매칭된 상품이 없는 1차 브랜드에만 하위 브랜드 추가가 가능합니다. \
📍1차 브랜드에 매칭된 상품이 있는 경우 하위 브랜드 추가 불가

하위 브랜드는 2epth까지 추가할 수 있습니다. (대표 브랜드 포함 총 3epth)
</Callout>

**④ 삭제**\
삭제할 브랜드를 선택하고 \[- 삭제] 버튼을 클릭합니다.&#x20;

<Callout type="danger">
매칭된 상품이 없는 브랜드의 삭제만 가능합니다. &#x20;
</Callout>

**⑤ 전체 열기 / 전체 닫기**\
1차 브랜드에 추가된 각각의 하위 브랜드를 전체 열기와 전체 닫기 할 수 있습니다.&#x20;

### 브랜드 기본 설정

**① 브랜드 번호**\
브랜드 저장 시 자동으로 부여되는 고유 번호입니다.

**② 대표 브랜드명**\
새로운 브랜드인 신규 브랜드의 이름을 입력하거나, 기존 브랜드명을 수정할 수 있습니다.\
\[중복확인] 버튼을 클릭해 브랜드명 사용 여부를 확인해 주세요.\
📍브랜명은 공백 포함 30자까지 입력 가능합니다.

**③ 보조 브랜드명**\
신규 브랜드의 보조 이름을 입력하거나, 기존 보조 브랜드명을 수정할 수 있습니다.\
📍보조 브랜드명은 대표 브랜드명 옆에 괄호 노출됩니다.\
&#x20;    <mark style="color:blue;background-color:yellow;">예 : 대표 브랜드명 shopby / 보조 브랜드명 샵바이 → shopby(샵바이)</mark>\
📍보조 브랜명은 공백 포함 30자까지 입력 가능합니다.

**④ 브랜드 이미지** <img src="/images/appearance/undefined/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FfCtrqbCqqHtkUFZ6PsUe%2FAPI1.png" alt="" data-size="line"/>\
브랜드 이미지를 등록할 수 있습니다.&#x20;

**⑤ 브랜드 설명** <img src="/images/appearance/undefined/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FfCtrqbCqqHtkUFZ6PsUe%2FAPI1.png" alt="" data-size="line"/>\
해당 브랜드에 대한 상세한 설명을 등록할 수 있습니다.

**⑥ 추가 설명** <img src="/images/appearance/undefined/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FfCtrqbCqqHtkUFZ6PsUe%2FAPI1.png" alt="" data-size="line"/>\
브랜드 관련 추가 정보를 등록할 수 있습니다.

**⑦ 등록 상품 수**\
\[조회] 버튼을 클릭하면 해당 브랜드에 매칭된 상품 수가 확인됩니다.

**⑧** \[변경사항 저장] 버튼을 클릭해 브랜드 설정을 완료합니다.

<Callout type="warning">
**브랜드 별 설정 사항은 반드시 각각의 \[변경사항 저장] 버튼을 클릭해 반영해 주세요.**\
**\[변경사항 저장] 버튼을 클릭하지 않고, 다른 브랜드로 이동하면 해당 설정 사항이 반영되지 않으므로 유의해 주시기 바랍니다.**
</Callout>

**⑨ 생성**\
\[생성] 버튼을 클릭하면 현재 설정되어 있는 브랜드 정보가 입력된 엑셀파일이 다운로드 됩니다.
