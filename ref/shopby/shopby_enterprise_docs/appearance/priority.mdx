# 전시상품 우선순위 관리

<Callout type="info">
**쇼핑몰 shop API 기능을 통해 ‘MD추천순’ 기준으로 쇼핑몰 내 상품 진열 설정 시 노출되는 상품 진열 순서를 설정할 수 있습니다.**
</Callout>

## 전시상품 관리 <a href="#undefined" id="undefined"></a>

<div data-full-width="false"><figure><img src="/images/appearance/priority/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FrKygHkSYTuNn704bnw32%2F1.%20%EC%A0%84%EC%8B%9C%EC%83%81%ED%92%88%20%EC%9A%B0%EC%84%A0%EC%88%9C%EC%9C%84%20%EA%B4%80%EB%A6%AC.png" alt=""/><figcaption></figcaption></figure></div>

**① 상품조회 삭제**\
해당 우선순위 관리에 등록된 상품 정보 조회 후 삭제를 원하는 경우 \[상품조회 삭제] 버튼 클릭 후 노출되는팝업창에서 가능합니다.

**② 상품등록**\
우선순위 관리에 추가할 상품 선택 시 등록된 상품을 조회하거나, 상품번호를 직접 입력하거나, 엑셀을 통해 상품번호를 일괄 등록하는 방법으로 등록할 수 잇습니다.

* 상품조회 등록: \[상품조회] 버튼을 클릭하여 쇼핑몰에 등록된 상품 정보를 조회하여 추가할 상품을 선택할 수 있습니다.
* 번호직접등록: 쇼핑몰에 등록된 상품에 대한 상품번호를 직접 입력 후 \[상품추가] 버튼 클릭을 통해 추가할 상품을 선택할 수 있으며, 상품번호 다중입력 시 Enter 또는 ‘,’로 구분하여 입력할 수 있습니다.
* 엑셀 일괄등록: 쇼핑몰에 등록된 상품에 대한 상품번호를 엑셀 파일을 통해 추가할 상품을 일괄 등록할 수 있습니다. \
  \- 상품번호 일괄등록 가능한 엑셀양식 파일을 다운로드 받을 수 있습니다. \
  \- 다운로드 받은 양식 파일을 작성하여 엑셀 일괄등록 버튼을 통해 추가할 상품을 일괄 등록할 수 있습니다.

**③ 전시상품 우선순위 리스트**\
전시상품 우선순위 등록된 상품 리스트를 확인할 수 있으며, 리스트에서 상품 체크 후 \[선택상품 삭제] 버튼 클릭 시 삭제 가능합니다.

* 상품 선택 후 진열순위를 변경하거나, 등록된 상품  리스트를 엑셀  파일로 다운로드 받을 수 있습니다.

**④ 변경사항저장**\
변경된 내용이 있는 경우 하단 변경사항저장 버튼을 클릭하는 경우에만 적용되며, 클릭하지 않고 메뉴 이동 시 변경 내용이 저장되지 않습니다.

***

<mark style="color:blue;">**■ 상품조회 등록(상품조회)**</mark>

* 쇼핑몰에 등록된 상품 중 여러 조건을 기준으로 조회 후 원하는 상품을 선택하여 우선순위 관리 리스트에 등록할 수 있습니다.

<div data-full-width="false"><figure><img src="/images/appearance/priority/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2Fxahbky0HAyG9uL6gVXG0%2F3.%20%EC%A0%84%EC%8B%9C%EC%83%81%ED%92%88%20%EC%9A%B0%EC%84%A0%EC%88%9C%EC%9C%84_%EC%83%81%ED%92%88%EC%A1%B0%ED%9A%8C%20%EB%93%B1%EB%A1%9D.png" alt=""/><figcaption><p>[상품조회] 버튼 클릭 시 팝업되는 화면으로 전시하고자 하는 상품을 조회하여 추가할 수 있습니다.</p></figcaption></figure></div>

<details>

<summary>상품조회 설명 자세히 보기</summary>

**① 쇼핑몰**\
선택한 카테고리가 등록된 쇼핑몰을 확인할 수 있습니다.

**② 검색어**\
상품명, 상품번호, 상품관리코드, 상품관리코드|부분검색을 기준으로 검색하여 상품을 조회할 수 있습니다.

**③ 브랜드명/파트너사**\
상품에 등록된 브랜드명과 파트너사를 기준으로 상품을 조회할 수 있습니다.

**④ 상품상태** \
선택하는 상품상태에 따른 판매상태와 판매설정을 기준으로 조회 가능합니다.

**⑤ 전시카테고리** \
상품에 등록된 전시카테고리를 기준으로 조회 가능합니다.

**⑥ 상세검색** \
재고, 프로모션, 전시상태, 추가할인, 플랫폼, 담당자, 판매방식, 판매수수료 방식 등 여러 상세 조건 기준으로 상품 조회 가능합니다.

* 재고여부: 재고여부나 5,10,50,100개 이하 중 선택
* 프로모션 여부: 프로모션 진행 가능 여부
* 전시상태: 상품의 전시(노출) 여부
* 추가할인여부: 상품에 설정된 추가 할인 여부
* 플랫폼구분: PC, 모바일 웹, 모바일 앱 중 선택
* 담당자: 상품을 등록한 담당자를 검색
* 판매방식: 매입, 사입 중 상품에 설정된 판매 방식을 지정하여 검색
* 판매수수료 방식: 상품에 설정된 판매 수수료 방식이 판매가 대비 수수료율과 공급가 중 선택

**⑦ 조회** \
모든 조건을 선택하고 \[조회] 버튼을 클릭하면 하단 조회 결과에 상품이 노출됩니다.

**⑧ 초기화** \
\[초기화] 버튼을 클릭하면 입력된 검색어 및 검색 조건이 기본값으로 초기화됩니다.

**⑨ 상품조회 리스트** \
지정한 조건에 해당하는 상품 리스트가 노출되며, 추가하고자 하는 상품을 선택할 수 있습니다.

* 리스트에서 상품 선택 후 \[선택상품 추가] 버튼을 클릭하거나 \[검색결과 전체 추가] 버튼 클릭을 통해 추가할 수 있습니다.

</details>
