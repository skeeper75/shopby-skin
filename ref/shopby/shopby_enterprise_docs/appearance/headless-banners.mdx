# 헤드리스 배너 관리

<Callout type="info">
**쇼핑몰 운영 방식이 '헤드리스'인 쇼핑몰에 신규 배너를 등록하여 노출할 수 있는 관리 기능을 제공합니다.**

**배너 구좌에 여러 배너 이미지를 등록하고 다양한 형태로 구성하여 쇼핑몰 여러 페이지에 노출할수 있습니다.**

**쇼핑몰 운영 방식이 '스킨'인 쇼핑몰은 배너를 등록해도 쇼핑몰에 노출이 되지 않는 점을 참고해 주세요.**&#x20;
</Callout>

## 헤드리스 배너 관리 검색조건

<figure><img src="/images/appearance/headless-banners/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FolMENdhupC1Y9KdDEHy7%2F%ED%95%B4%EB%93%9C%EB%A6%AC%EC%8A%A4%EB%B0%B0%EB%84%88%EA%B2%80%EC%83%89%EC%A1%B0%EA%B1%B4.png" alt=""/><figcaption><p>&#x3C;헤드리스 배너 관리 검색조건></p></figcaption></figure>

**① 쇼핑몰**\
등록된 배너를 조회할 쇼핑몰을 선택합니다.

**② 검색어**\
배너 그룹명, 배너명, ID 등의 검색어로 배너를 조회합니다. \
다중 검색과 일부 검색어는 지원하지 않습니다. 정확한 검색어를 입력해 주세요.

**③ 기간 검색**\
등록일, 수정일 기준으로 배너를 조회합니다.&#x20;

**④ 배너 그룹**\
배너의 특징으로 배너를 묶은 '배너 그룹'으로 배너를 조회합니다.&#x20;

## 헤드리스 배너관리 검색결과(리스트)

<figure><img src="/images/appearance/headless-banners/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FqWim7FKwmqSB1wKBq9kG%2F%ED%95%B4%EB%93%9C%EB%A6%AC%EC%8A%A4%EB%B0%B0%EB%84%88%EA%B2%80%EC%83%89%EA%B2%B0%EA%B3%BC%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;헤드리스 배너관리 검색결과(리스트)></p></figcaption></figure>

**①** [**배너 등록**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/appearance/headless-banners/add)\
&#x20;\[배너등록] 버튼을 클릭하면 신규 배너를 등록할 수 있는 '배너 등록' 페이지로 이동됩니다.

**② 선택 삭제**\
\[선택삭제] 버튼을 클릭 해 등록된 배너를 삭제할 수 있습니다.

**③** [**배너 그룹 관리**](#undefined-2)\
\[배너 그룹관리] 버튼을 클릭하면, 신규 배너 그룹을 등록하고 관리할수 있는 '배너 그룹 관리' 창이 노출 됩니다.

**④ 수정**\
\[수정] 버튼을 클릭하면 배너 설정 항목을 수정할 수 있는 '배너 수정' 페이지로 이동됩니다.

**⑤ 배너 구좌수**\
해당 배너에 등록된 배너 구좌수가 확인됩니다.

**⑥ 배너 그룹명**\
해당 배너에 매칭된 배너 그룹명이 확인됩니다.

<Callout type="success">
**헤드리스 운영방식이란, 결제, 재고, 배송, 고객 관리 등의 커머스 핵심을 제공하는 쇼핑몰 기능과** \
**고객에게 보여지는 외부 화면인 디자인 영역이 100% 분리되어 있는 구조의 쇼핑몰 운영방식을 의미합니다.**
</Callout>

***

#### <mark style="color:blue;">**■**</mark> <mark style="color:blue;"></mark><mark style="color:blue;">배너 그룹 관리</mark>

* 배너 그룹의 신규 등록 및 삭제 처리 등이 가능한 '배너 그룹 관리' 입니다.
* \[그룹 추가] 버튼을 클릭하고, 배너 그룹을 등록할 쇼핑몰을 선택한 다음 배너 그룹명을 입렵합니다.
* \[저장] 버튼을 클릭 해 배너 그룹 등록을 완료합니다.
* \[선택삭제] 버튼으로 배너 그룹을 삭제할 수 있습니다.
