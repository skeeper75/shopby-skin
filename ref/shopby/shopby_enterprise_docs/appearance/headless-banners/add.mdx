# 배너 등록

## 기본 정보

<figure><img src="/images/appearance/headless-banners/add/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FvFcsY6zh42f1QccKJFxu%2F%ED%97%A4%EB%93%9C%EB%A6%AC%EC%8A%A4%EB%B0%B0%EB%84%88%EA%B4%80%EB%A6%AC%20%EB%B0%B0%EB%84%88%EB%93%B1%EB%A1%9D%20%EA%B8%B0%EB%B3%B8%EC%A0%95%EB%B3%B4%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;기본 정보></p></figcaption></figure>

**① 쇼핑몰**\
배너를 등록하는 쇼핑몰을 선택합니다.

**② 배너 코드**\
배너 등록 시 자동으로 생성되는 고유한 번호입니다.

**③ 배너 ID**\
API 호출 및 조회가 가능하도록 배너ID를 지정할 수 있습니다. (최대 20자까지 입력 가능)\
📍 배너마다 다른 값을 지정해야 하므로, \[중복확인] 버튼을 클릭 해 사용여부를 확인해 주세요.

**④ 배너 그룹**\
배너 그룹은 등록하는 배너의 성격 및 특징에 맞게 묶음으로 관리하는 기능입니다.\
배너 그룹을 선택하거나, [헤드리스 배너 관리 > 배너 그룹 관리](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/appearance/headless-banners/..#undefined-2) 에서 새로운 배너 그룹을 등록하여 사용할 수 있습니다.

**⑤ 배너명**\
등록하는 배너명을 입력합니다.

**⑥ 배너 구좌수**\
배너 컨텐츠를 등록할 수 있는너배 구좌 수를 1 \~ 20건 중에서 선택할 수 있습니다.

**⑦ 플랫폼 구분**\
등록하는 배너가 노출되는 플랫폼을 설정합니다.&#x20;

**⑧ 회원등급 노출 설정**\
등록하는 배너를 노출 할 회원등급을 별도 설정할 수 있습니다. \
전체 회원등급 및 특정 회원등급의 선택이 가능합니다.

**⑨ 회원그룹 노출 설정**\
등록하는 배너를 노출 할 회원그룹을 별도 설정할 수 있습니다. \
전체 회원그룹 및 특정 회원그룹의 선택이 가능합니다.

## 배너 설정

<figure><img src="/images/appearance/headless-banners/add/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F0lnZXJROAbQR97sjK4PC%2F%ED%97%A4%EB%93%9C%EB%A6%AC%EC%8A%A4%EB%B0%B0%EB%84%88%EA%B4%80%EB%A6%AC%20%EB%B0%B0%EB%84%88%EB%93%B1%EB%A1%9D%20%EB%B0%B0%EB%84%88%EC%84%A4%EC%A0%95%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;배너 설정></p></figcaption></figure>

**① 구좌명**\
등록하는 배너의 구좌명을 입력하고,  배너의 가로/세로 사이즈(Pixel)를 설정합니다.

**②**[ **노출 설정**](#undefined-2)\
\[노출 설정] 버튼을 클릭하면, 배너 구좌의 노출방식 및 노출 플랫폼 등을 배너 구좌별로 설정할 수 있는 '배너 노출 설정' 창이 노출됩니다.&#x20;

**③ 콘텐츠 추가**\
등록하는 배너 구좌에 여러 개의 배너 이미지 등록이 필요한 경우 사용합니다.\
\[콘텐츠 추가] 버튼을 클릭하면 배너 콘텐츠 영역이 추가됩니다.\
📍 구좌당  배너 콘텐츠는 최대 30까지 추가 가능합니다.

<Callout type="warning">
한 개의 배너 구좌에 여러 개의 배너 이미지를 등록해, 배너 이미지 위치를 다양하게 구현할 수 있고 움직이는 배너 등으로 활용하는 등 다양한 배너 구성이 가능합니다.
</Callout>

**④ 배너 콘텐츠명**\
등록하는 배너 콘텐츠(배너 이미지 등)의 이름을 입력합니다.\
\[설명 등록] 버튼을 클릭하면 배너 콘텐츠 설명 문구를 입력할 수 있는 '[설명 등록](#undefined-3)' 창이 노출됩니다.\
'설명 등록' 창에 입력한 설명 문구가 배너 이미지에 텍스트로 추가 노출됩니다.

**⑤ 노출 기간** \
등록하는 배너의 노출 기간을 '상시'와 '기간' 중에서 선택합니다. \
기간노출 선택 시, 노출 시작일과 종료일을 지정해 주세요.

**⑥ 배너 이미지**\
배너 구좌에 배너이미지를 등록합니다. \
\[마우스 오버 이미지 등록] 버튼을 클릭하면, 마우스 오버 시 노출되는 배너 이미지 등록이 가능합니다.\
📍배너 이미지 좌우 여백 발생하는 경우, 여백 색상을 설정할 수 있습니다.\
📍좌우 여백 '컬러 피커' 를 선택해 '컬러 피커 창' 에서 여백 색상을 설정합니다.<br>

**⑦ 동영상 URL**\
배너 구좌에 동영상 콘텐츠를 등록해 전시할 수있습니다.  \
배너 콘텐츠로 전시할 동영상의 URL을 입력합니다.

**⑧ 랜딩 URL**\
배너 이미지 클릭 시 화면 및 페이지로 연결할 수 있는 다양한 방법의 랜딩 URL 설정 방식을 제공합니다.

* 일반 : 전시 카테고리 페이지 및 상품 상세 페이지 등 특정 페이지로 이동될 수 있도록 랜딩 페이지 URL 입력창이 노출되며, 페이지 연결 시 '현재 창'과 '새 창' 중에서 선택 가능합니다.
* 이미지 맵 : \[이미지 맵 등록] 버튼을 클릭하면  HTML 소스 (이미지 맵 태그) 입력으로 링크를 연결할 수 있는 '[이미지 맵 등록](#undefined-4)' 창이 노출됩니다. &#x20;
* 기획전 : 쇼핑몰에 등록된 '진행 대기',  '진행 중' 상태의 기획전 번호를 입력하여 기획전으로 랜딩할 수 있습니다. 페이지 연결 시 '현재 창'과 '새 창' 중에서 선택 가능합니다.

**⑨ 최종 수정**\
등록하는 배너 이미지를 수정한 최종 날짜가출노됩니다.\
📍 배너 등록 화면에서는 ' / ' 으로 노출되는 점을 참고해 주세요.

**⑩ 추가입력항목**\
등록하는 배너 이미지에 대한 추가 설명 및 참고사항을 입력할 수 있습니다.\
내부 관리용으로 쇼핑몰에 노출되지 않습니다.

**⑪ 삭제**\
\[삭제] 버튼을 클릭 해 배너 콘텐츠를 삭제할 수 있습니다.

**⑫** \[저장] 버튼을 클릭 해 배너 등록을 완료합니다.

***

#### <mark style="color:blue;">**■ 배너 노출 설정**</mark>

* 배너 구좌의 노출방식 및 노출 플랫폼 등을 배너 구좌별로 설정할 수 있습니다.
* 배너 값과 동일로 설정하면,  기본정보 항목에서 설정한 값과 동일한 설정으로 처리됩니다.

  설명문구  텍스트의 색상을 설정하고,  \[저장] 버튼을 클릭 해 배너설명 등록을 완료합니다.
