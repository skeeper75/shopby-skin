# 스킨 배너 수정

## 기본 정보

**① 배너명**\
배너 이름을 수정할 수 있습니다.\
배너명은 쇼핑몰 화면에 노출되지 않습니다.

**② 노출 여부**\
**수정**하는 배너의 노출 여부를 설정합니다.

**③ 노출 기간**\
수정하는 배너의 노출 기간을 '상시'와 '기간' 중에서 선택합니다. \
기간노출 선택 시, 노출 시작일과 종료일을 지정해 주세요.

<Callout type="danger">
**오로라 기본 스킨은 총 8개의 각기 다른 일반 배너 그룹이 기본값으로 등록되어 있습니다.**\
**수정하는 배너의 '배너 그룹'을 반드시 확인해 주세요.**
</Callout>

## 배너 이미지 설정

**✅ 배너 그룹에 따라 '배너 이미지 설정' 항목이 상이합니다.**

### 로고전용/일반배너

<figure><img src="/images/appearance/skin-banners/edit/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FmzeBESwQ30Np65kiwwzp%2F%EC%8A%A4%ED%82%A8%EB%B0%B0%EB%84%88%EC%88%98%EC%A0%95%20%EB%B0%B0%EB%84%88%EC%9D%B4%EB%AF%B8%EC%A7%80%20%EC%88%98%EC%A0%95%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;&#x3C;배너 이미지 설정 - 로고전용/일반배너 ></p></figcaption></figure>

**① 배너 이미지 사이즈**\
수정하는 배너의 이미지 사이즈를 설정할 수 있습니다.

* '등록한 이미지 사이즈대로 노출' 설정 : 수정하는 배너 이미지의 원본 사이즈로 쇼핑몰 화면에 노출됩니다.
* '등록한 이미지 사이즈대로 노출'  미설정 : 배너의 가로와 세로 사이즈를 설정할 수 있고, 설정한 배너 사이즈로 쇼핑몰 화면에 노출됩니다.

<Callout type="warning">
가로 사이즈 '%' 설정 : 세로 사이즈는 설정한 가로 사이즈에 따라 자동으로 비율이 조정됩니다. \
가로 사이즈 'Pixel' 설정 : 가로와 세로 사이즈를 'Pixel' 단위로 각각 입력합니다.
</Callout>

**② 배너 이미지**

* \[파일 삭제] 버튼을 클릭 해 등록된 샘플 배너 이미지를 삭제합니다.
* \[파일 찾기] 버튼을 클릭 해 배너 이미지를 등록합니다. \
  📍  배너로 등록 가능한 이미지 확장자는 JPG, JPEG, GIF, PNG, BMP, WEBP 입니다. (최대 10MB 파일)
* 링크 주소 : 배너 이미지 클릭 시 이동하는 페이지의 URL을 입력합니다.  페이지 이동 창을  '새 창' 과 '현재 창' 중에서 선택합니다.

<Callout type="warning">
로고전용 배너에는 홈 디렉토리 의미하는 ' /  ' 가 입력되어 있고, 쇼핑몰 메인 페이지로 이동됩니다.
</Callout>

* 이미지 설명 : 수정하는 배너 이미지 관련 부가 설명을 입력합니다.

<Callout type="success">
**이미지 설명은 배너 이미지를 위한 대체 텍스트로 배너 이미지가 이슈로 인해 표시되지 않을 경우 대신하여 노출되며, 배너 이미지 인식이 불가한 사용자를 위한 텍스트 정보로 제공됩니다 .**

**웹 접근성 준수 의무화에 따라 배너 이미지  등록 시 이미지 설명을 입력하는 것을 권장합니다.**&#x20;
</Callout>

**③** \[저장] 버튼을 클릭 해 배너 수정을 완료해 주세요.<br>

### 움직이는 배너

<figure><img src="/images/appearance/skin-banners/edit/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FtWUUEUxpIlAssXqBq6dn%2F%EC%8A%A4%ED%82%A8%EB%B0%B0%EB%84%88%EC%88%98%EC%A0%95%20%EC%9B%80%EC%A7%81%EC%9D%B4%EB%8A%94%EB%B0%B0%EB%84%88%EC%9D%B4%EB%AF%B8%EC%A7%80%20%EC%88%98%EC%A0%95%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;배너 이미지 설정 - 움직이는 배너 ></p></figcaption></figure>

**① 배너 이미지 사이즈**\
수정하는 움직이는 배너의 이미지 사이즈를 설정합니다.\
📍 로고전용/일반배너 항목과 동일한 기능을 제공합니다.

<Callout type="danger">
움직이는 배너를 수정하는 경우, 모두 동일한 이미지 사이즈로 설정해 주세요.
</Callout>

**② 배너 이미지**

<Callout type="warning">
오로라 스킨의 움직이는 배너는 총 6개의 배너 이미지가 기본값으로 등록되어 있습니다.
</Callout>

* \[파일 삭제] 버튼을 클릭 해 등록된 샘플 움직이는배너 이미지를 삭제합니다.
* \[파일 찾기] 버튼을 클릭 해 배너 이미지를 등록합니다. \
  📍  배너로 등록 가능한 이미지 확장자는 JPG, JPEG, GIF, PNG, BMP, WEBP 입니다. (최대 10MB 파일)
* 링크 주소 : 배너 이미지 클릭 시 이동하는 페이지의 URL을 입력합니다. 페이지 이동 창을  '새 창' 과 '현재 창' 중에서 선택합니다.\
  📍 로고전용/일반배너 항목과 동일한 기능을 제공합니다.
* 이미지 설명 : 수정하는 배너 이미지 관련 부가 설명을 입력합니다.\
  📍 로고전용/일반배너 항목과 동일한 기능을 제공합니다.
* 노출 여부 : 수정하는 움직이는 배너 이미지 별 노출 여부를 각각 설정할 수 있습니다.&#x20;
* 노출 기간 : 수정하는 움직이는 배너 이미지 별 노출 기간을 개별 설정할 수 있습니다.  노출 기간을 '상시'와 '기간' 중에서 선택합니다. 기간노출 선택 시, 노출 시작일과 종료일을 지정해 주세요.
* \[배너 추가] 버튼을 클릭하면 움직이는 배너 이미지 추가 등록란이 노출되며, \[선택 삭제] 버튼으로 삭제할 수 있습니다.
* '상/하 이동 아이콘' 으로 배너 이미지 노출 순서를 지정할 수 있습니다.&#x20;

**③ 배너 이미지 일괄 노출**\
수정하는 배너 이미지 노출 여부와 노출 기간을 설정하여 \[적용] 버튼으로 일괄 설정 처리할 수 있습니다.

## 배너 효과 설정

**✅ 움직이는 배너 그룹에 헤당하는 설정 항목입니다.**

<figure><img src="/images/appearance/skin-banners/edit/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FxAZnB1pa7jqUYvozBxwK%2F%EC%8A%A4%ED%82%A8%EB%B0%B0%EB%84%88%EC%9D%B4%EB%AF%B8%EC%A7%80%EC%84%A4%EC%A0%95%20%EC%9B%80%EC%A7%81%EC%9D%B4%EB%8A%94%20%EB%B0%B0%EB%84%88%EC%9D%B4%EB%AF%B8%EC%A7%80%20%EC%88%98%EC%A0%95%20%EB%B0%B0%EB%84%88%ED%9A%A8%EA%B3%BC%EC%84%A4%EC%A0%95%20%EB%B3%B5%EC%82%AC%202.png" alt=""/><figcaption><p>&#x3C;배너 효과 설정 - 움직이는 배너 ></p></figcaption></figure>

**① 전환 속도 선택**\
수정하는 움직이는 배너 이미지의 전환 속도를 빠르게/보통/느리게 중에서 선택합니다.&#x20;

**② 전환 시간 설정**\
수정하는 움직이는 배너 이미지의 전환 시간을 1\~10초, 수동 중에서 설정합니다.\
📍'수동' 으로 설정하면 고객이 배너 이미지를 마우스로 클릭 하는 경우에만 이미지가 전환됩니다.

**③ 효과 선택**\
수정하는 움직이는 배너 이미지의 전환 효과를 정적인 효과의 '슬라이드'와 동적인 효과의 '페이드' 중에서 선택합니다.

**④ 좌우 전환 버튼**\
수정하는 움직이는 배너 이미지 양 측면에 노출되는 좌우 전환 버튼 노출 여부와 버튼 색상을 설정할 수 있습니다.

**⑤ 네비게이션**\
수정하는 움직이는 배너 이미지 하단에 노출되는 네비게이션 노출 여부와 네비게이션 종류 및 크기, 네비게이션 활성 및 비활성 색상도 설정할 수 있습니다.&#x20;

**⑥** \[저장] 버튼을 클릭 해 스킨 배너 수정을 완료합니다.
