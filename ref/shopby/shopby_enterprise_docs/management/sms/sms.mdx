# SMS 발송

<Callout type="info">
**쇼핑몰 운영관련 광고성/정보성 SMS를 수동으로 발송할 수 있습니다.**

**광고성 SMS는 수신자가 수신거부를 할 수 있도록 별도 안내와 기능이 함께 제공되는, 필수 부가서비스 '080 수신거부 번호'가 등록되어 있어야 발송이 가능합니다.**
</Callout>

**① SMS 잔여 포인트** \
SMS 발송에 필요한 SMS 잔여 포인트 확인과 SMS 포인트 충전을 할 수 있습니다. \
\[조회하기] 버튼을 클릭하면 SMS 잔여 포인트가 조회됩니다. \
\[SMS 포인트 충전하기] 버튼을 클릭하면, SMS 포인트 충전 결제가 가능 'SMS 포인트 충전' 창이 노출됩니다.

**② SMS 유형** \
고객에게 발송하는 메시지 내용에 따라 SMS 유형을 광고성 또는 정보성 중에서 선택합니다.

<Callout type="success">
**광고성 : 운영자가 경제적 이득을 취할 목적으로 발송하는 메시지**

**정보성 : 정보통신망법 안내서에 ‘영리목적 광고성 정보의 예외’에 해당하는 메시지**\
예) 수신자가 구매한 상품에 대한 설명, 보증, 제품 리콜, 안전 또는 보안 관련 정보 제공 \
&#x20;     상품 및 서비스에 대한 조건 및 특징에 대한 변경 안내 (예 : 포인트 제도 변경 안내) 등
</Callout>

<Callout type="danger">
**광고성 SMS 발송 유의사항**\
\
광고성 SMS는 080 수신거부 번호가 등록되어 있어야 발송할 수 있어요.\
광고성 SMS를 수신 동의한 회원에게만 발송해야 해요.\
수신 동의한 회원에게도 별도 동의 없이 야간시간대(21:00\~익일 08:00)에는 광고성 SMS의 발송이 불가해요.

광고성 SMS 내용 시작 부분에 (광고) 문구를 표기하고,\
쇼핑몰명 등과 같이 광고성 SMS를 발송하는 발신자 명칭을 표기해야 합니다.\
또한 광고성 SMS의 무료 수신거부 또는 무료수신동의 철회 방식이  함께 안내되어야 합니다.\
예 :  (광고) 샵바이쇼핑몰 \
&#x20;       샵바이 쇼핑몰  광고!\
&#x20;       무료수신거부 080-XXXX-XXXX

080 수신거부 번호를 등록하고, 광고성 SMS 유형을 선택하면\
메시지 내용 앞에 (광고), 내용 마지막에 \[무료 수신거부] 080-XXXX-XXXX가 자동으로 추가되어  발송 처리됩니다.
</Callout>

**③ 발송유형** \
메세지 문장 길이에 따라 발송유형을 선택합니다.

* SMS : 90byte 이내 단문 메시지 전송 시 선택합니다. \
  SMS 전송은 내용만 전송되기 때문에,  SMS 선택 시 제목 입력란이 비활성화 되는 점을 참고해 주세요.
* LMS : 2000byte 이내 장문 메시지 전송 시 선택합니다. \
  LMS 전송은 제목과 내용을 작성하여 전송합니다.&#x20;

**④ 발신 번호**\
고객에게 발송되는 발신자 번호를 확인해 주세요.\
다수의 발신 번호를 등록했다면, 발신 번호 변경도 가능합니다.

**⑤ 수신 대상 선택** \
SMS을 수신할 회원을 선택해 주세요. 선택한 회원 중 회원가입 시 SMS 수신 동의를 하지 않은 '수신 거부 대상자 수'가 별도 노출됩니다.

* 전체회원 : 쇼핑몰에 가입한 전체 회원에게 SMS를 발송합니다.
* 회원선택 : 특정 회원등급 및 회원그룹에 속한 회원과 개별 회원을 대상으로 SMS를 발송합니다.
* CRM 그룹 선택  : 회원 추출 완료한 CRM 그룹 대상으로 SMS를 발송합니다. ([CRM그룹 관리 가이드](https://support-help.nhn-commerce.com/common/crm-group))
* 직접 입력: 휴대폰 번호를 직접 입력하여 SMS를 발송합니다. \
  휴대폰번호를 입력하고 \[추가] 버튼을 클릭하면, 발송 대상 인원이 확인됩니다. \
  직접 입력한 휴대폰 번호의 \[선택삭제] 및 \[전체삭제]가 가능합니다.

<Callout type="danger">
**휴대폰 번호를 직접 입력하여 SMS를 발송하는 경우, 해당 수신번호의 회원 여부와 수신동의 여부에 관계없이 SMS가 발송됩니다.**\
**비회원에게 SMS를 발송하려면, 발송 전 휴대폰 번호 개인정보 수집동의 및 광고성 SMS 수신동의가 필요하므로 발송에 유의해 주세요.**
</Callout>

**⑥ 발송 설정**\
즉시 발송과 예약 발송 중에서 선택합니다.

* 즉시 발송 :  SMS를 지금 즉시 발송합니다.
* 예약 지급 : 발송일자를 미래 시점으로 설정하여 예약 발송 SMS로 등록합니다.

**⑦ 제목**\
LMS 메시지 제목을 작성해 주세요.\
선택한 광고성/정보성 메시지 유형에 알맞은 제목을 작성합니다.

**⑧ 내용** \
선택한 광고성/정보성 메시지 유형과 발송유형에 알맞은 내용을 작성합니다.

**⑨ 미리보기**\
\[미리보기] 버튼을 클릭하면 설정한 SMS 발송 내용을 미리보기로 확인할 수 있습니다.

**⑩ 초기화**\
&#x20;\[초기화] 버튼을 클릭하면, 설정한 SMS 발송 내용이 초기화 처리 됩니다.

**⑪ 발송**\
\[발송] 버튼을 클릭해 수동 SMS 발송을 완료해 주세요. \
📍에약 발송 SMS은 등록 설정됩니다.
