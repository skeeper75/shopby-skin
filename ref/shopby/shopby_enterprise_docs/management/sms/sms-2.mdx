# SMS 발송내역 조회

<Callout type="info">
**자동 SMS 발송이 '사용함'으로 설정되어, 자동 발송된 SMS 발송내역을 조회할 수 있습니다.**

**발송실패와 발송성공으로 구분된 발송상태의 세부 발송내역도 확인이 가능해요.**
</Callout>

**① 발송요청 일시** \
발송 월을 지정하여 자동 SMS 발송내역을 검색합니다.

**② 검색어** \
수신번호, 내용 등을 입력하여 자동 SMS 발송내역을 검색합니다.

**③ 발송상태** \
자동 SMS 발송 상태로 발송내역을 검색합니다.

* 발송성공 : 자동 발송 요청이 정상으로 완료되어 수신을 받은 상태
* 발송실패 : 자동 발송 요청이 실패된 상태 또는 발송결과 실패 처리 상태

**④** 발송내역의  \[[발송상태](#sms)]를 클릭하면 'SMS 발송내역 상세 리스트' 창이 노출됩니다.

***

#### <mark style="color:blue;">■ SMS 발송내역 상세 리스트</mark>

* 선택한 SMS 발송내역 정보가 수신번호, 발송된 SMS 내용, 수신일시, 발송결과, 발송실패 사유 등으로 확인됩니다.&#x20;
* 발송요청일시에서 지정한 해당 월의 자동 SMS 발송내역을 발송결과, 수신번호 입력, 발송실패 사유 \
  선택으로 검색할 수 있습니다.&#x20;
* \[엑셀다운로드] 버튼을 클릭하면 검색한 발송내역을 엑셀파일로 다운로드 받을수 있습니다.
