# 자동 SMS 설정

<Callout type="info">
**자동 SMS 설정 메뉴에서는 쇼핑몰의 특정 상태 또는 다양한 이슈에서 자동으로 발송되는 SMS를 설정하고 관리합니다.**

**회원, 주문배송, 프로모션, 문의 관련하여 세부 상황별 SMS 템플릿을 제공합니다.**

**발송 상세설정 기능으로 항목별 야간시간(21:00 \~ 익일08:00)대 SMS 발송 제한을 설정할 수 있습니다.**
</Callout>

**① 자동 SMS 유형 (자동 SMS 발송 템플릿)** \
회원 관련, 주문배송 관련, 프로모션 관련, 문의 관련으로 구분된 SMS 템플릿을 확인할 수 있습니다. \
각각의 SMS 템플릿을 클릭하면 세부 상황별 템플릿이 확인됩니다.

**② 사용여부** \
세부 상황별 템플릿을 선택해 자동 SMS 발송 사용여부를 '사용함'과 '사용 안 함'으로 설정할 수 있습니다.\
'사용함'을 선택하면 템플릿 입력 영역이 활성되며 문구 내용의 수정이 가능합니다.

**③ 미리보기 / 편집하기**\
\[미리보기] 버튼을 클릭하면 치환텍스트가 일반 문구로 출력되면서 고객에게 발송되는 SMS 알림 내용을 \
점검할 수 있습니다. 미리보기 화면에서 \[편집하기] 버튼을 클릭하면 문구 내용을 수정할 수 있습니다.\
📍문구 내용을 수정하고, 템플릿 영역의 \[저장] 버튼을 클릭해 주세요.

<br>

**④** [**발송 상세설정 (SMS 야간발송 제한 설정)**](#sms)\
\[발송 상세설정] 버튼을 클릭하면 항목별 야간시간 발송제한을 설정하는 'SMS 발송 상세설정' 창이 노출됩니다.<br>

**⑤ 치환텍스트** \
\[치환텍스트] 버튼을 클릭하면 SMS에 활용할 수 있는 다양한 치환텍스트가 확인됩니다.

<Callout type="warning">
치환텍스트는 쇼핑몰 어드민 여러 메뉴 및 항목에 설정된 특정 정보를, 해당하는 내용으로 불러와 노출시키는 코드화로 만든 기능입니다.
</Callout>

**⑥ 저장** \
\[저장] 버튼을 클릭해 자동 SMS 설정을 완료해 주세요.

***

#### <mark style="color:blue;">■</mark> <mark style="color:blue;"></mark><mark style="color:blue;">**발송 상세 설정(SMS 야간발송 제한 설정)**</mark>

* 주문배송, 프로모션, 문의 관련 세부 상황별 SMS의 야간시간 발송을 제한할 수 있습니다. \
  📍<mark style="color:red;">야간시간 : 오후 9시부터 다음날 오전 8시</mark>
* 항목별 야간시간 발송 제한의 전체설정 또는 개별설정이 가능합니다.
* 야간발송 제한이 필요한 이슈 상태 템플릿을 선택하고 \[저장] 버튼을 클릭해 발송 제한 설정을 완료 주세요.

<Callout type="danger">
* **정보통신망법 제 50조 (영리목적의 광고성 정보 전송 제한)**\
  광고성 SMS 전송은 야간시간대인 오후 9시부터 그 다음 날 오전 8시까지 발송이 불가합니다.

* 프로모션 > 적립금과 쿠폰 지급 안내 템플릿은 영리 목적이 있는 광고성 내용의 문구로, \
  야간시간대 자동 발송을 제한해 주세요.

* 야간시간 SMS 발송이 필요한 경우, 반드시 고객에게 야간시간대 SMS 수신 동의를 받거나,\
  법에서 정한 명시사항 문구를 포함하여 발송해야 하는 점을 유의해 주세요.\
  \- SMS 내용 시작 부분에 (광고)임을 표기\
  \- 전송자의 명칭 (쇼핑몰명 등) 표기\
  \- SMS의 수신거부 또는 수신동의 철회 방식 방법과 해당 방식이 무료임을 안내하는 문구 삽입
</Callout>
