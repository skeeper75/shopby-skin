# 카카오톡 채널 생성

<Callout type="info">
**카카오 알림톡 발송을 위한 카카오 비즈니스 채널 생성 프로세스 입니다.**

**보다 자세한 내용은 카카오에서 제공하는** [**카카오톡 채널 이용가이드 > 채널만들기**](https://kakaobusiness.gitbook.io/main/channel/start)**에서 확인해 주세요.**
</Callout>

① [카카오 비즈니스 사이트](https://business.kakao.com/) > 상단 \[채널] 클릭 > \[카카오톡 채널 시작하기+] 클릭해 주세요.

<div align="left"><figure><img src="/images/management/kakao-notification/undefined/undefined/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FK3PuXuaZrOhKs3O0llCD%2F%EC%B9%B4%EC%B9%B4%EC%98%A4%20%EB%B9%84%EC%A6%88%EB%8B%88%EC%8A%A4.jpg" alt="" width="563"/><figcaption><p>&#x3C;카카오 비즈니스></p></figcaption></figure></div>

② 카카오 계정으로 로그인을 진행합니다.

* 카카오 채널은 카카오 비즈니스 통합회원 계정으로 생성할 수 있습니다. \
  📍카카오 계정 → 약관 동의 및 인증 진행 → 카카오 비즈니스 통합회원 전환

③ \[+ 새 채널 만들기] 버튼을 클릭해 주세요.

④ '비즈니스 채널 개설하기' 화면에서 사업자 등록번호 검색으로 채널 개설 여부를 조회합니다.

* 기존에 개설된 채널이 없으면, \[+ 새 채널 만들기] 버튼을 클릭합니다.
* 기존에 개설된 채널이 있다면, 기존 채널에 권한 요청을 진행합니다.

⑤  \[+ 새 채널 만들기] 버튼을 클릭하면 '비즈니스 정보 입력' 화면으로 이동합니다.

* 사업자 정보를 입력합니다.\
  📍카카오 비즈니스에 사전에 등록한 사업자 정보가 있다면 자동 연동되어 노출됩니다.
* 통신판매신고 정보를 입력합니다.&#x20;
* 개설하는 채널의 사업자등록증과 업종별 인허가 서류를 등록합니다.
* 채널을 개설하는 신청인 정보 확인을 진행합니다.

⑥ \[다음] 버튼을 클릭해 '채널 정보 입력' 화면으로 이동합니다.

* 카카오톡 프로필 사진, 채널 이름, 검색용 아이디 등 채널정보를 입력해 주세요.\
  📍검색용 아이디는 카카오톡에서 사용자가 검색하는 채널 검색어로 사용되며, \
  &#x20;    카카오 알림톡 사용 설정에 입력하는 플러스친구 아이디입니다.
* 채널과 비즈니스의 관계, 개인정보 수집 이용 동의 항목 등을 확인합니다.

⑦ \[완료] 버튼을 클릭하면, 비즈니스 채널 개설 신청이 완료되고 심사가 진행됩니다.

⑧ 심사가 완료되면, 카카오톡 채널 관리자 센터 > 내 채널 > 대시보드로 이동해 주세요.

* **프로필 > 채널홈 설정 > 기본정보** 항목에서 <mark style="background-color:green;">**고객센터 정보를 입력**</mark>합니다.\
  📍<mark style="color:red;">입력한 고객센터 번호로 카카오 채널 아이디(플러스친구 아이디) 등록에 필요한</mark>\
  &#x20;    <mark style="color:red;">관리자 인증번호가 발송됩니다.</mark> <br>
* **프로필 > 프로필 설정 > 옵션설정** 항목에서 **채널 공개**와 **검색 허용**을 <mark style="background-color:green;">**ON 상태로 변경**</mark>합니다.
* **프로필 > 프로필 설정 > 채널설정** 항목에서 <mark style="background-color:green;">**카카오 채널 검색용 아이디**</mark>를 확인할 수 있습니다.&#x20;
