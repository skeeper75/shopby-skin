# 메일 템플릿 설정

<Callout type="info">
**자주 발송하는 메일 제목과 내용을 템플릿으로 등록하고 관리하는 메일 템플릿 설정 메뉴 입니다.**

**등록한 메일 템플릿은 광고성/정보성 메일의 수동 발송에 사용할 수 있습니다.**
</Callout>

## 메일 템플릿 설정

**① 메일 템플릿 리스트**\
폴더와 템플릿 추가, 수정 및 삭제, 순서 변경 등 메일 템플릿 상세 설정이 가능합니다.

**② 폴더 추가**\
새로운 메일 템플릿을 그룹으로 묶어 폴더 별 관리가 가능합니다.  \
\[폴더 추가] 버튼을 클릭하면, 메일 템플릿 리스트 하단에 \[템플릿 폴더명 입력]으로 새로운 템플릿 폴더가 생성됩니다.

* 관리 용도에 맞는 이름을 폴더명에 입력하고, 사용여부를 설정해 주세요.
* \[저장] 버튼을 클릭해 폴더 추가를 완료해 주세요.

**③ 템플릿 추가**\
새로운 메일 템플릿 추가를 위해 템플릿 폴더를 선택하고, \[템플릿 추가] 버튼을 클릭합니다. \
메일 템플릿 리스트 하단에 \[템플릿명 입력]으로 새로운 템플릿이 생성됩니다.

* 사용 용도에 맞는 이름을 템플릿명에 입력하고, 사용여부를 설정해 주세요.
* 광고성/정보성 메일에 따른 알맞은 메일 제목과 내용을 작성합니다.
* \[저장] 버튼을 클릭해 템플릿 추가를 완료해 주세요.

<Callout type="warning">
템플릿 폴더 생성은 필수사항이 아니므로, 폴더를 생성하지 않고 메일 템플릿만 추가하여 사용할 수 있습니다.
</Callout>

**④ 상/하 이동 아이콘 (순서변경)**\
이동 아이콘으로 템플릿 폴더와 템플릿 순서를 변경할 수 있습니다.

**⑤ 삭제** \
삭제할 메일 템플릿 폴더 또는 템플릿을 선택하고 \[삭제] 버튼을 클릭합니다.
