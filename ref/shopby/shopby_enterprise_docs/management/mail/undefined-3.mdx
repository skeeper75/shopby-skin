# 자동 메일 설정

<Callout type="info">
**자동 메일 설정 메뉴에서는 쇼핑몰의 특정 상태 또는 다양한 이슈에서 자동으로 발송되는 이메일을 설정하고 관리합니다.**

**회원, 주문배송, 프로모션, 문의 관련하여 세부 상황별 메일 템플릿을 제공합니다.**
</Callout>

**① 자동메일 유형 (자동 메일 발송 템플릿)**\
회원 관련, 주문배송 관련, 프로모션 관련, 문의 관련으로 구분된 메일 템플릿을 확인할 수 있습니다.\
각각의 메일 템플릿 폴더를 펼치기하면 세부 상황별 템플릿이 확인됩니다.

**② 사용여부**\
세부 상황별 템플릿을 선택해 자동 메일 발송 사용여부를 설정할 수 있습니다.\
사용여부를 설정하면 '사용함'과  '사용 안 함'으로 표기됩니다.

**③ 발송 이메일 설정**\
세부 상황별 템플릿을 선택해 발신자 메일 주소로 사용할 이메일 주소를 설정할 수 있습니다.\
고객센터 이메일과 발송전용 이메일 중 선택합니다.\
📍고객센터 이메일과 발송전용 이메일은 [서비스관리 > 쇼핑몰 수정 > 고객센터 정보 ](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/mall/modification#undefined-8) 메뉴의\
&#x20;    고객센터 e메일 주소와 발송전용 e메일 주소에서 설정할 수 있어요.<br>

**④ 메일 제목 / 메일 본문**\
세부 상황별 템플릿의 메일 제목이 확인됩니다.\
에디터 영역에 치환텍스트와 HTML로 작성된 메일 본문이 확인됩니다.\
📍자동 메일 발송 템플릿(제목과 본문)은 쇼핑몰 운영상황 및 콘셉트에 맞게 수정 및 보완할 수 있어요.

**⑤ 기본 템플릿 복원**\
메일 제목 및 본문을 수정한 경우, \[기본 템플릿 복원] 버튼을 클릭하면 샵바이에서 제공하는 기본 템플릿 내용으로 복구됩니다.

**⑤ 치환텍스트**\
\[치환텍스트] 버튼을 클릭하면 메일 제목 및 본문에 활용할 수 있는 다양한 치환텍스트가 확인됩니다.

<Callout type="warning">
치환텍스트는 쇼핑몰 어드민 여러 메뉴 및 항목에 설정된 특정 정보를, 해당하는 내용으로 불러와 노출시키는 코드화로 만든 기능입니다.
</Callout>

**⑥ 저장**\
\[저장] 버튼을 클릭해 자동 메일 설정을 완료해 주세요.

**⑦ 테스트 전송**\
\[테스트 전송] 버튼을 클릭하면 선택한 자동 메일 템플릿이 운영자 이메일 주소로 테스트 전송됩니다.\
📍메일 제목 및 본문을 수정하였다면, 테스트 전송으로 점검 및 확인을 해 주세요.&#x20;

<Callout type="danger">
* **프로모션 관련** \
  법령 준수를 위해 메일 수신 동의를 한 회원에게만 메일이 자동발송되며, \
  거부를 한 회원에게는 자동 메일이 발송되지 않습니다.\
  메일을 발송하면 본문 하단에 수신 거부 안내문이 노출됩니다.

\ <br>

* **회원 관련 > 광고성 이메일 수신현황 안내 / 개인정보 이용내역 안내**\
  개인정보 보호법 제39조의 8에 따라 개인정보 이용내역을 주기적으로, 연 1회 이상 \
  고객에게 통지해야 합니다.\
  정보통신망법 제50조에 따라 2년마다 광고성 정보 수신자의 수신 동의 여부를 \
  확인하여야 합니다.\
  광고성 이메일 수신현황 안내 / 개인정보 이용내역 안내는 정보통신망법 법적 기준에 따라 \
  필수 발송해야하는 이메일로 템플릿 사용 여부를 '사용함' 으로 설정해야 합니다.\
  매 월 1일 안내 대상 회원에게 자동 메일이 발송되므로 참고하여 활용해 주세요.
</Callout>
