# 메일 발송

✅ e메일 수동 발송은 [서비스 관리 > 쇼핑몰 수정 >  e-mail 발송 사용 설정](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/mall/modification#e-mail) 메뉴에서 발송을 '사용함'으로 \
설정하고, e-mail Appkey키를 등록해야 사용할 수 있어요.

<Callout type="info">
**쇼핑몰 운영관련 광고성/정보성 메일을 수동으로 발송할 수 있습니다.**

**광고성 메일은 수신자가 수신거부를 선택할 수 있도록 별도 수신거부 기능이 제공됩니다.**
</Callout>

## 메일 발송

**① 메일 유형**\
수동으로 발송하는 메일 유형을 광고성 메일과 정보성 메일 중에 선택합니다.

<Callout type="success">
**광고성 : 운영자가 경제적 이득을 취할 목적으로 발송하는 메일**&#x20;

**정보성 : 정보통신망법 안내서에 ‘영리목적 광고성 정보의 예외’에 해당하는 메일**\
예) 수신자가 구매한 상품에 대한 설명, 보증, 제품 리콜, 안전 또는 보안 관련 정보 제공\
&#x20;    상품 및 서비스에 대한 조건 및 특징에 대한 변경 안내 (예 : 포인트 제도 변경 안내) 등
</Callout>

**② 템플릿 설정**\
자주 발송하는 광고성/정보성 메일은 템플릿으로 만들어 적용할 수 있습니다.&#x20;

* '사용 안 함'을 선택해 메일 제목과 내용을 직접 작성하여 메일을 발송합니다.
* '사용 함'을 선택하면 \[템플릿 선택] 버튼이 활성화 됩니다.  \
  \[템플릿 선택] 버튼을 클릭하면 '[메일 템플릿 선택](#undefined-1)' 창이 노출됩니다.

<Callout type="warning">
메일 템플릿을 사용하면 메일 제목과 내용은 비활성화 되고, 선택한 템플릿의 제목과 내용으로 자동 적용됩니다.\
'[e메일 관리 > 메일 템플릿 설정](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/management/mail/undefined-1)' 메뉴에 템플릿을 사전에 등록해야 선택할 수 있어요.
</Callout>

**③ 발송메일**\
발신자로 노출되는 발송 전용 메일 주소를 확인합니다.\
📍'[서비스 관리 > 쇼핑몰 수정 > 고객센터 정보](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/mall/modification#undefined-8)' 메뉴의 발송전용 e메일 주소에서 수정할 수 있어요.

**④ 수신 대상 회원 선택**\
메일을 수신할 회원을 선택해 주세요.\
선택한 회원 중 회원가입 시 이메일 수신 동의를 하지 않은 '수신 거부 대상자 수'가 별도 노출됩니다.

* 전체회원 : 쇼핑몰에 가입한 전체 회원에게 메일을 발송합니다. &#x20;
* 회원선택 : 특정 회원등급 및 회원그룹에 속한 회원과 개별 회원을 대상으로 메일을 발송합니다.&#x20;
* 엑셀 업로드 : 엑셀 양식에 작성한 회원번호와 이메일 주소로 메일을 발송합니다.
* CRM 그룹 선택  : 회원 추출 완료한 CRM 그룹 대상으로 메일 발송합니다. ([CRM그룹 관리 가이드](https://support-help.nhn-commerce.com/common/crm-group))

**⑤ 발송 설정**\
즉시 발송과 예약 발송 중에서 선택합니다.

* 즉시 발송 : 메일을 지금 즉시 발송합니다.
* 예약 지급 : 발송일자를 미래 시점으로 설정하여 예약 발송 메일로 등록합니다.

**⑥ 제목**\
선택한 광고성/정보성 메일 유형에 알맞은 메일 제목을 작성해 주세요.\ <mark style="color:red;">📍</mark>광고성 메일은 제목이 시작되는 앞부분에 (광고) 문구를 반드시 표시해야 해요.\ <mark style="color:red;">📍</mark>광고성 메일 유형 선택 시, 메일 제목 앞에 자동으로 (광고) 문구가 삽입됩니다.&#x20;

**⑦  내용**\
선택한 광고성/정보성 메일 유형에 알맞은 메일 내용을 작성해 주세요.

**⑧ \[미리보기]** \
\[미리보기] 버튼을 클릭하면 설정한 메일 발송 내용을 미리보기로 확인할 수 있습니다.

**⑨ \[초기화]** \
\[초기화] 버튼을 클릭하면, 설정한 메일 발송 내용이 초기화 처리 됩니다.

**⑩ \[보내기]** \
\[보내기] 버튼을 클릭해 수동 메일 발송을 완료해 주세요.\
📍에약 발송 메일은 등록 설정됩니다.<br>

***

#### <mark style="color:blue;">■ 메일 템플릿 선택</mark>

* '메일 템플릿 선택' 창에서 메일로 발송할 폴더와 템플릿을 선택합니다.
* 메일 제목과 내용을 확인하고  \[탬플릿 선택] 버튼을 클릭해 주세요.
