# 메일 발송내역 조회

<Callout type="info">
**자동 메일 발송이 '사용함'으로 설정되어, 자동 발송된 메일 발송내역을 조회할 수 있습니다.**

**테스트 전송 메일도 조회되는 점을 참고해 주세요.**
</Callout>

**① 발송대상**\
자동 메일이 발송된, 수신자 메일 주소를 입력하여 발송내역을 검색합니다.

**② 발송요청 일시**\
자동 메일 발송 일자를 지정하여 발송내역을 검색합니다.\
📍기간 지정은 최대 1개월 단위로 조회가  가능합니다.

**③ 발송상태**\
자동 메일 발송 상태로 발송내역을 검색합니다.

* 발송 중 : 자동 발송이 요청되었지만, 발송결과를 수신받지 못한 상태
* 발송실패 : 자동 발송 요청이 실패된 상태 또는 발송결과 실패 처리 상태
* 발송성공 : 자동 발송 요청이 정상으로 완료되어 수신을 받은 상태

**④** [**메일제목**](#undefined)\
\[메일제목]을 클릭하면 발송한 자동 '메일 발송내용 상세보기' 창이 노출됩니다.
