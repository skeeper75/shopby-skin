# 답변 템플릿 관리

<Callout type="info">
**자주 묻는 1:1 문의의 답변 또는 답변 양식을 템플릿으로 만들어 답변 내용에 활용할 수 있습니다.**
</Callout>

## 답변 템플릿 설정

<figure><img src="/images/management/inquiry/template/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FgBbZ1KxNXwI9g0JmyOH5%2F%EC%9D%BC%EB%8C%80%EC%9D%BC%EB%AC%B8%EC%9D%98%EA%B4%80%EB%A6%AC_%EB%8B%B5%EB%B3%80%ED%85%9C%ED%94%8C%EB%A6%BF%EC%B6%94%EA%B0%80.jpg" alt=""/><figcaption><p>&#x3C;답변 템플릿 설정></p></figcaption></figure>

**① 답변 템플릿 추가** \
답변 템플릿 등록을 위해 \[+ 답변템플릿 추가] 버튼을 클릭합니다. \
답변 템플릿 리스트에 \[새 답변템플릿] 으로 신규 답변 템플릿이 생성됩니다.&#x20;

**② 답변 템플릿 리스트**\
생성한 답변 템플릿의 설정, 삭제, 수정, 순서 변경 등이 가능합니다.&#x20;

**③ 삭제**\
삭제할 답변 템플릿을 선택하고 \[- 삭제] 버튼을 클릭합니다.&#x20;

**④ 상/하 이동 아이콘 (순서변경)**&#x20;

* 순서 변경이 필요한 답변 템플릿을 선택합니다.&#x20;
* ∧(상) 이동 아이콘을 클릭하면 현재 위치보다 한 단계 위, ∨(하) 이동 아이콘은 한 단계 아래로 해당 답변 템플릿이 이동됩니다.

**⑤** \[저장] 버튼을 클릭해 답변 템플릿 설정 및 등록을 완료합니다.

## 답변 템플릿 상세설정

<figure><img src="/images/management/inquiry/template/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F5ABnTarMUKA6AngBI1rT%2F%EC%9D%BC%EB%8C%80%EC%9D%BC%EB%AC%B8%EC%9D%98%EA%B4%80%EB%A6%AC_%EB%8B%B5%EB%B3%80%ED%85%9C%ED%94%8C%EB%A6%BF%EC%83%81%EC%84%B8%EC%84%A4%EC%A0%95.jpg" alt=""/><figcaption><p>&#x3C;답변 템플릿 상세설정></p></figcaption></figure>

**①** 답변 템플릿명, 답변내용을 입력하여 답변 템플릿 상세정보를 설정할 수 있습니다.

**②** 새롭게 추가한 \[새 답변템플릿] 을 클릭합니다.

* 1:1문의 답변에 활용할 답변 템플릿명과 답변 내용을 입력합니다.
* 반드시 \[저장] 버튼을 클릭해 답변 템플릿 상세설정을 완료해 주세요.

**③** 설정한 답변 템플릿 정보는 답변 템플릿명을 클릭해 상세설정에서 수정할 수 있습니다.
