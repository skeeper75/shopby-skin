# 문의유형/담당자관리

<Callout type="info">
**고객이 1:1 문의 등록 시 문의 내용 종류로 선택하는 문의유형을 다양하게 추가하고, 문의유형별 담당자를 별도 지정하여 관리합니다.**
</Callout>

## 문의유형 설정

<figure><img src="/images/management/inquiry/category/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F77x4xl0nBAi9IuzNW133%2F%EC%9D%BC%EB%8C%80%EC%9D%BC%EB%AC%B8%EC%9D%98%EA%B4%80%EB%A6%AC_%EB%AC%B8%EC%9D%98%EC%9C%A0%ED%98%95%EB%8B%B4%EB%8B%B9%EC%9E%90%EA%B4%80%EB%A6%AC_%EB%AC%B8%EC%9D%98%EC%9C%A0%ED%98%95%EB%A6%AC%EC%8A%A4%ED%8A%B8.jpg" alt=""/><figcaption><p>&#x3C;문의유형 설정></p></figcaption></figure>

**① 문의유형 리스트**

* 기타, 회원/정보관리, 주문/결제, 배송, 반품/환불/교환/AS, 상품/이벤트는 기본으로 제공되는 문의유형으로 지정된 담당자 수가 확인됩니다.
* 기본 문의유형도 문의유형 삭제와 순서 변경, 문의유형명 수정 등 모든 상세 설정이 가능합니다.\
  📍답변대기 및 진행중 처리상태에 해당하는 문의유형은 삭제가 불가해요.

**② 문의유형 추가**\
새로운 문의유형 등록을 위해 \[+ 문의유형 추가] 버튼을 클릭합니다.\
문의유형 리스트 하단에 \[새 문의유형] 으로 새로운 문의유형이 추가됩니다.

**③ 삭제**\
삭제할 문의유형을 선택하고 \[- 삭제] 버튼을 클릭합니다.

**④** [**상/하 이동 아이콘 (문의유형 순서변경)**](#undefined-2)\
이동 아이콘으로 문의유형의 순서를 변경할 수 있습니다.

**⑤** \[저장] 버튼을 클릭해 문의유형 설정 및 등록을 완료합니다.

## 문의유형 상세설정 / 담당자 설정

<figure><img src="/images/management/inquiry/category/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FDIxSOsFjVcjBgOcxZQ4w%2F%EC%9D%BC%EB%8C%80%EC%9D%BC%EB%AC%B8%EC%9D%98%EA%B4%80%EB%A6%AC_%EB%AC%B8%EC%9D%98%EC%9C%A0%ED%98%95%EB%8B%B4%EB%8B%B9%EC%9E%90%EA%B4%80%EB%A6%AC.jpg" alt=""/><figcaption><p>&#x3C;문의유형 상세설정 / 담당자 설정></p></figcaption></figure>

**①** 문의유형명, 문의유형 설명, 문의유형 담당자 지정 등 문의유형 상세정보를 설정합니다.

**②** 새롭게 추가한 \[새 문의유형] 을 클릭합니다.

* 문의유형 이름과 문의유형에 대한 설명을 입력합니다.
* 담당자 항목에서 해당 문의유형의 담당자를 지정합니다.\
  📍문의유형 담당자는 필수로 지정해야 해요.\
  📍1:1문의 관리 메뉴 권한이 있는 운영자만 담당자로 노출되는 점을 참고해주세요.\
  &#x20;    (운영자 등록 : [서비스 관리 > 운영자 관리 > 운영자 계정관리 > 신규 운영자 계정 등록](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/admin/list#undefined-2))
* 문의유형 담당자는 복수 지정이 가능합니다. \
  📍\[담당자를 선택해 주세요] 셀렉 박스를 클릭하면 쇼핑몰에 등록된 담당자 리스트가 노출되고, \
  &#x20;    담당자를 클릭해 선택합니다.

**③** 모든 상세정보를 설정하고, 반드시 \[저장] 버튼을 클릭해 문의유형 상세설정을 완료해 주세요.

**④** 설정한 문의유형 정보는 문의유형명을 클릭해 상세설정에서 수정할 수 있습니다.

***

#### <mark style="color:blue;">**■ 문의유형 순서변경**</mark>

* 순서 변경이 필요한 문의유형을 선택합니다.&#x20;
* ∧(상) 이동 아이콘을 클릭하면 현재 위치보다 한 단계 위, ∨(하) 이동 아이콘은 한 단계 아래로 해당 문의유형이 이동됩니다.
