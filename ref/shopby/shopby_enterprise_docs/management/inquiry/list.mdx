# 1:1문의 관리

<Callout type="info">
**고객이 쇼핑몰 고객센터 페이지에서 등록한 1:1 문의사항을 조회하고, 답변을 작성합니다.**
</Callout>

## 1:1문의 관리 검색조건

<figure><img src="/images/management/inquiry/list/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F516GRmPYD9uOySEoX3a3%2F%EC%9D%BC%EB%8C%80%EC%9D%BC%EB%AC%B8%EC%9D%98%EA%B4%80%EB%A6%AC_%EB%AC%B8%EC%9D%98%EA%B4%80%EB%A6%AC.jpg" alt=""/><figcaption><p>&#x3C;1:1문의 관리 검색조건></p></figcaption></figure>

**① 쇼핑몰**\
등록된 1:1 문의를 조회할 쇼핑몰을 선택합니다.

**② 문의일시**\
등록된 1:1문의는 최대 1년까지의 기간으로 검색이 가능합니다.

**③ 검색어**\
문의번호, 문의제목, 문의자, 1:1문의 유형의 담당자 등을 입력하여 1:1문의를 검색합니다.

**④ 문의유형/담당자**\
문의유형과 해당 문의유형의 담당자를 선택하여 검색합니다.

* 회원/정보관리, 주문/결제, 배송, 반품/환불/교환/AS, 상품/이벤트, 기타는 기본으로 제공되는 문의유형입니다.
* [문의유형/담당자 관리](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/management/inquiry/category) 메뉴에서 문의유형과 담당자를 추가 등록해야 검색조건에 노출됩니다.

**⑤ 처리상태**\
1:1문의 답변의 작성 및 고객 안내(전송) 상태입니다.

* 답변대기 : 답변을 작성하기 전 상태
* 진행중 : 답변을 작성하고 임시저장을 한 상태\
  📍 단, 작성한 답변이 있는 임시저장 상태 답변은 다시 임시저장 불가
* 답변완료 : 답변을 작성하고 고객 안내(전송)를 완료한 상태

## 1:1문의 관리 검색결과 (1:1문의 리스트)

<figure><img src="/images/management/inquiry/list/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F1JhbV27woU22oU6HCXMH%2F%EC%9D%BC%EB%8C%80%EC%9D%BC%EB%AC%B8%EC%9D%98%EA%B4%80%EB%A6%AC_%EB%AC%B8%EC%9D%98%EA%B4%80%EB%A6%AC%EA%B2%80%EC%83%89%EA%B2%B0%EA%B3%BC.jpg" alt=""/><figcaption><p>&#x3C;1:1문의 관리 검색결과 (1:1문의 리스트)></p></figcaption></figure>

**① 문의유형/담당자 변경**\
1:1문의를 선택하고 \[문의유형/담당자 변경]을 클릭하면 '[문의유형/담당자 변경](#undefined)' 창이 노출됩니다.

**② 문의번호 / 문의제목**\
\[문의번호] 및 \[문의제목] 을 클릭하면 '[1:1 문의 상세보기](#id-1-1-2)' 창이 노출됩니다.

**③ 문의자**\
문의자의 \[회원아이디] 를 클릭하면 고객의 최근 주문/1:1문의/상품문의 내역 등을 확인할 수 있는 고객정보관리시스템 CRM 창이 노출됩니다.

**④ 보기** \
검색한 1:1문의를 30, 50, 100, 200개 단위로 볼 수 있습니다.

**⑤ 생성**\
검색한 1:1문의 리스트가 엑셀파일로 생성되며, 압축된 엑셀파일을 다운로드 받을 수 있습니다.

***

#### <mark style="color:blue;">**■ 문의유형/담당자 변경**</mark>

* 선택한 1:1문의의 문의유형과 담당자를 변경할 수 있습니다.
* 단, 답변완료 상태의 1:1문의는 변경이 불가합니다.
* 문의유형과 담당자를 선택하고, 변경 사유를 입력한 다음 \[저장] 버튼을 클릭해 변경을 진행합니다.&#x20;
* [문의유형/담당자 관리](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/management/inquiry/category) 메뉴에 등록되어 있는 문의유형과 담당자만, 해당 창에서 노출되고 선택이 가능합니다.

#### <mark style="color:blue;">**■ 1:1문의 상세보기**</mark>

* 1:1 문의 정보 : 해당 1:1 문의 정보를 확인하고, \[변경] 버튼으로 문의유형과 담당자를 변경할 수 있습니다.
* 1:1 문의 내용 : 고객이 작성한 1:1 문의내용이 자세하게 확인됩니다. \
  또한 고객이 1:1 문의 등록 시 첨부한 파일을 돋보기 아이콘을 클릭해 확인할 수 있습니다.
* 1:1 문의 답변 : 입력란에 답변 내용을 작성합니다. \
  해당 1:1 문의에 적절한 ‘답변 템플릿’을 선택하면, 답변 템플릿에 등록한 답변 또는 답변 양식 내용이 자동 입력됩니다. \
  📍답변 템플릿을 선택하면 작성 중인 답변 내용은 초기화 되는 점을 유의해 주세요.
* \[임시저장] 버튼을 클릭하면 작성한 답변 내용이 임시저장되며, 처리상태가 진행 중으로 변경됩니다.
* \[답변완료] 버튼을 클릭하면 작성한 답변 내용이 고객에게 안내(전송)되며, 처리상태가 답변완료로 변경됩니다.
* \[취소] 버튼을 클릭하면 ‘1:1 문의 상세보기’ 창이 닫힙니다.
