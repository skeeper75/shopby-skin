# 카카오 친구톡

<Callout type="info">
**카카오 채널을 추가한 사용자(채널 친구)에게 캠페인성 광고 메시지를 발송할 수 있는 카카오 친구톡 기능입니다.**&#x20;

**카카오 친구톡은 비즈니스 인증을 받고, 채널공개과 검색이 허용된 '카카오톡 채널'을 카카오톡 발신 프로필로 사용해야 합니다.**
</Callout>

&#x20;**✅&#x20;**<mark style="background-color:yellow;">**카카오 친구톡 매뉴얼은**</mark> [<mark style="color:green;background-color:yellow;">**'친구톡 앱 사용 가이드'**</mark>](https://store-help.nhn-commerce.com/app/counsel/kakao_friend_talk) <mark style="background-color:yellow;">**에서 보다 자세하게 안내해 드립니다.**</mark>

&#x20;     (카카오톡 비즈니스 채널 생성은 ['shop by 매뉴얼 > 카카오톡 채널 생성'](https://nhn-commerce.gitbook.io/shopby_premium_manual/management/kakao-notification/undefined/undefined) 에서도 확인 가능해요.)
