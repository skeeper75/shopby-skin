# 게시판 관리

<Callout type="info">
**쇼핑몰 운영에 필요한 공지사항을 전달하고, 이벤트 진행 및 커뮤니티 용도 등으로 활용하는 게시판의 기능을 개별 설정할 수 있습니다.**

**게시판 종류와 특성에 맞게 게시판명과 게시판 카테고리 등을 등록하고, 글쓰기 및 읽기 권한 설정, 비밀글/답글 작성 가능 여부 등을 설정합니다.**
</Callout>

## 게시판 설정

<figure><img src="/images/management/board/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FDUgPjJK9ZfdQYGmdyGbZ%2F%EC%9A%B4%EC%98%81%EA%B4%80%EB%A6%AC_%EA%B2%8C%EC%8B%9C%ED%8C%90%EA%B4%80%EB%A6%AC_%EA%B2%8C%EC%8B%9C%ED%8C%90%EC%84%A4%EC%A0%95.jpg" alt=""/><figcaption><p>&#x3C;게시판 설정></p></figcaption></figure>

**① 게시판 리스트**

* FAQ, 공지사항, 자유 게시판, 이벤트/혜택, 메모 게시판은 기본으로 제공되는 게시판입니다.
* 기본 게시판도 게시판명의 수정과 게시판 삭제 및 순서 변경 등 모든 상세 설정이 가능합니다.

**② 게시판 추가**\
새로운 게시판 추가를 위해 \[+ 게시판 추가] 버튼을 클릭합니다.\
게시판 리스트 하단에 \[게시판명 입력]으로 새로운 게시판이 추가됩니다.

**③ 삭제**\
삭제할 게시판을 선택하고 \[- 삭제] 버튼을 클릭합니다.&#x20;

<Callout type="danger">
게시글이 등록되어 있지 않은 게시판만 삭제가 가능해요.
</Callout>

**④ 상/하 이동 아이콘 (게시판 순서변경)**

* 순서 변경이 필요한 게시판을 선택합니다.&#x20;
* ∧(상) 이동 아이콘을 클릭하면 현재 위치보다 한 단계 위, ∨(하) 이동 아이콘은 한 단계 아래로 해당 게시판이 이동됩니다.

## 게시판 상세설정 / 게시판 카테고리

<figure><img src="/images/management/board/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FdkFaxKkne1es2a5oFu5M%2F%EC%9A%B4%EC%98%81%EA%B4%80%EB%A6%AC_%EA%B2%8C%EC%8B%9C%ED%8C%90%EA%B4%80%EB%A6%AC_%EA%B2%8C%EC%8B%9C%ED%8C%90%EC%83%81%EC%84%B8%EC%84%A4%EC%A0%95.%EC%B9%B4%ED%85%8C%EA%B3%A0%EB%A6%AC.jpg" alt=""/><figcaption><p>&#x3C;게시판 상세설정 / 게시판 카테고리></p></figcaption></figure>

**① 게시판 번호**\
게시판에 부여된 시스템 고유번호로 수정이 불가합니다.

**② 게시판 ID**\
쇼핑몰에서 노출되는 해당 게시판의 페이지(주소) 구분 값으로 사용됩니다.

<Callout type="success">
공지사항 게시판 ID → notice\
쇼핑몰 주소 → <https://abcde.co.kr&#x20>;

**공지사항 게시판 페이지(주소) → <https://abcde.co.kr/notice>**
</Callout>

**③ 게시판 사용설정**\
해당 게시판의 사용 가능 대상을 설정합니다.&#x20;

* 전체(회원+비회원) : 회원과 비회원 모두 게시판 사용이 가능해요.
* 회원 전용 : 쇼핑몰에 회원가입한 고객만 게시판 사용이 가능해요.
* 특정 회원 전용 : 쇼핑몰 회원 등급 또는 회원 그룹에 해당하는 고객만 게시판 사용이 가능해요.\
  📍\[회원등급/그룹 선택] 버튼을 클릭해 특정 회원 등급과 그룹을 설정할 수 있어요.
* ‘사용 안 함’으로 설정된 게시판 페이지는 접근이 불가해요.

**④ 게시판명**\
게시판 특성에 맞는 이름을 입력합니다.

**⑤ 게시판 설명**\
게시판 사용 용도 등 게시판 관련 내용을 입력할 수 있습니다. \
📍게시판 설명은 내부 관리용으로 쇼핑몰에 노출되지 않아요.

**⑥ 게시글 대표 이미지 사용**\
게시글에 노출되는 대표 이미지 사용 여부를 설정합니다.

**⑦ 카테고리 사용**\
해당 게시판에 작성되는 게시글 유형을 카테고리로 분류하여 사용할 수 있습니다.

* '사용함’을 선택하면 카테고리를 등록할 수 있는 게시판 카테고리 항목이 하단에 노출되요.
* 게시판 카테고리는 1개 이상 등록이 가능해요.
* '카테고리1 ' 입력란에 카테고리명을 입력해요.
* \[카테고리 추가] 버튼을 클릭해 카테고리를 추가하고 \[삭제] 버튼으로 삭제할 수 있어요.

<Callout type="warning">
카테고리를 등록하여 사용 중인 게시판의 카테고리 사용을 '사용 안 함'으로 설정할 경우, 해당 게시판 카테고리에 등록된 기존 게시글은 자동으로 이동되지 않는 점을 참고해주세요.
</Callout>

**⑧ 글쓰기 권한 설정**\
해당 게시판에 글쓰기가 가능한 대상을 설정합니다.

**⑨ 읽기 권한 설정**\
해당 게시판에 작성된 게시글 읽기가 가능한 대상을 설정합니다.

**⑩ 비밀 글쓰기** ![](/images/management/board/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FELvojgd5TjDfxqfuS3tO%2FAPI%20%EB%B3%B5%EC%82%AC7.png) \
비밀글 작성 가능 여부를 설정합니다.

**⑪ 답글**\
답글 작성 가능 여부를 설정합니다.

**⑫ 답글 쓰기 권한 설정**\
해당 게시판에 답글 쓰기가 가능한 대상을 설정합니다.

**⑬ 답글 읽기 권한 설정**\
해당 게시판에 작성된 답글글 읽기가 가능한 대상을 설정합니다.

**⑭ 첨부파일**\
첨부파일 등록 가능 여부를 설정합니다.

**⑮ 작성자 표시 방법**\
해당 게시판에 게시글을 작성한 작성자 정보를 회원명 / 회원아이디 / 닉네임 / 회원 이메일 주소 중에서 설정할 수 있습니다.

**⑯** \[저장] 버튼을 클릭해 게시판 상세 설정을 완료해 주세요.
