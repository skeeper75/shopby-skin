# 신규 게시글 등록

## 게시글 등록

<figure><img src="/images/management/post/undefined/register/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FCqpmR5duPhjCfJQuFFhR%2F%EC%9A%B4%EC%98%81%EA%B4%80%EB%A6%AC_%EA%B2%8C%EC%8B%9C%EA%B8%80%EA%B4%80%EB%A6%AC_%EA%B2%8C%EC%8B%9C%EA%B8%80%EB%93%B1%EB%A1%9D.jpg" alt=""/><figcaption><p>&#x3C;게시글 등록></p></figcaption></figure>

**①** \[신규 게시글 등록] 버튼을 클릭하면, '게시글 등록' 창이 노출됩니다.

**②** 등록하는 게시글의 제목을 입력합니다.

**③** 게시글이 등록되는 쇼핑몰과, 게시판 종류 및 게시판 카테고리를 선택합니다.&#x20;

**④** 등록하는 게시글의 공개여부, 공지 및 비밀글 등을 설정합니다.

* 공지 게시글은 조회 결과(게시글 리스트)에서 일반 게시글보다 상단에 항상 위치하며, '[공지 아이콘](#undefined-1)'이 표시됩니다.

<Callout type="warning">
여러 개의 공지 게시글이 있는 경우 최종 수정 일시를 기준으로 최근 공지 게시글이 상단에 위치합니다.  \
최종 수정 일시가 없는 경우 작성 일시를 기준으로 위치합니다.
</Callout>

**⑤ 태그** ![](/images/management/post/undefined/register/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FELvojgd5TjDfxqfuS3tO%2FAPI%20%EB%B3%B5%EC%82%AC7.png)\
게시글의 연관 주제 키워드인 태그를 생성하여 게시글에 노출할 수 있습니다.

* 입력란에 태그를 입력하고 'Enter(엔터) 키'를 누르면 태그가 생성됩니다.&#x20;
* 태그는 특수문자 입력이 불가한 점을 참고해 주세요.
* 태그는 최대 20개까지 생성할 수 있으며, 생성된 태그의 \[X]를 클릭하면 삭제됩니다.

**⑥** 입력란에 게시글 내용을 작성합니다.

**⑦** 고객에게 제공되는 첨부파일은 최대 10개까지 등록 가능합니다.

**⑧** \[저장] 버튼을 클릭하면 선택한 게시판에 게시글이 등록됩니다.
