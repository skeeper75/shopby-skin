# 회원 등록

<Callout type="info">
**쇼핑몰 운영자가 직접 수동으로 회원 등록을 할 수 있습니다.**

**고객에게  전달받은 기본정보 및 부가정보로 회원을 등록하고, 개인정보 수집·이용 동의 내역을 확인합니다.**&#x20;

**회원가입 시 (필수) 동의 항목에 대해서는 ‘동의’로 처리되어 가입되는 점을 참고해 주세요.**
</Callout>

## 회원등록&#x20;

### &#x20;기본정보

<div data-full-width="false"><figure><img src="/images/member/register/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FWWtMgwjIz8ZHDNWwgrmr%2F%ED%9A%8C%EC%9B%90%EB%93%B1%EB%A1%9D_%EA%B8%B0%EB%B3%B8%20%EC%A0%95%EB%B3%B4%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C; 회원등록-기본 정보></p></figcaption></figure></div>

**① 쇼핑몰**\
회원을 등록할 쇼핑몰을 선택합니다.

**② 회원등급**\
등록하는 회원에게 지정할 회원등급을 선택합니다. 참고로 회원가입 시 최초 부여되는 회원등급은 '기본등급'입니다.

**③ 회원명**\
등록하는 회원의 이름을 입력합니다.

**④ 회원번호**\
회원 가입 시 자동으로 부여되는 관리번호로, 별도 설정할 필요가 없습니다.

**⑤ 아이디**\
등록하는 회원의 아이디를 입력합니다. 동일 쇼핑몰에서 중복된 아이디로 가입이 불가하므로 중복확인을 필수로 진행해 주세요.

**⑥ 비밀번호**\
비밀번호는 영문, 숫자, 특수문자 모두 조합하여 8\~20자리 사용이 가능합니다.

* 10자리 이상 → 2종류 조합
* 10자리 미만 → 3종류 모두 조합

**⑦ 이메일**\
등록하는 회원의 이메일을 입력합니다. 동일 쇼핑몰에서 중복된 이메일 사용 불가하므로 중복확인을 필수로 진행해 주세요.

**⑧ 휴대폰번호**\
등록하는 회원의 휴대폰번호를 입력합니다.

### 부가정보 및 개인정보 수집·이용 동의

<div data-full-width="false"><figure><img src="/images/member/register/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F37Rng7mDNEgLf7Clv92r%2F%ED%9A%8C%EC%9B%90%EB%93%B1%EB%A1%9D_%2C%EB%B6%80%EA%B0%80%EC%A0%95%EB%B3%B4%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;부가정보와개인정보 수집⋅이용 동의></p></figcaption></figure></div>

**① 환불 계좌 정보**\
등록하는 회원의 환불 계좌 정보를 입력 할 수 있습니다.&#x20;

<Callout type="warning">
환불 계좌 정보는 샵바이 어드민에서 운영자만 등록과 수정이 가능합니다.
</Callout>

**② 회원 정보 관련 메모**\
회원과 관련된 운영자의 참고 및 체크사항을 작성하여 활용할 수 있는 메모란입니다.

**③ 개인정보 수집⋅이용 동의** \
[서비스관리 > 약관/개인정보처리 방침 관리](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/terms) 메뉴에 등록한 내용이 노출되며 (필수)사항은 동의로 처리됩니다.

**④** \[저장] 버튼을 클릭해 회원 수기 등록을 완료합니다.
