# 회원등급 관리

<Callout type="info">
**쇼핑몰에 가입한 회원을 구매 금액과 구매 횟수 등의 실적 활동별로 세분화하고, 다양한 혜택을 지급하고 관리하는 회원등급 관리 기능입니다.**

**모든 회원은 최초 회원 가입 시 신규 회원등급인 기본등급으로 자동 설정됩니다.**
</Callout>

## 회원등급 관리

**①** [**등급 추가**](#undefined-4)\
회원등급 등록이 가능하며, 회원등급 개수는 제한을 두지 않고 있습니다.

**②** [**등급 평가 설정**](#undefined-2)\
회원등급을 등록하기 전, 회원등급을 분류할 구체적인 평가 기준인 회원등급 평가 조건을 설정합니다.

**③ 선택 삭제**\
회원등급 삭제가 가능합니다.\
📍 회원이 존재하는 회원등급과 기본등급은 삭제 처리가 불가한 점을 참고해 주세요.

**④ 등급 혜택 즉시 지급**\
회원등급에 설정된 적립금과 쿠폰을 수동으로 즉시 지급할 수 있습니다.

**⑤ 등급명**\
\[등급명]을 클릭하면 해당 회원등급 정보를 수정할 수 있는 '회원등급 수정'창이 노출됩니다.

**⑥ 사용여부**\
해당 회원등급의 사용여부를 설정합니다.

**⑦ 회원수**\
회원이 존재하는 등급은 회원수가 카운팅되어 노출됩니다.

**⑧ 상/하 이동 아이콘 (등급 순위 조정)**\
회원등급의 '우선순위'를 조정할 수 있습니다.

<Callout type="danger">
**등급순위**<br>

* 등급 순위의 숫자가 작을 수록 높은 등급입니다.&#x20;
* 등급평가 시 두 개 이상의 등급 평가 조건에 중복되어 해당되는 경우, 우선순위가 높은 등급(순위 숫자가 작은 등급)으로 적용됩니다.&#x20;

예)\
\[순위1] gold : 30만원 이상\
\[순위2] silver : 15만원 이상&#x20;

⇀ 구매금액 32만원 회원 : gold와 silver에 모두 속하지만 평가 시 순위가 높은 gold 등급 적용&#x20;

<br>

* 등급 순위에 맞춰 등급평가 조건을 차등으로설정해 주시기 바랍니다.
* 기본등급은 가장 낮은 등급으로, 우선순위의 조정/삭제/등급 평가 조건 설정이 제한됩니다.

예)  \
\[순위1] vip : 100만원 이상\
\[순위2] gold : 30만원 이상\
\[순위3] silver : 15만원 이상\
\[순위-] 기본등급 : 조건없음&#x20;
</Callout>

#### <mark style="color:blue;">**■ 회원등급 평가 설정**</mark>

* 자동 등급 평가 사용 : '사용함' 설정 시, 매월 등급 조정일에 자동으로 회원등급 평가가 진행됩니다.
* 등급 평가 조건  : 등급을  분류하는 구매실적 조건으로 설정한 평가 조건에  맞추어 등급별 평가 기준을 등록합니다.
  * 구매 금액 : 옵션가가 포함된 상품의 최종 실 결제 금액 + 적립금 사용금액 + 배송비포함\
    📍클레임이 진행 중인 주문(옵션포함) 건은 구매 실적에 포함되지 않으며,\
    &#x20;     주문번호 내 모든 주문상품옵션번호가 설정된 등급평가 조건 상세 기준을 충족하는 시점을 기준으로 반영됩니다.&#x20;
  * 구매 횟수 : 주문상품옵션번호 기준으로 카운팅
* 평가 기간 : 등급 조정 시 평가 조건 달성 여부를 판단하는 기간으로 전체 또는 1\~12/24/36개월의 기간중에서 선택합니다.&#x20;
* 등급 유지 기간 : 평가 결과에 따라 조정된 회원등급을 유지하는 기간으로 1\~12/24/36개월 중에서 기간을 설정합니다.&#x20;
* 등급 조정일 : 평가 결과에 따라 등급 조정이 반영되는 일자로 매월 1/5/10/15/20일 중에서 설정합니다.

<Callout type="success">
**자동 등급 평가 : 설정한 등급 평가 조건 및 방법 등에 따라 시스템에서 자동으로 회원등급이 산정·평가되는 방식입니다. (자동 등급 평가 사용 안함 설정 시 모든 항목 비활성화 처리)**&#x20;

**설정한 '평가 기간'의 월 말까지 '등급 평가 조건'으로 선택한 실적 활동이 산정되고, 익 월에 회원원등급 조정이 적용돼요.**

**'등급 유지 기간'중에 있는 회원이 '등급 평가 조건'을 충족하더라도, 등급 조정 심사 대상에 포함되지 않고 다음 등급 조정일에 등급이 변경되어 적용돼요.**

**설정한 '등급 조정일' 기준으로 회원등급 적립금이 지급되고, 만약 등급 조정일을 수정했다면 다음 등급 조정 시 적용돼요.**
</Callout>

<Callout type="warning">
'등급 조정일' 당시 등급 조정 여부에 관계없이 등급 평가 대상 회원별로 마지막 등급 조정일이 업데이트 됩니다.\
\
예) 회원등급 C인 A회원 / B회원

* 등급 조정일 당시 등급 평가 대상이었으나 구매 실적에 변동 사항이 없어 그대로 'C' 등급을 \
  유지한 'A'회원 ⇀ 등급 조정일이 업데이트 됩니다.
* 등급 조정일 당시 등급 유지 기간으로 인해 등급 평가 대상이 아니었기 때문에 'C' 등급을 \
  유지한 'B'회원 ⇀ 등급 조정일이 업데이트 되지 않습니다.
</Callout>

#### <mark style="color:blue;">**■ 회원등급 추가 (등급 등록)**</mark>

* 등급명 : 회원등급에 사용할 명칭을 입력합니다. \
  📍회원등급명은 쇼핑몰 마이페이지에서도 확인이 가능하므로 입력에 유의해 주세요.<br>
* 사용 여부 : '사용함' 이 기본값으로 설정되어 있습니다.  \
  📍 '사용 안함' 설정 시 해당 등급은 평가 설정 시 제외 처리됩니다.\
  📍 회원이 지정되어 있는 회원등급은 '사용함' 으로 재설정됩니다.<br>
* 등급 평가 조건 : '회원등급 평가 설정' 창에서 설정한 등급 평가 조건이 노출됩니다. \
  설정한 등급 평가 조건에 따라 구매 금액 또는 구매 횟수를 입력합니다.
* 적립금 자동 지급 여부 : 해당 회원등급에게 지급하는 적립금 자동 지급 여부를 설정합니다.
* 적립금 자동 지급 주기 : 자동으로 지급하는 적립금 자동 지급 주기를 '매월' 지급과 '1회' 지급 중에서 설정합니다.
* 정기 지급 적립금 : 적립금 자동 지급 여부 설정에 따라 자동 등급 평가 시 지급되는 적립금입니다. \
  정기 지급 주기에 따라 매월 또는 1회 지급이 가능합니다.
* 쿠폰 혜택 적용 :  자동/수동 등급 조정으로 회원등급이 변경된 경우에만 지급됩니다.\
  📍 등급이 유지된 회원은 등급 평가 시 제공되는 쿠폰 혜택이 지급되지 않습니다.<br>
* 구매 시 지급 적립금 : 정기 지급 혜택 이외에 해당 회원등급에게 구매 적립금을 추가로 지급할 수 있습니다. 구매 금액 기준에서 적용되는 적립률을 ' %' 단위로 입력합니다.
* 등급 설명 : 회원등급을 대표하는 등급 설명을 입력합니다. 등급 설명은 내부 관리용입니다.

적립금 자동 지급 상태를 '사용 안함'에서 '사용함'으로 변경/설정할 경우, 설정일 이후 돌아오는 등급 조정일부터 설정한 혜택이 지급됩니다.

**적립금 자동 지급 주기**\
'매월' 지급 선택 : '매월' 등급 조정일에 해당 회원 등급 회원을 대상으로 혜택 지급\
'1회' 지급 선택 : 매월 등급 조정일에 해당 회원 등급으로 조정된 회원을 대상으로 혜택 '1회' 지급
</Callout>
