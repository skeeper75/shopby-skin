# 회원가입 항목 관리

<Callout type="info">
**쇼핑몰 회원가입 시 회원에게 수집할 회원 기본 정보 항목을 설정할 수 있습니다.**\
**아이디, 이름, 비밀번호, 이메일 항목은 필수 수집 항목으로 설정 변경이 불가하며, 그 이외 항목은 사용 설정 및 필수 값으로 관리합니다.**

**기본 정보 이외에 운영자가 추가로 수집이 필요한 회원 정보를 등록하고 관리할 수도 있습니다.**&#x20;
</Callout>

## 기본 정보

**① 회원가입 항목 설정**\
회원가입 시 회원에게 수집할 회원 정보 항목을 설정할 수있습니다.\
'사용 함'을 선택하면 회원가입 정보 항목이 노출되며, 항목 별 설정이 가능합니다.

<Callout type="warning">
**사용 : 해당 항목을 '사용'하는 항목으로 설정합니다.**\
**필수 : 해당 항목을 반드시 수집해야 하는 '필수 값'으로 설정합니다.**

\[사용]만 설정하면 해당 항목은 고객의 선택에 의해 작성 및 입력할 수 있는 회원가입 정보 항목으로 설정됩니다.\
\[사용 + 필수]를 설정하면 해당 항목은 고객이 반드시 작성 및 입력해야 하는 회원가입 정보 항목으로 설정됩니다.
</Callout>

**② 필수 사용 항목**\
아이디, 이름, 비밀번호, 이메일은 회원가입 시 반드시 수집해야 하는 회원 정보 항목으로 설정 변경이 불가합니다.

**③ 사용 설정 항목**\
이메일 수신동의, SMS 수신동의 항목은 사용 설정만 가능합니다.

**④ 사용 + 필수 설정 항목**\
닉네임, 휴대폰번호, 주소, 전화번호, 생년월일, 성별 항목은 사용 설정 및 필수 값 설정이 가능합니다.

## 추가 정보

<figure><img src="/images/member/collection-setting/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FsuqENB8sp1N89lmyHoyW%2F%ED%9A%8C%EC%9B%90%EA%B0%80%EC%9E%85%20%ED%95%AD%EB%AA%A9%20%EA%B4%80%EB%A6%AC_%EC%B6%94%EA%B0%80%EC%A0%95%EB%B3%B4%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;추가 정보></p></figcaption></figure>

**① 항목명**\
추가로 수집할 회원 정보 항목명을 입력합니다.\
📍\[+] 버튼을 클릭하면 추가 정보 항목을 입력할 수 있는 입력란이 추가되고, \[-] 버튼을 클릭하면 입력란이\
&#x20;    삭제되요.

**② 사용여부**\
해당 추가 정보의 사용 여부를 체크박스로 설정합니다.

**③ 필수여부**\
해당 추가 정보의 필수 여부를 체크박스로 설정합니다.

**④ 항목 타입**\
해당 추가 정보를 입력하는 타입(방식)을 설정할 수 있습니다.

* 텍스트박스 : 고객이 직접 텍스트로 정보를 입력하는 방식
* 이미지 : 고객이 직접 이미지 파일을 첨부하여 정보를 입력하는 방식
* 라디오버튼 / 드롭다운 : 여러 옵션 값 중 1개의 정보를 선택하여 입력하는 방식                                                                                                                                                                                                                                                                 &#x20;
* 체크박스 : 여러 옵션 값 중 복수개의 정보를 선택하여 입력하는 방식   &#x20;

**⑤ 옵션값**\
해당 추가 정보의 선택지를 등록할 수 있습니다.\
📍\[+] 버튼을 클릭하면 여러 옵션값을 입력할 수 있는 입력란이 추가되고, \[-] 버튼을 클릭하면 입력란이 \
&#x20;    삭제되요.

**⑥** \[저장] 버튼을 클릭해 회원가입 항목 설정을 완료합니다.

<Callout type="danger">
**회원가입 정보 항목을 변경하였다면,  '**[**서비스관 > 약관/개인정보 처리방침 관리 > 개인정보 수집/동의 항목**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/service/terms/collection)**' 메뉴에 해당 내용을 반드시 적용해 주세요.**
</Callout>
