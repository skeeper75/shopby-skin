# 탈퇴회원 조회

<Callout type="info">
**탈퇴회원은 고객이 마이페이지의 회원 탈퇴를 통해 직접 회원 탈퇴를 진행하거나, 쇼핑몰 운영 정책에 따라 운영자가 강제탈퇴 처리를 진행하여 쇼핑몰 이용 자격이 상실된 회원을 의미합니다.**

**탈퇴회원 조회 메뉴에서는 특정 탈퇴회원 정보의 조회 및 삭제처리, 탈퇴를 철회할 수 있는 기능 등의 관리가 가능합니다.**
</Callout>

## 탈퇴회원 조회

<figure><img src="/images/member/expel/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FKofWbLMHmLxWwUvFVV8O%2F%ED%83%88%ED%87%B4%ED%9A%8C%EC%9B%90%EC%A1%B0%ED%9A%8C%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;탈퇴회원 조회></p></figcaption></figure>

**① 쇼핑몰**\
탈퇴회원을 조회할 쇼핑몰을 선택합니다.

**② 검색어**\
아이디, 회원번호, 회원명, 닉네임, 휴대폰번호, 이메일의 검색어로 탈퇴회원을검색합니다.

**③ 가입유형**\
일반 회원, 연동 아이디 회원, 간편로그인 회원(페이코, 네이버, 카카오, 페이스북, 라인) 중 가입유형을 선택하여 탈퇴회원 조회가 가능합니다.

* 일반회원 : 쇼핑몰 회원가입을 통해 가입한 회원
* 연동 아이디 회원 : 외부 회원 연동을 이용하는 업체 대상 API로 회원가입과 로그인이 가능한 회원
* 간편로그인 회원 : SNS 간편로그인을 통해 가입한 회원 (네이버, 카카오, 페이코, 페이스북, 라인)

**④ 상세검색**\
탈퇴유형(강제탈퇴/직접탈퇴)과 쇼핑몰 재가입 가능여부(가능/불가능)로 탈퇴회원을 검색합니다.

## 탈퇴회원 검색결과(리스트)

<figure><img src="/images/member/expel/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FYeg8jlYdMFschrtj2vqB%2F%ED%83%88%ED%87%B4%ED%9A%8C%EC%9B%90%EC%A1%B0%ED%9A%8C%20%EA%B2%80%EC%83%89%EA%B2%B0%EA%B3%BC%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;탈퇴회원 검색결과(리스트)></p></figcaption></figure>

**① 탈퇴철회**\
탈퇴회원의 탈퇴를 철회할 수 있습니다. 탈퇴 철회 시 회원정보가 복구됩니다.

**② 삭제처리**\
탈퇴회원 정보를 삭제합니다.

**③ 강제탈퇴**\
운영자의 강제탈퇴 처리 여부가 'Y'와  'N'으로 구분되어 표시됩니다.

* Y : 운영자 강제탈퇴
* N : 회원 직접탈퇴

**④ 탈퇴사유**\
회원이 직접 탈퇴 시 작성한 탈퇴사유 또는 운영자 강제탈퇴 시 작성한 강제탈퇴 사유가 확인됩니다.

<Callout type="danger">
탈퇴회원 정보는 분리보관 5일 후 즉시 파기됩니다.

주문자 정보는 전자상거래 등에서의 소비자보호에 관한 법률에 따라 5년 간 보관후 파기합니다.

간편로그인 회원은 강제탈퇴 처리 이후 탈퇴철회가 불가합니다.
</Callout>
