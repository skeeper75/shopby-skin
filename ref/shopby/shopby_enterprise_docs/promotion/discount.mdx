# 추가할인 관리

<Callout type="info">
**추가할인은 상품의 옵션 판매가를 기준으로 할인을 한 번 더 적용하는 프로모션형 혜택 입니다.**

**추가할인은 상품에 적용되는 할인기간을 설정할 수 있어 특정시점에만 할인 판매를 진행하는 타임세일 등의 프로모션으로 활용할 수 있고, 상품 구매수량에 따라 할인금액을 다르게 설정할 수 있는 구매수량 설정 기능으로 대량구매할인 프로모션도 가능합니다.**
</Callout>

## 추가할인  관리 검색조건

<figure><img src="/images/promotion/discount/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FFymSqnt889knzF1D97QU%2F%EC%B6%94%EA%B0%80%ED%95%A0%EC%9D%B8%EA%B2%80%EC%83%89%EC%A1%B0%EA%B1%B4.png" alt=""/><figcaption><p>&#x3C;추가할인  관리 검색조건></p></figcaption></figure>

**① 쇼핑몰**\
추가할인 프로모션을 조회할 쇼핑몰을 선택합니다.

**② 검색어**\
추가할인명, 추가할인번호 등의 검색어로 등록한 추가할인 프로모션을 조회합니다.\
일부 검색어는 지원하지 않으며 정확한 검색어를 입력해 주세요.

**③ 기간**\
등록일, 할인시작일, 할인종료일 기준으로 등록한 추가할인 프로모션을 조회합니다.

**④ 추가할인상테**\
등록한 추가할인 프로모션을 진행대기, 할인중, 기간만료, 할인종료 등의 추가할인상태로 검색할 수 있습니다.

* 진행대기 : 추가할인 시작일이 도래하지 않은 상태
* 할인 중 : 추가할인 시작일이 도래하여 추가할인 기간안에 있는 상태
* 기간만료 : 추가할인 종료일이 지나 추가할인이 종료된 상태
* 할인종료 : 이슈발생 등으로 할인 중지 및종료가 필요하여 추가할인을 강제 종료처리한 상태

**⑤ 검색/초기화**\
모든 조건을 선택하고 \[조회] 버튼을 클릭하면 하단 조회 결과에 주문 건이 노출됩니다. \
\[초기화] 버튼을 클릭하면 입력된 검색어 및 검색조건이 기본값으로 초기화됩니다.

#### <mark style="color:blue;">**■ 추가할인 수정**</mark>

* 기본정보 항목의 추가할인명/추가할인설명/추가할인 종료기간, 대상 설정 항목의 추가할인 적용회원/할인대상 상품의 수정 및 변경이 가능합니다.
* '할인 중' 상태의 추가할인 프로모션은 할인 중지 및 종료가 필요한 경우, 추가할인 수정 페이지에서 \[할인종료] 버튼을 클릭 해 추가할인 프로모션을 종료 처리 할 수 있습니다.&#x20;

<Callout type="warning">
종료된 추가할인 프로모션은 정보사항 수정 및 다른 상태 값으로 변경이 불가능합니다.
</Callout>
