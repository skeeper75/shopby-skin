# 지급등록

## 쿠폰 지급 설정

<figure><img src="/images/promotion/coupon/issue-plan/form/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2F40WfnfPdQw1A4B1ziBt5%2F%ED%95%A0%EC%9D%B8%EC%BF%A0%ED%8F%B0%EC%A7%80%EA%B8%89%EA%B4%80%EB%A6%AC%20%EC%A7%80%EA%B8%89%EB%93%B1%EB%A1%9D%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;쿠폰 지급 설정></p></figcaption></figure>

**①** \[지급등록] 버튼을 클릭하면, '쿠폰 지급 설정' 창이 노출됩니다.

**②** 쿠폰 지급 건을 등록하는 쇼핑몰을 선택합니다.

**③ 지급유형**\
즉시지급, 예약지급, 자동지급 중에서 선택합니다.

* 즉시지급 : 쿠폰지급 건을 등록하는 즉시 쿠폰이지급됩니다.
* 예약지급 : 지급일자를 미래 시점으로 설정하여 쿠폰의 예약 지급을 등록합니다.
* 자동지급 : 생일과 회원가입 조건을 설정하여, 조건을 충족한 회원에게 쿠폰이 지급됩니다.

**④ 대상 회원**\
쿠폰을 지급 할 특정 회원을 선택합니다. 회원을 선택하면 선택된 회원 수가 표기됩니다.

* 회원등급/ 그룹 : 특정 회원등급 및 회원그룹에 속한 회원을 대상으로 쿠폰을 지급합니다.
* 회원 조회 : 개별 회원을 대상으로 쿠폰을 지급합니다.
* 엑셀 일괄 등록 : 엑셀파일 내 회원번호 항목에 쿠폰을 지급할 회원의 회원번호를 입력하고 저장한 다음 엑셀파일을 업로드합니다.

**⑤ 지급 시점**\
예약지급 건의 쿠폰지급일(예약지급 시작일)과 시간을 설정합니다. \
📍 즉시지급과 자동지급 선택 시 지급 시점 미노출

<Callout type="warning">
예약시점이 도래하기 전 선택한 쿠폰의 발급상태가 '발급종료'로 변경될 경우, 해당 쿠폰은 발급되지 않습니다.

예약시점이 도래하기 전 선택한 회원이 '휴면회원' 혹은 '탈퇴회원'으로 전환될 경우, 해당 회원은 제외하고 쿠폰을 발급합니다.
</Callout>

**⑥ 지급 조건**\
자동지급 건의 쿠폰 지급 조건을 '생일'과 회원가입' 중에서 선택합니다.\
📍 즉시지급과 예약지급 선택 시 지급 조건 미노출

<Callout type="warning">
생일 :  지급간기 동안 당일이 생일인 회원을 대상으로 쿠폰이 지급됩니다.

회원가입  : 회원가입 시점에 가입한 회원을 대상으로 쿠폰이 즉시지급됩니다.
</Callout>

**⑦ 지급 기간**\
자동지급 건의 쿠폰 지급 시작일시와 종료일시를 설정합니다. \
📍 즉시지급과 예약지급 선택 시 지급 기간 미노출

**⑧ 지급 사유**\
회원에게 쿠폰을 지급하는 사유를 입력란에 작성합니다.

**⑨** [**지급쿠폰 선택 (쿠폰 조회)**](#undefined-1)\
\[쿠폰 조회] 버튼을 클릭하면 '쿠폰 조회' 창이 노출됩니다.

**⑩ 알림설정**\
쿠폰 지급 알림을 SMS와 이메일로 발송하는 기능입니다.\
📍 회원별 수신동의여부에 따라 알림이 발송되지 않을 수 있는 점을 참고해 주세요.

\

#### <mark style="color:blue;">■ 쿠폰 조회</mark>

* '쿠폰 조회' 창에서 지급할 쿠폰을 조회하고, 선택한 다음 \[선택 추가] 버튼을 클릭 해 지급 쿠폰을 추가합니다.
