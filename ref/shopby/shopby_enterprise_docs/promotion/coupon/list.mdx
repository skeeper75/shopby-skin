# 할인쿠폰 발행 관리

<Callout type="info">
**프로모션 혜택으로 회원에게 지급되는 쿠폰을 신규 등록하고, 지급 조건을 설정하여 쿠폰을 즉시지급 또는 예약지급 할 수 있는 할인쿠폰 발행관리 메뉴 입니다.**

**등록된 쿠폰을 여러 조건을 통해 검색하고 쿠폰 수정 및 발급 상태 등도 관리할 수 있습니다.**&#x20;
</Callout>

## 할인쿠폰 발행 관리 검색조건

<figure><img src="/images/promotion/coupon/list/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FxAx7H84nwnJpTUtLFnnA%2F%ED%95%A0%EC%9D%B8%EC%BF%A0%ED%8F%B0%EB%B0%9C%ED%96%89%EA%B4%80%EB%A6%AC%20%EA%B2%80%EC%83%89%EC%A1%B0%EA%B1%B4%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;할인쿠폰 발행 관리 검색조건></p></figcaption></figure>

**① 쇼핑몰**\
쿠폰을 조회할 쇼핑몰을 선택합니다.

**② 검색어**\
쿠폰명, 쿠폰번호, 쿠폰  등록자 등의 검색어로 등록한 쿠폰을 조회합니다.\
일부 검색어는 지원하지 않으며 정확한 검색어를 입력해 주세요.

**③ 기간**\
쿠폰등록일, 발급시작일, 발급종료일 기준으로 등록한 쿠폰을 조회합니다.

**④ 혜택구분**\
등록한 쿠폰의 혜택 적용 범위에 따라 '상품적용 쿠폰'과 '주문적용 쿠폰'으로 구분하여 검색이 가능합니다.

* 상품적용 쿠폰 : 상품 개별 단위, 즉 각각의 상품에 적용되는쿠폰입니다.
* 주문적용 쿠폰 : 주문 단위, 즉 배송비를 제외한 총 주문 결제금액에 적용되는 쿠폰입니다.

**⑤ 발급유형**\
등록한 쿠폰이 발급되는 유형인 다운로드 발급, 코드 발급(직접입력), 코드 발급(자동생성), 자동 발급 유형으로 구분하여 검색이 가능합니다.

* 다운로드 발급 : 회원이 쇼핑몰에서 직접 쿠폰을 다운로드하여 발급받는 형태입니다.
* 코드 발급(직접입력) : 운영자가 쿠폰 등록 시 직접 생성한 코드 번호를 회원이 입력하여 쿠폰을 발급받는 형태입니다.
* 코드 발급(자동생성) : 원하는 수량만큼 시스템에서 임의로 자동 생성한 코드 번호를 회원이 입력하여 쿠폰을 발급받는 형태입니다.
* 자동 발급 : 쿠폰 지급 관리 및 회원 등급 관리 등의 메뉴에서 설정한 조건에 따라 자동으로 쿠폰이 발급되는 형태입니다.
* 기프트 쿠폰 : 쇼핑몰에서 사용 가능한 적립금을 지급하는 형태의 쿠폰입니다.

**⑥ 발급상태**\
등록한 쿠폰을 발급대기, 발급중, 발급중지, 발급종료의 발급상태로 검색할 수 있습니다.

* 발급대기 : 쿠폰발급 시작일이 도래하지 않은 상태입니다.
* 발급 중 : 쿠폰이 정상적으로 발급 중인 상태입니다.
* 발급중지 : 이슈발생 등으로 쿠폰발급을 일시 중지 처리한 상태로 쿠폰발급 재시작이 가능합니다.
* 발급종료 : 쿠폰발급 종료일이 지나 쿠폰발급이 종료된 상태 또는 이슈발생 등으로 쿠폰발급을 강제 종료 처리한 상태입니다.

## 할인쿠폰 발행 관리 검색결과(리스트)

<figure><img src="/images/promotion/coupon/list/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FApw8Pne4cRPYQTc9wsTg%2F%ED%95%A0%EC%9D%B8%EC%BF%A0%ED%8F%B0%EB%B0%9C%ED%96%89%EA%B4%80%EB%A6%AC%20%EA%B2%80%EC%83%89%EA%B2%B0%EA%B3%BC%EB%A6%AC%EC%8A%A4%ED%8A%B8%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;할인쿠폰 발행 관리 검색결과(리스트)></p></figcaption></figure>

**①** [**쿠폰등록**](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/promotion/coupon/list/addform)\
쿠폰을 신규로 등록할 수 있습니다.

**②** [**지급쿠폰 선택**](#undefined-2)\
특정 쿠폰을 선택하고, \[지급쿠폰 선택] 버튼을 클릭하면 스크롤이 아래로 이동되면서 지급 조건 설정 영역이 나타납니다. 해당 영역에서 선택된 쿠폰을 즉시지급 또는 예약지급 할 수 있습니다.

**③** [**수정 (쿠폰 수정)**](#undefined-3)\
등록된 쿠폰 수정이 가능한 쿠폰수정 페이지로 이동됩니다.

**④ 복사** \
쿠폰 등록 시 설정한 쿠폰 정보를 복사하여 동일한 쿠폰을 (복사)등록할 수 있는 쿠폰 복사 페이지로 이동됩니다.

**⑤ 발급상태**\
쿠폰 발급 시작일과 종료일에 따른 발급상태 또는 '일시 중지' 및 '강제 종료' 처리에 의한 발급상태가 확인됩니다.

<Callout type="warning">
발급중지 상태는 '느낌표' 아이콘이 표시되며, 클릭 시 발급중지 사유 안내가 노출됩니다.
</Callout>

**⑥ 발급유형**\
등록한 쿠폰의 발급유형이 확인됩니다. \
코드발급(자동생성/직접입력) 쿠폰은 \[코드관리]와 \[코드복사] 버튼이 각각 노출됩니다.

* [코드발급(자동생성) 코드관리](#undefined-4) : \[코드관리] 버튼을 클릭하면 생성된 쿠폰 코드 조회 및 신규 코드 생성이 가능한 '쿠폰코드관리' 창이 노출됩니다.
* 코드발급(직접입력) 코드복사 : \[코드복사] 버튼을 클릭해 운영자가 직접 생성한 코드를 복사할 수 있습니다.

**⑦ 발급현황** \
쿠폰 등록 시 입력한 '총 발급수량'에서 해당 쿠폰에 설정된 발급방식으로 쿠폰이 발급된 '일반 발급수량'이 카운팅 되어 노출됩니다. \
발급현황 숫자를 클릭하면 '[쿠폰 발급/사용 현황](#undefined-5)' 창이 노출됩니다.

**⑧ 운영자 발급현황**\
운영자가 회원을 직접 선택하여 쿠폰을 발급한 수량이 카운팅 되어 노출됩니다. \
발급현황 숫자를 클릭하면 '[쿠폰 발급/사용 현황](#undefined-5)' 창이 노출됩니다.\
📍 <mark style="color:orange;">운영자 발급은 쿠폰 등록 시 입력한 '총 발급수량' 제한에 관계없이 발급이 가능합니다.</mark>

**⑨ 사용현황**\
회원의 '쿠폰 사용 완료 수량'이 카운팅 되어 노출됩니다.\
사용현황 숫자를 클릭하면 '[쿠폰 발급/사용 현황](#undefined-5)' 창이 노출됩니다.

<Callout type="danger">
**발급현황 = 일반 발급수량 / 총 발급수량**\
\
총 발급수량 : [쿠폰등록 > 발급정보](https://nhn-commerce.gitbook.io/shopby_enterprise_manual/promotion/coupon/addform#undefined-4) > 총 발급수량 항목에 입력한 값\
일반 발급 : 쿠폰에 설정된 발급방식에 따라 쿠폰이 발급된 건\
운영자 발급 : 운영자가 회원을 직접 선택하여 쿠폰을 발급한 건
</Callout>

#### <mark style="color:blue;">**■ 지급 쿠폰 선택**</mark>

* 지급 쿠폰으로 선택한 지급예정 쿠폰의 쿠폰번호, 쿠폰명, 할인금액이 확인됩니다. \
  \[삭제] 버튼으로 쿠폰의 삭제처리도 가능합니다.
* 지급조건 설정에서 쿠폰을 지급할 대상 회원을 회원조회와 엑셀 일괄등록 방식 중에서 선택합니다.\
  쿠폰 지급사유를 입력하고, 쿠폰 지급시점을 즉시지급과 예약지급 중에서 선택합니다.\
  📍 예약지급을 선택하면 예약지급 일시를 설정할 수 있습니다.
* \[지급] 버튼을 클릭하면, 설정한 지급조건에 맞춰 쿠폰이 지급됩니다.

#### <mark style="color:blue;">**■ 수정 (쿠폰 수정)**</mark>

* 기본정보 항목의 쿠폰명/쿠폰설명, 발급정보 항목의 발급수량/발급 종료기간의 수정 및 변경이 가능합니다.
* 쿠폰 발급상태 및 발급 종료기간 따라 쿠폰 수정 페이지에서 처리할 수 있는 발급상태값이 각기 상이합니다.

  발급종료 : 발급종료 처리 시 쿠폰 재 발급 불가\
  복사등록 : 설정한 쿠폰 정보가 복사되어 동일한 쿠폰의 등록 가능
* **발급 중 상태 쿠폰**\
  발급종료 : 발급종료 처리 시 쿠폰 재 발급 불가\
  발급중지 : 발급중지 처리 시 쿠폰 발급 일시 중지\
  복사등록 : 설정한 쿠폰 정보가 복사되어 동일한 쿠폰의 등록 가능

  발급 재시작 : 발급 재시작 처리 시 발급 중 상태로 변경되어 쿠폰 재 발급 가능\
  발급종료 : 발급종료 처리 시 쿠폰 재 발급 불가\
  복사등록 : 설정한 쿠폰 정보가 복사되어 동일한 쿠폰의 등록 가능
* **발급중지 ( + 발급 종료기간 과거 시점) 상태 쿠폰**\
  발급종료 : 발급종료 처리 시 쿠폰 재 발급 불가\
  복사등록 : 설정한 쿠폰 정보가 복사되어 동일한 쿠폰의 등록 가능

#### <mark style="color:blue;">**■**</mark> <mark style="color:blue;"></mark><mark style="color:blue;">코드발급(자동생성) 코드관리</mark>

* 시스템에서 코드 발급(자동생성) 쿠폰에  자동 생성된 코드 번호를 쿠폰 발급여부로조회할 수 있습니다.
* 쿠폰 수량을 입력하고, \[추가생성] 버튼을 클릭 해 미발급 상태 코드 번호를 신규 생성할 수 있습니다.\
  📍미발급 상태 코드 번호는 최대 100,000개까지 생성이 가능합니다.
* 자동 생성 코드 번호 리스트의 엑셀 파일 다운로드도 가능합니다.&#x20;

#### <mark style="color:blue;">**■ 쿠폰 발급/사용 현황**</mark>

* 발급현황, 운영자 발급현황, 사용현황의 숫자를 클릭하면 공통으로 노출되는 '쿠폰 발급/사용 현황' 창에서 쿠폰의 발급 및 사용 현황, 철회 현황 관련 정보등을 조회할 수 있습니다.
* 각각의 현황 건수를 클릭하면 내역 리스트에 해당 쿠폰의 발급/사용/철회 내역의 상세 내용이 노출됩니다. (총 발급/일반 발급/운영자 발급 대상 회원정보, 쿠폰 사용 상태, 실 할인금액, 쿠폰 발급일과 만료일, 쿠폰 최종사용일, 쿠폰 철회일, 지급 및 철회 사유 등)
* '사용 가능' 상태의 쿠폰은 \[지급 철회]를 클릭해 쿠폰 지급을 철회할 수 있습니다. '쿠폰 철회' 창에서 철회사유를 입력하고, \[확인]을 클릭해 쿠폰 지급을 철회합니다.

<Callout type="success">
**발급된 쿠폰 사용에 따른 사용상태 값**

* 사용 가능 : 발급된 쿠폰을 사용하기 전 상태
* 사용 완료 : 발급된 쿠폰이 사용된 상태
* 기간 만료 : 발급된 쿠폰이 아직 사용되지 않은 상태에서 쿠폰 사용기간이 종료된 상태
* 철회 완료 : 운영자가 사용가능 상태 쿠폰의 지급을 철회하여 철회완료된 상태
</Callout>
