# 쿠폰등록

## 기본정보 <a href="#undefined-2" id="undefined-2"></a>

<figure><img src="/images/promotion/coupon/list/addform/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FzGRjR88CYVBWaCxqtVMS%2F%ED%95%A0%EC%9D%B8%EC%BF%A0%ED%8F%B0%EB%B0%9C%ED%96%89%EA%B4%80%EB%A6%AC%20%EC%BF%A0%ED%8F%B0%EB%93%B1%EB%A1%9D%20%EA%B8%B0%EB%B3%B8%EC%A0%95%EB%B3%B4%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;기본정보></p></figcaption></figure>

**① 쇼핑몰** \
쿠폰을 등록하는 쇼핑몰을 선택합니다.

**② 쿠폰명**\
쿠폰 특성에 맞는 이름을 입력합니다. \
📍 쿠폰명은 쇼핑몰에서 회원에게 노출되므로 작성에 유의해 주세요.

**③ 쿠폰 설명** \
쿠폰 사용용도 등 쿠폰 혜택관련 부가설명을 입력할 수 있습니다. \
운영자 관리 목적의 입력 항목으로 쇼핑몰에 노출되지 않습니다.

## 발급정보 <a href="#undefined-4" id="undefined-4"></a>

**✅&#x20;**<mark style="color:green;background-color:yellow;">**쿠폰 발급유형에 따라 설정하는 발급정보 항목이 상이합니다.**</mark>

### 다운로드 발급

<figure><img src="/images/promotion/coupon/list/addform/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FYaLt3DfH2Nzv4hbCs8zr%2F%ED%95%A0%EC%9D%B8%EC%BF%A0%ED%8F%B0%EB%B0%9C%ED%96%89%EA%B4%80%EB%A6%AC%20%EC%BF%A0%ED%8F%B0%EB%93%B1%EB%A1%9D%20%EB%B0%9C%EA%B8%89%EC%A0%95%EB%B3%B4%20%EB%8B%A4%EC%9A%B4%EB%A1%9C%EB%93%9C%20%EC%BF%A0%ED%8F%B0%20%EB%B3%B5%EC%82%AC%203.png" alt=""/><figcaption><p>&#x3C;발급정보 - 다운로드 쿠폰></p></figcaption></figure>

**① 총 발급수량**\
발급 가능한 쿠폰의 총 수량을 입력합니다.

**② 일별 총 발급 수량**\
쿠폰의 하루 발급 수량을 제한할 수 있습니다. \
'발급 제한'을 선택하고 하루동안 발급할 쿠폰 수량을 입력해 주세요.

**③ 회원별 발급 수량**\
회원 1인에게 발급되는 쿠폰의 발급 수량을 발급 전체기간과 하루 기준으로 구분하여 제한할 수 있습니다. \
'발급 제한'을 선택하고 '전체 발급기간 기준'과 '하루 기준' 중에서 선택해 입력합니다.

<Callout type="warning">
일별 총 발급수량과 회원별 발급수량 설정의 '하루 기준'은 동일한 일자를 의미합니다.\
오늘 날짜를 기준으로 쿠폰의 '일별 총 발급수량'과 '회원별 하루 발급수량'을 모두 발급받은 경우,\
**익일 00:00부터 쿠폰의 재발급이 가능**합니다.
</Callout>

**④ 발급기간**\
쿠폰의 발급 전체기간을 발급 시작일과 발급 종료일로 설정할 수 있습니다.&#x20;

**⑤ 발급가능 요일**\
쿠폰 발급 요일을 설정할 수 있습니다. '설정'을 선택하고 요일을 선택합니다.

**⑥ 발급가능 시간**\
쿠폰 발급 시간대를 설정할 수 있습니다. '설정'을 선택하고 시간대를 설정합니다.

**⑦ 발급대상 회원**\
쿠폰 발급대상 회원등급과 회원그룹을 별도 설정할 수 있습니다. '회원 선택' 을 선택하고 \[회원등급/그룹] 버튼을 클릭해 쿠폰을 발급받을 회원등급 및 회원그룹을 설정합니다.

**⑧ 발급가능 플랫폼**\
쿠폰 발급 플랫폼을 설정할 수 있습니다. '설정'을 선택하고 플랫폼을 선택합니다.

**⑨ 발급/사용 채널제한**\
가격비교사이트 등 외부 채널을 통해 상품 상세 페이지로 유입된 경우, 채널별로 할인 쿠폰 다운로드 및 사용을 제한하는 기능입니다.&#x20;

<Callout type="warning">
안내되어 있는 네이버쇼핑/페이스북/카탈로그/다나와/에누리/쿠차/위메프/원더쇼핑의 EP 생성 시 상품 상세 URL에 '채널타입 코드값'이 포함되어 생성됩니다.

그 밖에 채널에 연동한 경우 상품 상세 URL에 채널타입 코드값을 직접 입력하고, 채널 추가입력 체크박스를 선택한 다음, 추가한 채널타입 코드값을 직접 입력하여 쿠폰 발급이 가능한 채널을 설정할 수 있습니다.
</Callout>

**⑩ 상품상세 노출여부**\
쿠폰의 상품 상세 페이지 노출여부를 설정합니다. \
특정 페이지에서만 쿠폰 노출이 필요한 경우 활용할 수 있습니다.\
📍 단, 발급/사용 채널제한의 사용을 '설정' 한 경우, 상품상세 노출여부는 '노출'로 고정 처리되요.\
&#x20;   (외부 채널 연동에서 발생할 수 있는 패널티 등의 이슈 발생 방지 목적)

### 코드발급(직접입력)

<figure><img src="/images/promotion/coupon/list/addform/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2Fzqx698JX9yRjKkgU3Jp8%2F%ED%95%A0%EC%9D%B8%EC%BF%A0%ED%8F%B0%EB%B0%9C%ED%96%89%EA%B4%80%EB%A6%AC%20%EC%BF%A0%ED%8F%B0%EB%93%B1%EB%A1%9D%20%EB%B0%9C%EA%B8%89%EC%A0%95%EB%B3%B4%20%EC%BD%94%EB%93%9C%EB%B0%9C%EA%B8%89%20%EC%A7%81%EC%A0%91%20%EC%BF%A0%ED%8F%B0%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;발급정보 - 코드 발급(직접입력) 쿠폰></p></figcaption></figure>

**① 프로모션 코드**\
쿠폰 발급을 위한 코드번호 생성을 위해 코드를 입력하고 \[중복확인] 버튼으로 사용여부를 확인해 주세요.\
📍 프로모션 코드 입력문자는 제한이 없으며, 최대 20자까지 입력 가능합니다.

### 코드발급(자동생성)

<figure><img src="/images/promotion/coupon/list/addform/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FmqIq2WHOQ4Yc1evI7Jrv%2F%ED%95%A0%EC%9D%B8%EC%BF%A0%ED%8F%B0%EB%B0%9C%ED%96%89%EA%B4%80%EB%A6%AC%20%EC%BF%A0%ED%8F%B0%EB%93%B1%EB%A1%9D%20%EB%B0%9C%EA%B8%89%EC%A0%95%EB%B3%B4%20%EC%BD%94%EB%93%9C%EB%B0%9C%EA%B8%89%20%EC%9E%90%EB%8F%99%20%EC%BF%A0%ED%8F%B0%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;발급정보 - 코드 발급(자동생성) 쿠폰></p></figcaption></figure>

**① 프로모션 코드**\
쿠폰 발급을 위한 코드번호의 자동 생성은 자리수 지정이 가능합니다. \
'지정 자리수'를 선택하고 8/12/16자리 중에서 선택하거나, '자리수 입력'을 선택하고 코드번호 자리수를 입력합니다.\
📍 프로모션 코드 자리수는  5\~20자리 까지 지정 가능합니다.

또는 발급 수량 입력도 가능합니다.

## 금액설정 <a href="#undefined-5" id="undefined-5"></a>

**✅&#x20;**<mark style="color:green;background-color:yellow;">**쿠폰 혜택 적용 범위에 따라 금액설정 항목이 상이합니다.**</mark>

### 상품적용 쿠폰

<figure><img src="/images/promotion/coupon/list/addform/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FM6oY2MTuj61jucl6EZtE%2F%ED%95%A0%EC%9D%B8%EC%BF%A0%ED%8F%B0%EB%B0%9C%ED%96%89%EA%B4%80%EB%A6%AC%20%EC%BF%A0%ED%8F%B0%EB%93%B1%EB%A1%9D%20%EA%B8%88%EC%95%A1%EC%84%A4%EC%A0%95%20%EC%83%81%ED%92%88%EC%A0%81%EC%9A%A9%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;금액설정 상품적용 쿠폰></p></figcaption></figure>

**① 할인금액**\
쿠폰 사용 시 적용되는 할인금액을 '원' 또는 '%'로 설정할 수 있습니다.

<Callout type="warning">
**할인금액은 원 단위 절삭됩니다.** \
\
예) 130원 상품에 10% 할인금액 적용\
&#x20;     130\*0.1 = 13원  → 원 단위 절삭하여 최종 10원 할인
</Callout>

**② 비용부담 설정** \
파트너사 입점으로 판매되는 상품에 쿠폰 할인을 적용하는 경우, 할인금액에 대해 쇼핑몰과 파트너사간 비용 분담 비율을 관리하는 기능입니다. 쇼핑몰 부담 비율을 입력하면 파트너사 부담 비율은 자동 산정되어 반영됩니다.

**③ 최소기준금액** \
쿠폰 할인을 받는 상품 금액의 최소 기준을 설정할 수 있습니다. \
쿠폰 할인 대상 상품의 최종 합계 금액이 설정한 최소 금액 기준을 만족한 경우에만 쿠폰 할인 적용이 가능합니다.

**④ 최대기준금액** \
쿠폰 할인을 받는 상품 금액의 최대 기준을 설정할 수 있습니다. \
쿠폰 할인 대상 상품의 최종 합계 금액이 설정한 최대 금액 기준을 만족한 경우에만 쿠폰 할인 적용이 가능합니다.

**⑤ 혜택제한** \
해당 쿠폰 사용 시 다른 혜택이 중복으로 적용될 수 없도록 제한하는 기능입니다.\
쿠폰 사용과 동시에 상품 구매 적립금의 적립과 주문 쿠폰 사용이 불가하도록 설정할 수 있습니다.\
📍 상품 적용 쿠폰의 경우,  해당 상품의 적립금 지급만 제한됩니다.

<Callout type="danger">
적립금 설정 메뉴에서 '쿠폰 사용 시 적립금 적립 제한'을 설정한 경우, 상품 적용 쿠폰에 적립금 지급 혜택 제한을 설정하지 않아도 해당 쿠폰 사용 시 적립금 지급이 불가합니다.
</Callout>

주문금액 할인과 기본 배송비 금액 할인 중에서 선택할 수 있습니다.\
'기본 배송비 금액 할인'으로 설정하면, 배송비 할인 쿠폰으로 활용이 가능해요.

**④ 최대기준금액** \
쿠폰 할인을 받는 주문 금액의 최대 기준을 설정할 수 있습니다. \
최종 주문 결제 금액이 설정한 최대 금액 기준을 만족한 경우에만 쿠폰 할인 적용이 가능합니다.

**⑤ 혜택제한** \
해당 쿠폰 사용 시 다른 혜택이 중복으로 적용될 수 없도록 제한하는 기능입니다.\
쿠폰 사용과 동시에 상품 구매 적립금의 적립과 상품 쿠폰 사용이 불가하도록 설정할 수 있습니다.

### **기프트 쿠폰**

<figure><img src="/images/promotion/coupon/list/addform/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FMDpmtrcH3IPqsUwT5DrQ%2F%ED%95%A0%EC%9D%B8%EC%BF%A0%ED%8F%B0%EB%B0%9C%ED%96%89%EA%B4%80%EB%A6%AC%20%EC%BF%A0%ED%8F%B0%EB%93%B1%EB%A1%9D%20%EA%B8%88%EC%95%A1%EC%84%A4%EC%A0%95%20%EA%B8%B0%ED%94%84%ED%8A%B8%EC%A0%81%EC%9A%A9%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;금액설정 기프트  쿠폰></p></figcaption></figure>

**① 혜택금액**  \
쿠폰으로 지급하는 적립금(혜택금액)을 입력합니다.

**② 혜택 유효기간 설정**\

<figure><img src="/images/promotion/coupon/list/addform/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FoTmOYrVK6orSRhfpq03C%2F%ED%95%A0%EC%9D%B8%EC%BF%A0%ED%8F%B0%EB%B0%9C%ED%96%89%EA%B4%80%EB%A6%AC%20%EC%BF%A0%ED%8F%B0%EB%93%B1%EB%A1%9D%20%EB%8C%80%EC%83%81%EC%84%A4%EC%A0%95%20%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;대상설정 상품적용 쿠폰></p></figcaption></figure>

쿠폰 할인 적용 대상 상품을 설정할 수 있습니다.

* 전체상품 : 쇼핑몰에 등록된 모든 상품을 대상으로 쿠폰 할인을 적용합니다.
* 브랜드 : 상품 등록 시 설정한 브랜드 대상으로 쿠폰 할인을 적용할 수 있습니다.  '브랜드'를 선택하면 브랜드 검색창이 노출됩니다.
* 카테고리 : 상품 등록 시 설정한 전시카테고리 대상으로 쿠폰 할인을 적용할 수 있습니다.  '카테고리'를 선택하면 카테고리 찾기 또는 카테고리 검색이 가능합니다.
* 파트너 : 파트너사 상품을 대상으로 쿠폰 할인을 적용할 수 있습니다.  '파트너'를 선택하면 파트너 검색창이 노출됩니다.
* 개별상품 : 상품조회 또는 엑셀 일괄등록 방식으로 쿠폰할인 적용 상품을 개별 설정합니다.

<Callout type="danger">
'승인상태 - 등록대기'와 '판매상태 - 판매금지'인 상품은 할인대상 상품으로선택이 불가합니다.\
상품 등록 시 프로모션 적용을 '불가능'으로 설정한 상품은 상품 적용 쿠폰을 사용할 수 없습니다.
</Callout>

**② 할인대상 제외**\
특정 상품만 쿠폰 할인 대상 제외 상품으로 별도 설정이 가능합니다. 할인대상 제외 사용을 '설정함'으로 선택하고, 할인 대상 제외 상품을 설정합니다.

**② 사용가능 결제수단** \
쿠폰 사용이 가능한 결제수단을 개별 설정할 수 있습니다. '설정'을 선택하고 쿠폰 사용 결제수단을 설정합니다.

**③ 사용기간** \
쿠폰을 사용할 수 있는 기간을 설정할 수 있습니다.

* 쿠폰 발급일로부터, 당일/1\~30일/60일/90일/180일/1년/해당월 말일 중에서 사용가능 기간을 설정합니다.
* 쿠폰 사용 종료일자를 특정일시로 설정할 수있습니다.  \

  &#x20;          ⇀ 사용 종료일은 2024-04-24  23시 59분(까지 사용가능) 부터 설정이 가능해요.
