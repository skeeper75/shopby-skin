# 추가할인 등록

### 기본정보

<figure><img src="/images/promotion/discount/additional/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FsG8FEWKAPLlDYD3P1Gkc%2F%EC%B6%94%EA%B0%80%ED%95%A0%EC%9D%B8%EB%93%B1%EB%A1%9D-%EA%B8%B0%EB%B3%B8%EC%A0%95%EB%B3%B4%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;기본정보></p></figcaption></figure>

**① 쇼핑몰**\
추가할인을 등록하는 쇼핑몰을 선택합니다.

**② 추가할인명**\
추가할인 프로모션 특성에 맞는 이름을 입력합니다.

**③ 추가할인 설명**\
추가할인 사용용도 등 프로모션 관련 부가설명을 입력할 수 있습니다.  \
운영자 관리 목적의 입력 항목으로 쇼핑몰에 노출되지 않습니다.

**④ 할인기간**\
추가할인이 적용되는 기간을 시작일과 종료일로정설합니다.

### 금액  설정

<figure><img src="/images/promotion/discount/additional/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2Fwx8lv0gzG8NmOSXRrn40%2F%EC%B6%94%EA%B0%80%ED%95%A0%EC%9D%B8%EB%93%B1%EB%A1%9D-%EA%B8%88%EC%95%A1%EC%84%A4%EC%A0%95%EB%B3%B5%EC%82%AC2.jpg" alt=""/><figcaption><p>&#x3C;금액 설정></p></figcaption></figure>

**① 기본(상세 조건 미적용)**\
추가 할인금액을 설정합니다.

**② 구매 수량별**\
상품 동일 옵션을 기준으로 수랑 별 할인금액을 다르게 설정할 수 있습니다.\
옵션 기준 구매수량 범위에 따른 할인금액을  '원' 또는 '%' 단위로 설정 합니다. &#x20;

<Callout type="warning">
구매수량이 해당되는 구간에 설정된 할인금액이 적용되므로, \[추가] 버튼을 클릭 해 입력란을 생성하여 수량 구간과 해당 구간에 적용되는할인금액을 입력합니다. \[삭제] 버튼을 클릭하면 입력란이 삭제됩니다.

구매수량 구간 별 정액(원) 할인금액은  설정한 최소기준금액보다 작아야 합니다.
</Callout>

**③ 회원등급/그룹별**\
상품 동일 옵션을 기준으로 회원등급/회원그룹 별 할인금액을 설정할 수 있습니다.\
회원 등급과 회원 그룹 조건에 따른 할인금액을 '원' 또는 '%' 단위로 설정 합니다. &#x20;

<Callout type="warning">
회원 등급/회원 그룹 조건에 해당되는 구간에 설정된 할인금액이 적용되므로, \[추가] 버튼을 클릭 해 입력란을 생성하여 해당 조건에 적용되는 할인금액을 입력합니다. \[삭제] 버튼을 클릭하면 입력란이 삭제됩니다.

회원등급/그룹 별 정액(원) 할인금액은 설정한 최소기준금액보다 작아야 합니다.
</Callout>

**④ 최소기준금액**\
추가할인을 적용받는 상품 금액의 최소 기준을 설정할 수 있습니다. \
추가할인 대상 상품의 최종 합계 금액이 설정한 최소 금액 기준을 만족한 경우에만 추가할인 적용이 가능합니다.

&#x20;**⑤ 최대기준 금액** \
추가할인을 적용받는 상품 금액의 최대 기준을 설정 할 수 있습니다. \
추가할인 대상 상품의 최종 합계 금액이 설정한 최대 금액 기준을 만족한 경우에만 추가할인 적용이 가능합니다.

**⑥ 비용분담 설정**\
파트너사 입점으로 판매되는 상품을 추가할인하는 경우, 추가할인 금액에 대해 쇼핑몰과 파트너사간 비용 분담 비율을 관리하는 기능입니다. 쇼핑몰 부담 비율을 입력하면 파트너사 부담 비율은 자동 산정되어 반영됩니다.

<Callout type="success">
**최소기준금액 8,000원 설정**

상품 ⓐ : 판매가 9,000원 → 추가할인 적용 가능\
상품 ⓑ : 판매가 7,000원  → 추가할인 적용 불가능
</Callout>

<Callout type="success">
**최대기준금액 50,000원 설정**

상품 ⓐ : 판매가 48,000원 → 추가할인 적용 가능\
상품 ⓑ : 판매가 52,000원 → 추가할인 적용 불가능
</Callout>

### 대상 설정

<figure><img src="/images/promotion/discount/additional/spaces%2F5MtRLrF338WdIfIJP9jq%2Fuploads%2FRsMmGhqfkqyxunv38rdn%2F%EC%B6%94%EA%B0%80%ED%95%A0%EC%9D%B8%EB%93%B1%EB%A1%9D-%EB%8C%80%EC%83%81%EC%84%A4%EC%A0%95%EB%B3%B5%EC%82%AC.png" alt=""/><figcaption><p>&#x3C;대상설정></p></figcaption></figure>

**① 할인적용회원**\
추가할인을 적용 받을 수 있는 회원을 별도 설정할 수 있습니다.&#x20;

* 전체 선택 : 회원과 비회원 모두를 대상으로 한 전체회원과 비회원을 제외한 회원 전용 설정이 가능합니다.
* 회원 선택 : \[회원등급/그룹] 버튼을 클릭하면 '회원등급/그룹 선택'창이 노출됩니다. 추가할인을 적용받을 회원등급 또는 회원그룹을 설정합니다.&#x20;

**② 할인대상**\
추가할인을 적용할 상품을 설정합니다.&#x20;

* 전체상품 : 쇼핑몰에 등록된 모든 상품을 대상으로 추가할인을 적용합니다.&#x20;
* 개별상품 : 상품조회 또는 엑셀 일괄등록 방식으로 추가할인 적용 상품을 개별 설정합니다.

<Callout type="danger">
추가할인은 동일 기간 내, 동일 상품에 중복적용이 불가능합니다.\
여러 개의 추가할인 프로모션을 등록하는 경우, 동일 기간 내 추가할인대상으로 중복 선택된 개별 상품은 '등록일'이 최근인 추가할인 프로모션의 할인 대상이 되는 점을 참고해 주세요.
</Callout>

**③ 할인 대상 제외**\
할인대상 상품을 '전체상품'으로 설정한 경우, 특정 상품만 할인 대상 제외 상품으로 별도 설정이 가능합니다. 할인 대상 제외 사용을 '설정함'으로 선택하고, 추가할인 대상 제외 상품을 설정합니다.
