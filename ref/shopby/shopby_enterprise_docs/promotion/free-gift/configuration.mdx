# 사은품 기본설정

<Callout type="info">
**사은품은 고객이 특정 조건을 달성한 경우 혜택으로 제공되는 프로모션형 상품입니다.**

**사은품 기본설정 메뉴에서는 사은품 사용여부와 중복 지급 허용 여부를 설정할 수 있습니다.**&#x20;
</Callout>

## 사은품기본설정

**① 사은품 사용여부**\
사은품 지급 사용여부를 설정합니다.<br>

**③ 사은품 중복 지급 허용**\
사은품 중복 지급 허용 여부를설정합니다.

* 허용함 : 1개의 상품에 복수의 사은품 지급 조건을 적용 할 수 있습니다.
* 허용 안 함 : 1개의 상품에 1개의 사은품 지급 조건만 적용하여, 사은품 중복 지급을 제한할 수 있습니다.

<Callout type="danger">
**사은품 사용여부와 지급 조건 정책**<br>

사은품 사용여부 '사용 안 함' + 사은품 지급 조건 등록\
:   사은품 지급 조건을 달성하더라도 사은품 지급이 불가합니다.

사은품 사용여부 '사용함' + 상품 주문 완료\
&#x20;:   사은품 지급 조건 달성 이후 사용여부를 '사용 안 함'으로 변경하더라도 사은품은 제공됩니다.

사은품 사용여부 '사용 안 함' + 상품 주문 완료\
:   사은품 지급 조건 달성 이후 사용여부를 '사용함'으로 변경한 경우,  지급 조건을 달성하더라도\
&#x20;   사은품은 제공되지 않습니다.

사은품 중복 지급 '허용 안 함' + 복수 개 사은품 지급 조건 충족\
&#x20;:   사은품 지급 조건 등록일 기준으로 가장 최근에 등록된 1개의 지급 조건만 적용됩니다.
</Callout>
