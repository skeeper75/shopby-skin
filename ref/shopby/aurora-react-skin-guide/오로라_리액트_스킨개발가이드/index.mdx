# 오로라 리액트 스킨개발가이드

### 오로라 리액트(Aurora React) 스킨개발가이드에 오신 것을 환영합니다.

오로라 리액트 스킨은 javascript와 react를 통해 고객들의 최접전에 있는 쇼핑몰 프론트화면을 자유롭게 커스텀할 수 있도록
NHN커머스에서 제작한 스킨입니다.

* 스킨(skin)이란? 쇼핑몰 고객들이 보는 쇼핑몰 프론트 화면 디자인 (템플릿, 레이아웃)을 뜻합니다.



>  :exclamation:본 스킨개발가이드는 오로라(Aurora) 기본 스킨을 기준으로 안내하고 있습니다.
> 
> 본 가이드는 `쇼핑몰 운영방식이 '스킨'`인 몰을 기준으로 안내합니다.
> shop by premium의 헤드리스(headless)몰은 [API 문서](https://docs.shopby.co.kr/)를 확인하여 스킨개발 진행이 필요합니다.

### `sample 스킨`  오로라 리액트(Aurora React)

* shop by basic, pro, premium 모두 라인업 상관없이 적용 가능합니다.
    * 쇼핑몰 신규 생성 시, 쇼핑몰에 기본 스킨으로 적용됩니다.
* 본 가이드는 쇼핑몰 운영방식이 `스킨`인 몰을 기준으로 안내합니다.
    * shop by premium은 쇼핑몰 생성 시 디폴트로 `스킨`으로 적용됩니다.
    * shop by premium 쇼핑몰 관리자 [디자인> 디자인 관리> 디자인스킨리스트] '쇼핑몰 운영방식'을 `헤드리스`(headless)로 변경할 경우, [API 문서](https://docs.shopby.co.kr/)를 확인하여 스킨개발 진행이 필요합니다.
* react 기반
* NHN커머스에서 제공하는 `@shopby패키지`를 활용하여 보다 손 쉬운 개발이 가능합니다
    * `@shopby/shared` : 서버상태 관리와 복잡한 비즈니스 로직을 처리하는 타입스크립트 기반 패키지를 제공합니다
    * `@shopby/react-components` : 리액트 기반의 `ui 컴포넌트`와 `프로바이더`를 제공합니다.
* PC/모바일 구분이 없는 `통합형` 스킨으로, 모바일 퍼스트(mobile first) 스킨으로 제작되었습니다.
