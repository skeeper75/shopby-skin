## ? ```상품 상품 상세정보/ ? 배송 교환 반품 안내```

### 쇼핑몰 관리자(어드민) 경로 안내

![image.png](https://nhnent.dooray.com/share/tree/WoOk8q6KT1K3MZcLSuyZsw/pages/3520341741059334058/attach-files/3525574959384615220)
[상품 등록/수정] 화면에서 아래 항목들에 대한 에디터(editor)기능을 제공하고 있습니다.
에디터 기능을 통해 쇼핑몰 관리자는 상품 상세 정보 및 배송/반품/교환 영역에 노출할 내용을 작성할 수 있습니다.

> 단  ⓐ 상품상세-사용안함값과 ⓑ 배송안내/AS안내/환불안내/교환안내는 shop by pro에서만 제공하는 기능입니다.
> (모든 shop by 라인업에 제공할 수 있도록 개선 중)

* 상품 상세 (상단)
* 상품 상세
* 상품 상세 (하단)
* 배송 안내 `->` shop by pro 전용
* AS 안내 `->` shop by pro 전용
* 환불 안내 `->` shop by pro 전용
* 교환 안내 `->` shop by pro 전용

> [GET /products/{productNo}](https://docs.shopby.co.kr/?url.primaryName=product/#/Product/get-products-product)

```
▶ 상품 상세 조회하기
해당 상품번호(productNo)에 대한 상세 정보를 조회합니다.
```

앞서 본 문서 상단 ? 상품기본 정보 영역에서 소개했던 API입니다.
해당 상품번호에 대한 상품 기본정보, 상세정보, 배송/반품/교환, 후기, 문의, 관련상품 등을 정보를 조회할 수 있습니다.

어드민에 작성한 데이터를 그대로 쇼핑몰 고객들이 보는 화면에 노출합니다.
해당 정보들은 `GET /products/{productNo}` 리턴 값에 포함되어 있으며, 에디터로 작성한 내용은 html 코드 형태로 제공하므로, 곧바로 상품 페이지 html 에 추가할 수 있습니다.

* `contentFooter` 상품 상세 하단
* `contentHeader`상품 상세 상단

### 등록된 옵션 이미지 사용

상품 상세 정보 영역에서 '등록된 옵션 이미지 사용' 기능에 대해 소개합니다.

어드민에서 '상품 상세 사용' 토글 대신 '등록된 옵션 이미지 사용' 항목을 선택할 수 있습니다.
등록된 옵션 이미지 사용 선택 시, 옵션 이미지가 등록되어 있는 옵션의 이미지 등록 여부 및 사용 여부에 따라
[판매정보 > 옵션 > 옵션 수정]에서 등록해 두었던 옵션의 첫 번째 옵션 값과 이미지가 노출됩니다.
![](https://rlyfaazj0.toastcdn.net/works/images/api-dev/prd-detail16.jpg)

> [GET /products/{productNo}/options/{optionNo}/images](https://docs.shopby.co.kr/?url.primaryName=product/#/Product/get-products-option-images)

```
▶ 옵션 이미지 목록 조회
해당 상품번호(productNo)에 대한 옵션 이미지 목록 정보를 조회합니다.
```

'등록된 옵션 이미지 사용' 사용 여부는 응답 값 내 `baseInfo.optionImageViewable` 값을 통해 확인 할 수 있습니다.
옵션 번호 `OptionNo`를 기준으로 옵션 이미지 리스트를 호출할 수 있습니다.

## ? ```상품 후기```

![image.png](https://nhnent.dooray.com/share/tree/WoOk8q6KT1K3MZcLSuyZsw/pages/3520341741059334058/attach-files/3525546010547824966)

### 상품 후기 조회

상품 후기 메뉴 접근 시 상품 후기 목록을 조회합니다.

> [GET /products/{productNo}/product-reviews](https://docs.shopby.co.kr/?url.primaryName=display/#/Review/get-product-product-reviews)

```
▶ 상품후기 목록 조회하기 
해당 상품번호에 작성된 상품후기를 조회합니다.
```

해당 상품 번호 `productNo`기준으로 작성된 상품후기 목록을 조회하며, 아래의 항목으로 리스트를 구성합니다.

* 상품 정보 (상품 썸네일 / 상품명 / 옵션명)
* 별점
* 작성일
* 후기 정보 (후기 내용 / 대표 이미지)

### 상품후기 수정

> [PUT  /products/{productNo}/product-reviews/{reviewNo}](https://docs.shopby.co.kr/?url.primaryName=display/#/Review/put-product-review)

```
▶ 상품후기 수정하기
작성자 본인의 상품 후기을 수정합니다.
```

해당 상품 번호`productNo`에 대한 작성자 본인의 상품후기를 수정합니다.
상품 후기는 작성자 본인만 수정 가능하여 회원 엑세스 토큰 `accessToken` 은 요청 시 필수로 입력해야 합니다.
수정이 가능한 범위는 주문상품, 평점, 내용, 첨부파일 입니다.

### 상품후기 삭제

> [DELETE /products/{productNo}/product-reviews/{reviewNo}](https://docs.shopby.co.kr/?url.primaryName=display/#/Review/delete-product-review)

```
▶ 상품후기 삭제하기
해당 상품번호에 대한 작성자 본인의 상품 후기를 삭제합니다.
```

상품 후기 삭제는 상품후기 상세 페이지에서 기능 제공합니다.
상품 후기는 작성자만 삭제 가능하여 요청 시 회원 엑세스 토큰 `accessToken`을 필수 값으로 전송해야 합니다.

## ?```상품 문의```

### 어드민 설정

상품문의는 [어드민> 게시판 관리> 게시판 리스트] 설정에 따릅니다.
상품문의는 회원만 작성 가능하며, 비회원은 작성이 불가합니다.

> 현재 shop by pro 에서만 상품문의 게시판에 대한 설정기능을 제공합니다. (모든 shop by 라인업에서 제공할 수 있도록 개선 중)

### 상품 문의 조회

> [GET /products/{productNo}/inquiries](https://docs.shopby.co.kr/?url.primaryName=display/#/ProductInquiry/get-product-inquiries)

```
▶ 상품 문의 목록 조회하기
해당 상품 번호에 대한 상품 문의 목록을 조회합니다.
```

위 화면과 같이 해당 상품 번호 `productNo`에 대한 상품 문의 목록을 조회합니다.
답변이 완료 되지 않은 문의 글에 대해서만 수정 및 삭제가 가능합니다.

### 상품 문의 등록

> [POST /products/{productNo}/inquiries ](https://docs.shopby.co.kr/?url.primaryName=display/#/ProductInquiry/post-product-inquiries)

```
▶ 상품 문의 목록 등록하기
해당 상품 번호에 대한 상품 문의를 등록합니다.
```

기본 스킨에서는 아래의 항목을 제공합니다.

* 문의유형
    * 문의유형은 어드민에서 등록한 문의 유형을 select-box로 제공합니다.
* 상품
    * 문의할 상품이 `productNo` 기준으로 고정되어 있습니다.
* 내용
    * 최대 1000자 입력 가능합니다.
* 비밀글 설정
    * 어드민에서 비밀글 쓰기를 사용함으로 설정하는 경우 등록시 비밀글 여부를 선택할 수 있도록 기능이 제공됩니다.
    * 어드민에서 비밀글 쓰기를 사용안함으로 설정하는 경우 등록시 비밀글 여부선택 항목은 제공되지 않아야 합니다.
    * ※ 단, 상품문의 비밀 글쓰기 사용여부는 현재 shop by pro 에서만 제공되는 스펙입니다.

### 상품 문의 수정

> [PUT /products/inquiries/{inquiryNo}](https://docs.shopby.co.kr/?url.primaryName=display/#/ProductInquiry/put-product-inquiry)

```
▶ 상품 문의 수정하기
해당 문의번호 inquiryNo에 대한 상품 문의를 수정합니다.
```

'더보기' 버튼 클릭 시, 작성자 본인만 상품 문의를 수정할 수 있습니다.
제목 / 내용 / 문의 유형만 수정 가능합니다. 답변이 완료된 문의는 수정할 수 없습니다.

### 상품 문의 삭제

> [DELETE /products/inquiries/{inquiryNo}](https://docs.shopby.co.kr/?url.primaryName=display/#/ProductInquiry/delete-product-inquiry)

```
▶ 상품 문의 삭제하기
작성자의 상품 문의를 삭제합니다.
```

'더보기' 버튼 클릭 시, 작성자 본인만 상품 문의를 삭제할 수 있습니다.
답변이 완료 되지 않은 문의건에 대해서만 삭제 가능합니다.
상품문의 상세 조회 시 `inquiryStatus` (답변상태) 값이 답변 완료인 경우 삭제 버튼을 미노출 하여 삭제 기능을 제공하지 않습니다.

## ? ```관련 상품```

어드민에서 설정한 관련 상품 목록을 상품 상세 페이지 최하단에 노출합니다.
디스플레이 유형은 스와이프 형으로, 세로 1줄로 설정한 관련 상품이 3개씩 출력되며 스와이프를 통해 관련상품 확인이 가능합니다. 

> [GET /products/{productNo}/related-products](https://docs.shopby.co.kr/?url.primaryName=product/#/Product/get-products-related-products)

```
▶ 관련 상품 목록 조회하기
관련 상품 정보를 조회할 수 있습니다.
```

어드민 [상품 > 상품 등록 / 수정] 에서 관련 상품을 설정할 수 있습니다.
관련 상품 사용 안함 설정 시 영역이 미노출됩니다.
