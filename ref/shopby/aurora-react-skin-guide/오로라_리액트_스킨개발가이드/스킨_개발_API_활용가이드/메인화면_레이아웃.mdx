# 메인화면_레이아웃

> :bangbang:[동영상 활용가이드](https://nhnent.dooray.com/share/pages/WoOk8q6KT1K3MZcLSuyZsw/3524081956753040600)를 참고해보세요. javascript와 react만 안다면 쉽게 화면 커스텀이 가능합니다.



기본 스킨에서 메인화면의 레이아웃은 아래 3가지 영역으로 구성되어있습니다.

* ? [공통 영역](https://nhnent.dooray.com/share/pages/WoOk8q6KT1K3MZcLSuyZsw/3520339276669442643): 쇼핑몰 내에서 화면이 바뀌어도, 계속 유지되는 요소를 의미합니다. 
* ? [배너 영역](https://nhnent.dooray.com/share/pages/WoOk8q6KT1K3MZcLSuyZsw/3520339109559697129): 쇼핑몰에서 로고 혹은 프로모션 이벤트를 노출하기 위한 공간이자 이미지입니다.
* ? [상품 진열 영역](https://nhnent.dooray.com/share/pages/WoOk8q6KT1K3MZcLSuyZsw/3520339180345088313): 메인 페이지 내에서 특별한 목적에 따라 상품 리스트를 노출 하기 위한 공간입니다
