# PageScriptsProvider

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | --- | --- | --- | --- |
| platform | X | PC, MOBILE | PC | 조회할 스크립트의 플랫폼 타입 |
| typesToExclude | X | Array | [] | 스크립트를 조회할 때 제외할 스크립트 종류
ㄴ 'ADMIN': 어드민에서 설정한 스크립트
ㄴ 'EXTERNAL': 외부 앱용 스크립트 |
| children | O | ReactNode | - | - |

## 상태

* pageScriptsState 은 usePageScriptsStateContext 를 호출해 사용한다.

``` typescript
const pageScriptsState = usePageScriptsStateContext();
```

| 이름 | 타입 | 기본 값 | 설명 |
| --- | --- | --- | --- |
| pageScripts | [response 타입](https://docs.shopby.co.kr/?url.primaryName=manage/#/Page/search-external-script) \| null | null | 페이지별 적용해야하는 스크립트들 |

## 액션

* pageScriptsAction 은 usePageScriptsActionContext 를 호출해 사용한다.

``` typescript
const pageScriptsAction = usePageScriptsActionContext();
```

| 이름 | 인자타입 | 필수여부 | 기본값 | 설명 | 관리하는 상태 |
| --- | --- | --- | --- | --- | --- |
| applyPageScripts | object | O | - | 페이지 스크립트 적용하기 | - |
| ㄴ pageType | PageTypeToShopbyGlobalVariable(하기 내용 참고) 의 key 타입 | O | - | 적용할 스크립트의 페이지 타입 | - |
| ㄴ data | PageTypeToShopbyGlobalVariable의 value 타입 
(pageType 인자에 따라 정해집니다.) | - | - | 적용할 스크립트가 적용되기 전 sb 글로벌 변수에 세팅되어야 하는 데이터값 | - |
