# TermsProvider

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| termsTypes | false | string[] | [] | 약관 타입
ㄴ[request](https://docs.shopby.co.kr/?url.primaryName=manage/#/Terms/search-used-terms).termsTypes 타입 참고 |
| termsHistoryTypes | false | string[] | [] | 약관 변경이력 타입
ㄴ [request](https://docs.shopby.co.kr/?url.primaryName=manage/#/Terms/search-terms-histories).termsType 타입 참고 |
| futureDaysToShow | false | string | '' | 조회할 현재부터 미래 날짜 |
| hasOnlyUsedTerms | false | boolean | false | 조회된 약관 중 현재 사용중인 약관만 필터링 여부 |
| children | true | ReactNode |  |  |

## 상태

* termsState 는 useTermsStateContext 를 호출해 사용한다.

```typescript
const termsState = useTermsStateContext();
```

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| terms | false | object |  | 해당 쇼핑몰의 약관 정보
[response](https://docs.shopby.co.kr/?url.primaryName=manage/#/Terms/search-used-terms) 타입과 동일 |
| termsDetail | false | object |  | 특정 약관(약관번호 기준)의 상세정보
[response](https://docs.shopby.co.kr/?url.primaryName=manage/#/Terms/get-term) 타입과 동일 |
| termsHistory | false | object[] | [] | 해당 쇼핑몰 약관의 변경이력 정보
[response](https://docs.shopby.co.kr/?url.primaryName=manage/#/Terms/search-terms-histories) 타입과 동일 |
| customTerms | false | object[] | [] | 해당 쇼핑몰의 추가 약관 정보
[response](https://docs.shopby.co.kr/?url.primaryName=manage/#/Terms/get-custom-terms) 타입과 동일 |
| termsUsed | false | object |  | 해당 쇼핑몰의 현재 적용 중인 약관
[response](https://docs.shopby.co.kr/?url.primaryName=manage/#/Terms/search-used-terms-only-used) 타입과 동일 |

## 액션

* termsAction 은 useTermsActionContext 를 호출해 사용한다.

```typescript
const termsAction = useTermsActionContext();
```

| --- | ---- | ---- | --- | --- | ------- |
| fetchTerms | string | true |  | 해당 쇼핑몰의 약관을 조회함.

| ㄴ futureDaysToShow | string | false | - | 조회할 현재부터 미래 날짜 |  |
| ㄴ cacheOption | [object](#cacheOption) |  | - | 브라우저 캐시 옵션 관련 설정

 | termsState.termsDetail |

| fetchCustomTerms | object |  |  | 해당 쇼핑몰의 추가 약관을 조회함. | termsState.customTerms |

### cacheOption[#](#cacheOption) 

| 이름 | 타입 | 기본값 | 설명 |
| --- | --- | --- | --- |
| key | string |  | 캐시 설정 시 활용할 키 값 |
| type | string |  | 캐시 설정 시 저장할 저장소 종류
MEMORY: 메모리에 저장, LOCAL\_STORAGE: 로컬스토리지에 저장, SESSION\_STORAGE: 세션스토리지에 저장, null |
| timeBySeconds | string |  | 캐시 설정 시 저장할 시간(초 단위) |
