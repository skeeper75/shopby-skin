# ProfileAccumulationProvider

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | --- | --- | --- | --- |
| children | true | ReactNode |  |  |

## 상태

* profileAccumulationState 는 useProfileAccumulationStateContext 를 호출해 사용한다.

``` typescript
const profileAccumulationState = useProfileAccumulationStateContext();
```

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | --- | --- | --- | --- |
| profileAccumulation | false | null \| object | null | 적립금 이력정보
[response](https://docs.shopby.co.kr/?url.primaryName=manage/#/Accumulation/search-accumulation-histories) 타입과 동일 |
| profileAccumulationSummary | false | null \| object | null | 적립금 요약정보
[response](https://docs.shopby.co.kr/?url.primaryName=manage/#/Accumulation/search-accumulations-summary) 타입과 동일 |
| profileEstimatedAccumulation | false | null \| object | null | 해당 회원의 예상 적립금(적립대기) 정보 |
| ㄴ waitingAccumulation | false | number | - | 해당 회원의 예상 적립금 |

## 액션

* profileAccumulationAction 은 useProfileAccumulationActionContext 를 호출해 사용한다.

``` typescript
const profileAccumulationAction = useProfileAccumulationActionContext();
```

ㄴ [request](https://docs.shopby.co.kr/?url.primaryName=manage/#/Accumulation/search-accumulations-summary) 타입과 동일 | profileAccumulationState.profileAccumulationSummary |
| fetchProfileEstimatedAccumulation | - | - | - | 해당 회원의 예상 적립금을 조회함. | profileAccumulationState.profileEstimatedAccumulation |
