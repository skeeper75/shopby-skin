# ArticleProvider

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| children | true | ReactNode |  |  |

## 상태

* articleState 는 useArticleStateContext 를 호출해 사용합니다.

```typescript
const articleState = useArticleStateContext();
```

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| article | false | object |  | 특정 게시판(게시판 번호 기준)의 게시글 목록 정보
[response](https://docs.shopby.co.kr/?url.primaryName=manage/#/Board/search-posts) 타입과 동일 |
| articleDetail | false | object |  | 득정 게시글(게시판 번호 기준)의 상세정보
[response](https://docs.shopby.co.kr/?url.primaryName=manage/#/Board/get-post) 타입과 동일 |

## 액션

* articleAction 은 useArticleActionContext 를 호출해 사용합니다.

```typescript
const articleAction = useArticleActionContext();
```
