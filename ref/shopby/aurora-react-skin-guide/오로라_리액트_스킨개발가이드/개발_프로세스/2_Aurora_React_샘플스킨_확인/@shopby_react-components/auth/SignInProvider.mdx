# SignInProvider

* 로그인 기능을 제공합니다.

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| children | true | ReactNode | - | - |

## 상태

* signInValueState 는 useSignInValueContext 를 호출해 사용합니다.

```typescript
const signInValueState = useSignInValueContext();
```

| 이름 | 타입 | 기본 값 | 설명 |
| --- | --- | ---- | --- |
| signInInfo | object |  | 회원 정보 |
| ㄴ memberId | string | '' | 회원 아이디 |
| ㄴ password | string | '' | 회원 비밀번호 |
| hasGuestOrderSheetUrl | boolean | false | 비회원 주문서 url 존재여부 |
| guestOrderInfo | object |  | 비회원 주문정보 |
| ㄴ orderNo | string | '' | 주문번호 |
| ㄴ orderPassword | string | '' | 주문번호 비밀번호 |
| isSignedIn | boolean | false | 로그인 여부 |
| dormantMemberResponse | object |  | 휴면 회원 정보 |
| ㄴ memberName | null \| string | null | 휴면 회원명 |
| ㄴ mobileNo | null \| string | null | 휴면 회원 휴대폰 번호 |
| ㄴ email | null \| string | null | 휴면 회원 이메일 |
| daysFromLastPasswordChange | null \| number | null | 비밀번호 변경일로부터 경과된 일 수 |




## 액션

* signInAction 은 useSignInActionContext 를 호출해 사용합니다.

```typescript
const signInAction = useSignInActionContext();
```
