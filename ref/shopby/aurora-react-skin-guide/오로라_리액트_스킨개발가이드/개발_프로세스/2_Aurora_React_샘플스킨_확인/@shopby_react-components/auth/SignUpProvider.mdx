# SignUpProvider

* 회원가입하는 기능을 제공합니다.

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| children | true | ReactNode | - |  |
| terms | true | Record\ | - | 약관 정보
ㄴ TermsType은 [termsTypes](https://docs.shopby.co.kr/?url.primaryName=manage/#/Terms/search-used-terms)와 동일함.
ㄴ TermsKey는 [key](https://docs.shopby.co.kr/?url.primaryName=manage/#/Terms/search-used-terms)와 동일함. |
| PI\_TERMS\_MAP | true | Record\ | - | 회원가입 약관의 key와 key에 따른 약관명을 가짐.
ㄴ key, value는 [termsTypes 참조](https://docs.shopby.co.kr/?url.primaryName=manage/#/Terms/search-used-terms)
`ex) { PI_COLLECTION_AND_USE_OPTIONAL: '개인정보 수집/이용' }` |

* `signUpState` 는 `useSignUpStateContext` 를 호출해 사용합니다.
