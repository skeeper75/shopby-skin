# ImageFileProvider

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | --- | --- | --- | --- |
| children | false | ReactNode | - | - |

## 상태

* imageFileState 는 useImageFileStateContext 를 호출해 사용한다.

``` typescript
const imageFileState = useImageFileStateContext();
```

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | --- | --- | --- | --- |
| postedImages |  | object[] | [] | ToastImage 저장소에 업로드한 이미지 목록 |
| ㄴ imageUrl | true | string |  | 이미지 URL |
| ㄴ originName | true | string |  | 원본 파일명 |

## 액션

* imageFileAction 은 useImageFileActionContext 를 호출해 사용한다.

``` typescript
const imageFileAction = useImageFileActionContext();
```

| 이름 | 인자타입 | 필수여부 | 기본값 | 설명 | 관리하는 상태 |
| --- | --- | --- | --- | --- | --- |
| postImages | object | true | - |  | imageFileState.postedImages |
| ㄴ files | [File](https://developer.mozilla.org/en-US/docs/Web/API/File)[] | true | [] | ToastImage 저장소에 업로드할 이미지 파일 목록 |  |
| ㄴ images | postedImages | false | [] | 저장소에 업로드 된 이미지 파일 목록ㄴ imageFileState.postedImages 의 타입과 동일함. |  |
| ㄴ limitFileSizeInMegaBytes | number | false |  | 저장 가능한 이미지 파일 최대 용량 (MB 단위) |  |
| ㄴ limitCount | number | false |  | 저장 가능한 이미지 파일 수 |  |
| deleteImage | object | true | - | 이미지를 삭제함. |  |
| ㄴ imageUrl | ㄴstring | true | - | 삭제할 이미지 Url |  |
| deleteImages | object | true | - | 복수개의 이미지를 삭제함. |  |
| ㄴ imageUrls | ㄴ string[] | true | - | 삭제할 이미지 Urls |  |
