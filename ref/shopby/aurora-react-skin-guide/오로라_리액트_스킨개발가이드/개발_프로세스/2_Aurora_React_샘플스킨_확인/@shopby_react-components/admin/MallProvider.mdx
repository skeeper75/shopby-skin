# MallProvider

* `clientId` 와 `mallProfile` 은 결제모듈, 방문자 통계 등에서 사용합니다.
    * `clientId` 는 쇼핑몰 사이트를 출력하기 위해 할당된 각 상점의 쇼핑몰 구분 값입니다.
    * [shop api](https://docs.shopby.co.kr/#/Mall/get-malls) 호출 시 해당하는 쇼핑몰을 판단할 수 있는 `key` 값입니다.
    * `도메인/environment.json` 으로 확인할 수 있습니다.
        * 예) `example.shopby.co.kr/environment.json`

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| clientId | o | string | - | environment.json > clientId |
| mallProfile | o | alpha \| real | - | environment.json > profile |
| children | x | ReactNode | - |  |

## 상태

* `mallState` 는 `useMallStateContext` 를 호출해 사용합니다.

```typescript
const mallState: MallState = useMallStateContext();
```
