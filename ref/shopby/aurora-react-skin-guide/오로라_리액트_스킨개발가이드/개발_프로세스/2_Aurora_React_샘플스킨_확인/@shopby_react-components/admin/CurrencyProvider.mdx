# CurrencyProvider

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| currencyCode | x | string | KRW | 화폐 단위를 결정할 코드 |
| currencySymbolMap | x | object | { KRW: '원', USD: '$', JPY: '¥' } | 화폐 단위를 결정할 코드 맵 |
| children | x | ReactNode | - |  |




## 상태

* currencyState 는 useCurrencyStateContext 를 호출해 사용합니다.

```javascript
const currencyState = useCurrencyStateContext();
```

| 이름 | 타입 | 기본 값 | 설명 |
| --- | --- | ---- | --- |
| currencySymbol | string | 원 | 화폐 단위 |




## 액션

* currencyAction 은 useCurrencyActionContext 를 호출해 사용합니다.

```javascript
const currencyAction = useCurrencyActionContext();
```
