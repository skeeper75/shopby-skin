# CouponProvider

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| children | false | ReactNode | - | - |

## 상태

* couponState 는 useCouponStateContext 를 호출해 사용한다.

```typescript
const couponState = useCouponStateContext();
```

## 액션

* couponAction 은 useCouponActionContext 를 호출해 사용한다.

```typescript
const couponAction = useCouponActionContext();
```

| 이름 | 인자타입 | 필수여부 | 기본값 | 설명 | 관리하는 상태 |
| --- | ---- | ---- | --- | --- | ------- |
| fetchCoupons | object | true |  | 로그인한 사용자가 보유한 쿠폰중 사용가능한 쿠폰과 이미 사용한 쿠폰을 구분하여 조회함. | couponState.coupon |
| ㄴ pageNumber | number | true | 1 |  |  |
| ㄴ pageSize | number | true | 10 |  |  |
| ㄴ sortDirection | DESC \| ASC | false | DESC | 내림차순 정렬 |  |
| ㄴ canUse | null \| boolean | false | - | 사용가능 여부 
ㄴ true: 사용 가능
ㄴ false: 사용 불가능
ㄴ null: 조건 없음 |  |
| ㄴstartYmd | string | false | '' | 검색 시작 일 |  |
| fetchProfileCouponSummary | object | false |  | 로그인한 사용자가 보유한 쿠폰의 정보를 요약하여 조회함. | couponState.profileCouponSummary |
| ㄴ expireDay | string | false |  | 만료일
ㄴ (현재 + 만료일)이 검색 조건에 포함됨. |  |
| fetchIssuableCoupons | - |  |  | 상품과 상관없이 오늘 날짜 기준으로 다운로드 가능한 쿠폰을 모두 조회함. | couponState.issuableCoupons |
| couponIssueNoUse | string | true |  | 쿠폰 사용 (혜택 지급) |  |
