# CouponByProductProvider

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| productNo | true | number |  | 상품번호 |
| children | false | ReactNode | - |  |

## 상태

* couponByProductState 는 useCouponByProductStateContext 를 호출해 사용한다.

```typescript
const couponByProductState = useCouponByProductStateContext();
```

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| coupons | false | object[] | [] | [response](https://docs.shopby.co.kr/?url.primaryName=promotion/#/Coupon/get-issuable-coupon-by-products) 타입과 동일 |
| issuedCouponNos | false | object[] | [] | [response](https://docs.shopby.co.kr/?url.primaryName=promotion/#/Coupon/post-download-coupons)[] 타입과 동일 |
| downloadedCouponResult | false | object |  | [response](https://docs.shopby.co.kr/?url.primaryName=promotion/#/Coupon/post-download-coupons) 타입과 동일 |
| downloadedAllCouponResult | false | object |  | [response](https://docs.shopby.co.kr/?url.primaryName=promotion/#/Coupon/post-download-coupon-by-product) 타입과 동일 |

## 액션

* couponByProductAction 은 useCouponByProductActionContext 를 호출해 사용한다.

```typescript
const couponByProductAction = useCouponByProductActionContext();
```
