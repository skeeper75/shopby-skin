# ClaimProvider

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| claimType | X | string | - | 클레임 목록 조회 시 사용하는 클레임 타입
ㄴ [request.claimType](https://docs.shopby.co.kr/?url.primaryName=claim/#/Guest/get-guest-order-options-no-claims) 과 동일함 |
| children | O | ReactNode | - |  |

## 상태

* claimState 는 useClaimStateContext 를 호출해 사용합니다.

```typescript
const claimState = useClaimStateContext();
```

| 이름 | 타입 | 기본 값 | 설명 |
| --- | --- | ---- | --- |
| claimInfo | null \| object | null | 선택한 옵션에 대한 클레임을 신청할 때 필요한 정보임.
ㄴ [response](https://docs.shopby.co.kr/?url.primaryName=claim/#/Member/get-profile-order-options-no-claims) 와 동일 |

| ㄴ isChecked | boolean | - | ㄴ 클레임 신청 선택 여부 |
| ㄴ claimAmount | number | - | ㄴ 클레임 신청한 수량 |
| allClaimableOptions | object[] | [] | 클레임 신청 가능한 옵션 정보 목록
ㄴ [response. originalOption](https://docs.shopby.co.kr/?url.primaryName=claim/#/Member/get-profile-order-options-no-claims) 타입과 동일 |

| claimReason | string | '' | 클레임 사유

| ㄴ invoiceNo | string | '' | 송장번호 |
| accountForRefund | object | {} | 환불계좌정보
ㄴ [request.bankAccountInfo](https://docs.shopby.co.kr/?url.primaryName=claim/#/Member/post-profile-claims-return) 타입에서 bankName 을 제외한 타입과 동일 |

## 액션

* claimAction 은 useClaimActionContext 를 호출해 사용합니다.

```typescript
const claimAction = useClaimActionContext();
```
