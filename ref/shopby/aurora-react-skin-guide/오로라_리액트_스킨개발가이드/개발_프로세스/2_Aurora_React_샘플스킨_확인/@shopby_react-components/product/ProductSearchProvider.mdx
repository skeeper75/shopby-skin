# ProductSearchProvider

* 상품 목록을 검색 한다.

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | --- | --- | --- | --- |
| children | true | ReactNode | - |  |

## 상태

* `productSearchState`는 `useProductSearchStateContext`를 호출해 사용한다.

``` tsx
const eventState = useProductSearchStateContext();
```

| 이름 | 타입 | 기본 값 | 설명 |  |
| --- | --- | --- | --- |
| productSearchResponse | object | null | 상품 검색 데이터 |  |
| pageCount | number | 0 | 페이지 수 |  |
| brand | [object&#91;&#93;](https://docs.shopby.co.kr/?url.primaryName=product/#/Product/get-products-search) |  | 브랜드 정보 response와 동일 |  |
| category | [object](https://docs.shopby.co.kr/?url.primaryName=product/#/Product/get-products-search) |  | 카테고리 정보 response (depth1 \~ 4Categories, multiLevelCategories) |  |
| totalCount | number | 0 | 총 개수 |  |
| minPrice | number | 0 | 최소 가격 |  |
| displayableStock | boolean | false | 재고 노출 여부 |  |
| maxPrice | number | 0 | 최대 가격 |  |
| clickUrlPrefix | object | null |  |  |
| items | [object&#91;&#93;](https://docs.shopby.co.kr/?url.primaryName=product/#/Product/get-products-search) | [] | 상품 목록 response와 동일 |  |
| accumulationProducts | [object&#91;&#93;](https://docs.shopby.co.kr/?url.primaryName=product/#/Product/get-products-search) | [] | 상품 목록 response 와 동일 |  |
| products | [object&#91;&#93;](https://docs.shopby.co.kr/?url.primaryName=product/#/Product/get-products-search) | [] | 상품 목록 response와 동일 |  |
| pageNumber | number | 1 | 페이지 번호 |  |
| sortType | POPULAR, DISCOUNTED\_PRICE, REVIEW, SALE\_CNT, RECENT\_PRODUCT, MD\_RECOMMEND, LIKE\_CNT | POPULAR | 정렬 조건
POPULAR:판매인기순(검색엔진 도입), DISCOUNTED\_PRICE:가격순, REVIEW:상품평, SALE\_CNT:총판매량순, RECENT\_PRODUCT:최근상품순, MD\_RECOMMEND:MD추천순, LIKE\_CNT: 좋아요 |  |

## 액션

* `eventAction`는 `useEventActionContext`를 호출해 사용한다.

``` tsx
const productSearchAction = useProductSearchActionContext();
```
