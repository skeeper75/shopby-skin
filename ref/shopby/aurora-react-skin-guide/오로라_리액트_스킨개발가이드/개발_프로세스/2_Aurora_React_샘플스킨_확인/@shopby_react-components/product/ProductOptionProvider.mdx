# ProductOptionProvider

* 첨부 이미지와 같이 선택 가능한 옵션과 텍스트 옵션 관리 기능을 제공한다.

[선택 가능한 옵션 이미지]
![image](https://github.nhnent.com/storage/user/2230/files/ba658380-e994-11ed-95c4-158d455bb102)



[텍스트 옵션 이미지]
![image](https://github.nhnent.com/storage/user/2230/files/e7199b00-e994-11ed-956a-3514142a42f9)



## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | --- | --- | --- | --- |
| productNo | false | number |  | 상품번호 |
| initialState | false | productOptionState | productOptionState | productOptionState 타입과 동일함.
ㄴ 기본값은 하기 productOptionState 초기 상태와 동일함 |
| children | false | ReactNode | - |  |

## 상태

* productOptionState 는 useProductOptionStateContext 를 호출해 사용한다.

``` typescript
const productOptionState = useProductOptionStateContext();
```
