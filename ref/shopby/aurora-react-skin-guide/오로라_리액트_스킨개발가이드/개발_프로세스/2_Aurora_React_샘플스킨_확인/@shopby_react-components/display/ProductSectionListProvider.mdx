# ProductSectionListProvider

* 메인페이지 섹션 더보기 클릭시 상품 목록을 제공합니다.

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| children | true | ReactNode | - |  |
| platformType | true | PC, MOBILE\_WEB |  | PC 또는 MOBILE\_WEB의 섹션코드를 얻기 위한 타입 |

## 상태

* `productSectionListState`는 `useProductSectionListStateContext` 를 호출해 사용합니다.

```ts
const productSectionListState = useProductSectionListStateContext();
```

| 이름 | 타입 | 기본 값 | 설명 |
| --- | --- | ---- | --- |
| displaySectionResponse | object | null | 진열 상품 목록 전체 응답 값 |
| productTotalCount | number | 0 | 전체 상품 수 |
| label | string | - | 진열 명 |
| products | [object\[\]](https://nhnent.dooray.com/#products)아래 products 참조 | [] | 상품 목록 |
| accumulationProducts | [object\[\]](https://nhnent.dooray.com/#products)아래 products 참조 | [] |  |
| pageNumber | number | 1 | 페이지 번호 |
| sortType | SALE, LOW\_PRICE, HIGH\_PRICE, REVIEW, REGISTER | SALE | 상품 목록 정렬 타입
SALE(판매량 순), LOW\_PRICE(낮은 가격순), HIGH\_PRICE(높은 가격순), REVIEW(상품후기순), REGISTER(등록일순) |

## 액션

* `productSectionListAction`은 `useProductSectionListActionContext`를 호출해 사용합니다.

```tsx
const productSectionListAction = useProductSectionListActionContext();
```
