# ProductInquiryProvider

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |

## 상태

* productInquiryState 는 useProductInquiryStateContext 를 호출해 사용합니다.

```typescript
const productInquiryState = useProductInquiryStateContext();
```

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| totalCount | true | number | 0 | [response.totalCount](https://docs.shopby.co.kr/?url.primaryName=display/#/ProductInquiry/get-products-inquiries) |
| inquiries | true | object[] | [] | [response.items](https://docs.shopby.co.kr/?url.primaryName=display/#/ProductInquiry/get-products-inquiries) |
| inquiryConfig | false | object |  | [response](https://docs.shopby.co.kr/?url.primaryName=display/#/ProductInquiry/get-inquiries-configurations) |

## 액션

* productInquiryAction 은 useProductInquiryActionContext 를 호출해 사용합니다.

```typescript
const productInquiryAction = useProductInquiryActionContext();
```

| 이름 | 인자타입 | 필수여부 | 기본값 | 설명 | 관리하는 상태 |

productInquiryState.inquiries |
