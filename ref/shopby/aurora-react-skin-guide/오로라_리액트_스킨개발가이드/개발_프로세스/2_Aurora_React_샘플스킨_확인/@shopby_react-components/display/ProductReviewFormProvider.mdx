# ProductReviewFormProvider

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| children | true | ReactNode |  |  |

## 상태

* productReviewFormState 는 useProductReviewFormStateContext 를 호출해 사용합니다.

```typescript
const productReviewFormState = useProductReviewFormStateContext();
```

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| reviewDetail | true | object |  | [response](https://docs.shopby.co.kr/?url.primaryName=display/#/Review/get-products-product-reviews) 와 동일 |
| reviewableOption | true | object |  | [response](https://docs.shopby.co.kr/?url.primaryName=display/#/Review/get-product-reviews-configurations) 와 동일 |

## 액션

* productReviewFormAction 은 useProductReviewFormActionContext 를 호출해 사용합니다.

```typescript
const productReviewFormAction = useProductReviewFormActionContext();
```
