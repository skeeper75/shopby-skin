# ProfileProductReviewProvider

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| children | true | ReactNode |  |  |

## 상태

* profileProductReviewState 는 useProfileProductReviewStateContext 를 호출해 사용합니다.

```typescript
const profileProductReviewState = useProfileProductReviewStateContext();
```

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| reviewableProduct | false | object |  | 작성가능한 상품평정보 |
| ㄴ totalCount | false | number | 0 | 작성가능한 상품평 수 |
| ㄴ items | false | object[] | [] | 작성가능한 상품평 목록
[request.items](https://docs.shopby.co.kr/?url.primaryName=display/#/Review/get-product-reviewable) 에 하기 타입 추가 |
| ㄴ isDeliverable | false | boolean |  | 배송상품여부 |
| ㄴ usesOption | false | boolean |  | 옵션사용여부 |
| ㄴ canDeliveryInternational | false | boolean |  | 해외 배송가능 여부 |
| ㄴ isRefundable | false | boolean |  | 환불 가능 여부 |
| ㄴ canExchange | false | boolean |  | 교환 가능 여부 |
| reviewedProduct | false | object |  | 작성된 상품평 정보 |
| ㄴ totalCount | false | number | 0 | 작성된 상품평 수 |
| ㄴ items | false | object[] | [] | 작성된 상품평 목록
[response.items](https://docs.shopby.co.kr/?url.primaryName=display/#/Review/get-profile-product-reviews). 에 하기 타입 추가 |
| ㄴ isBestReview | false | boolean |  | 우수 상품평 여부 |
| ㄴ isExternalReview | false | boolean |  | 외부 리뷰 작성 여부 |

## 액션

* profileProductReviewAction 은 useProfileProductReviewActionContext 를 호출해 사용합니다.

```typescript
const profileProductReviewAction = useProfileProductReviewActionContext();
```

| 이름 | 인자타입 | 필수여부 | 기본값 | 설명 | 관리하는 상태 |
| --- | ---- | ---- | --- | --- | ------- |
| fetchProfileReviewableProducts | object | false |  | 내 상품평 작성 가능 목록을 조회함.
[request](https://docs.shopby.co.kr/?url.primaryName=display/#/Review/get-product-reviewable) 타입과 동일 | profileProductReviewState.reviewableProduct |
| fetchProfileReviewedProducts | object | false |  | [reques](https://docs.shopby.co.kr/?url.primaryName=display/#/Review/get-profile-product-reviews) 타입에 하기 타입 추가 | profileProductReviewState.reviewedProduct |
| ㄴ isBestReview | boolean | false |  | 우수상품평 여부 |  |
