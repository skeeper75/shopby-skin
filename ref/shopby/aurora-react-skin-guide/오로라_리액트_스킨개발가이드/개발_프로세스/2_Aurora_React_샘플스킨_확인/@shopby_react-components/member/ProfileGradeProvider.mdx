# ProfileGradeProvider

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | --- | --- | --- | --- |
| children | false | ReactNode | - | - |

## 상태

* profileGradeState 는 useProfileGradeStateContext 를 호출해 사용한다.

``` typescript
const profileGradeState = useProfileGradeStateContext();
```

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | --- | --- | --- | --- |
| profileGrade | true | object |  | 회원 등급 정보
ㄴ [response](https://docs.shopby.co.kr/?url.primaryName=member/#/Profile/get-profile-grade) 에 하기 프로퍼티가 추가됨. |
| ㄴ reserveAutoSupplying | false | object |  | [response.reserveAutoSupplying](https://docs.shopby.co.kr/?url.primaryName=member/#/Profile/get-profile-grade) 에 하기 프로퍼티가 추가됨. |
|     ㄴ canUse | false | boolean |  | 적립금 자동지급 사용여부 |
| ㄴ reserveBenefit | false | object |  | [response.reserveBenefit](https://docs.shopby.co.kr/?url.primaryName=member/#/Profile/get-profile-grade) 에 하기 프로퍼티가 추가됨. |
|     ㄴ canUse | false | boolean |  | 적립금 혜택 사용여부 |

## 액션

* profileGradeAction 은 useProfileGradeActionContext 를 호출해 사용한다.

``` typescript
const profileGradeAction = useProfileGradeActionContext();
```
