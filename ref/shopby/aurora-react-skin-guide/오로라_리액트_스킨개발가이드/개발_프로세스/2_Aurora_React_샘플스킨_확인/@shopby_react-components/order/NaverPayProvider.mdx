# NaverPayProvider

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | --- | --- | --- | --- |
| mallProfile | O | string | real | 몰 서빙 환경 |
| clientId | O | string | - | 몰 clientId |
| platform | O | 'PC' \| 'MOBILE\_WEB' \| 'IOS' \| 'AOS' | - | 클라이언트 플랫폼 |
| children | O | ReactNode | - | - |

## 상태

* NaverPayProvider는 내려주는 상태값이 없습니다.

## 액션

* naverPayAction 은 useNaverPayActionContext 를 호출해 사용한다.

``` typescript
const naverPayAction = useNaverPayActionContext();
```
