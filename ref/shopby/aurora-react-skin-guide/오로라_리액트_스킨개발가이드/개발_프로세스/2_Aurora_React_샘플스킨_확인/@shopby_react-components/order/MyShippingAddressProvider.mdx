# MyShippingAddressProvider

* 배송지 주소 정보를 관리한다.

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | --- | --- | --- | --- |
| children | true | ReactNode | - | - |

## 상태

* myShippingAddressState는 useMyShippingAddressStateContext 를 호출해 사용한다.

``` typescript
const myShippingAddressState = useMyShippingAddressStateContext();
```

| 이름 | 타입 | 기본 값 | 설명 |
| --- | --- | --- | --- |
| bookedAddresses | AddressDetail[] | [] | 등록된 배송지 주소 목록 |
| defaultAddress | null \| AddressDetail | null | 기본 배송지 주소 |
| recentAddresses | AddressDetail[] | [] | 최근 배송지 주소 목록 |
| recentAddress | null \| AddressDetail | null | 최근 배송지 주소 |

AddressDetail은 [api Response](https://docs.shopby.co.kr/?url.primaryName=order/#/ShippingAddress/get-profile-shipping-addresses) 값과 동일합니다.

## 액션

* myShippingAddressAction은 useMyShippingAddressActionContext 를 호출해 사용한다.

``` typescript
const myShippingAddressAction = useMyShippingAddressActionContext();
```

| 이름 | 인자타입 | 필수여부 | 기본값 | 설명 | 관리하는 상태 |
| --- | --- | --- | --- | --- | --- |
| fetchMyShippingAddress | - | false | - | 배송지 주소를 조회함. | `myShippingAddressState.defaultAddress`
`myShippingAddressState.recentAddresses`
`myShippingAddressState.bookedAddresses`
`myShippingAddressState.recentAddress` |
| getShippingAddressByAddressNo | string | true | - | 배송지 번호로 세부 정보를 조회함.
ㄴ 인자로 받은 배송지 번호를 기준으로 조회함. |  |
| ㄴ addressNo | string | true | - | 배송지 번호 |  |
| saveMyShippingAddress | object | true | - | 배송지 주소를 등록합니다. |  |
| ㄴ request.body |  |  |  | [request.body](https://docs.shopby.co.kr/?url.primaryName=order/#/ShippingAddress/put-profile-shipping-addresses-address-no) 타입 |  |
| modifyMyShippingAddress | object | true | - | 배송지 주소를 수정합니다. |  |
| ㄴ request.body |  |  |  | [request.body](https://docs.shopby.co.kr/?url.primaryName=order/#/ShippingAddress/put-profile-shipping-addresses-address-no) 타입 |  |
| ㄴ addressNo | string | true |  | 배송지 번호 |  |
| deleteMyShippingAddress | string | true | - | 배송지 주소를 삭제합니다. |  |
| ㄴ addressNo | string | true |  | 배송지 번호 |  |
