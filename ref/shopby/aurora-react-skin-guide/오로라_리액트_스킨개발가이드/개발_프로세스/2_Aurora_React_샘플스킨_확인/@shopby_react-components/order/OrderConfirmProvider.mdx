# OrderConfirmProvider

```
주문서 페이지에서 결제 진행 후 리다이렉팅 되는 confirmUrl의 페이지 컴포넌트를 감싸서 사용 가능한 프로바이더입니다.
```

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | --- | --- | --- | --- |
| orderRequestType | X | 'ALL' \| 'CLAIM' \| 'NORMAL' | 'ALL' | 주문 상세를 조회할 때 사용되는 값임. (deprecated 예정) |
| children | O | ReactNode | - | - |

## 상태

* orderConfirmState 는 useOrderConfirmStateContext 를 호출해 사용한다.

``` typescript
const orderConfirmState = useOrderConfirmStateContext();
```
