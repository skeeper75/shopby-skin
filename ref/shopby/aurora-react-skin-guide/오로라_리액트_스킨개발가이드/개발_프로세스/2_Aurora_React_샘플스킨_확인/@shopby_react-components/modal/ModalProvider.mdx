# ModalProvider

## 속성

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| children | true | ReactNode |  |  |

## 상태

* modalState 는 useModalStateContext 를 호출해 사용한다.

```typescript
const modalState = useModalStateContext();
```

| 이름 | 필수여부 | 타입 | 기본 값 | 설명 |
| --- | ---- | --- | ---- | --- |
| modalState |  |  |  | 현재 오픈 된 모달 정보 |
| ㄴ type | true | null \| alert \| confirm \| customModal | null | 모달 타입 |
| ㄴ name | true | null string | null | 모달의 고유한 이름 |
| alertState | true | object |  |  |
| ㄴ isOpen | true | boolean | false | 모달 오픈 여부 |
| ㄴ label | false | string | 확인 | onClose 이벤트를 발생시키는 버튼의 라벨 |
| ㄴ message | true | string | '' | 모달 내용 |
| ㄴContent | false | null \| ReactNode | null | 모달 내용이 컴포넌트로 들어가는 경우 사용 |
| ㄴ onClose | false | function | () => null |  |
| confirmState | object |  |  |  |
| ㄴ isOpen | true | boolean | false | 모달 오픈 여부 |
| ㄴ message | true | string | '' | 모달 내용 |
| ㄴ Content | false | null \| ReactNode | null | 모달 내용이 컴포넌트로 들어가는 경우 사용 |
| ㄴ confirmLabel | false | string | 등록 | onConfirm 이벤트를 발생시키는 버튼의 라벨 |
| ㄴ cancelLabel | false | string | 취소 | onCancel 이벤트를 발생시키는 버튼의 라벨 |
| ㄴ onConfirm | false | function | () => null |  |
| ㄴ onCancel | false | function | () => null |  |

## 액션

* modalAction 은 useModalActionContext 를 호출해 사용한다.

```typescript
const modalAction = useModalActionContext();
```

| 이름 | 인자타입 | 필수여부 | 기본값 | 설명 | 관리하는 상태 |
| --- | ---- | ---- | --- | --- | ------- |
| openAlert | object |  |  |  | modalState.alertState |
| ㄴ message | string | false | '' | 알럿 모달에 노출될 내용 |  |
| ㄴ Content | null \| ReactNode | false | null | 알럿 모달에 노출될 컴포넌트 |  |
| ㄴ label | string | false | 확인 | 버튼에 들어갈 라벨 |  |
| ㄴ onClose | function | false | () => null | 버튼 클릭 시 발생하는 이벤트핸들러 |  |
| closeAlert | - |  |  |  | modalState.alertState |
| openConfirm | object |  |  |  | modalState.confirmState |
| ㄴ message | string | false | '' | 컨펌 모달에 노출될 내용 |  |
| ㄴ Content | null \| ReactNode | false | null | 컨펌 모달에 노출될 컴포넌트 |  |
| ㄴ confirmLabel | string | false | 등록 | confirm 버튼에 들어갈 라벨 |  |
| ㄴ cancelLabel | string | false | 취소 | cancel 버튼에 들어갈 라벨 |  |
| ㄴ onConfirm | function | false | () => null | onfirm 버튼 클릭 시 발생하는 이벤트핸들러 |  |
| ㄴ onCancel | function | false | () => null | cancel 버튼 클릭 시 발생하는 이벤트핸들러 |  |
| closeConfirm | - |  |  |  |  |
