# 개발 프로세스

> 단, 디자인 파트너 계정이 존재하는 스킨판매 에이전시의 경우, [Gitlab스킨 판매 프로세스 가이드](https://nhnent.dooray.com/share/pages/RwdJEHPDQtCV6688_odoxQ)를 확인하시길 바랍니다.

## 이해하기

shop by 라인업의 쇼핑몰 신규 생성 시, 기본스킨으로 ```NHN커머스에서 제작한 샘플스킨 (스킨명: Aurora)```가 세팅됩니다.
해당 스킨 커스텀하여 "나만의 새로운 스킨"을 제작할 수 있도록 가이드 드립니다.

* step3) [개발](https://nhnent.dooray.com/share/pages/WoOk8q6KT1K3MZcLSuyZsw/3524105094555627383)
* step4) [배너 정보 (asset파일) 이란?](https://nhnent.dooray.com/share/pages/WoOk8q6KT1K3MZcLSuyZsw/3524126083456407491)
* step5) [설치](https://nhnent.dooray.com/share/pages/WoOk8q6KT1K3MZcLSuyZsw/3520342416475553875)
