# 후니프린팅 IA - shopby 기능 매핑 분석

## 분류 범례

| 분류 | 의미 | 설명 |
|------|------|------|
| ✅ shopby Native | 기본 제공 | shopby에서 별도 개발 없이 사용 가능 |
| 🔧 Skin Custom | 스킨 커스텀 | shopby API 활용 + 커스텀 UI/스킨 작업 필요 |
| ⚙️ Custom Dev | 커스텀 개발 | shopby에서 제공하지 않는 기능, 별도 개발 필요 |
| 🔌 External | 외부 연동 | PG, SMS, 외부 서비스 연동 필요 |
