## 3. 정책 결정 사항

### 3.1 비회원 주문 허용 여부 (★ 최우선 결정)

#### 옵션 A: 비회원 주문 허용 (권장)

**장점**:
- 첫 주문 진입 장벽 낮춤 (회원가입 없이 바로 주문 가능)
- 레드프린팅과 동일 수준의 고객 편의성 제공
- 급한 주문 시 즉시 결제 가능

**단점**:
- 고객 데이터 수집 제한 (마케팅 활용 어려움)
- 적립금/쿠폰 혜택 미적용으로 재주문율 낮음
- CS 응대 시 본인 확인 절차 추가 필요

**비회원 주문 시 수집 정보**:
- 주문자명 (필수)
- 휴대전화번호 (필수, 주문조회 키)
- 이메일 (필수, 주문확인 발송)
- 배송지 정보 (필수)
- 비밀번호 (선택, 주문조회용)

**비회원 주문조회 방법**:
- 주문번호 + 휴대전화번호로 조회 (레드프린팅 방식)

#### 옵션 B: 회원 전용 (비회원 주문 불가)

**장점**:
- 회원 전환율 100%
- 고객 데이터 완전 확보
- 적립금/쿠폰 체계 일관 적용

**단점**:
- 첫 주문 이탈율 높음 (회원가입 귀찮아서 이탈)
- 경쟁사 대비 불편한 UX

#### 정책 결정 체크리스트

- [ ] 비회원 주문 허용 여부 결정
- [ ] 비회원 주문 시 수집 정보 범위 결정
- [ ] 비회원 → 회원 전환 유도 방안 (주문 완료 후 "회원가입하면 적립금 N원 지급" 등)
- [ ] 비회원 주문 건 CS 처리 절차 정의

### 3.3 회원 등급 체계

#### 권장안: 4단계 (업계 표준)

| 등급 | 조건 (6개월 기준) | 혜택 |
|------|------------------|------|
| **일반** | 가입 시 | 기본 혜택 |
| **VIP** | 5만원 이상 + 2회 이상 | 적립률 상향, 할인 쿠폰 |
| **VVIP** | 20만원 이상 + 5회 이상 | 추가 적립, 배송비 할인 |
| **SVIP** | 50만원 이상 + 10회 이상 | 최대 적립, 무료배송, 강등 없음 |

참고: BizHows 모델 (FAMILY → VIP → VVIP → SVIP)

#### 정책 결정 체크리스트

- [ ] 등급 단계 수 (3단계 vs 4단계 vs 10단계)
- [ ] 등급 산정 기간 (6개월 롤링 vs 연간)
- [ ] 등급 갱신 주기 (매월 1일 vs 분기)
- [ ] 등급별 혜택 항목 (적립률, 쿠폰, 배송비, 전용 상담)
- [ ] 최고 등급 강등 여부 (SVIP 강등 없음?)
- [ ] B2B 고객 별도 등급 체계 필요 여부

### 3.6 주문 프로세스 (회원 vs 비회원)

#### 회원 주문 플로우

```
상품 선택 → 옵션 설정 → 파일 업로드 → 장바구니
→ 주문서 작성 (배송지 자동완성, 쿠폰/적립금 적용)
→ 결제 → 주문 완료 → 파일 검수 → 제작 → 배송
```

#### 비회원 주문 플로우 (도입 시)

```
상품 선택 → 옵션 설정 → 파일 업로드 → 장바구니
→ 주문서 작성 (주문자 정보 직접 입력, 쿠폰/적립금 미적용)
→ 결제 → 주문 완료 (회원가입 유도 배너)
→ 파일 검수 → 제작 → 배송
```

## 4. 운영 관점 고려사항

### 4.1 CS 대응 방안

**비회원 CS 처리**:
- 주문번호 + 주문자 휴대전화번호로 본인 확인
- 비회원 주문 건은 회원 전환 없이 CS 처리 가능하도록 어드민 지원
- 비회원 주문 건에 대해 "회원가입 후 적립금 전환" 유도

**회원 전환 유도**:
- 주문 완료 페이지: "회원가입하면 1,000P 적립!"
- 주문 확인 이메일: 하단에 회원가입 배너
- 비회원 재주문 시: "이전 주문 이력을 편하게 관리하세요" 안내

### 4.2 마케팅 활용

**회원 데이터 기반**:
- 구매 이력 기반 상품 추천
- 등급별 타겟 프로모션 (VIP 전용 쿠폰 등)
- 생일/기념일 쿠폰 자동 발송

**비회원 한계**:
- 이메일/SMS 마케팅 불가 (동의 미수집 시)
- 재주문 유도 어려움
- 고객 행동 분석 제한

### 4.3 재주문 관리

| 항목 | 레드프린팅 | 후니프린팅 권장 |
|------|-----------|--------------|
| 재주문 기간 | 1개월 보관 | 3개월 보관 (차별화) |
| 재주문 방식 | 간편재주문 | 옵션보관함 + 간편재주문 |
| 파일 보관 | 1개월 | 3개월 (회원 한정) |

## 6. 추천 정책안 (종합)

### 핵심 방향

> "비회원 주문을 허용하되, 회원 전환을 적극 유도하여 재주문율을 높인다"

### 구체 정책

1. **비회원 주문 허용** - 주문 진입장벽 최소화
2. **SNS 로그인** - 카카오 + 네이버 필수 도입
3. **4단계 회원등급** - 일반/VIP/VVIP/SVIP (6개월 롤링)
4. **가입 혜택** - 쿠폰 2,000원 + 적립금 1,000P + 첫 주문 무료배송
5. **적립금 체계** - 결제금액 3% 기본 적립, 등급별 추가
6. **결제 수단** - 카드/무통장/카카오페이/네이버페이 필수
7. **비회원 → 회원 전환** - 주문 완료 시 적립금 전환 유도

### 기술 구현 포인트

1. **비회원 장바구니**: shopby 비회원 장바구니 API 활용 (쿠키/세션 기반)
2. **비회원 → 회원 전환**: 주문 완료 후 회원가입 시, 기존 주문 데이터 연결 필요
3. **SNS 로그인**: shopby OAuth 연동 설정 (어드민 > 서비스관리)
4. **회원등급 자동 산정**: shopby 등급 자동 승급 설정 활용

### API 참조

- 비회원 주문: `POST /order-shop/guest-orders`
- 비회원 주문조회: `GET /order-shop/guest-orders/{orderNo}`
- 회원등급 조회: `GET /member-shop/profile/grades`
- 적립금 조회: `GET /member-shop/profile/accumulations`
