# SPEC-STATS-001: 수용 기준

## 1. 모듈별 수용 기준

### 1.1 모듈 1: 상품 통계 (Product Statistics)

#### AC-STS-001: 인쇄 상품통계 기간별 조회

```gherkin
GIVEN 관리자 계정으로 로그인하고
AND 인쇄/제본 통계 페이지에 접근하면
WHEN 기간 필터에서 "2026년 1월~3월"을 선택하면
THEN 해당 기간의 주문건수, 매출액(공급가 기준)이 표시되고
AND 용지별/코팅별/후가공별/수량별/인쇄도수별/사이즈별 분석 차트가 렌더링된다
```

- 관련 요구사항: REQ-STS-002, REQ-STS-003, REQ-STS-004, REQ-STS-005
- 검증 방법: E2E 테스트 (Playwright)

#### AC-STS-002: 제본 상품 분석 축 전환

```gherkin
GIVEN 인쇄/제본 통계 페이지에서
WHEN 제본 탭을 선택하면
THEN 제본방식별(중철/무선/사철/스프링/양장) 분포 차트가 표시되고
AND 페이지수별, 표지용지별, 합판/독판별, 부수별 분석이 전환 가능하다
```

- 관련 요구사항: REQ-STS-006
- 검증 방법: E2E 테스트

#### AC-STS-003: 굿즈 상품통계 인쇄방식 분석

```gherkin
GIVEN 굿즈 통계 페이지에 접근하면
WHEN 인쇄방식별 분석 탭을 선택하면
THEN 실크/UV/전사/각인/디지털 각 인쇄방식의 주문건수와 매출 비율이 표시된다
```

- 관련 요구사항: REQ-STS-007
- 검증 방법: E2E 테스트

#### AC-STS-004: 패키지 상품통계 톰슨 분석

```gherkin
GIVEN 패키지 통계 페이지에 접근하면
WHEN 톰슨별 분석을 조회하면
THEN 기존톰슨/신규톰슨 비율과 칼형 제작 건수가 표시된다
```

- 관련 요구사항: REQ-STS-008
- 검증 방법: E2E 테스트

#### AC-STS-005: 수작 상품통계 공정/시간 분석

```gherkin
GIVEN 수작 통계 페이지에 접근하면
WHEN 소요시간별 분석을 조회하면
THEN 1시간 이내/반일/1일/2일+ 각 구간의 주문건수와 비율이 표시된다
```

- 관련 요구사항: REQ-STS-009
- 검증 방법: E2E 테스트

#### AC-STS-006: 차트 유형 전환

```gherkin
GIVEN 상품 통계 페이지에서 바 차트가 표시된 상태에서
WHEN 라인 차트 전환 버튼을 클릭하면
THEN 동일 데이터가 라인 차트로 전환되어 표시된다
```

- 관련 요구사항: REQ-STS-010
- 검증 방법: E2E 테스트

#### AC-STS-007: 데이터 테이블 정렬/필터

```gherkin

WHEN 매출액 열 헤더를 클릭하면
THEN 매출액 기준 내림차순으로 정렬되고
```

- 검증 방법: E2E 테스트

#### AC-STS-008: 빈 데이터 안내

```gherkin

THEN "해당 기간의 데이터가 없습니다" 메시지가 표시되고
```

- 관련 요구사항: REQ-STS-013
- 검증 방법: E2E 테스트

#### AC-STS-009: 미결제 주문 제외 검증

```gherkin

WHEN 해당 기간의 통계를 조회하면
THEN 입금대기 주문의 금액은 매출 합계에 포함되지 않는다
```

- 관련 요구사항: REQ-STS-003, REQ-STS-014

## 2. 품질 게이트

### 2.1 Definition of Done

- [ ] 모든 AC 테스트 시나리오 통과
- [ ] 집계 로직 단위 테스트 커버리지 95% 이상

- [ ] 엑셀 내보내기 1만 행 5초 이내 (성능 테스트 통과)
- [ ] RBAC 권한 검증 (운영자 타팀 접근 차단)
- [ ] 감사 로그 기록 검증 (엑셀 다운로드)
- [ ] 접근성 검증 (데이터 테이블, 차트 대체 텍스트)
- [ ] PC-First 반응형 검증 (1280px, 1920px)
