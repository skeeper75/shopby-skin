# SPEC-SKIN-006: 관리자 상품관리

> **SPEC ID**: SPEC-SKIN-006
> **상태**: Draft
> **작성일**: 2026-03-14
> **우선순위**: High
> **Phase**: 2
> **IA 항목**: No. 50~63
> **기술 스택**: React 18 + @shopby/react-components + shadcn/ui + Tailwind CSS

## 1. 개요

관리자 상품관리 전체를 구현한다. 인쇄/제본 상품등록(미리보기/등록/수정, 기본값설정), 사이즈/소재/종이 선택 팝업, 가격관리 팝업 8종, 사이즈/소재/용지 마스터 데이터 관리, 굿즈/수작/포장재/디자인 상품 등록을 포함한다. 가격 시뮬레이터가 핵심이다.

## 6. 인수 조건 (EARS Format)

### REQ-SKIN-006-001: 인쇄/제본 상품 등록

**WHEN** 관리자가 상품 정보와 인쇄 옵션을 설정하고 "등록하기" 버튼을 클릭하면
**THE SYSTEM SHALL** 상품을 등록하고 사용자 쇼핑몰에 노출한다

**수용 기준**:
- [ ] 상품명, 분류, 옵션(사이즈/용지/코팅/후가공) 설정
- [ ] 기본값 설정 (사용자 초기 선택값)
- [ ] 상세 설명 에디터 (이미지 포함)
- [ ] 미리보기 기능

### REQ-SKIN-006-002: 가격관리 매트릭스

**WHEN** 관리자가 가격관리 팝업에서 수량별 단가를 입력하고 저장하면
**THE SYSTEM SHALL** 가격 매트릭스를 저장하고 사용자 가격 계산에 반영한다

**수용 기준**:
- [ ] 사이즈 x 용지 x 코팅 조합별 가격 입력
- [ ] 수량 구간별 단가 (단면/양면)
- [ ] 행 추가/삭제/일괄 적용
- [ ] 저장 시 즉시 사용자 측 가격에 반영

### REQ-SKIN-006-003: 마스터 데이터 관리

**WHEN** 관리자가 사이즈/소재/용지 관리 페이지에서 항목을 추가/수정/삭제하면
**THE SYSTEM SHALL** 마스터 데이터를 업데이트하고 상품 등록 팝업에 반영한다

**수용 기준**:
- [ ] CRUD 테이블 (추가/수정/삭제/사용중지)
- [ ] 사용중인 항목 삭제 방지 (경고)
- [ ] 정렬 순서 변경 지원

### REQ-SKIN-006-004: 사이즈/소재/종이 선택 팝업

**WHEN** 관리자가 상품 등록 시 "사이즈 선택" 팝업을 열면
**THE SYSTEM SHALL** 등록된 사이즈 목록에서 다중 선택을 허용하고, 선택 결과를 상품에 적용한다

**수용 기준**:
- [ ] 체크박스 다중 선택
- [ ] 검색/필터 기능
- [ ] 선택 완료 시 상품 폼에 반영

## UI/UX 개선사항

### 관리자 상품관리 (IA No. 50~63) Improvements

#### Product management improvements

1. **Product form wizard**: 상품 등록을 멀티스텝 폼으로 구성 -- (1)기본정보 → (2)가격설정 → (3)옵션구성 → (4)이미지 → (5)미리보기
2. **Option tree builder**: 드래그앤드롭 옵션 계층 빌더 (@hello-pangea/dnd 또는 @dnd-kit 사용)
3. **Price matrix table**: 수량x옵션 가격 조합을 그리드 에디터로 구현. 스프레드시트 스타일 인라인 편집.
4. **Image upload zone**: 썸네일 미리보기 + 순서 변경이 가능한 드래그앤드롭 이미지 업로드 영역 (redprinting 관리자 패턴 참고)
5. **Preview mode**: "쇼핑몰에서 보기" 버튼으로 고객 시점의 상품 미리보기 제공
6. **Price popup management**: 8종의 가격관리 팝업 -- 각각 모달 형태로 데이터 테이블 + 인라인 편집
7. **Bulk price import**: 가격 매트릭스용 CSV/Excel 업로드 기능
