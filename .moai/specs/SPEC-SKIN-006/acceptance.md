# SPEC-SKIN-006 v2: 관리자 상품관리 - 인수 기준

## 1. 인쇄/제본 상품 등록 (REQ-006-001)

### AC-006-001: 상품 등록 성공
```gherkin
Given 관리자가 상품등록 페이지에 접속하고
And 상품명 "A4 컬러전단지"를 입력하고
And 분류를 "디지털"로 선택하고
And 사이즈 팝업에서 A4, A3를 선택하고
And 용지 팝업에서 "모조지 150g"를 선택하고
And 기본값(사이즈: A4, 용지: 모조지 150g, 코팅: 무코팅, 수량: 100)을 설정하면
When "등록하기" 버튼을 클릭하면
Then 상품이 성공적으로 등록되고
And Snackbar로 "상품이 등록되었습니다" 알림이 표시되고
And 상품 목록 페이지로 이동한다
```

### AC-006-002: 필수 필드 누락 시 등록 실패
```gherkin
Given 관리자가 상품등록 페이지에서
And 상품명을 입력하지 않으면
When "등록하기" 버튼을 클릭하면
Then Field 컴포넌트의 에러 상태가 활성화되고
And "상품명은 필수입니다" 에러 메시지가 표시된다
```

### AC-006-003: 임시저장
```gherkin
Given 관리자가 상품등록 폼에 일부 정보를 입력하고
When "임시저장" 버튼을 클릭하면
Then 입력된 데이터가 저장되고
And 다음 접속 시 복원 가능하다
```

## 2. 선택 팝업 (REQ-006-002)

### AC-006-004: 사이즈 선택 팝업 동작
```gherkin
Given 관리자가 상품등록 폼에서 "사이즈 선택" 버튼을 클릭하면
When Dialog가 열리고 마스터 데이터 목록이 표시되면
Then 관리자는 Checkbox로 복수 사이즈를 선택할 수 있고
And 검색 TextField로 필터링할 수 있고
And "확인" 클릭 시 선택된 사이즈가 상품 폼에 Chip으로 표시된다
```

### AC-006-005: 빈 마스터 데이터 선택 팝업
```gherkin
Given 사이즈 마스터 데이터가 0건인 상태에서
When 관리자가 "사이즈 선택" 팝업을 열면
Then "등록된 사이즈가 없습니다. 사이즈관리에서 먼저 등록해주세요." 안내가 표시된다
```

## 3. 가격 매트릭스 (REQ-006-003, REQ-006-004)

### AC-006-006: 가격 매트릭스 인라인 편집
```gherkin
Given 관리자가 DP02(디지털 출력) 가격관리 팝업을 열고
And 사이즈 "A4", 용지 "모조지 150g", 코팅 "무코팅" 조합을 선택하면
When 수량 100의 단면단가 셀을 클릭하고 "250"을 입력하면
Then 해당 셀에 250이 표시되고
And 합계가 자동 계산(100 x 250 = 25,000)되어 표시된다
```

### AC-006-007: 가격 매트릭스 행 추가/삭제
```gherkin
Given 가격관리 팝업에서 기존 수량 구간이 100, 200, 500이고
When "행 추가" 버튼을 클릭하고 수량 1000을 입력하면
Then 테이블에 수량 1000 행이 추가되고 오름차순 정렬된다
```

### AC-006-008: 수량 구간 중복 방지
```gherkin
Given 가격관리 팝업에 수량 구간 100, 200, 500이 있는 상태에서
When 관리자가 수량 200을 추가로 입력하면
Then "이미 존재하는 수량 구간입니다" 에러가 표시되고
And 행이 추가되지 않는다
```

### AC-006-009: 일괄 적용
```gherkin
Given 관리자가 A4/모조지150g/무코팅 조합의 가격을 설정하고
When "일괄 적용" 버튼을 클릭하고 대상 조합을 선택하면
Then 선택된 모든 조합에 동일한 가격이 적용된다
```

## 4. 마스터 데이터 (REQ-006-005, REQ-006-010)

### AC-006-010: 마스터 데이터 추가
```gherkin
Given 관리자가 사이즈관리 페이지에서 "+ 사이즈 추가" 버튼을 클릭하고
And 이름 "B4", 가로 257, 세로 364, 분류 "디지털"을 입력하면
When "저장" 버튼을 클릭하면
Then 사이즈 목록에 B4가 추가되고
And Snackbar로 "사이즈가 추가되었습니다" 알림이 표시된다
```

### AC-006-011: 사용중 마스터 데이터 삭제 방지
```gherkin
Given 사이즈 "A4"가 3개 상품에서 사용중인 상태에서
When 관리자가 A4 사이즈를 삭제하려 하면
Then Dialog로 "A4는 3개 상품에서 사용중입니다. 삭제할 수 없습니다." 경고가 표시되고
And 삭제가 실행되지 않는다
```

### AC-006-012: 마스터 데이터 사용중지
```gherkin
Given 사이즈 "A4"가 사용중 상태이고
When 관리자가 Switch를 토글하여 "사용중지"로 변경하면
Then A4의 상태가 "사용중지"로 변경되고
And 이후 신규 상품등록 시 사이즈 선택 팝업에서 A4가 표시되지 않는다
```

### AC-006-013: 상위 옵션 변경 시 하위 리셋
```gherkin
Given 상품등록 폼에서 사이즈 "A4", 용지 "모조지 150g", 코팅 "유광"이 선택된 상태에서
When 관리자가 사이즈를 "A3"로 변경하면
Then 용지와 코팅 선택이 초기화되고

```

## 7. 컴포넌트 렌더링 검증

### AC-006-016: Huni DS 컴포넌트 렌더링
```gherkin
Given 관리자 상품관리 페이지에 접속하면
Then Dialog는 Huni Dialog 컴포넌트로 렌더링되고 (lazyMount 적용)
And TextField는 Huni TextField로 렌더링되고 (7 slots 구조)
And Switch는 Huni Switch로 렌더링되고 (data-checked 상태 속성)
And 모든 스타일은 --huni-* 토큰을 사용한다
```
