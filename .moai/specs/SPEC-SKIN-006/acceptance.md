# SPEC-SKIN-006: 관리자 상품관리 - 인수 조건

> **SPEC ID**: SPEC-SKIN-006

## 1. 테스트 시나리오

### Scenario 1: 인쇄 상품 등록
- **Given** 상품등록 폼에서
- **When** 모든 필수정보와 옵션을 설정하고 등록하면
- **Then** 사용자 쇼핑몰에서 해당 상품이 노출된다

### Scenario 2: 가격 매트릭스 수정
- **Given** 가격관리 팝업에서
- **When** A4/모조지/무코팅/100부 단가를 250원으로 변경하면
- **Then** 사용자 측 해당 옵션 가격이 25,000원으로 표시된다

### Scenario 3: 마스터 데이터 삭제 방지
- **Given** 상품에서 사용중인 사이즈가 있을 때
- **When** 해당 사이즈를 삭제하려 하면
- **Then** "사용중인 상품이 있어 삭제할 수 없습니다" 경고가 표시된다

## 2. 품질 게이트

- [ ] 8종 가격 팝업 일관된 UI/UX
- [ ] 마스터 데이터 변경 시 참조 무결성 유지
- [ ] 상품 미리보기와 실제 표시 일치
