## 4. 모바일 반응형 고려사항

| 화면 | PC (1280px) | 모바일 (375px) |
|------|-------------|---------------|
| 레이아웃 | 좌측 메뉴 + 우측 콘텐츠 | 상단 탭 or 드로어 메뉴 |
| 주문 목록 | 테이블 형태 | 카드 스택 형태 |
| 쿠폰 목록 | 2열 그리드 | 1열 풀폭 카드 |
| 프린팅머니 | 잔액 카드 + 내역 테이블 | 잔액 카드 + 내역 리스트 |
| 리뷰 작성 | 좌측 상품 + 우측 폼 | 상품 위 + 폼 아래 |
| 증빙서류 | 테이블 형태 | 카드 리스트 |
