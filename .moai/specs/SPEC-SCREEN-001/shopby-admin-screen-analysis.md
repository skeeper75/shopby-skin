# Shopby 관리자 화면 구성요소 분석

> SPEC-SCREEN-001 보조 리서치 문서
> 생성일: 2026-03-19

## Shopby 관리자 화면 분석 종합 리포트\n\n이 분석은 Shopby Enterprise 문서의 모든 카테고리에서 추출한 43개의 주요 관리자 화면을 대상으로 합니다. 각 화면의 구성요소, 데이터 필드, 사용자 액션, 레이아웃 패턴을 문서화하고 후니프린팅의 요구사항과 매핑합니다.\n\n### 분석 결과 요약\n\n다음은 제가 읽고 분석한 문서 목록입니다:\n\n**주요 카테고리 개요 파일:**\n- `/order.mdx` - 17개 주문 관리 화면\n- `/product.mdx` - 21개 상품 관리 화면  \n- `/member.mdx` - 7개 회원 관리 화면\n- `/management.mdx` - 18개 운영 관리 화면 (1:1문의, 게시판, 적립금, 메일, SMS)\n- `/claim-order.mdx` - 8개 클레임 관리 화면\n- `/promotion.mdx` - 7개 프로모션 화면\n- `/appearance.mdx` - 14개 전시 관리 화면\n- `/service.mdx` - 15개 서비스/설정 화면\n- `/partner.mdx` - 5개 파트너 관리 화면\n\n**상세 화면 분석 파일:**\n- `/order/list.mdx` - 주문 검색 및 일괄 처리 UI (필터, 표, 모달)\n- `/order/list/detail.mdx` - 주문 상세 보기 (구조화된 정보 섹션)\n- `/product/list.mdx` - 상품 목록 및 일괄 변경 (복잡한 검색 조건)\n- `/member/list.mdx` - 회원 관리 (검색, 등급 변경, 패널티 처리)\n- `/management/inquiry/list.mdx` - 1:1문의 관리 (상태 관리, 템플릿)\n\n**참고 문서:**\n- `/feature-matrix.md` - NATIVE/SKIN/CUSTOM 분류 (43개 기능, 88% NATIVE)\n\n### 핵심 발견사항\n\n#### 1. 공통 UI 패턴 (모든 화면에서 반복)\n\n**필터/검색 섹션**\n```\n컴포넌트:\n- 쇼핑몰 선택 (드롭다운)\n- 검색어 입력 (선택 가능한 타입: 번호, 명칭, 코드 등)\n- 기간 검색 (date range picker, 최대 3개월)\n- 상태/카테고리 선택 (드롭다운, 멀티셀렉트)\n- 상세검색 (접기/펼치기)\n- 초기화/조회 버튼 (CTA)\n```\n\n**결과 리스트 섹션**\n```\n컴포넌트:\n- 데이터 테이블 (기본 스크롤 가능)\n- 행 선택 체크박스 (일괄 작업용)\n- 정렬 가능 컬럼 헤더\n- 톱니바퀴 아이콘 (컬럼 설정)\n- 페이지네이션 (\"30/50/100/200개 보기\")\n- 액션 버튼 (편집, 삭제, 상태변경)\n- 생성 버튼 (엑셀 다운로드)\n```\n\n**상세보기 섹션 (모달/새 페이지)**\n```\n구성:\n- 제목 + 기본정보 영역\n- 탭 또는 섹션 구분 (정보 카테고리별)\n- 읽기전용 정보 + 편집 가능 필드\n- 저장/임시저장/취소 버튼\n- 관련 정보로의 링크 (CRM, 상품 수정 등)\n```\n\n---\n\n### 2. 카테고리별 상세 분석\n\n#### A. 주문관리 (ORDER)\n\n**주요 화면:**\n1. **전체주문 조회** (핵심 화면)\n   - 화면 구성: 필터 → 배송번호/주문번호 기준 탭 → 리스트 → 상세보기\n   - 필터 필드: 쇼핑몰, 파트너사, 검색어(7가지), 기간, 주문/클레임, 결제수단, 상세검색\n   - 테이블 컬럼: 배송번호, 주문번호, 주문자, 상품, 결제정보, 상태, 액션\n   - 액션: 상태변경, 송장번호 입력, SMS 발송, 거래명세서 출력, 일괄처리\n   - 후니프린팅 매핑: \"접수 상태 추적\" + \"파일 처리 진행도 표시\" + \"배송 상태\" + \"품질 검증 결과\"\n\n2. **주문상세**\n   - 섹션: 주문상품 정보 → 결제정보 → 혜택지급 → 주문자정보 → 수령자정보\n   - 특징: 클릭 가능한 링크 (파트너사, 상품번호, CRM)\n   - 후니프린팅 매핑: \"주문 상세 정보 + 첨부 파일 목록 + 처리 이력 + CS 메시지\"\n\n3. **상태별 주문 화면** (입금대기, 결제완료, 상품준비중, 배송준비중 등)\n   - 동일한 리스트 UI + 상태에 따른 특화 액션\n   - 예: 배송준비중 → 송장번호 일괄 업로드 기능\n   - 후니프린팅 매핑: \"각 작업 단계별 전문화된 대시보드\"\n\n**컴포넌트 요구사항:**\n- 다단계 상태 머신 표시 (주문 흐름 시각화)\n- 링크 기반 네비게이션 (CRM, 상품 정보 연계)\n- 대량 작업 UI (선택 후 일괄 상태변경)\n- SMS/메일 발송 모달\n\n---\n\n#### B. 상품관리 (PRODUCT)\n\n**주요 화면:**\n1. **상품정보 조회/수정** (핵심)\n   - 필터: 쇼핑몰, 파트너사, 검색어(4가지: 상품명, 번호, 관리코드, 그룹코드), 기간(6가지), 상품상태(8가지), 판매상태(4가지), 판매설정(3가지), 카테고리, 재고수량, 배송정보, 담당자, 브랜드, 상세검색\n   - 테이블 컬럼: 상품 이미지, 상품명, 상품번호, 카테고리, 재고, 판매가, 상태, 액션\n   - 액션: 수정, 삭제, 일괄변경 (그룹코드, 배송정보, 판매가/할인)\n   - 후니프린팅 매핑: \"상품 = 인쇄상품 사양\" + \"파일 템플릿 관리\" + \"견적서 생성\"\n\n2. **상품등록** (복잡한 폼)\n   - 섹션: 기본정보 → 판매정보 → 옵션 → 이미지정보 → 배송정보\n   - 특징: 옵션 의존성(다단계), 이미지 업로드, 배송비 템플릿 선택\n   - 후니프린팅 매핑: \"인쇄 파일 템플릿 등록\" + \"다양한 옵션 조합 (용지, 색상, 크기, 마무리 등)\"\n\n3. **상품이미지 대량등록**\n   - 기능: 파일 업로드 → 진행상황 표시 → 결과 확인\n   - 후니프린팅 매핑: \"일괄 파일 검증 및 처리\" (PitStop와의 통합)\n\n**컴포넌트 요구사항:**\n- 복잡한 다단계 폼 (옵션 의존성)\n- 파일 업로드 (단일 + 대량)\n- 이미지 미리보기\n- 일괄 변경 UI (엑셀 업로드)\n- 상품 심사 상태 표시\n\n---\n\n#### C. 회원관리 (MEMBER)\n\n**주요 화면:**\n1. **회원 조회**\n   - 필터: 쇼핑몰, 검색어(6가지), 가입일, 가입유형(3가지), 회원상태(4가지), 회원등급, 회원그룹, 적립금, 로그인횟수, 플랫폼, 상세검색\n   - 테이블: 회원번호, 회원명, 아이디, 이메일, 휴대폰, 가입일, 상태, 액션\n   - 액션: 가입승인, 등급변경, 그룹변경, 적립금 지급/차감, 쿠폰 지급, 이용정지, 강제탈퇴\n   - 특징: CRM 연계 (클릭 시 회원 상세 + 주문/혜택 이력)\n   - 후니프린팅 매핑: \"B2B 계약 거래처 관리\" + \"구매 이력\" + \"신용 등급\"\n\n2. **회원상세정보 - CRM**\n   - 섹션: 회원정보 → 주문내역 → 혜택내역 → 게시글 작성 → CS 이력\n   - 특징: 읽기전용 데이터 + 수정 액션\n   - 후니프린팅 매핑: \"거래처 360도 뷰\" + \"거래 이력\"\n\n**컴포넌트 요구사항:**\n- 검색 결과 개인정보 보호 (검색 후에만 표시)\n- 모달 기반 일괄 작업 (등급변경, 이용정지 등)\n- 다중 탭 구조 (CRM 상세보기)\n\n---\n\n#### D. 운영관리 (MANAGEMENT)\n\n**주요 화면:**\n1. **1:1문의 관리**\n   - 필터: 쇼핑몰, 문의일시(1년), 검색어, 문의유형/담당자, 처리상태(3가지)\n   - 테이블: 문의번호, 제목, 문의자, 유형, 담당자, 상태, 액션\n   - 상세보기: 문의 내용 + 첨부파일 + 답변 입력 + 템플릿 선택\n   - 액션: 유형/담당자 변경, 답변작성 (임시저장/완료)\n   - 후니프린팅 매핑: \"고객 문의 관리\" + \"품질 불만 처리\"\n\n2. **게시글 관리**\n   - 필터: 게시판, 검색어, 상태\n   - 테이블: 제목, 작성자, 작성일, 신고 수, 액션\n   - 액션: 수정, 삭제, 신고 처리\n   - 후니프린팅 매핑: \"공지사항 관리\" + \"FAQ\"\n\n3. **적립금 관리**\n   - 화면: 적립금 설정 → 지급/차감 현황 → 수동 지급/차감\n   - 후니프린팅 매핑: 제한적 (인쇄업은 사용 안 함)\n\n4. **메일/SMS 관리**\n   - 기능: 발송, 자동 발송 설정, 발송내역 조회, 템플릿 관리\n   - 후니프린팅 매핑: \"자동 알림\" (주문 상태 변화 등)\n\n**컴포넌트 요구사항:**\n- 템플릿 선택 UI (메일/SMS/알림톡)\n- 자동 발송 규칙 설정 (상태 변화 트리거)\n- 발송 이력 조회 (상태, 수신인, 일시)\n\n---\n\n#### E. 클레임관리 (CLAIM-ORDER)\n\n**주요 화면:**\n1. **취소/반품/교환/환불 관리**\n   - 동일한 UI 패턴: 필터 → 리스트 → 상태별 상세보기\n   - 상태: 취소, 반품, 교환, 환불 (각 4-5단계 상태)\n   - 액션: 상태 변경, 이유 입력, 승인/거절\n   - 후니프린팅 매핑: \"인쇄 주문 취소/환불 처리\" + \"파일 폐기 정책\"\n\n2. **CS처리내역 관리**\n   - 리스트: 클레임 번호, 상품, 고객, 상태, 처리 일자, 결과\n   - 후니프린팅 매핑: \"품질 이슈 해결 이력 추적\"\n\n**컴포넌트 요구사항:**\n- 상태 머신 시각화 (각 클레임별 단계)\n- 거절 사유/승인 메모 입력 (필수 필드)\n\n---\n\n#### F. 프로모션관리 (PROMOTION)\n\n**주요 화면:**\n1. **할인쿠폰 관리**\n   - 화면: 쿠폰 목록 + 발행 관리 + 지급 관리\n   - 후니프린팅 매핑: 제한적 (기본 설정만 필요)\n\n---\n\n#### G. 전시관리 (APPEARANCE)\n\n**주요 화면:**\n1. **기획전 관리** - 기획전 등록 폼\n2. **상품 진열 관리** - 진열 등록/수정\n3. **배너 관리** (헤드리스/스킨)\n4. **팝업 관리**\n5. **전시카테고리 관리**\n\n**후니프린팅 매핑:** 대부분 SKIP (프론트엔드용 컨텐츠)\n\n---\n\n#### H. 서비스/설정 (SERVICE)\n\n**주요 화면:**\n1. **쇼핑몰 관리** - 기본 정보, SNS 연동, 약관 관리\n2. **운영자 관리** - 계정 등록, 부서/직책\n3. **도메인/보안서버 관리**\n4. **개인정보 접근 기록**\n\n**후니프린팅 매핑:** 기본 설정만 필요 (NATIVE)\n\n---\n\n#### I. 파트너관리 (PARTNER)\n\n**주요 화면:**\n1. **파트너 조회/등록**\n   - 필터: 파트너 검색\n   - 테이블: 파트너명, 담당자, 연락처, 상태, 액션\n   - 액션: 신규등록, 정보 수정, 공지사항\n   - 후니프린팅 매핑: **[매우 중요]** \"거래처 관리\" (파트너 = 외주 인쇄소)\n\n2. **파트너 정산** - 매출/정산 조회\n   - 후니프린팅 매핑: \"원재료 공급처 정산\" + \"외주 수수료 관리\"\n\n---\n\n### 3.
