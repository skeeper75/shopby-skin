> 자동 생성: 2026-03-19 (Playwright 자동 크롤링)
> 대상 사이트: https://service.shopby.co.kr
> 계정: huniprinting (huniprinting48 쇼핑몰)
> 로그인 상태: 성공 (NHN Commerce OAuth2 OIDC 인증)
> 총 캡처: 10개 성공 / 0개 실패/스킵
> 뷰포트: 1280x800 (데스크톱)

---

## 로그인 흐름 분석

shopby 서비스어드민은 **2단계 인증 구조**를 사용합니다:

1. **shopby 로그인 페이지** (`service.shopby.co.kr/login`): 아이디/비밀번호 입력
2. **NHN Commerce 통합 로그인** (`accounts.nhn-commerce.com`): OAuth2 OIDC 방식으로 리다이렉트
3. **OAuth 콜백** (`service.shopby.co.kr/oauth/redirect`): 토큰 처리 후 대시보드로 이동

**로그인 UI 패턴:**
- shopby 로그인: 로고 + 아이디 입력 + 비밀번호 입력 + 로그인 버튼(빨간색) + NHN커머스 통합계정 버튼(검정색)
- NHN Commerce 로그인: 흰 배경 + 아이디 입력 + 비밀번호 입력 + 로그인 버튼(검정색)

## 실제 스크린샷 기반 상세 UI 분석

### 대시보드 (02-dashboard.png)

**실제 확인된 UI:**
- 상단: shop by 서비스어드민 로고, 매뉴얼 링크, APP 다운로드, enterprise 배지, 전체메뉴 열기, 채훈희 님, 로그아웃
- 상단 배너: 쇼핑몰 남은 사용기간 314일, SMS 잔여포인트 0 포인트, 알림 상태
- 4개 요약 카드 (가로 배치):
  - 판매 현황 (최근 1개월): 입금대기/결제완료/상품준비/배송준비/배송중/배송완료/배송보류
  - 처리지연 현황 (최근 3개월): 발송지연/취소지연/반품지연/교환지연/답변지연/배송보류지연
  - 클레임 현황 (최근 3개월): 취소주문/반품주문/교환주문
  - 상품 현황 (최근 3개월): 승인대기/수정 후 승인대기/판매중인 상품/품절상품/종료 7일전/재고 10개 이하
- 빠른 실행 그리드 (2x4): 상품등록, 상품조회, 주문조회, 회원조회, 쿠폰등록, 쿠폰조회, 파트너사조회, 공지등록
- 공지사항 목록 (날짜 + 제목)
- 업무메세지 현황
- 프로모션 현황 (발행중쿠폰/발행종료 7일전/할인중 추가할인/진행종료 7일전)
- 상품문의 섹션

**레이아웃 특징:** 사이드바 없음 (접힘 상태), 풀너비 카드 그리드, 반응형

### 전체주문 조회 (03-order-list.png)

**실제 확인된 UI:**
- 헤더: "전체주문 조회" 페이지 타이틀
- 검색 필터 패널:
  - 쇼핑몰 드롭다운 (huniprinting48)
  - 파트너사: 전체/쇼핑몰 자체 상품/파트너사 라디오 버튼 + 파트너사명 검색
  - 검색어: "주문번호" 드롭다운 + 텍스트 입력 (최대 100개 복수 검색)
  - 기간검색: 오늘/어제/3일/1주일/1개월/3개월 버튼 + 날짜 범위 피커
  - "펼치기" 버튼 (고급 필터 토글)
  - 검색(빨간 버튼) + 초기화(흰 버튼)
- 결과 탭: "배송번호 기준 리스트" | "주문번호 기준 리스트"
- 액션 버튼 행: 주문상태 선택 드롭다운, 일괄처리, 송장번호 저장, SMS 발송, 거래명세서 출력, 30개 보기, 엑셀 생성
- 데이터 테이블: 체크박스, 배송번호, 배송메모, 수령자명, 수령자연락처, 우편번호, 주소, 개인통관고유부호

**레이아웃 특징:** 사이드바 + 콘텐츠, 검색영역 + 테이블 패턴

### 후니프린팅 어드민 스킨 전략

#### NATIVE 화면 (스킨 커스터마이징)
- 주문 관리, 상품 관리, 회원 관리, 통계, 설정
- 브랜드 컬러(파란계열 또는 후니프린팅 시그니처), 폰트(Noto Sans KR), 아이콘 교체
- 레이아웃 구조는 shopby 기본 유지 (검색패널+테이블 패턴)

#### SKIN 확장 (컴포넌트 추가)
- 대시보드: 인쇄 작업 현황 카드 추가 (파일 접수중/교정중/인쇄중/출고완료)
- 주문 상세: 인쇄 파일 프리뷰 패널, 작업 상태 타임라인 추가
- 상품 목록: 실제 인쇄 상품 데이터 (3단접지카드, 2단접지카드, 스탠다드업서) 기반 컬럼 설계

#### CUSTOM 화면 (별도 개발 필요)
- 인쇄 파일 관리 (업로드/교정/승인)
- 작업 진행 현황 (Kanban 보드)
- 견적/인쇄 옵션 설정

### 재사용 가능한 UI 컴포넌트
- 데이터 테이블 (정렬/필터/일괄선택/엑셀생성)
- 날짜 범위 피커 (오늘/1주일/1개월/3개월 등 단축 버튼 포함)
- 상태 탭 필터 바 (상품 상태별 카운트 표시)
- 검색 필터 패널 (기본/고급 접힘/펼침)
- 일괄처리 버튼 그룹
- 페이지네이션
- 확인 모달
- 파일 업로드 (드래그앤드롭)
