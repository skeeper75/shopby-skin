# SPEC-SKIN-002: 마이페이지

> **SPEC ID**: SPEC-SKIN-002
> **상태**: Draft
> **작성일**: 2026-03-14
> **우선순위**: High
> **Phase**: 1
> **IA 항목**: No. 8~19
> **기술 스택**: React 18 + @shopby/react-components + shadcn/ui + Tailwind CSS

## 1. 개요

로그인 사용자의 개인 정보 관리, 주문 조회, 쿠폰/포인트 관리, 리뷰/문의, 회원 정보 수정 등 마이페이지 전체 기능을 구현한다. Aurora Skin의 MyPage, OrderDetail, MemberModification 등을 기반으로 확장하되, 인쇄 전문몰 특화 기능(옵션보관함, 프린팅머니, 증빙서류)을 추가한다.

### 3.3 반응형 브레이크포인트

- **Mobile (< 768px)**: 사이드바 숨김, 그리드 메뉴로 전환, 주문 카드 단일 컬럼

- **Desktop (>= 1024px)**: 좌측 사이드바 고정 (200px), 메인 콘텐츠 확장

## 6. 인수 조건 (EARS Format)

### REQ-SKIN-002-001: 마이페이지 메인

**WHEN** 로그인 사용자가 마이페이지에 접속하면
**THE SYSTEM SHALL** 사용자 이름, 주문 현황 요약(입금대기/제작중/배송중/배송완료 건수), 최근 주문 목록을 표시한다

**수용 기준**:
- [ ] 주문 상태별 건수 정확히 표시
- [ ] 최근 주문 5건 카드 형태로 표시
- [ ] Desktop: 좌측 사이드바 네비게이션 표시
- [ ] Mobile: 그리드 메뉴로 전환

### REQ-SKIN-002-002: 주문 조회

**WHEN** 사용자가 "주문조회" 메뉴를 클릭하면
**THE SYSTEM SHALL** 전체 주문 목록을 날짜 역순으로 표시하고, 기간/상태별 필터를 제공한다

**수용 기준**:
- [ ] 기간 필터: 1주일/1개월/3개월/6개월/전체
- [ ] 상태 필터: 전체/입금대기/제작중/배송중/배송완료
- [ ] 무한 스크롤 또는 페이지네이션
- [ ] 주문 카드 클릭 시 상세 페이지 이동

### REQ-SKIN-002-003: 주문 상세

**WHEN** 사용자가 주문 카드를 클릭하면
**THE SYSTEM SHALL** 주문 상품, 인쇄 옵션, 배송 정보, 결제 정보, 주문 상태 타임라인을 표시한다

**수용 기준**:
- [ ] 인쇄 옵션 (사이즈/용지/코팅/수량) 표시
- [ ] "인쇄옵션 미리보기" 버튼 제공
- [ ] 주문 상태 타임라인 (접수 -> 파일확인 -> 제작 -> 배송 -> 완료)
- [ ] [옵션저장] [재주문] [취소요청] 버튼

### REQ-SKIN-002-004: 옵션보관함

**WHEN** 사용자가 "옵션보관함" 메뉴를 클릭하면
**THE SYSTEM SHALL** 저장된 인쇄 옵션 목록을 표시하고, 불러오기/수정/삭제를 제공한다

**수용 기준**:
- [ ] 각 옵션: 상품명, 사이즈, 용지, 코팅, 수량 정보 표시
- [ ] "불러오기" 클릭 시 해당 상품 주문 페이지로 옵션 적용하여 이동
- [ ] "삭제" 클릭 시 확인 다이얼로그 후 삭제
- [ ] 빈 상태: 안내 메시지 표시

### REQ-SKIN-002-005: 프린팅머니

**WHEN** 사용자가 "프린팅머니" 메뉴를 클릭하면
**THE SYSTEM SHALL** 현재 잔액, 충전/사용 내역, 충전하기 버튼을 표시한다

**수용 기준**:
- [ ] 현재 잔액 상단 크게 표시
- [ ] 충전/사용 내역 목록 (날짜, 구분, 금액, 잔액)
- [ ] "충전하기" 클릭 시 충전 금액 선택 + PG 결제

### REQ-SKIN-002-006: 리뷰 작성

**IF** 주문 상태가 "배송완료"가 아닌 경우
**THEN** 시스템은 리뷰 작성 버튼을 **비활성화해야 한다**

**WHEN** 배송완료된 주문의 "리뷰쓰기" 버튼을 클릭하면
**THE SYSTEM SHALL** 별점(1~5), 텍스트, 사진(최대 5장) 입력 폼을 제공하고, 등록 시 리뷰 쿠폰을 자동 발행한다

**수용 기준**:
- [ ] 별점 필수, 텍스트 10자 이상
- [ ] 사진 업로드 최대 5장 (각 10MB 이하)
- [ ] 등록 즉시 노출
- [ ] 리뷰 쿠폰 자동 발행
- [ ] 삭제 시 쿠폰 자동 회수

### REQ-SKIN-002-007: 증빙서류 발급

**WHEN** 사용자가 "증빙서류" 메뉴를 클릭하면
**THE SYSTEM SHALL** 주문별 증빙서류 발급 상태(신청가능/발급완료/불가/신용카드)를 표시하고, 사업자 정보 관리를 제공한다

**수용 기준**:
- [ ] 4가지 상태 뱃지로 구분 표시
- [ ] 신청가능 주문: "신청하기" 버튼
- [ ] 사업자 정보 CRUD (상호, 사업자등록번호, 대표자명 등)
- [ ] 현금영수증 / 세금계산서 탭 분리

### REQ-SKIN-002-008: 회원정보수정

**WHEN** 사용자가 "회원정보수정" 메뉴를 클릭하면
**THE SYSTEM SHALL** 먼저 비밀번호 확인 화면을 표시하고, 확인 후 정보 수정 폼을 제공한다

**수용 기준**:
- [ ] 비밀번호 입력 -> 확인 -> 수정 폼 전환
- [ ] 이메일은 수정 불가 (읽기 전용)
- [ ] 이름, 휴대전화, 주소 수정 가능

## 8. UI/UX 개선사항 (경쟁사 분석 기반)

> 경쟁사 3사 (redprinting.co.kr, buysangsang.com, wowpress.co.kr) 분석 결과 도출

### 8.1 대시보드 summary cards

마이페이지 상단에 3개 요약 카드를 배치한다 (Toss 계좌 요약 스타일, buysangsang.com 패턴 참고):

```
┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│ 주문 진행중   │ │ 쿠폰         │ │ 프린팅머니   │
│    3건       │ │    5장       │ │  120,000원   │
│ [상세보기 >] │ │ [사용하기 >] │ │ [충전하기 >] │
└──────────────┘ └──────────────┘ └──────────────┘
```

- 배경: `#EEEBF9` (primary-light-3)
- 숫자: `#5538B6` (Primary), font-weight bold, 24px
- 라벨: `#424242` (Text Dark), 14px
- 카드 radius: 8px, 패딩: 16px
- 모바일: 가로 스크롤 또는 1행 3열 컴팩트 그리드

### 8.2 주문 상태 타임라인 (HuniOrderStatus)

주문 상세 페이지에서 단계별 진행 상태를 시각적 타임라인으로 표시한다:

```
● 주문접수 ─── ● 파일확인 ─── ◉ 제작중 ─── ○ 배송중 ─── ○ 배송완료
  03.12          03.12         03.13
```

**컴포넌트 사양**:
- 5단계 수평 프로그레스 바
- 완료 단계: `#5538B6` 채워진 원 + 체크마크
- 현재 활성 단계: `#5538B6` 펄스 애니메이션 원
- 미완료 단계: `#CACACA` 아웃라인 원
- 연결선 완료: `#5538B6` 실선
- 연결선 미완료: `#CACACA` 점선
- 각 단계 아래 날짜 표시 (완료된 단계만)
- 모바일: 세로 타임라인으로 전환 (768px 미만)

### 8.3 1-tap 재주문 (HuniReorderFloat 연동)

주문 목록의 각 주문 라인 옆에 "재주문" 버튼을 추가한다:

- 클릭 시 동일 옵션으로 장바구니에 상품을 추가 (옵션 사전 적용)
- 버튼 스타일: `#5538B6` outline, h-[36px], font-size 13px
- 아이콘: 새로고침(refresh) 아이콘 + "재주문" 텍스트
- redprinting의 floating 재주문 CTA 컨셉을 주문 목록 인라인으로 적용

### 8.4 옵션 보관함 thumbnail grid

저장된 옵션 세트를 일반 리스트가 아닌 시각적 썸네일 카드 그리드로 표시한다:

```
┌─────────────┐ ┌─────────────┐ ┌─────────────┐
│ [상품 이미지]│ │ [상품 이미지]│ │ [상품 이미지]│
│ 전단지 A4   │ │ 명함 90x50  │ │ 스티커 원형  │
│ 모조지 150g │ │ 스노우 250g │ │ 유포지 80g  │
│ 100부       │ │ 200부       │ │ 500부       │
│ [불러오기]   │ │ [불러오기]   │ │ [불러오기]   │
└─────────────┘ └─────────────┘ └─────────────┘
```

- 카드: white bg, border `#CACACA`, radius 8px, 패딩 12px
- 상품 이미지: 상단 비율 4:3, object-fit cover
- 그리드: Desktop 3열, Tablet 2열, Mobile 1열

### 8.5 프린팅머니 내역 chart

월별 적립/사용 내역을 바 차트로 시각화한다:

- shadcn/ui charts 또는 경량 차트 라이브러리 활용
- 적립 바: `#5538B6` (Primary)
- 사용 바: `#9580D9` (Secondary)
- X축: 최근 6개월
- Y축: 금액 (원)
- 차트 높이: 200px, 패딩 16px

### 8.6 Mobile bottom nav 연동

HuniBottomNav가 표시되는 모바일 화면에서 마이페이지 라우트 진입 시:

- active 색상: `#5538B6`
- inactive 색상: `#979797`
- 마이페이지 하위 라우트 (`/mypage/*`) 모두 "마이" 탭 활성화 유지
