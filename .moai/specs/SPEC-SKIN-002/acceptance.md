# SPEC-SKIN-002: 마이페이지 - 인수 조건

> **SPEC ID**: SPEC-SKIN-002
> **문서 유형**: 인수 조건 (acceptance.md)

## 1. 테스트 시나리오

### Scenario 1: 마이페이지 접근 제어
- **Given** 비로그인 사용자가 마이페이지 URL에 접근하면
- **When** 페이지 로드 시
- **Then** 로그인 페이지로 리다이렉트된다

### Scenario 2: 주문 조회 및 상세
- **Given** 주문 내역이 있는 회원이 마이페이지에 접속하면
- **When** 주문조회 메뉴를 클릭하면
- **Then** 주문 목록이 날짜 역순으로 표시되고, 카드 클릭 시 상세 페이지로 이동한다

### Scenario 3: 옵션보관함 불러오기
- **Given** 저장된 인쇄 옵션이 있을 때
- **When** "불러오기" 버튼을 클릭하면
- **Then** 해당 상품 주문 페이지로 옵션이 적용된 상태로 이동한다

### Scenario 4: 리뷰 작성 후 쿠폰 발행
- **Given** 배송완료된 주문이 있을 때
- **When** 리뷰를 작성하고 등록하면
- **Then** 리뷰가 즉시 노출되고, 리뷰 쿠폰이 자동 발행된다

### Scenario 5: 증빙서류 신청
- **Given** 현금 결제 주문이 있을 때
- **When** 현금영수증 신청을 하면
- **Then** 상태가 "신청가능"에서 "발급완료"로 변경된다
