## 1. 개요

## 6. 인수 조건 (EARS Format)

### REQ-SKIN-007-001: 견적문의 답변

**WHEN** 관리자가 견적문의 상세에서 답변을 작성하고 등록하면
**THE SYSTEM SHALL** 답변을 저장하고 고객에게 답변 알림을 발송한다

**수용 기준**:
- [ ] 답변 에디터 (이미지 첨부 가능)
- [ ] 답변 등록 시 상태 "대기" -> "완료" 변경
- [ ] 고객에게 이메일/알림톡 발송

### REQ-SKIN-007-002: 회원관리

**WHEN** 관리자가 회원관리 목록에서 회원을 클릭하면
**THE SYSTEM SHALL** 회원 상세 정보, 주문 내역, 프린팅머니, 쿠폰 현황을 표시한다

**수용 기준**:
- [ ] 회원 검색 (이름/이메일/전화번호)
- [ ] 회원 상세: 기본정보 + 주문내역 + 적립금 + 쿠폰
- [ ] "프린팅머니 지급" / "쿠폰 발급" 버튼

### REQ-SKIN-007-003: 쿠폰 발행

**WHEN** 관리자가 쿠폰 정보를 입력하고 "발행하기"를 클릭하면
**THE SYSTEM SHALL** 쿠폰을 생성하고 매칭 대상 상품에 적용한다

**수용 기준**:
- [ ] 할인유형: 정율(%) / 정액(원)
- [ ] 매칭: 출력상품 전체 / 굿즈 카테고리 / 특정 상품
- [ ] 발행수량, 유효기간 설정
- [ ] 생성 즉시 사용자 측에서 적용 가능

### REQ-SKIN-007-004: 체험단관리

**WHEN** 관리자가 체험단 모집글을 등록하고 당첨 처리하면
**THE SYSTEM SHALL** 당첨자에게 알림을 발송하고 후기 작성 권한을 부여한다

**수용 기준**:
- [ ] 모집글 HTML 직접 입력 가능
- [ ] 신청자 목록 확인
- [ ] 당첨 처리 (다건 선택)
- [ ] 당첨자만 후기 등록 활성화

## UI/UX 개선사항

### 관리자 게시판/회원/쿠폰 (IA No. 64~78) Improvements

#### Board (게시판) improvements

1. **Rich text editor**: 공지사항/QA 답변 에디터로 TipTap 또는 Quill 통합 (이미지 업로드 포함)
2. **Board list table**: 정렬 가능한 컬럼 (작성일, 조회수, 상태), 인라인 상태 토글
3. **QA quick reply**: 행 클릭 시 우측 패널이 열려 페이지 이동 없이 빠른 Q&A 답변 가능

#### Member management improvements

1. **Member search**: 이름/전화번호/이메일 실시간 검색 (debounce 300ms)
2. **Member detail drawer**: 회원 상세를 별도 페이지가 아닌 우측 Drawer로 표시
3. **Activity timeline**: 회원 상세 페이지에 활동 로그 (주문, 쿠폰사용, 로그인 등) 타임라인 표시
4. **Printing money management**: 잔액 조정 폼에 사유 텍스트 필수 입력

#### Coupon management

1. **Coupon wizard**: 멀티스텝 쿠폰 생성 (조건설정 → 할인설정 → 대상설정 → 확인)
2. **Coupon status toggle**: 원클릭 쿠폰 활성/비활성 토글 스위치
3. **Coupon usage chart**: 기간별 사용량을 보여주는 Bar chart
