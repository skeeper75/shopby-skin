# SPEC-SKIN-007: 관리자 게시판/회원/쿠폰 - 구현 계획

> **SPEC ID**: SPEC-SKIN-007

## 1. 마일스톤

### Primary Goal: 회원관리 + 쿠폰
- 회원관리 (목록/상세/주문내역)
- 쿠폰관리 (발행/매칭/사용내역)
- 견적문의 답변

### Secondary Goal: 게시판 관리
- 공지사항/FAQ CRUD
- Q&A/1:1문의 답변
- 이용후기 관리

### Final Goal: 부가 기능
- 프린팅머니 관리
- 체험단 관리
- 탈퇴회원 관리

## 2. 기술적 접근
- Shopby Admin API 최대 활용 (게시판, 회원, 쿠폰)
- 커스텀 API: 견적문의/상담 답변, 프린팅머니, 체험단

## 3. 리스크

| 리스크 | 영향도 | 대응 |
|--------|--------|------|
| 쿠폰 매칭 복잡도 | Medium | 출력상품+굿즈+개별상품 3단계 매칭 UI 설계 |
| 체험단 HTML 입력 | Low | 에디터 컴포넌트 재활용 |
