# SPEC-SKIN-007: 관리자 게시판/회원/쿠폰 - 인수 조건

> **SPEC ID**: SPEC-SKIN-007

## 1. 테스트 시나리오

### Scenario 1: 견적문의 답변
- **Given** 미답변 견적문의가 있을 때
- **When** 관리자가 답변을 작성하고 등록하면
- **Then** 상태가 "완료"로 변경되고 고객에게 알림이 발송된다

### Scenario 2: 쿠폰 발행 및 사용
- **Given** 관리자가 10% 할인 쿠폰을 발행하면
- **When** 사용자가 주문 시 해당 쿠폰을 적용하면
- **Then** 10% 할인이 적용되고 쿠폰 사용내역에 기록된다

### Scenario 3: 프린팅머니 지급
- **Given** 회원 상세에서
- **When** "프린팅머니 지급" 버튼으로 5,000원을 지급하면
- **Then** 회원의 프린팅머니 잔액이 5,000원 증가한다

### Scenario 4: 회원 검색
- **Given** 회원관리 페이지에서
- **When** "홍길동"을 검색하면
- **Then** 해당 이름의 회원 목록이 표시된다

## 2. 품질 게이트
- [ ] 게시판 답변 등록 후 고객 측 확인 가능
- [ ] 쿠폰 발행/매칭/사용 전체 플로우 정상
- [ ] 프린팅머니 지급/차감 잔액 정확
- [ ] 회원 검색 성능 (1만건 이상)
