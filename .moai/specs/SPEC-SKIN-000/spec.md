# SPEC-SKIN-000: 후니프린팅 Shopby Skin 마스터 플랜

> **SPEC ID**: SPEC-SKIN-000
> **상태**: Draft
> **작성일**: 2026-03-14
> **우선순위**: High
> **Phase**: 전체
> **IA 항목**: No. 1~94 (전체)
> **기술 스택**: React 18 + @shopby/react-components + shadcn/ui + Tailwind CSS

---

## 1. 개요

후니프린팅(HuniPrinting)은 인쇄 전문 쇼핑몰로, NHN Commerce의 Shopby 플랫폼 위에 커스텀 스킨을 구축하는 프로젝트이다. Aurora Skin을 기반으로 확장하되, Toss/당근마켓 스타일의 깔끔하고 모바일 퍼스트한 UI/UX를 지향한다.

전체 IA는 94개 항목으로 구성되며, 쇼핑몰(사용자) 43개 + 관리자 51개로 나뉜다. 이를 8개 SPEC으로 그룹핑하여 Phase 1(사용자 핵심)과 Phase 2(관리자)로 단계적으로 개발한다.

### 그룹핑 논리

- **Phase 1 (사용자)**: 고객이 직접 사용하는 쇼핑몰 프론트엔드. 매출 발생의 핵심 경로.
- **Phase 2 (관리자)**: 운영자가 사용하는 백오피스. Phase 1 완료 후 순차 개발.
- **P0**: 반드시 필요한 핵심 기능 (로그인, 상품, 주문, 결제)
- **P1**: 중요하지만 후순위 가능 (고객센터, 마케팅, 통계)

### SPEC별 핵심 컴포넌트 참조

- **SPEC-SKIN-003**: HuniPriceCalculator, HuniOptionPreview를 핵심 컴포넌트로 포함 (상품 옵션 선택 시 실시간 가격 계산 및 시각적 프리뷰 제공). 상세 정의는 Section 11 참조.

### 4.2 UI 스타일 방향 (Toss/당근마켓 참조)

| 원칙 | 설명 |
|------|------|
| Clean & Minimal | 불필요한 장식 요소 제거, 여백 활용 |
| Mobile-First | 모바일 레이아웃 우선 설계, 데스크탑 확장 |
| Clear CTA Hierarchy | 주요 액션 버튼 시각적 강조 (보라색 단일 CTA) |
| Card-Based Layout | 정보를 카드 단위로 구분, 그림자 최소화 |
| Bottom Sheet | 모바일에서 모달 대신 바텀시트 활용 |
| Skeleton UI | 로딩 시 Skeleton 패턴 적용 |
| Micro Interaction | 버튼 press, 토글 전환 등 미세 애니메이션 |

### 4.3 반응형 브레이크포인트

| 구분 | 범위 | 레이아웃 |
|------|------|----------|
| Mobile | < 768px | 단일 컬럼, 바텀 네비게이션 |
| Tablet | 768~1024px | 2컬럼, 사이드바 토글 |
| Desktop | >= 1024px | 다중 컬럼, 고정 사이드바 |

3. **재주문 기능**: 인쇄업 특성상 재주문 빈도 높음. HuniReorderFloat으로 재주문 전환율 향상.
4. **가격 투명성**: 수량별 단가 실시간 표시가 업계 표준. HuniPriceCalculator 필수 구현.
5. **뱃지 시스템**: 경쟁사 모두 상품 카드에 뱃지 활용. HuniBadge로 통일된 뱃지 시스템 구축.
