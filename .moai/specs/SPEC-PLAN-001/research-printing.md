## 1. 상품 옵션 UI/UX 패턴

### 1.1 와우프레스 패턴 (wowpress.co.kr)

- **옵션 구조**: 캐스케이딩 드롭다운 방식
  - 선택 순서: 규격 -> 인쇄도수 -> 용지 -> 당일출고 -> 수량 -> 부속품
  - 상위 옵션 변경 시 하위 자동 리셋 + "규격 변경 시 용지 초기화" 경고 메시지
- **가격 계산**: AJAX `getJobCost(rt)` 호출로 실시간 가격 업데이트
- **특장점**: 이지템플릿 시스템 (90+ 프리디자인 템플릿 제공)
- **후가공 선택**: 각 옵션별 설명 문구(tooltip) 제공
- **회원등급**: 6단계 (3개월 접수액 기준)
- **신규혜택**: 쿠폰 4종 + 무료배송

### 1.2 레드프린팅 패턴 (redprinting.co.kr)

- **옵션 구조**: 라디오 버튼 + 드롭다운 혼합 방식
  - 선택 순서: 인쇄방식 -> 용지사이즈 -> 디자인수량 -> 주문수량
  - 조건부 필수 필드: 상위 선택에 따라 하위 옵션 동적 노출
- **가격 표시**: 우측 플로팅 가격 요약 박스 (sticky sidebar)
- **특장점**: 간편 재주문 배지, 360도 프리뷰
- **파일 업로드**: 드래그앤드롭 + 포맷 자동검증 + PDF 템플릿 다운로드
- **옵션보관함**: 1개월 보관, 재주문 지원
- **회원등급**: 6단계 (3개월 접수액 기준)
- **SNS 로그인**: 4종 (카카오/네이버/구글/애플)

### 1.3 Vistaprint 패턴 (vistaprint.com)

- **옵션 구조**: 순차 선택 방식
  - 선택 순서: 두께 -> 용지 -> 모서리 -> 수량
  - Swan 디자인시스템 기반 모듈형 컴포넌트
- **진입 경로**: 디자인 업로드 vs 템플릿 양갈래 진입
- **모바일 UX**: Baymard Institute 모바일 UX 평가 상위 1%
- **특장점**: 대규모 템플릿 라이브러리, 내장 디자인 에디터

### 1.4 MOO 패턴 (moo.com)

- **옵션 구조**: 머티리얼 퍼스트(Material-First) 접근
  - 용지 등급 선택이 최우선: Original / Super / Luxe
  - 용지 선택 후 나머지 옵션이 자동 필터링
- **특장점**: Printfinity (카드마다 다른 디자인 가능)
- **샘플 전략**: 무료 샘플 팩으로 촉각적(tactile) 평가 기회 제공
- **가격 표시**: URL 파라미터 기반 가격 인코딩

### 1.5 Best Practice (업계 표준)

Baymard Institute, Nebulab, Factory.dev 등의 이커머스 UX 리서치 종합:

- **Start Over 버튼 필수**: 복잡한 옵션 조합 시 초기화 기능 필수
- **단계적 공개 (Progressive Disclosure)**: 상위 옵션 선택 전 하위 옵션 비활성화
- **실시간 시각 검증**: 옵션 변경 시 미리보기 즉시 업데이트
- **가격/배송 동적 업데이트**: 옵션 변경마다 가격 + 예상 배송일 갱신
- **모바일 sticky 헤더/푸터**: 가격 요약과 주문 버튼 항상 노출

| 스노우지 | 무광 코팅, 부드러운 질감 | 명함, 리플릿, 고급 인쇄물 |

## 3. 후가공 종류

### 3.1 기본 후가공

| 후가공 | 설명 | 용도 |
|--------|------|------|
| 오시(scoring) | 종이를 접기 쉽게 눌러 자국을 내는 가공 | 접지 전 처리, 두꺼운 용지 |
| 미싱(perforation) | 점선 형태로 잘라내기 쉽게 구멍을 뚫는 가공 | 쿠폰, 티켓, 영수증 |
| 타공(hole punching) | 원형 구멍을 뚫는 가공 | 바인더 제본, 태그 |
| 라운딩(round corners) | 모서리를 둥글게 처리 | 명함, 카드 |
| 넘버링(numbering) | 일련번호 인쇄 | 티켓, 쿠폰, 상품권 |

### 3.2 특수 후가공

| 후가공 | 설명 | 종류 |
|--------|------|------|
| 박(foil stamping) | 금속 박지를 열과 압력으로 전사 | 금박, 은박, 홀로그램박 |
| 형압(embossing/debossing) | 양각 또는 음각으로 입체감 표현 | 양각(embossing), 음각(debossing) |
| 도무송(die-cutting) | 특수 칼형으로 원하는 모양 재단 | 특수 형태 카드, 스티커 |
| UV 코팅 | 자외선 경화 코팅으로 부분 광택 | 부분 UV, 전체 UV |
| 에폭시 | 투명 수지 코팅으로 볼록한 효과 | 스티커, 뱃지 |

### 3.3 접지(folding) 종류

| 접지 방식 | 설명 | 용도 |
|-----------|------|------|
| 반접(half fold) | 가운데 1회 접기 | 카드, 간단 리플릿 |
| 3단접지(tri-fold) | Z자 또는 C자로 2회 접기 | 리플릿, 브로슈어 |
| 병풍접지(accordion fold) | 지그재그로 여러 번 접기 | 메뉴판, 안내서 |
| 십자접지(cross fold) | 가로세로 각 1회씩 접기 | 지도, 대형 전단지 |
| 대문접지(gate fold) | 양쪽을 안으로 접기 | 고급 브로슈어 |

## 4. 파일 업로드/검수

### 4.1 검증 항목

| 항목 | 기준 | 오류 결과 |
|------|------|----------|
| 해상도 | 300 DPI (대형은 100~150 DPI) | 흐림/픽셀화 |
| 컬러모드 | CMYK 필수 (RGB 불가) | 색상 변질 (특히 녹색/파랑 계열) |
| 도련(bleed) | 사방 3mm | 흰색 테두리 노출 |
| 안전영역 | 재단선 안쪽 3mm | 콘텐츠 잘림 |
| 폰트 | 아웃라인 변환 또는 임베드 | 텍스트 깨짐/대체 폰트 |
| 포맷 | AI/PDF 권장, PSD/JPG 허용 | 품질 저하 가능 |
| 별색(spot color) | CMYK 변환 필수 | 의도하지 않은 색상 |
| 오버프린트 | 확인 필수 (특히 흰색 오브젝트) | 인쇄 누락 |

### 4.2 업로드 프로세스 벤치마크

**레드프린팅 방식** (가장 우수):
1. 드래그앤드롭 또는 파일 선택으로 업로드
2. 포맷 자동 검증 (PDF/AI/PSD/JPG)
3. 해상도/색상모드 자동 체크
4. 검증 결과 즉시 표시 (통과/경고/오류)
5. PDF 템플릿 다운로드 제공 (상품별)

**와우프레스 방식**:
1. AI/EPS 포맷만 허용 (제한적)
2. 이지템플릿 시스템으로 온라인 편집 대안 제공
3. 파일 검수는 관리자가 수동 확인

**후니프린팅 권장 방식**:
1. 드래그앤드롭 업로드 (레드프린팅 참조)
2. 클라이언트 사이드 기본 검증 (포맷, 파일 크기)
3. 서버 사이드 상세 검증 (PitStop 연동)
4. 검증 결과 3단계 표시: 통과(녹색) / 경고(노란색) / 오류(빨간색)
5. 상품별 PDF 가이드라인 템플릿 다운로드

### 4.3 PitStop 연동 고려사항

- **PitStop Server**: Enfocus PitStop Server는 PDF 프리플라이트 자동화 솔루션
- **검증 항목**: 해상도, 색상모드, 도련, 폰트 임베딩, 오버프린트
- **처리 시간**: 파일 크기에 따라 1~30초
- **API 연동**: REST API 또는 Hot Folder 방식
- **비용 고려**: 라이선스 비용이 있으므로 대안(자체 PDF 파싱) 검토 필요

## 5. 인쇄 생산 공정

### 5.1 8단계 공정 흐름

```
접수 -> 파일확인 -> 인쇄 -> 후가공 -> 제본 -> 포장 -> 출고 -> 배송
```

| 단계 | 설명 | 소요시간 | 상태 코드 |
|------|------|---------|----------|
| 1. 접수 | 주문 접수 및 결제 확인 | 즉시 | ORDER_RECEIVED |
| 2. 파일확인 | 인쇄 파일 검수 (PitStop) | 1~4시간 | FILE_CHECKING |
| 3. 인쇄 | 디지털/옵셋 인쇄 | 2~8시간 | PRINTING |
| 4. 후가공 | 코팅, 박, 형압 등 | 1~4시간 | POST_PROCESSING |
| 5. 제본 | 무선제본, 중철제본, 스프링 등 | 1~4시간 | BINDING |
| 6. 포장 | 검수 및 포장 | 30분~1시간 | PACKAGING |
| 7. 출고 | 택배사 인계 | 당일 | SHIPPED |
| 8. 배송 | 택배 배송 | 1~2일 | DELIVERING |

### 5.2 상태 변경 시 알림

| 상태 변경 | 알림 방식 | 내용 |
|-----------|----------|------|
| 접수 완료 | 이메일 + 알림톡 | 주문번호, 예상 제작일 |
| 파일 확인 완료 | 알림톡 | 검수 결과, 제작 시작 안내 |
| 파일 재업로드 요청 | 알림톡 + SMS | 문제 내용, 재업로드 링크 |
| 제작 완료 | 알림톡 | 출고 예정일 |
| 출고 완료 | 알림톡 | 송장번호, 배송조회 링크 |

## 7. 가격 구조

### 7.1 가격 결정 요소

인쇄 상품의 가격은 다음 요소들의 조합으로 결정된다:

```
최종 가격 = 기본 출력비 + 용지비 + 코팅비 + 후가공비 + 제본비 + 배송비
```

| 요소 | 변동 기준 | 예시 |
|------|----------|------|
| 기본 출력비 | 수량, 크기, 인쇄면(편면/양면) | 100매 A4 편면: 10,000원 |
| 용지비 | 종류, 평량, 수량 | 스노우지 250g: +5,000원 |
| 코팅비 | 코팅 종류, 면적 | 유광 양면: +3,000원 |
| 후가공비 | 종류, 수량, 난이도 | 금박 1도: +15,000원 |
| 제본비 | 방식, 페이지 수 | 무선제본 100p: +20,000원 |

### 7.2 수량별 단가 체감

| 수량 | 단가(예시: A4 전단) | 체감율 |
|------|-------------------|--------|
| 100매 | 120원/매 | 기준 |
| 500매 | 60원/매 | 50% |
| 1,000매 | 35원/매 | 71% |
| 5,000매 | 15원/매 | 87% |
| 10,000매 | 10원/매 | 92% |

### 7.3 특수 가격 정책

- **당일출고 할증**: 기본 가격 대비 +30~50%
- **B2B 특별 가격**: 법인 회원 전용 할인율 적용 (5~20%)
- **프린팅머니 보너스**: 충전 금액에 따라 보너스 적립금 지급

- 법인 계약 기반 후불 결제
- 대량 인쇄/복사/제본 원스톱 서비스
- 후니프린팅의 온/오프라인 통합 원장관리와 유사한 구조

## 10. 후니프린팅 핵심 권장사항

### 10.1 상품 옵션 UI

**권장**: 와우프레스 캐스케이딩 의존 드롭다운 + 레드프린팅 조건부 필수 필드 패턴 결합

- 상위 옵션 선택 시 하위 옵션 동적 로딩 (와우프레스 방식)
- 조건부 필수/선택 필드 분기 (레드프린팅 방식)
- 단일 페이지 스크롤 폼 (피그마 option_NEW 패턴, Step Wizard 미사용)
- 하단 고정 가격 요약 + 주문 버튼
- Start Over 버튼으로 옵션 전체 초기화

### 10.2 파일 업로드/검수

**권장**: 드래그앤드롭 + 자동 검증 하이브리드 방식

- 드래그앤드롭 업로드 UI (레드프린팅 참조)
- 클라이언트: 파일 포맷/크기 기본 검증
- 서버: PitStop 연동 상세 검증 (해상도, CMYK, 도련, 폰트)
- 검증 결과 3단계 시각화: 통과 / 경고 / 오류
- 상품별 PDF 가이드라인 템플릿 다운로드

### 10.3 주문 상태 트래킹

**권장**: 8단계 인쇄 전용 타임라인 스테퍼

- 인쇄 공정에 맞춘 커스텀 상태 코드
- shopby 기본 상태와 매핑 (shopby 5단계 -> 인쇄 8단계)
- 상태 변경 시 알림톡 자동 발송
- 마이페이지에서 시각적 프로그레스 바 표시

### 10.4 B2B 기능

**권장**: 법인 포탈 + 프린팅머니 + 후불결제 (단계적 도입)

- P1: 프린팅머니 충전 (PG -> 적립금 전환)
- P1: 사업자정보 등록/관리
- P2: 후불결제 시스템 (법인 전용)
- P2: 원장관리 시스템 (ERP 도입 시기 결정 후)

### 10.5 관리자 대시보드

**권장**: 역할별 관리자 대시보드

- 파일 확인 담당자: 파일 검수 대기열 + 빠른 승인/반려
- 제작 담당자: 공정별 현황 + 상태 일괄 변경
- 운영 관리자: 매출 통계 + 주문 현황 종합
- 대표: 월별 매출 + 팀별 실적 요약

### 10.6 인쇄 가이드

**권장**: 4단 계층 인쇄 가이드 구조

| 계층 | 대상 | 내용 |
|------|------|------|
| 1단계 | 초보 사용자 | 파일 포맷/해상도 기초 안내 |
| 2단계 | 일반 사용자 | 상품별 파일 제작 가이드 |
| 3단계 | 디자이너 | 도련/안전영역/색상 프로파일 상세 |
| 4단계 | 전문가 | 인쇄 규격, 후가공 조합 가이드 |
