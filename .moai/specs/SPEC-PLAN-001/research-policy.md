#### 정책 11: 쿠폰 동시사용 제한

| 항목 | 내용 |
|------|------|
| **현재 제안** | 최대 3개 |
| **선택지** | 1개 / 2개 / 3개 / 무제한 |
| **shopby 설정** | O (쿠폰 정책 설정) |

**shopby 쿠폰 구조 분석**:
- shopby는 상품 쿠폰(상품에 적용) + 주문 쿠폰(주문 전체에 적용)으로 구분
- 동시 사용: 상품 쿠폰 1개 + 주문 쿠폰 1개 = 총 2개가 구조적 한도
- 3개 이상은 별도 개발 필요

**추천안**: **상품 1개 + 주문 1개 (총 2개)**
- shopby 기본 구조에 맞춘 설정
- 추가 개발 없이 즉시 적용 가능
- 고객 입장에서 충분한 혜택 (상품 할인 + 주문 할인 중복)

| 승인 후 노출 | 품질 관리 가능 | 작성자 이탈, 운영 부담 |

**추천안**: **즉시 노출**
- shopby 기본 설정으로 즉시 적용
- 스팸/악의적 리뷰는 관리자가 사후 삭제 (shopby 관리자에서 가능)
- 인쇄 업종 특성상 리뷰 작성률이 낮으므로 최대한 장벽 제거

#### 정책 14: 리뷰 작성 보상

| 항목 | 내용 |

| **shopby 설정** | 없음 (개발 필요) |

- 적립금 방식이 쿠폰보다 운영 단순 (최소주문 설정 불필요)
- 인쇄업 평균 객단가 대비 적절한 보상 수준
- 포토 리뷰 차등 보상으로 양질의 리뷰 유도
- 리뷰 삭제 시 적립금 자동 회수 로직 필수 (악용 방지)
