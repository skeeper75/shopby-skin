# SPEC-SKIN-008: 관리자 거래처/원장/통계

> **SPEC ID**: SPEC-SKIN-008
> **상태**: Draft
> **작성일**: 2026-03-14
> **우선순위**: Medium
> **Phase**: 2
> **IA 항목**: No. 45~49, 79~85
> **기술 스택**: React 18 + @shopby/react-components + shadcn/ui + Tailwind CSS

## 1. 개요

오프라인 거래처 관리(등록/수정/처리, 업체목록 .txt 생성), 매장게시판, 원장관리(계좌관리, 오프라인 거래등록, Excel/청구서/매칭, 업체별 미수금), 통계(인쇄/제본/굿즈/패키지/수작 상품통계, 월별 매출통계, 굿즈 발주/정산, 팀별통계)를 구현한다. 월별 매출통계가 핵심 경영지표이다.

## 6. 인수 조건 (EARS Format)

### REQ-SKIN-008-001: 월별 매출통계

**WHEN** 관리자가 통계 메뉴에서 기간을 선택하면
**THE SYSTEM SHALL** 해당 기간의 총매출, 주문수, 평균단가, 신규회원을 요약 카드로 표시하고, 월별 매출 차트와 상품별 매출 비율을 시각화한다

**수용 기준**:
- [ ] 요약 카드: 총매출, 주문수, 평균단가, 신규회원
- [ ] 막대 차트: 월별 매출 추이
- [ ] 테이블: 상품분류별 매출/비율/건수
- [ ] Excel 다운로드

### REQ-SKIN-008-002: 거래처관리

**WHEN** 관리자가 거래처를 등록하면
**THE SYSTEM SHALL** 거래처 정보를 저장하고 원장관리에서 활용 가능하게 한다

**수용 기준**:
- [ ] 거래처 CRUD (업체명, 대표자, 사업자번호, 연락처, 주소)
- [ ] 업체목록 .txt 파일 생성 (다운로드)
- [ ] 거래 상태 관리 (진행/중단)

### REQ-SKIN-008-003: 원장관리

**WHEN** 관리자가 오프라인 거래를 등록하면
**THE SYSTEM SHALL** 원장에 기록하고 업체별 잔액을 자동 계산한다

**수용 기준**:
- [ ] 거래 등록: 날짜, 업체, 내용, 금액(매출/입금)
- [ ] 업체별 잔액 자동 계산
- [ ] Excel 다운로드
- [ ] 청구서 PDF 생성
- [ ] 판매내역 매칭 기능

### REQ-SKIN-008-004: 업체별 미수금

**WHEN** 관리자가 미수금 메뉴에 접속하면
**THE SYSTEM SHALL** 거래처별 미수금 합계를 표시하고 Excel 다운로드를 제공한다

**수용 기준**:
- [ ] 업체별 미수금 합계 목록
- [ ] 상세 내역 드릴다운
- [ ] Excel 다운로드

### REQ-SKIN-008-005: 상품별 통계

**WHEN** 관리자가 상품통계(인쇄/굿즈/패키지/수작) 메뉴에 접속하면
**THE SYSTEM SHALL** 기간별 상품 판매 현황을 테이블과 차트로 표시한다

**수용 기준**:
- [ ] 기간 필터
- [ ] 상품별 판매수량, 매출, 비율
- [ ] 상세보기 드릴다운
- [ ] Excel 다운로드

### REQ-SKIN-008-006: 팀별통계

**WHEN** 관리자가 팀별통계 메뉴에 접속하면
**THE SYSTEM SHALL** 제작팀별 처리 건수, 매출을 표시하여 공정관리에 활용하게 한다

**수용 기준**:
- [ ] 팀별 처리건수, 매출 테이블
- [ ] 기간 필터
- [ ] Excel 다운로드

## UI/UX 개선사항

### 관리자 거래처/원장/통계 (IA No. 45~49, 79~85) Improvements

#### 거래처 management

1. **거래처 card view**: 기본 뷰로 회사 카드 그리드 (로고, 이름, 잔액, 상태) 표시. 테이블 뷰 토글 지원.
2. **미수금 alert**: 거래처 목록 상단에 연체 거래처 수를 Badge로 표시
3. **거래처 statement export**: 원장 명세서 PDF/Excel 다운로드 버튼

#### 통계 (Statistics) improvements

1. **Dashboard charts**:
   - 일별매출 line chart (최근 30일)
   - 상품별 매출 donut chart (상위 5개 상품)
   - 팀별 실적 bar chart (horizontal)
2. **Date range picker**: shadcn DateRangePicker를 활용한 기간 필터
3. **Export**: 각 차트마다 "엑셀 다운로드" 버튼
4. **KPI cards**: 이번달 매출 | 전월 대비 ±% | 주문건수 | 평균단가 (상단 4개 KPI 카드)

#### Chart Library

Recharts 권장 (shadcn/ui chart 컴포넌트에서 이미 사용 중) 또는 경량 Chart.js
