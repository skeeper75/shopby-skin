# SPEC-SKIN-008: 관리자 거래처/원장/통계 - 인수 조건

> **SPEC ID**: SPEC-SKIN-008

## 1. 테스트 시나리오

### Scenario 1: 월별 매출통계 조회
- **Given** 관리자가 통계 페이지에 접속하면
- **When** 2026년 1월~3월을 선택하면
- **Then** 해당 기간의 총매출, 주문수, 평균단가, 신규회원 요약이 표시되고 월별 차트가 렌더링된다

### Scenario 2: 원장 Excel 업로드 및 매칭
- **Given** 원장관리 페이지에서
- **When** 은행 거래내역 Excel을 업로드하면
- **Then** 파싱된 거래 목록이 표시되고 기존 주문과 자동매칭 결과가 제시된다

### Scenario 3: 업체별 미수금 확인
- **Given** 미수금 관리 페이지에서
- **When** 특정 거래처를 선택하면
- **Then** 해당 거래처의 미결제 건 목록과 총 미수금액이 표시되고 Excel로 다운로드 가능하다

### Scenario 4: 거래처 등록 및 .txt 생성
- **Given** 거래처관리 페이지에서
- **When** 새 거래처 정보를 입력하고 등록하면
- **Then** 거래처가 목록에 추가되고 업체목록 .txt 파일 생성이 가능하다

### Scenario 5: 굿즈 발주/정산
- **Given** 굿즈 발주/정산 페이지에서
- **When** 특정 기간의 발주 내역을 조회하면
- **Then** 외주처별 발주 수량, 단가, 정산 금액이 표시되고 Excel 다운로드 가능하다

### Scenario 6: 팀별 통계
- **Given** 팀별통계 페이지에서
- **When** 특정 월을 선택하면
- **Then** 팀별 제작 수량, 매출, 공정별 현황이 표시된다

## 2. 품질 게이트
- [ ] 월별 매출통계 차트 정확도 (수동 계산과 일치)
- [ ] Excel 업로드 시 유효성 검사 및 에러 표시
- [ ] 원장 매칭 정확도 (자동매칭 성공률 70% 이상)
- [ ] 미수금 합계 정확성 (거래내역과 일치)
- [ ] 모든 통계 페이지 Excel 다운로드 정상 동작
- [ ] 대용량 데이터 조회 성능 (3년치 데이터 3초 이내)
