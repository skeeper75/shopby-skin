# SPEC-SKIN-008 v2: 관리자 거래처/원장/통계 - 인수 기준

## 1. 거래처 관리 (REQ-008-001, REQ-008-012)

### AC-008-001: 거래처 등록
```gherkin
Given 관리자가 거래처등록 페이지에서
And 업체명 "A인쇄", 대표자 "김대표", 사업자번호 "123-45-67890"을 입력하고
And 유형을 "인쇄소"로 선택하고
And 결제 조건을 "월정산"으로 설정하면
When "등록" 버튼을 클릭하면
Then 거래처가 저장되고
And 초기 등급 "C"로 설정되고
And Snackbar로 "거래처가 등록되었습니다" 알림이 표시된다
```

### AC-008-002: 업체목록 .txt 생성
```gherkin
Given 관리자가 거래처관리 페이지에서
When "업체목록 .txt 생성" 버튼을 클릭하면
Then 활성 거래처 목록이 .txt 파일로 다운로드된다
```

### AC-008-003: 거래처 등급 표시
```gherkin
Given 거래처 목록 페이지에서
Then 각 거래처의 등급이 Chip으로 표시되고
And S등급은 보라색, A등급은 파란색, B등급은 회색, C등급은 연회색으로 구분된다
```

## 2. 매장게시판 (REQ-008-002)

### AC-008-004: 매장게시판 비공개
```gherkin
Given 관리자가 "A인쇄" 거래처의 매장게시판에 글을 등록하면
When "A인쇄" 거래처가 게시판에 접속하면
Then 해당 글이 표시되고
And 다른 거래처("B디자인")는 해당 글을 볼 수 없다
```

## 3. 계좌/원장 관리 (REQ-008-003, REQ-008-004)

### AC-008-005: 계좌 등록
```gherkin
Given 관리자가 계좌관리 페이지에서 "계좌 추가" 버튼을 클릭하고
And 은행 "국민은행", 계좌번호 "123-456-789", 예금주 "후니프린팅", 용도 "매출용"을 입력하면
When "저장" 버튼을 클릭하면
Then 계좌가 등록되고 원장관리에서 선택 가능하다
```

### AC-008-006: 오프라인 거래 등록 및 잔액 계산
```gherkin
Given 관리자가 원장관리에서 "거래 등록" 버튼을 클릭하고
And 날짜 "2026-03-14", 업체 "A인쇄", 내용 "전단지 5,000장", 금액 "250,000원(매출)"을 입력하면
When "등록" 버튼을 클릭하면
Then 원장에 거래가 기록되고
And A인쇄의 미수금 잔액이 250,000원으로 표시된다
```

### AC-008-007: 원장 입금 처리
```gherkin
Given A인쇄의 미수금이 250,000원인 상태에서
And 관리자가 입금 거래를 등록(금액: 200,000원)하면
When 저장이 완료되면
Then A인쇄의 미수금 잔액이 50,000원으로 갱신된다
```

### AC-008-008: 원장 Excel 다운로드
```gherkin
Given 관리자가 원장관리에서 기간 "2026-01-01 ~ 2026-03-31", 업체 "전체"를 선택하고
When "Excel 다운로드" 버튼을 클릭하면
Then 해당 기간의 원장 데이터가 xlsx 파일로 다운로드된다
```

## 4. 미수금 관리 (REQ-008-005)

### AC-008-009: 미수금 4단계 상태 표시
```gherkin
Given 관리자가 업체별 미수금 페이지에 접속하면

And 정상(녹색), 주의(노란색), 경고(주황색), 거래중지(빨간색)로 구분된다

Then A인쇄의 전체 거래 내역(매출/입금)이 시간순으로 표시되고
And 미수금 합계가 표시된다
```

### AC-008-011: 미수금 Excel
```gherkin
Given 관리자가 미수금 페이지에서
When "미수금 Excel 다운" 버튼을 클릭하면
Then 거래처별 미수금 합계가 xlsx 파일로 다운로드된다
```

## 5. 대시보드/월별 매출 (REQ-008-006, REQ-008-007)

### AC-008-012: 대시보드 KPI 카드
```gherkin
Given 관리자가 통계 대시보드에 접속하면
Then 4개 KPI 카드가 표시되고
And 이번달 매출 카드에 금액이 #5538B6 24px 600 weight로 표시되고
And 전월 대비 증가 시 녹색(#22C55E), 감소 시 빨간색(#EF4444)으로 트렌드가 표시된다

When 차트가 렌더링되면
Then Recharts BarChart로 월별 매출 막대그래프가 표시되고
And 차트 primary 색상은 #5538B6이고
And 하단 테이블에 상품분류별 매출/비율/건수가 표시된다
```

### AC-008-014: 매출 Excel 다운로드
```gherkin
Given 관리자가 월별 매출통계 페이지에서
When "Excel 다운로드" 버튼을 클릭하면
Then 선택 기간의 매출 데이터가 xlsx 파일로 다운로드된다
```

## 10. 컴포넌트 렌더링 검증

### AC-008-022: Huni DS 컴포넌트 렌더링
```gherkin
Given 관리자 거래처/원장/통계 페이지에 접속하면
Then Chip은 Huni Chip으로 등급/상태를 표시하고 (data-selected)
And Dialog는 Huni Dialog로 렌더링되고 (lazyMount)
And Skeleton은 차트 로딩 시 wave 애니메이션으로 표시되고
And Tabs는 Huni Tabs로 탭 전환하고 (line variant)
And 모든 스타일은 --huni-* 토큰을 사용한다
```

### AC-008-023: 차트 Huni 토큰 적용
```gherkin
Given 통계 차트가 렌더링되면
Then BarChart primary 색상은 #5538B6이고
And secondary 색상은 #9580D9이고
And KPI 카드 배경은 #F6F6F6이고
And 증감 표시는 증가(#22C55E)/감소(#EF4444)으로 구분된다
```
