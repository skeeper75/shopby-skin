## 2. 화면 플로우 다이어그램

### 2.1 인증 모듈 (Authentication) 플로우

```
[비로그인 사용자]
    |
    v
[SCR-MBR-001: 로그인]
    |
    +---> [이메일+비밀번호 입력] ---> 성공 ---> [메인 페이지]
    |       |
    |       +---> 실패 ---> [SCR-MBR-002: 로그인 실패 상태]
    |       |       |
    |       |       +---> 5회 실패 ---> [계정 잠금 안내 (30분)]
    |       |
    |       +---> 잠금 계정 ---> [잠금 해제 시간 안내]
    |
    +---> [카카오/네이버 SNS 버튼]
    |       |
    |       +---> OAuth 인증 완료
    |               |
    |               +---> 기존 계정 있음 ---> 자동 로그인 ---> [메인 페이지]
    |               |
    |               +---> 이메일 중복 ---> [SCR-MBR-004: 연동 확인 모달]
    |               |       +---> 연동 승인 ---> [메인 페이지]
    |               |
    |               +---> 신규 계정 ---> [SCR-MBR-003: 추가정보 입력]
    |                       +---> 정보 입력 완료 ---> [메인 페이지]
    |
    +---> [아이디 찾기 링크] ---> [SCR-MBR-005: 아이디 찾기]
    |       |
    |       +---> 이름+휴대전화 입력 ---> 일치 ---> [SCR-MBR-006: 결과]
    |       |                               +---> "이메일 전체 발송" 클릭 ---> 이메일 발송
    |       |
    |       +---> 불일치 ---> [SCR-MBR-007: 실패]
    |
    +---> [비밀번호 찾기 링크] ---> [SCR-MBR-008: 비밀번호 찾기]
            |
            +---> 이메일+이름+휴대전화 입력 ---> 재설정 링크 발송
                    |
                    +---> 링크 클릭 (24시간 이내) ---> [SCR-MBR-009: 재설정]
                    |       +---> 새 비밀번호 설정 ---> [SCR-MBR-001: 로그인]
                    |
                    +---> 링크 만료/사용됨 ---> [SCR-MBR-010: 만료 안내]

[로그인 상태에서 로그인 페이지 접근]
    +---> [SCR-MBR-012에 의해 메인 페이지로 리다이렉트 (REQ-MEMBER-012)]
```

### 2.2 회원가입 모듈 (Registration) 플로우

```
[비회원]
    |
    v
[SCR-MBR-011: 약관동의]
    |
    +---> 전체동의 / 개별동의
    |       |
    |       +---> 필수 약관 미동의 ---> "다음" 버튼 비활성화
    |       +---> 필수 약관 전체 동의 ---> "다음" 버튼 활성화
    |
    +---> 약관 "보기" 클릭 ---> 아코디언 펼치기
    |
    +---> "다음" 클릭
            |
            v
[SCR-MBR-012: 정보입력]
    |
    +---> 이메일 입력 (300ms debounce)
    |       +---> [SCR-MBR-013: 사용 가능] (초록 체크)
    |       +---> [SCR-MBR-014: 중복] (빨간 X)
    |
    +---> 비밀번호 입력
    |       +---> [SCR-MBR-018: 강도 표시] (약함/보통/강함)
    |
    +---> "인증번호 발송" 클릭
    |       +---> [SCR-MBR-015: SMS 인증 진행중] (3분 타이머)
    |       |       +---> 올바른 코드 ---> [SCR-MBR-016: 인증 완료]
    |       |       +---> 잘못된 코드 ---> 인라인 오류
    |       |       +---> 타이머 만료 ---> [SCR-MBR-017: 인증 만료]
    |       +---> 1분 이내 재발송 ---> "1분 후 재시도" 안내
    |
    +---> 유효성 검증 실패 ---> [SCR-MBR-020: 필드별 인라인 오류]
    |
    +---> "가입하기" 클릭 (이메일 확인 + SMS 인증 + 모든 필드 유효)
            |
            v
[SCR-MBR-019: 가입완료]
    |
    +---> 신규 가입 쿠폰 안내 (10,000원)
    +---> "로그인하기" ---> [SCR-MBR-001]
    +---> "메인으로" ---> [메인 페이지]
```

### 2.3 회원관리 모듈 (Member Management) 플로우

```
[비회원 주문]
    |
    +---> 굿즈 상품 ---> [SCR-MBR-023: 비회원 정보입력] ---> 주문 완료
    |
    +---> 인쇄물 상품 ---> [SCR-MBR-022: 주문 차단 안내 모달]
    |       +---> "로그인" CTA ---> [SCR-MBR-001]
    |       +---> "회원가입" CTA ---> [SCR-MBR-011]
    |
    +---> 주문조회 ---> [SCR-MBR-021: 비회원 주문조회]
            +---> 주문번호+휴대전화 ---> 주문 상세

[로그인 회원]
    |
    +---> 마이페이지
    |       +---> [SCR-MBR-024: 등급 섹션] (배지 + 프로그레스 바)
    |       +---> "등급 혜택 보기" ---> [SCR-MBR-025: 등급별 비교표]
    |
    +---> 회원정보 수정
    |       +---> [SCR-MBR-030: 비밀번호 재확인 모달]
    |       +---> 인증 성공 ---> [SCR-MBR-026: 회원정보 수정]
    |
    +---> 비밀번호 변경 ---> [SCR-MBR-027: 비밀번호 변경]
    |       +---> 변경 완료 ---> 재로그인 유도
    |
    +---> 회원탈퇴 ---> [SCR-MBR-028: 회원탈퇴]
            +---> 미완료 주문 있음 ---> [SCR-MBR-029: 탈퇴 차단]
```
