## 1. 개요

고객센터(공지사항, FAQ, Q&A, 견적문의, 기업상담, 디자인상담, 비회원주문조회), 정보(회사소개, 이용약관, 찾아오시는 길), 가이드(작업 유의사항 11종), 마케팅(랜딩페이지 5종, 이용후기, 체험단)을 구현한다. 콘텐츠 중심의 정적 페이지가 많으며, B2B 견적문의가 비즈니스 핵심이다.

## 6. 인수 조건 (EARS Format)

### REQ-SKIN-004-001: 공지사항

**WHEN** 사용자가 공지사항 메뉴에 접속하면
**THE SYSTEM SHALL** 이벤트/공지/상품 카테고리로 구분된 공지 목록을 표시한다

**수용 기준**:
- [ ] 카테고리 탭: 전체/이벤트/공지/상품
- [ ] 각 항목 뱃지로 카테고리 구분
- [ ] 클릭 시 상세 내용 표시
- [ ] 페이지네이션 또는 무한 스크롤

### REQ-SKIN-004-002: FAQ

**WHEN** 사용자가 FAQ 메뉴에 접속하면
**THE SYSTEM SHALL** 카테고리별 질문/답변을 아코디언 형태로 표시한다

**수용 기준**:
- [ ] 카테고리: 가입/로그인, 주문/배송/결제, 취소/환불, 증빙서류발급
- [ ] 아코디언 클릭 시 답변 펼침
- [ ] 검색 기능 제공

### REQ-SKIN-004-003: 대량주문 견적문의

**WHEN** 사용자가 견적문의 폼을 작성하고 "견적 문의하기" 버튼을 클릭하면
**THE SYSTEM SHALL** 문의 내용을 저장하고 관리자에게 알림을 발송한다

**수용 기준**:
- [ ] 필수 필드: 회사명, 담당자, 연락처, 이메일, 인쇄물 종류, 수량
- [ ] 파일 첨부 지원 (최대 30MB)
- [ ] 등록 성공 시 "문의가 접수되었습니다" + 확인 이메일

### REQ-SKIN-004-004: 비회원 주문조회

**WHEN** 비회원이 주문번호, 이메일, 휴대전화를 입력하고 조회하면
**THE SYSTEM SHALL** 해당 주문의 상세 정보와 진행 상태를 표시한다

**수용 기준**:
- [ ] 3가지 정보 모두 일치해야 조회
- [ ] 불일치 시 "주문 정보를 찾을 수 없습니다"
- [ ] 주문 상세 + 증빙서류 발급 연동

### REQ-SKIN-004-005: 작업 가이드

시스템은 **가능하면** 작업 가이드 11종을 카드 그리드 형태로 제공하고, 각 가이드 상세 페이지에 이미지와 설명을 포함해야 한다

**수용 기준**:
- [ ] 11종 가이드 카드 그리드 (모니터색상/칼라설정/재단선/용지 등)
- [ ] 각 가이드: 제목, 아이콘, 상세 콘텐츠 (이미지+텍스트)
- [ ] 반응형 그리드 (모바일 2열, 데스크탑 3-4열)

### REQ-SKIN-004-006: 랜딩페이지

시스템은 **가능하면** 마케팅용 랜딩페이지 5종(종이/제본/캘린더/파우치/스티커)을 히어로 배너 + 상품 추천 + CTA 구조로 제공해야 한다

**수용 기준**:
- [ ] 히어로 배너: 풀스크린 이미지 + 텍스트 + CTA 버튼
- [ ] 추천 상품 섹션: 관련 상품 카드
- [ ] CTA: 주문하기 / 견적 문의
- [ ] 관리자가 배너 이미지 교체 가능
