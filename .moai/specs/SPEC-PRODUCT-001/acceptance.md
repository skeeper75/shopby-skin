# SPEC-PRODUCT-001: 수용 기준

## 1. 모듈별 수용 기준

### 1.1 모듈 1: 옵션 마스터 관리

#### AC-PRD-001: 사이즈 마스터 CRUD

```gherkin
GIVEN 관리자가 사이즈 관리 페이지에 접근하고
WHEN 상품유형 "인쇄/제본"을 선택하고 사이즈명 "A4", 가로 210mm, 세로 297mm를 입력하여 저장하면
THEN 사이즈 마스터 목록에 새 항목이 추가되고
AND 상품유형, 사이즈명, 치수, 활성 상태가 정확히 표시된다
```

- 관련 요구사항: REQ-PRD-001, REQ-PRD-005
- 검증 방법: E2E 테스트

#### AC-PRD-002: 마스터 데이터 비활성화 시 기존 참조 유지

```gherkin
GIVEN 사이즈 "A4"를 참조하는 상품이 3개 존재하고
WHEN 관리자가 사이즈 "A4"를 비활성화하면
THEN 기존 3개 상품에서는 사이즈 "A4"가 정상 표시되고
AND 신규 상품등록 시 사이즈 선택 팝업에서 "A4"가 표시되지 않는다
```

- 관련 요구사항: REQ-PRD-008, REQ-PRD-009
- 검증 방법: 통합 테스트

#### AC-PRD-003: 참조 중인 마스터 삭제 차단

```gherkin
GIVEN 활성 상품에서 참조 중인 소재 "아트지"가 있고
WHEN 관리자가 "아트지" 삭제를 시도하면
THEN 시스템은 "3개 상품에서 사용 중이므로 삭제할 수 없습니다. 비활성화를 사용해주세요." 메시지를 표시하고
AND 삭제가 차단된다
```

- 관련 요구사항: REQ-PRD-009
- 검증 방법: 단위 테스트 + E2E 테스트

#### AC-PRD-004: 마스터 데이터 검색

```gherkin
GIVEN 사이즈 마스터에 20개 항목이 등록되어 있고
WHEN 관리자가 검색창에 "A"를 입력하면
THEN "A4", "A5", "A3" 등 이름에 "A"가 포함된 항목만 필터링되어 표시된다

- 관련 요구사항: REQ-PRD-010

```

- 관련 요구사항: REQ-PRD-071, 072
