### 1.3 상품별 고유 필드

#### 디지털인쇄 전용 필드
- **판수**: 국4절 기준 판걸이 수 (4, 6, 8, 10, 12, 15)
- **종이(필수)**: 36종 (`*별도설정` = MES 연동 별도 관리)
- **인쇄(옵션)**: 단면, 양면
- **별색인쇄(옵션)**: 화이트/클리어/핑크/금색/은색 - 각각 (없음/단면/양면)
- **코팅(옵션)**: 코팅없음, 무광코팅(단면/양면), 유광코팅(단면/양면)
- **커팅(옵션)**: 25종 (기본형, 원형, 삼각, 팔각, 라운딩, 자유형 등)
- **접지(옵션)**: 19종 (2단/3단/4단/6단/반접지/상하접지 등)
- **후가공(옵션)**: 모서리(직각/둥근), 오시(없음/1~3줄), 미싱(없음/1~3줄), 가변텍스트(0~3개), 가변이미지(0~3개)
- **박/형압(옵션)**: 박(없음/있음), 형압(없음/양각/음각) + 크기 + 박칼라(9종)
- **추가상품(옵션)**: 28종 (OPP봉투, PP투명케이스, 카드봉투, 트레싱지봉투 등)
- **건수(옵션)**: Y (가변인쇄 건수 카운트)

#### 스티커 전용 필드

- **표지파일사양**: 별도 MES ITEM_CD, 블리드, 작업사이즈 등

| 옵션결합 단가 | 사이즈, 인쇄, 표지, 수량 → 단가 | (사이즈, 인쇄, 표지, 수량) |
| 수량할인율 | 수량 구간 → 할인율(%) | (구간) |

## 5. 기존 화면 대비 GAP 요약

### 5.1 데이터 구조 복잡도

| 영역 | 현재 상태 | 관리자 화면 필요 기능 |
|------|----------|---------------------|
| 용지 관리 | 약 120종 (출력소재관리 시트) | 용지 CRUD + 상품별 적용 매핑 UI |
| 사이즈 관리 | 약 200종 (상품별 흩어짐) | 사이즈 마스터 + 판걸이수 연동 |
| 가격 관리 | 14개 가격 시트 (다양한 매트릭스) | 가격 매트릭스 편집기 (수량x옵션) |
| 상품 관리 | 244개 상품 (11개 시트) | 상품별 옵션 조합 관리 |
| 견적 산정 | 8가지 견적 방식 | 견적 로직 설정 + 테스트 UI |

### 5.2 핵심 GAP 항목

1. **다중 가격 매트릭스**: 단순 행x열이 아닌, 3~4차원 매트릭스 (사이즈x인쇄x표지x수량) 편집 필요
2. **용지-상품 매핑**: 15개 카테고리별 ● 매핑 관리 (현재 엑셀의 L~Z열)
3. **비규격 사이즈**: 실사/아크릴의 가로x세로 범위 입력 + 면적 기반 가격 계산
4. **견적 방식 다양성**: 자동견적(수량별 계산) vs 고정단가 vs 수량할인율 적용 - 3가지 방식 혼재
5. **MES 연동 필드**: ITEM_CD, 파일명약어, 폴더, 출력파일 등 MES 시스템과의 데이터 동기화
6. **조건부 옵션**: ★ 마크로 표시된 제약 조건 (사이즈 선택시 커팅 다름, 180g이상 코팅가능 등)
7. **복합 상품**: 책자(내지+표지 별도 관리), 포토북(편집기 연동), 캘린더(가공+부자재 결합)
8. **상품악세사리**: 메인 상품 주문시 추가 선택 가능한 부자재 연동
