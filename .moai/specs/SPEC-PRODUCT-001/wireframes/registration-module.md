## 2. Module 4: 인쇄/제본 상품등록

### SCR-PRD-020: 인쇄 상품 목록

#### 화면 개요

인쇄/제본 상품 전체 목록 페이지. DataTable 기반으로 상품코드, 상품명, 상품유형, 견적방식, MES ITEM_CD, 판매상태(Badge), 생성일, 수정일을 표시한다. 상품유형/견적방식/상태 필터와 검색을 제공하며, 등록/복제/삭제/상태변경(일괄) 액션을 지원한다.

**유형:** Page
**경로:** `/admin/product/print`
**API:** `GET /api/admin/product/print`

#### API 바인딩

| API | Method | 설명 |
|-----|--------|------|
| `/api/admin/product/print` | GET | 상품 목록 (filter, search, page, pageSize) |
| `/api/admin/product/print/:id` | DELETE | 상품 삭제 |
| `/api/admin/product/print/bulk-status` | PATCH | 일괄 상태 변경 |
| `/api/admin/product/print/bulk-delete` | DELETE | 일괄 삭제 |

#### 인터랙션

- [+ 신규등록] 클릭: `/admin/product/print/new` 로 이동 (SCR-PRD-021)
- [수정] 클릭: `/admin/product/print/:id/edit` 로 이동
- [복제] 클릭: SCR-PRD-025 (복제 확인 Dialog) 표시
- [삭제] 클릭: 삭제 확인 Dialog 표시
- 필터 변경: debounce 300ms 후 목록 재조회
- 체크박스 선택: Bulk Actions 바 표시

### SCR-PRD-021: 기본정보 탭

#### 화면 개요

상품등록 폼의 첫 번째 탭. 상품 기본 정보(상품명, 상품코드, 상품유형), 대표이미지, 상세설명(에디터), MES 연동 필드(ITEM_CD, 파일명약어, 출력파일, 폴더), 견적방식 선택(자동견적/고정단가/수량할인 3종 라디오), 판매설정, PDF 가이드라인을 입력한다.

**유형:** Tab (상품등록 폼의 첫 번째 탭)
**경로:** `/admin/product/print/new` (신규) / `/admin/product/print/:id/edit` (수정)
**API:** `POST /api/admin/product/print` (신규), `PUT /api/admin/product/print/:id` (수정)

### SCR-PRD-022: 옵션설정 탭

#### 화면 개요

상품등록 폼에서 가장 복잡한 화면. 상품유형에 따라 **단일 상품 구조**(디지털인쇄, 스티커, 실사, 아크릴, 굿즈, 문구, 악세사리, 캘린더, 디자인캘린더)와 **복합 상품 구조**(책자/제본, 포토북: 내지+표지+제본 3중 구조)로 분기된다. 사이즈/소재/용지 선택 팝업 호출, 후가공 옵션 체크박스, 커팅/접지/별색인쇄 옵션, 조건부 옵션 규칙 편집기, 수량 제약 조건 설정을 포함한다.

**유형:** Tab (상품등록 폼의 두 번째 탭)
**핵심:** 단일/복합 상품 구조 분기 + 조건부 옵션 규칙 편집기 + 수량 제약 조건

### SCR-PRD-023: 가격관리 탭

#### 화면 개요

기본정보 탭에서 선택한 견적방식에 따라 UI가 분기된다. 자동견적: 14종 가격 유형 선택 후 해당 가격관리 팝업(SCR-PRD-014~017) 호출. 고정단가: 옵션 조합별 직접 단가 입력 테이블. 수량할인: 기본가 입력 + 수량 구간별 할인율 설정. 공통으로 당일출고 할증 설정과 가격 시뮬레이터(SCR-PRD-019) 호출을 제공한다.

**유형:** Tab (상품등록 폼의 세 번째 탭)
**핵심:** 견적방식별 UI 분기 + 14종 가격 유형 + 시뮬레이터

## 3. Module 5: 일반 상품등록

### SCR-PRD-028: 굿즈 카테고리 관리

#### 화면 개요

shopby 카테고리 API 기반 트리형 CRUD 화면. 좌측 트리 뷰 + 우측 편집 패널 2컬럼 구조. 드래그로 순서 변경 가능.

**유형:** Page
**경로:** `/admin/product/goods/category`
**API:** `GET/POST/PUT/DELETE shopby /admin/categories`

#### ASCII 와이어프레임

```
+----------------------------------------------------------+
| Breadcrumb: 상품관리 > 굿즈 > 카테고리 관리               |

| | 굿즈 카테고리 관리                 [+ 카테고리 추가]    | |
| +------------------------------------------------------+ |
|                                                          |
| +-- 2컬럼 레이아웃 -----------------------------------+ |
| |                                                      | |
| | +-- 좌측: 트리 뷰 (w:50%) -----+  +-- 우측: 편집 --+| |
| | |                              |  |                 || |
| | | v 후니프린팅                   |  | 카테고리 정보    || |
| | |   v 굿즈                     |  |                 || |
| | |     > 에코백      (드래그)    |  | 카테고리명 *     || |
| | |     > 텀블러      (드래그)    |  | [텀블러       ]  || |
| | |     > 머그컵      (드래그)    |  |                 || |
| | |     > 노트/다이어리 (드래그)   |  | 정렬 순서       || |
| | |     > 폰케이스    (드래그)    |  | [2           ]  || |
| | |                              |  |                 || |
| | |                              |  | 전시 상태       || |
| | |                              |  | (o) 전시함      || |
| | |                              |  | ( ) 전시안함    || |
| | |                              |  |                 || |
| | |                              |  | 카테고리 이미지  || |
| | |                              |  | [이미지 업로드]  || |

| | |                              |  | [삭제](ghost)   || |
| | |                              |  | [저장](primary) || |

| +------------------------------------------------------+ |
+----------------------------------------------------------+
```

#### 컴포넌트 트리

```
GoodsCategoryPage
  Breadcrumb
  PageTitle + Button["+카테고리 추가"]
  TwoColumnLayout (50%:50%)
    TreeView (react-dnd / @dnd-kit)
      TreeNode[]: 접기/펼치기, 드래그 정렬
      ActiveNode: bg:#EDE9FE, border-left: 2px solid #5538B6
    EditPanel
      Input: 카테고리명 (required)
      NumberInput: 정렬 순서
      RadioGroup: 전시 상태 (전시함/전시안함)
      FileUpload[image]: 카테고리 이미지
      Button[ghost]: 삭제
      Button[primary]: 저장
```

#### API 바인딩

| API | Method | 설명 |
|-----|--------|------|
| `shopby /admin/categories` | GET | 카테고리 트리 조회 |
| `shopby /admin/categories` | POST | 카테고리 생성 |
| `shopby /admin/categories/:id` | PUT | 카테고리 수정 |
| `shopby /admin/categories/:id` | DELETE | 카테고리 삭제 |
| `shopby /admin/categories/sort` | PUT | 카테고리 순서 변경 |

#### 인터랙션

- 노드 클릭: 우측 편집 패널에 해당 카테고리 정보 로드
- 드래그 앤 드롭: 순서 변경 (자동 저장)
- [+ 카테고리 추가]: 선택된 노드의 하위로 추가 또는 최상위 추가

### SCR-PRD-029: 굿즈 상품등록

#### 화면 개요

shopby 기본 상품등록 + 색상/사이즈 옵션 조합형. 구매수량별 할인 설정 테이블(구간: 10~49, 50~99, 100~499, 500~999, 1000+)을 포함한다. 엑셀 기반 93종 굿즈 상품, 21개 하위 카테고리 대응.

**유형:** Page
**경로:** `/admin/product/goods/new`
**API:** `POST shopby /admin/products`

#### SKIN 확장 필드 와이어프레임

```
+-- 섹션: 굿즈 확장 옵션 --------------------------------+
|                                                        |
| 색상 옵션 *       [색상 입력 후 Enter                 ] |
|                   [빨강 x] [파랑 x] [검정 x] [흰색 x]  |
|                                                        |
| 사이즈 옵션 *     [사이즈 입력 후 Enter               ] |
|                   [S x] [M x] [L x] [XL x]            |
|                                                        |
| 소재              [...                              ]  |
|                   "예: 면 100%, 폴리에스터 등"          |
|                                                        |
| 인쇄 방식         [인쇄 방식 선택       v]              |
|                   (실크스크린, 디지털프린팅, 자수,       |
|                    열전사, UV 프린팅)                    |
|                                                        |
| 최소 주문 수량    [1          ] 개                     |
|                                                        |
| +-- 구매수량별 할인 설정 ----------------------------+  |
| | (토글) 수량 할인 적용 [ON/OFF]                     |  |
| |                                                    |  |
| | (ON일 때)                                          |  |
| | | 구간              | 할인율(%) |                   |  |
| | |-------------------+----------|                   |  |
| | | 10 ~ 49           | [3     ] |                   |  |
| | | 50 ~ 99           | [5     ] |                   |  |
| | | 100 ~ 499         | [10    ] |                   |  |
| | | 500 ~ 999         | [15    ] |                   |  |
| | | 1,000+            | [20    ] |                   |  |
| +----------------------------------------------------+  |
+--------------------------------------------------------+
```

#### 컴포넌트 트리

```
GoodsProductForm (extends CommonProductForm)
  Section: 굿즈 확장 옵션
    TagInput: 색상 옵션 (required)
    TagInput: 사이즈 옵션 (required)
    Input: 소재 (optional)
    Select: 인쇄 방식
    NumberInput: 최소 주문 수량
  Section: 구매수량별 할인
    Toggle: 수량 할인 적용 ON/OFF
    DiscountTable (conditional)
      columns: [구간(readonly), 할인율(%)(input)]
      rows: 10~49, 50~99, 100~499, 500~999, 1000+
```

#### API 바인딩

| API | Method | 설명 |
|-----|--------|------|
| `shopby /admin/products` | POST | shopby 상품 등록 |
| `/api/admin/product/goods/:id/discount` | PUT | 수량 할인 설정 |

#### 인터랙션

- 옵션 조합: 색상 x 사이즈 자동 생성 (shopby 조합형 옵션)
- 수량 할인 토글 ON: 구간별 할인율 테이블 표시
- 할인율 변경: 할인가 자동 계산 표시
