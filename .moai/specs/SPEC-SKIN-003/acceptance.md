# SPEC-SKIN-003: 상품/주문/결제 - 인수 조건

> **SPEC ID**: SPEC-SKIN-003
> **문서 유형**: 인수 조건 (acceptance.md)

## 1. 테스트 시나리오

### Scenario 1: 인쇄 옵션 선택 후 가격 계산
- **Given** 출력상품 상세 페이지에서
- **When** 사이즈 A4, 용지 모조지, 양면, 무광코팅, 100부를 선택하면
- **Then** 0.3초 내에 예상 가격이 표시된다

### Scenario 2: 전체 구매 플로우
- **Given** 로그인 사용자가 상품 상세에서 옵션을 선택하고
- **When** 장바구니 -> 주문서 -> 결제를 완료하면
- **Then** 주문완료 페이지에 주문번호가 표시되고 확인 메일이 발송된다

### Scenario 3: 파일 업로드 검증
- **Given** 주문서 작성 시
- **When** 200MB PDF 파일을 업로드하면
- **Then** "최대 100MB까지 업로드 가능합니다" 에러가 표시된다

### Scenario 4: 도서산간 배송비
- **Given** 주문서에서 제주도 주소를 입력하면
- **When** 배송비가 계산되면
- **Then** 추가 배송비가 자동으로 적용된다

### Scenario 5: 프린팅머니 전액 결제
- **Given** 프린팅머니 잔액이 결제 금액 이상일 때
- **When** "프린팅머니 전액결제"를 선택하고 결제하면
- **Then** PG 없이 즉시 주문이 완료된다
