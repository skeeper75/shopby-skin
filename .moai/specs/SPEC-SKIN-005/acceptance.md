# SPEC-SKIN-005: 관리자 기반/주문관리 - 인수 조건

> **SPEC ID**: SPEC-SKIN-005

## 1. 테스트 시나리오

### Scenario 1: 관리자 로그인
- **Given** 등록된 관리자 계정으로
- **When** 로그인하면
- **Then** 대시보드에 오늘의 현황이 표시된다

### Scenario 2: 주문 검색
- **Given** 주문관리 목록에서
- **When** 주문번호로 검색하면
- **Then** 해당 주문만 필터링되어 표시된다

### Scenario 3: 일괄 상태변경
- **Given** 주문 목록에서 3건을 선택하고
- **When** "제작진행중"으로 일괄 변경하면
- **Then** 3건 모두 상태가 변경되고 고객에게 SMS가 발송된다

### Scenario 4: 파일확인 후 재업로드 요청
- **Given** 파일 미리보기에서 문제를 발견하고
- **When** "재업로드 요청"을 클릭하면
- **Then** 고객에게 재업로드 안내 SMS가 발송된다

## 2. 품질 게이트
- [ ] 관리자 권한별 메뉴 접근 제한
- [ ] 주문 목록 1000건 이상 성능 정상
- [ ] 일괄 처리 동시 최대 50건
- [ ] SMS 발송 성공/실패 로그 기록
