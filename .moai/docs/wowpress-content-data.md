### 4-2. 파일 준비 가이드 상세

#### 재단사이즈와 작업사이즈
- **재단사이즈**: 실제 인쇄물의 크기
- **작업사이즈**: 재단 여유분 포함 (재단 시 1~2mm 오차 발생)
- **안전영역**: 재단사이즈에서 약 3~5mm 안쪽 (중요 글자/이미지는 반드시 이 안에 배치)
- **도련(블리드)**: 사방 3mm 여유 추가 (Photoshop: Image > Canvas Size에서 3mm씩 추가)

#### RGB vs CMYK
- 인쇄 데이터는 **반드시 CMYK** 칼라모드로 작업
- RGB로 인쇄 시 모니터 색상과 실제 인쇄물 색상 차이 큼 (재인쇄/환불 불가)
- RGB에서 CMYK 변환 시 채도 감소 -> 처음부터 CMYK로 작업 권장
- **변환 방법:**
  - Photoshop: Image > Mode > CMYK Color Mode
  - Illustrator: File > Document Color Mode > CMYK Color
  - CorelDraw: Edit > Find & Replace > Replace

#### CMYK 잉크량 제한
- CMYK 합계 250% 이상 시 문제 발생:
  - 건조 속도 저하
  - 잉크 묻어남/비침
  - 스크래치
  - 재단 시 찢어짐
- 배경색과 패턴 색상은 최소 30% 이상 색상값 차이 필요
- 색상 허용오차: +-10% 미만 (재작업 대상 아님)

#### 해상도 기준
| 콘텐츠 유형 | 칼라모드 | 최소 DPI |
|-------------|----------|----------|
| 텍스트/선만 | Bitmap | 1200dpi 이상 |
| 흑백 사진/일러스트 | Grayscale | 300dpi 이상 |
| 컬러 사진/일러스트 | CMYK | 350dpi 이상 |
| 텍스트+배경이미지 | CMYK 또는 Grayscale | 600dpi 이상 |
| 현수막/배너 이미지 | - | 700dpi (1/10 축소 작업 기준) |

#### 최소 인쇄 가능 사양
- 선 두께: 최소 0.5pt
- 폰트 크기: 최소 5pt
- 벨벳코팅 제품: 선 1pt 이상, 폰트 12pt 이상
- 카드명함 박인쇄: 선 1pt 이상

#### 리치블랙
- 일반 K100보다 깊고 진한 블랙 표현
- 권장 배합: **C20, M20, K100**
- 작은 텍스트/가는 선에는 사용 불가 (정렬 오차로 시각적 불안정)
- PET 재질에는 K100만 사용 (리치블랙 사용 시 블랙 인쇄 불가)

### 4-3. Adobe Illustrator 가이드

#### 칼라모드
- 반드시 CMYK Color로 작업
- 변경: File > Document Color Mode > CMYK Color

#### 서체 아웃라인 (필수)
1. Object > Unlock All로 모든 잠금 해제
2. 전체 선택 후 Type > Create Outline

**주의사항:**
- Envelope Distortion 적용 후에는 아웃라인 불가 -> Object > Envelope Distort > Expand 먼저 실행
- Graph tool 텍스트는 그룹 해제 후 아웃라인
- 패턴 텍스트는 패턴 생성 전에 아웃라인
- 세리프체 테두리 적용 시 뾰족한 모서리 발생 -> Stroke: Miter Join을 Round Join으로 변경

#### 이미지 임베드
**직접 임베드:** 복사/붙여넣기로 문서에 직접 삽입
- Bitmap: 1200dpi
- Grayscale: 300dpi+
- CMYK: 350dpi+

**링크 방식:** 외부 참조로 파일 크기 줄이나 전송 시 이미지 누락 위험
- 링크를 임베드로 전환: Relink > 이미지 선택 > Link 체크 해제

**주의:** AI/PDF 파일을 이미지로 임베드하면 안 됨 (선 두께 오류 발생)

#### 저장 주의사항
- CS~CS6 파일을 9~10 버전으로 다운그레이드 저장 시 색상값 오류 (K100으로 변환) 발생 가능
- 저장 후 최소 2회 이상 파일 재확인

### 4-6. 후가공 가이드 상세

#### 도무송
- 목형(나무틀)에 칼날을 꽂아 원하는 모양으로 재단
- 오차 1~3mm 발생 가능 (재작업 불가)
- 파일 3종 제출 필요: 원본 데이터, 칼선, 원본+칼선 합본

#### 오시(Crease)
- 접이명함/리플렛 등 접는 제품에 사용
- 접는 위치에 홈을 내어 종이가 터지지 않도록 함
- 150g 이상 종이는 반드시 오시 선택 필수
- 위치 미지정 시 중앙 기준 가공

#### 접지(Folding)
- 전단/명함 등에서 표지와 내지를 구분하는 후가공
- 150g 이상 종이는 반드시 오시와 함께 사용
- 샘플파일과 CMYK 인쇄파일을 나란히 배치, 접지 수량/위치 표기

### 교환/반품/환불 정책

#### 재작업(AS) 신청
- **신청 경로:** 마이페이지 > 재작업신청
- **신청 기한:** 수령일로부터 **7일 이내**
- **조건:** 수령한 제품 모두 반송 필요, 원본 파일로 재작업 원칙

#### 재작업 불가 사유
- 서체 아웃라인 미처리
- 이미지/텍스트가 안전영역(2.5~5mm) 밖에 배치
- 이미지 해상도 300dpi 미만
- RGB 칼라모드 (CMYK 필수)
- CMYK 합계 250% 초과
- 미세 백점 (1mm 이하, 5개 이하)
- 색상 편차 +-10% 미만

#### 취소/환불 가능 조건
- 본사 생산/물류 이상으로 기본 출고일 기준 **5영업일 이상 지연** 시
- 배송 누락/오배송으로 재작업 진행 시 지연될 경우
- 동일 사유로 불량 재발생 시
- 주문제작 상품은 교환/반품 불가
