## A. 상품 옵션 선택 화면 (Product Configurator)

### A-1. 옵션 구성 구조

**공통 패턴**: 모든 사이트가 위에서 아래로 순차 선택하는 세로형 옵션 구조를 채택

```
[규격/사이즈] → [인쇄도수] → [용지 선택] → [코팅/후가공] → [수량] → [추가옵션]
```

**레드프린팅 패턴**:
- 상품 상세 페이지에서 전체 옵션을 한 화면에 노출
- 인쇄 방식(단면/양면), 사이즈, 코팅 옵션을 라디오 버튼형으로 선택
- 옵션 선택 시 PDF 템플릿 다운로드 버튼 표시
- 파일 업로드와 결제가 같은 페이지에서 처리

**와우프레스 패턴** (가장 상세한 구조):
- 규격: 90×50mm, 50×90mm, 86×52mm, 비규격 옵션
- 인쇄도수: 단면 칼라4도, 양면 칼라8도
- 재단사이즈/작업사이즈 mm 단위 표기
- 용지 선택: 규격·인쇄도수 선택 후 가능한 용지만 필터링 표시
  - **중요**: "규격 및 인쇄도수를 변경하면 선택한 용지가 초기화"됨
- 후가공: 오시, 미싱, 타공, 라운딩, 넘버링, 박, 형압
  - 각 후가공 옵션에 설명 문구 제공 (예: "오시는 잘 접을 수 있도록...")
- 당일출고 신청 여부 (신청안함/신청)
- 수량: 1건~50건

**비즈하우스 패턴**:
- 미리캔버스 에디터 연동형
- 디자인 완료 후 "인쇄물 제작" 버튼 클릭
- 옵션 선택: "이 옵션으로 제작할래요" 확인 팝업
- 장바구니 추가 → 결제 흐름

**성원애드피아 패턴**:
- 셀프디자인 메뉴 별도 운영
- 별도견적 메뉴로 맞춤형 주문 지원

### A-2. 가격 표시 방식

**실시간 계산형** (주류):
- 옵션 선택마다 하단 또는 우측에 즉시 가격 업데이트
- 수량 변경 시 단가 × 수량 실시간 계산
- 후가공 추가 시 추가 비용 분리 표시

**주요 발견사항**:
- 모든 사이트가 "선결제 후제작" 시스템 운영
- 대부분 VAT 별도 또는 포함 명시
- 배송비 조건 명시 (3만원 이상 무료 등)

### A-3. 옵션 의존성 처리

**핵심 패턴**: 상위 옵션 선택이 하위 옵션 범위를 결정

```
규격 선택 → 가능한 인쇄도수 필터링
인쇄도수 선택 → 가능한 용지 필터링
용지 선택 → 가능한 코팅 필터링
코팅 선택 → 가능한 후가공 필터링
```

**와우프레스의 명시적 안내 방식**:
- "규격 및 인쇄도수를 변경하면 선택한 용지가 초기화됩니다"
- 변경 시 팝업 또는 인라인 경고 텍스트 표시

### A-4. 파일 업로드 통합 지점

**사이트별 업로드 위치**:
- 레드프린팅: 옵션 선택 → 즉시 파일 업로드
- 와우프레스: 주문 완료 후 별도 파일 업로드 (또는 셀프디자인)
- 비즈하우스: 에디터에서 디자인 → 자동 연동
- 성원애드피아: 셀프디자인 또는 파일 별도 업로드

**파일 포맷 요구사항** (공통):
- PDF 권장 (CMYK, 300dpi 이상)
- AI, PSD 지원 사이트도 있으나 PDF가 표준
- 작업 사이즈: 재단 + 도련 3mm 포함 사이즈
- 안전 영역: 재단선 안쪽 3mm
- 서체: 아웃라인(곡선화) 처리 필수

## B. 주문 플로우 (Order Flow)

### B-1. 전체 주문 프로세스

```
상품 탐색 → 옵션 선택 → 파일 업로드 → 장바구니 → 결제 → 접수확인 → 제작 → 배송
```

**레드프린팅 플로우**:
1. 상품 검색
2. 상품 상세 페이지에서 인쇄 사양 선택
3. PDF 템플릿 다운로드 (해당 시)
4. 파일 업로드
5. 결제 (선결제 필수)
6. 주문완료 → 제작 시작

**비즈하우스 플로우** (디자인 연동형):
1. 미리캔버스에서 디자인 작업
2. "인쇄물 제작" 버튼 클릭
3. 제품 유형 선택
4. 옵션 선택 ("이 옵션으로 제작할래요")
5. 장바구니 추가
6. 결제
7. 12시 이전 주문 시 당일 제작접수

### B-2. 장바구니 / 보관함

**공통 기능**:
- 인쇄 옵션 저장 (용지, 수량, 후가공 등)
- 업로드 파일 연결
- 수량 변경 시 가격 재계산
- 임시저장 기능 (결제 전 보관)

### B-3. 파일 업로드 UI/UX

**업로드 방식**:
- 드래그앤드롭 + 파일 탐색기 선택
- 업로드 진행바 표시
- 썸네일 미리보기 제공
- 업로드 후 기본 검수 (해상도, 색상모드 체크)

**주요 파일 가이드 제공 방식**:
- 작업 가이드 PDF 다운로드
- 인라인 가이드 (사이즈, 도련, 색상 모드 안내)
- 비디오 가이드 (일부 사이트)

### B-4. 주문서 (Order Sheet) 구조

```
주문 상품 정보
└── 상품명, 옵션 요약
└── 수량, 단가, 합계
└── 업로드 파일 정보

배송 정보
└── 받는 사람, 주소, 연락처
└── 배송 방법 선택
└── 배송 요청사항

결제 정보
└── 결제 금액 요약 (상품 + 배송비)
└── 쿠폰/포인트 적용
└── 결제 수단 선택
```

### B-5. 결제 방식

**공통 지원**:
- 신용카드, 계좌이체, 무통장입금
- 네이버페이, 카카오페이
- 법인카드 지원 (기업 고객용)

## C. 마이페이지 (MyPage) 설계

### C-1. 주문 추적과 인쇄 특화 상태

**인쇄 주문 상태 단계** (비즈하우스 기준):
```
주문완료 → 결제확인 → 파일검수 → 제작접수 → 제작중 → 출고 → 배송중 → 배송완료
```

**주문 보류 (특수 상태)**:
- 파일 오류 발견 시 자동 전환
- 카카오 알림톡 / 문자 발송
- 고객이 직접 파일 수정 후 "제작시작" 버튼 클릭 필요
- "[제작시작] 버튼을 누르지 않으면 계속 보류 상태"

**인쇄 특화 상태 추가 권장**:
- 파일접수: 파일 업로드 완료, 검수 대기
- 파일검수중: 자동/수동 검수 진행
- 검수완료: 제작 진행 가능 상태
- 제작접수: 공장 제작 대기열
- 제작중: 실제 인쇄 진행
- 후가공중: 코팅, 재단 등 후처리
- 출고준비: 포장 및 배송 준비

### C-2. 파일 관리 / 재업로드

**주요 기능**:
- 업로드된 파일 목록 조회
- 파일 교체 (주문 보류 시)
- 파일 다운로드 (원본 백업)
- 파일 재사용 (동일 파일로 재주문)

**비즈하우스 파일 수정 플로우**:
1. 주문보류 사유 확인
2. 파일 주문 → "첨부파일 변경" 클릭
3. 디자인 주문 → "디자인 수정하기" 클릭
4. 수정 완료 후 "[제작시작]" 버튼 클릭

### C-3. 옵션 저장 (저장된 설정)

**패턴**:
- 자주 주문하는 옵션 저장 (즐겨찾기형)
- 재주문 버튼: 이전 옵션 그대로 새 주문 생성
- 주문 내역에서 동일 상품 재주문 원클릭

### C-4. 인쇄머니 / 적립금

**관찰된 패턴** (성원애드피아, 일반 인쇄몰):
- 인쇄머니: 선불 충전형 포인트 (현금처럼 사용)
- 적립금: 주문 금액의 일정 비율 적립
- 쿠폰 시스템 병행

**권장 기능**:
- 인쇄머니 충전 내역
- 적립금 사용/적립 이력
- 만료 예정 포인트 알림

## D. 관리자 기능 (Admin/Management)

### D-1. 주문 관리

**인쇄 워크플로 반영 주문 관리**:
- 주문 상태별 필터링 (검수중, 제작중 등)
- 긴급 주문 / 당일출고 주문 별도 표시
- 출고일 기준 우선순위 정렬

**파일 검수 인터페이스**:
- 업로드 파일 썸네일 목록
- 검수 통과 / 보류 버튼
- 보류 사유 템플릿 선택 (+ 직접 입력)
- 고객 알림 자동 발송 연동

### D-2. 파일 관리

**어드민 관점 파일 검수**:
- 해상도 자동 체크 (300dpi 미만 경고)
- 색상모드 체크 (RGB → CMYK 변환 필요 경고)
- 재단선 영역 침범 체크
- 폰트 아웃라인 미처리 감지

### D-3. 가격 관리

**옵션별 가격 테이블 구조**:
```
상품 기본가
+ 규격별 차등가
+ 수량별 단가 (볼륨 디스카운트)
+ 용지 추가가
+ 후가공별 추가가
+ 배송비
= 최종 가격
```

## 핵심 UI 패턴 요약

### 1. 옵션 선택기 (Option Configurator)

```
[선택형 칩/버튼] 옵션 그룹
├── 활성 옵션: 강조색 배경 + 체크 아이콘
├── 비활성 옵션: 회색 처리 + 비활성 사유 툴팁
└── 상위 옵션 변경 시: 하위 옵션 초기화 + 안내 텍스트
```

### 2. 실시간 가격 계산기

```
[사이드 패널 또는 하단 고정]
선택 옵션 요약: 용지명 | 수량 | 후가공
────────────────────────
상품 가격:        12,000원
후가공 (라미):     3,000원
배송비:           2,500원
────────────────────────
합계:            17,500원 (VAT 포함)

[장바구니 담기] [바로 구매]
```

### 3. 파일 업로드 UI

```
[드래그앤드롭 영역]
  파일을 여기에 드래그하거나 클릭하여 선택
  지원 형식: PDF (권장), AI, PSD
  최대 파일 크기: 500MB

[업로드 후]
  파일명.pdf  [12.3MB] [×]
  ✓ 해상도 300dpi 이상
  ✓ CMYK 색상모드
  ⚠ 폰트 아웃라인 확인 필요
```

### 4. 인쇄 특화 주문 상태 배지

```
결제완료 → [파일접수] → [검수중] → [제작접수]
→ [인쇄중] → [후가공] → [출고준비] → [배송중] → 배송완료

특수상태: [주문보류] - 빨간색 배지 + 수정 요청
```

## Shopby Enterprise 구현 시 고려사항

### 커스터마이징 필요 영역

1. **상품 옵션 구조**
   - 인쇄 특화 옵션 (규격, 도수, 용지, 후가공) 계층 구조
   - 옵션 의존성 로직 (상위 변경 시 하위 초기화)
   - 실시간 가격 계산 연동

2. **주문 상태 확장**
   - 기본 Shopby 상태 + 인쇄 특화 상태 추가
   - 파일검수 상태 및 알림 연동

3. **파일 업로드 시스템**
   - 대용량 파일 처리 (PDF 500MB+)
   - 파일 검수 자동화 (해상도, 색상모드)
   - 파일-주문 연결 구조

4. **마이페이지 확장**
   - 인쇄 파일 관리 섹션
   - 재주문 기능 (옵션 + 파일 함께 저장)
   - 인쇄 진행 상태 상세 표시

5. **어드민 커스터마이징**
   - 파일 검수 워크플로
   - 제작 상태 관리
   - 출고일 관리

### 경쟁사 대비 차별화 포인트 제안

| 항목 | 경쟁사 현황 | 훈이인쇄 차별화 |
|------|------------|----------------|
| 옵션 선택 | 텍스트 중심 | 시각적 미리보기 강화 |
| 가격 표시 | 선택 후 계산 | 옵션 선택 즉시 실시간 |
| 파일 검수 | 수동 / 지연 | 자동 검수 + 즉각 피드백 |
| 주문 상태 | 단순 배송 상태 | 인쇄 공정별 상세 표시 |
| 재주문 | 단순 상품 재주문 | 파일+옵션 세트 저장 |
| 디자인 연동 | 미리캔버스(비즈하우스) | 자체 간편 템플릿 |
