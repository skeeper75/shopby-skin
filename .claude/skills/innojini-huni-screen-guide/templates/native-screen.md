## 5. shopby API 연동

> NATIVE 화면은 shopby Provider를 통해 데이터를 자동으로 받습니다.

**사용 가능한 shopby Provider/Hook**:
- `useMallState()` - 몰 설정 정보
- `useAuthState()` - 인증 상태
- `useProductListState()` - 상품 목록
- `useCartState()` - 장바구니
- `useOrderState()` - 주문 상태
